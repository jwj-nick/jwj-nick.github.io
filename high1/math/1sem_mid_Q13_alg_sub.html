Q13 — 대수 치환

공통수학1 · 5.3점

문제를 읽고 스스로 접근 방향을 생각해 보세요.
$x + y + z = 9$일 때, $$\frac{(x-3)(y-3) + (y-3)(z-3) + (z-3)(x-3)}{(x-3)^2 + (y-3)^2 + (z-3)^2}$$ 의 값은? (단, $x \neq y,\; y \neq z,\; z \neq x$이다.)
① $-\dfrac{1}{2}$ ② $-\dfrac{1}{3}$ ③ $\dfrac{1}{4}$ ④ $\dfrac{2}{3}$ ⑤ $1$
🤔 잠깐 생각해 봐요
분자·분모에 $(x-3),(y-3),(z-3)$이 반복되네요. 조건 $x+y+z=9$와 이 반복 패턴, 어떻게 연결할 수 있을까요?

Q13 풀이

1단계 / 4단계

1단계
치환으로 변수 단순화
💡 분자·분모에 $(x-3)$ 형태가 반복되고, $x+y+z=9$에서 $x+y+z-9=0$이에요. "각각에서 3을 빼면 합이 0"이 되는 치환이 핵심이에요.

$a = x-3,\quad b = y-3,\quad c = z-3$으로 놓으면:

$$a + b + c = (x-3)+(y-3)+(z-3) = (x+y+z)-9 = 0$$

이제 주어진 분수식을 $a,b,c$로 바꾸면:

$$\frac{(x-3)(y-3)+(y-3)(z-3)+(z-3)(x-3)}{(x-3)^2+(y-3)^2+(z-3)^2} = \frac{ab+bc+ca}{a^2+b^2+c^2}$$
✅ 치환 후 목표: $a+b+c=0$ 조건을 이용해 분자와 분모의 관계를 찾아요.

Q13 풀이

2단계 / 4단계

2단계
핵심 항등식으로 분모 변환
💡 $a+b+c=0$이라는 조건이 있으면, 이 식을 제곱해서 $a^2+b^2+c^2$과 $ab+bc+ca$의 관계를 뽑아낼 수 있어요.

$(a+b+c)^2$을 전개하면:

$$(a+b+c)^2 = a^2+b^2+c^2 + 2(ab+bc+ca)$$

$a+b+c=0$이므로 좌변은 0:

$$0 = a^2+b^2+c^2 + 2(ab+bc+ca)$$ $$\therefore\quad a^2+b^2+c^2 = -2(ab+bc+ca)$$
✅ 분모 $a^2+b^2+c^2$을 $-2(ab+bc+ca)$로 바꿀 수 있어요!

Q13 풀이

3단계 / 4단계

3단계
분수 약분 → 정답
💡 분자 $ab+bc+ca$, 분모 $-2(ab+bc+ca)$이므로 바로 약분할 수 있어요.
$$\frac{ab+bc+ca}{a^2+b^2+c^2} = \frac{ab+bc+ca}{-2(ab+bc+ca)} = -\frac{1}{2}$$
⚠️ 약분 조건 확인
$ab+bc+ca=0$이면 약분 불가. 만약 $ab+bc+ca=0$이면 $a^2+b^2+c^2=0$이므로 $a=b=c=0$, 즉 $x=y=z=3$. 그런데 문제 조건 $x \neq y$ 위반! → 조건이 약분을 보장해요.
검산
구체적인 수로 확인

$x=4,\; y=2,\; z=3$이면 $x+y+z=9$, 모두 다름.

$a=1,\; b=-1,\; c=0$

$$\frac{(1)(-1)+(-1)(0)+(0)(1)}{1+1+0} = \frac{-1}{2} = -\frac{1}{2}\checkmark$$

4단계 / 4단계

4단계
왜 틀렸나 — 핵심 정리

이 문제의 핵심은 2가지예요:

1️⃣ 치환 패턴 인식
"$\sum = k$" 꼴 조건 → 각 변수에서 $k/n$을 빼면 합이 0이 됨.
$x+y+z=9$ → $a=x-3$, 합 $= 0$.
2️⃣ 항등식 암기
$a+b+c=0$이면 $a^2+b^2+c^2=-2(ab+bc+ca)$
이것을 바로 떠올릴 수 있으면 이 유형은 30초에 풀 수 있어요.
흔한 실수: 분자·분모를 직접 $x,y,z$로 계산하려다 막혀서 포기.
→ 치환 먼저, 항등식 다음!

Q13 — 한페이지 요약

공통수학1 · 5.3점 · 대수 치환

정답
① $-\dfrac{1}{2}$

핵심 풀이 흐름

① $a=x-3,\, b=y-3,\, c=z-3$ 치환 → $a+b+c=0$
② 식 변환: $\dfrac{ab+bc+ca}{a^2+b^2+c^2}$
③ $(a+b+c)^2=0$ → $a^2+b^2+c^2 = -2(ab+bc+ca)$
④ 약분: $\dfrac{ab+bc+ca}{-2(ab+bc+ca)} = -\dfrac{1}{2}$

왜 틀리나

• 치환 힌트($x+y+z=9$)를 놓침
• $(a+b+c)^2$ 항등식을 모름
• 분모≠0 확인을 건너뜀
