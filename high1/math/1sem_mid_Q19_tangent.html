Q19 — 이차함수 동시접선

공통수학1 · 논술형1 · 6.0점

두 이차함수:

$$y = x^2-x-4, \quad y = -x^2+5x+m$$

이 직선 $y=x+n$에 동시에 접할 때, $m+n$의 값은?

두 포물선과 공통접선 (정성적 그림)

🤔 "접한다"는 것은 교점이 딱 1개. 이차방정식으로 만들면 중근 조건 = 판별식 $D=0$!

Q19 풀이

1단계 / 3단계

1단계
$y=x^2-x-4$와 $y=x+n$의 접선 조건
💡 두 그래프를 연립 → 이차방정식 → 중근 조건 $D=0$으로 $n$ 결정. 슬라이더로 직선을 움직여 접하는 순간을 확인해 보세요!
슬라이더를 움직여 n을 조정해 보세요
$$x^2-x-4 = x+n$$ $$x^2-2x-(4+n) = 0$$

판별식 $D = (-2)^2 - 4\cdot(-(4+n)) = 4(5+n) = 0$:

$$\boxed{n = -5}$$

Q19 풀이

2단계 / 3단계

2단계
$y=-x^2+5x+m$과 $y=x-5$의 접선 조건
💡 $n=-5$가 확정됐어요. 이제 $m$을 움직여 두 번째 포물선이 직선 $y=x-5$에 접하는 순간을 찾아봐요!
슬라이더를 움직여 m을 조정해 보세요
$$-x^2+5x+m = x+n$$ $$x^2-4x+(n-m) = 0$$

판별식 $D = 16 - 4(n-m) = 36+4m = 0$ (단, $n=-5$ 대입):

$$m = n-4 = -5-4 = \boxed{-9}$$

Q19 풀이

3단계 / 3단계

3단계
정답 + 검산
$$m+n = -9+(-5) = \mathbf{-14}$$
검산: 직선 $y=x-5$.
$y=x^2-x-4$: $x^2-2x+1=0$ → $(x-1)^2=0$ → $x=1$, 접점 $(1,-4)$ ✓
$y=-x^2+5x-9$: $x^2-4x+4=0$ → $(x-2)^2=0$ → $x=2$, 접점 $(2,-3)$ ✓
실수 ①: 두 포물선 중 하나에만 $D=0$ 적용 → "동시에" 강조
실수 ②: $x^2-2x-(4+n)=0$에서 상수항 부호 오류

Q19 — 한페이지 요약

공통수학1 · 논술형1 · 6.0점

정답
m+n = −14

핵심 풀이 흐름

① 접한다 ↔ 이차방정식 중근 ↔ $D=0$
② $y=x^2-x-4$와 $y=x+n$ 연립: $x^2-2x-(4+n)=0$, $D=0$ → $n=-5$
③ $y=-x^2+5x+m$과 $y=x+n$ 연립: $x^2-4x+(n-m)=0$, $D=0$ → $m=n-4=-9$
④ $m+n = -9+(-5) = -14$
