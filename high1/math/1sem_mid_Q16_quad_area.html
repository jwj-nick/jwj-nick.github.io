Q16 — 이차함수와 넓이

공통수학1 · 5.7점

이차함수 $f(x)=\dfrac{1}{2}x^2-2x$의 그래프가 $x$축과 만나는 두 점을 각각 $O$, $A$라 하자.

$\dfrac{1}{2} \le m \le \dfrac{3}{2}$인 실수 $m$에 대하여 점 $O$를 지나고 기울기가 $m$인 직선 $l_1$이 이차함수 $y=f(x)$의 그래프와 만나는 점 중 $O$가 아닌 점을 $B$, 점 $A$를 지나고 기울기가 $m$인 직선 $l_2$가 이차함수 $y=f(x)$의 그래프와 만나는 점 중 $A$가 아닌 점을 $C$라 하자.

두 점 $B$, $C$에서 $x$축에 내린 수선의 발을 각각 $D$, $E$라 하고, 두 삼각형 $AEC$, $BOD$의 넓이를 각각 $S_1$, $S_2$라 하자. $S_2-S_1$의 최댓값은? (단, $O$는 원점이다.)

① $2$ ② $4$ ③ $8$ ④ $18$ ⑤ $36$
🤔 $f(x)$의 $x$절편을 먼저 구해보세요. $B$와 $C$의 좌표를 $m$으로 어떻게 나타낼 수 있을까요?

Q16 풀이

1단계 / 4단계

1단계
B, C의 좌표를 $m$으로 표현
💡 핵심 전략: B와 C의 위치가 $m$에 따라 어떻게 변하는지 좌표로 정확히 표현해야 넓이를 구할 수 있어요.

점 B: $l_1: y=mx$를 $f(x)$와 연립

$$mx = \tfrac{1}{2}x^2-2x \implies x(\tfrac{1}{2}x-(2+m))=0$$ $$x = 0 \text{ (O)} \;\;\text{또는}\;\; x = 2m+4$$ $$B = (2m+4,\;\; 2m^2+4m)$$

점 C: $l_2: y=m(x-4)$를 $f(x)$와 연립

$$m(x-4) = \tfrac{1}{2}x^2-2x \implies (x-4)(x-2m)=0$$ $$x = 4 \text{ (A)} \;\;\text{또는}\;\; x = 2m$$ $$C = (2m,\;\; 2m^2-4m)$$
⚠️ $\frac{1}{2} \le m \le \frac{3}{2}$이면 $2m(m-2) = 2m^2-4m < 0$. C의 $y$좌표는 음수! (포물선 아래에 있어요.)

Q16 풀이

2단계 / 4단계

2단계
$S_1$, $S_2$ 넓이 계산
💡 D=$(2m+4,0)$, E=$(2m,0)$. 삼각형 넓이 = $\frac{1}{2}|\text{밑변}|\times|\text{높이}|$.

$S_2 = \triangle BOD$: 밑변 $OD = 2m+4$, 높이 $BD = 2m^2+4m$

$$S_2 = \tfrac{1}{2}(2m+4)(2m^2+4m) = (m+2)\cdot 2m(m+2) = 2m(m+2)^2$$

$S_1 = \triangle AEC$: 밑변 $AE = 4-2m$, 높이 $CE = |2m^2-4m| = 2m(2-m)$

$$S_1 = \tfrac{1}{2}(4-2m)\cdot 2m(2-m) = (2-m)\cdot 2m(2-m) = 2m(2-m)^2$$

Q16 풀이

3단계 / 4단계

3단계
$S_2-S_1$ 정리 + 최댓값
$$S_2-S_1 = 2m\bigl[(m+2)^2-(2-m)^2\bigr]$$
합차 공식: $(a+b)^2-(a-b)^2 = 4ab$. 여기서 $a=2, b=m$ → $(2+m)^2-(2-m)^2 = 4\cdot 2\cdot m = 8m$.
$$S_2-S_1 = 2m\cdot 8m = 16m^2$$

$m \in \left[\frac{1}{2}, \frac{3}{2}\right]$에서 $16m^2$은 $m=\frac{3}{2}$일 때 최대:

$$16\times\left(\frac{3}{2}\right)^2 = 16\times\frac{9}{4} = \mathbf{36}$$

Q16 풀이

4단계 / 4단계

핵심 체크
흔한 실수 정리
실수 ①: C의 $y$좌표 $2m^2-4m$이 음수임을 모르고 $S_1$ 계산 오류
실수 ②: $(m+2)^2-(2-m)^2$ 전개를 직접 하다 실수 → 합차 공식 사용!
실수 ③: 최댓값을 $m=\frac{1}{2}$에서 찾음 → $16m^2$은 순증가, 최댓값은 $m=\frac{3}{2}$
풀이 핵심 요약:
B: $x=2m+4$ | C: $x=2m$
$S_2=2m(m+2)^2$, $S_1=2m(2-m)^2$
$S_2-S_1=16m^2$ → 최댓값 $36$

① $O=(0,0)$, $A=(4,0)$ (영점)

③ $S_2=2m(m+2)^2$, $S_1=2m(2-m)^2$
