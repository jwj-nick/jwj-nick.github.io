Q18 — 복소수와 켤레복소수

공통수학1 · 5.9점

$z_1=a+bi$, $z_2=c+di$, $a,b,c,d$는 자연수. $z_1\overline{z_1}=5$일 때:

ㄱ. $a^2+b^2=25$
ㄴ. $z_1-z_2=-4i$이면 $c+d=7$
ㄷ. $(z_1+z_2)(\overline{z_1}+\overline{z_2})=52$이면 $z_2\overline{z_2}$의 최댓값은 $25$

옳은 것만을 있는 대로 고른 것은?

Q18 풀이

1단계 / 4단계

1단계
기본 조건 파악
💡 $z_1\overline{z_1}=|z_1|^2=a^2+b^2$. 이 값이 5이고 $a,b$가 자연수이므로 경우의 수가 딱 2가지입니다.
$$z_1\overline{z_1} = a^2+b^2 = 5$$ $$a,b \in \mathbb{N} \implies (a,b) \in \{(1,2),\; (2,1)\}$$ $$\text{두 경우 모두: } a+b = 3$$

복소평면(아르강 평면): $a^2+b^2=5$를 만족하는 자연수 쌍

⚠️ ㄱ: $a^2+b^2=5 \ne 25$. ㄱ은 거짓.

Q18 풀이

2단계 / 4단계

2단계
ㄴ 판별: $z_1-z_2=-4i$
💡 복소수 등호: 실수부=실수부, 허수부=허수부.
$$z_1-z_2 = (a-c)+(b-d)i = 0+(-4)i$$ $$\implies a = c, \quad b - d = -4 \implies d = b+4$$
$$c+d = a+(b+4) = \underbrace{(a+b)}_{=3}+4 = \mathbf{7}$$
✓ ㄴ은 참. $(a,b)=(1,2)$이든 $(2,1)$이든 $a+b=3$이므로 항상 $c+d=7$.

ㄷ 판별: $z_2\overline{z_2}$ 최댓값
💡 $(z_1+z_2)(\overline{z_1}+\overline{z_2}) = |z_1+z_2|^2 = |z_1|^2+|z_2|^2+2\text{Re}(z_1\bar{z_2})$
$$5 + (c^2+d^2) + 2(ac+bd) = 52$$ $$(c^2+d^2) + 2(ac+bd) = 47$$

핵심 풀이 흐름

① $z_1\overline{z_1}=a^2+b^2=5$ → $(a,b)=(1,2)$ or $(2,1)$, $a+b=3$
② ㄱ: $a^2+b^2=5\ne25$ → ✗
③ ㄴ: $z_1-z_2=-4i$ → $a=c$, $d=b+4$ → $c+d=3+4=7$ → ✓
④ ㄷ: $(c+1)^2+(d+2)^2=52$ → 최대 $c^2+d^2=29\ne25$ → ✗
