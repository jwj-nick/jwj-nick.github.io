Q15 — 이차함수 최대·최소

공통수학1 · 5.5점

두 자연수 $a, b$에 대하여 $0 \le x \le 3$에서 이차함수 $f(x) = (x-a)^2 + 3b$의 최댓값을 $M$, 최솟값을 $m$이라 하자.

$M \le 25$이고 $m \ge 6$을 만족시키는 모든 순서쌍 $(a, b)$의 개수는?

① $16$ ② $17$ ③ $18$ ④ $19$ ⑤ $20$

꼭짓점 위치에 따른 최대·최솟값 위치 (정성적 그림)

경우 1: a=1
경우 2: a=2, 3
경우 3: a≥4
🤔 꼭짓점 $x=a$가 구간 $[0,3]$ 안에 있을 때와 밖에 있을 때, 최솟값이 어떻게 달라지나요?

Q15 풀이

1단계 / 3단계

1단계
꼭짓점 위치로 경우 분류
💡 $f(x)=(x-a)^2+3b$는 $x=a$에서 최솟값을 가져요. 이 꼭짓점이 구간 $[0,3]$ 안에 있는지 밖에 있는지에 따라 최대·최솟값이 달라져요. 슬라이더로 직접 확인해 보세요!
슬라이더를 움직여 조건을 확인해보세요
$a$ 범위최솟값 $m$최댓값 $M$
$a=1$ (구간 내부)$f(1)=3b$$f(3)=4+3b$
$a=2$ (구간 내부)$f(2)=3b$$f(0)=4+3b$
$a=3$ (오른쪽 끝)$f(3)=3b$$f(0)=9+3b$
$a=4$ (오른쪽 밖)$f(3)=1+3b$$f(0)=16+3b$
$a \ge 5$$f(3)=(a-3)^2+3b$$f(0)=a^2+3b \ge 28$
⚠️ $a=1$일 때 최댓값: $f(0)=1+3b$와 $f(3)=4+3b$ 중 $f(3)$이 더 큼 (꼭짓점에서 더 멀리 있는 끝점이 최댓값).
✅ $a \ge 5$이면 $M = a^2+3b \ge 25+3 = 28 > 25$. 조건 $M \le 25$ 불만족 → $a \ge 5$는 전부 제외.

Q15 풀이

2단계 / 3단계

2단계
각 경우에 $M \le 25$, $m \ge 6$ 적용
💡 $b$는 자연수($\ge 1$)이고, 두 조건을 동시에 만족하는 $b$의 범위를 구한 뒤 정수 개수를 셉니다.
$a$$m \ge 6$$M \le 25$$b$ 범위개수
1$3b \ge 6$ → $b \ge 2$$4+3b \le 25$ → $b \le 7$$\{2,3,4,5,6,7\}$6
2$3b \ge 6$ → $b \ge 2$$4+3b \le 25$ → $b \le 7$$\{2,3,4,5,6,7\}$6
3$3b \ge 6$ → $b \ge 2$$9+3b \le 25$ → $b \le 5$$\{2,3,4,5\}$4
4$1+3b \ge 6$ → $b \ge 2$$16+3b \le 25$ → $b \le 3$$\{2,3\}$2

3단계 / 3단계

3단계
합산 + 검산
$$6 + 6 + 4 + 2 = \mathbf{18}$$
검산 포인트
• $a=4,\, b=3$: $M=16+9=25 \le 25$ ✓, $m=1+9=10 \ge 6$ ✓
• $a=4,\, b=4$: $M=16+12=28 > 25$ → 제외 ✓
• $a=3,\, b=1$: $m=3 < 6$ → 제외 ✓
흔한 실수: $a=4$ 경우를 누락하거나, $a=3$에서 최솟값을 $f(3)=3b$가 아닌 다른 값으로 계산.

핵심 흐름

① 꼭짓점 위치($a$ 값)에 따라 최솟값·최댓값 공식이 달라짐
② $a=1,2$: $m=3b,\; M=4+3b$ → 6가지씩
③ $a=3$: $m=3b,\; M=9+3b$ → 4가지
④ $a=4$: $m=1+3b,\; M=16+3b$ → 2가지
⑤ $a \ge 5$: $M > 25$ → 0가지
합계 = 18
