Q20 — 인수분해와 계수비교

공통수학1 · 논술형2 · 6.0점

$$x^3+2ax^2+(6a+2)x+4a+12 = (x+b)(x+c)^2$$

실수 $a, b, c$에 대하여 $a+b+c$의 최댓값 $M$, 최솟값 $m$일 때, $M+m$의 값은?

이중근 vs 단순근 — $(x+b)(x+c)^2$의 그래프 형태

🤔 오른쪽을 전개해서 계수비교! $x^2$, $x^1$, $x^0$ 계수를 각각 맞춰보세요.

Q20 풀이

1단계 / 4단계

1단계
$(x+b)(x+c)^2$ 전개 후 계수비교
💡 항등식 = 모든 $x$에서 성립 = 각 차수 계수가 동일.
$$(x+b)(x+c)^2 = x^3+(2c+b)x^2+(c^2+2bc)x+bc^2$$
차수원래전개방정식
$x^2$$2a$$2c+b$$b=2a-2c$ ...(I)
$x^1$$6a+2$$c^2+2bc$(II)
$x^0$$4a+12$$bc^2$(III)

Q20 풀이

2단계 / 4단계

2단계
$a$ 소거 → $c$만의 방정식
💡 (I)로 $b$를 $a, c$로 표현 → (II), (III)에 대입 → $a$를 $c$로 표현 → 등치.

(III)에 대입: $(2a-2c)c^2=4a+12$

$$a = \frac{c^3+6}{c^2-2}$$

(II)에 대입: $c^2+4ac-4c^2=6a+2$

$$a = \frac{3c^2+2}{2(2c-3)}$$

두 식 등치: 교차곱 후 정리

$$c^4-6c^3+4c^2+24c-32 = 0$$ $$(c-2)^2(c+2)(c-4) = 0$$ $$c = 2,\; -2,\; 4$$

3단계 / 4단계

3단계
각 $c$에 대한 $a, b$ 계산
💡 각 경우의 삼차함수 그래프를 직접 확인해 보세요. 이중근(초록)에서 x축에 접하고, 단순근(빨강)에서 x축을 통과해요.
c=−2, a=−1, b=2: a+b+c = −1 (최솟값)
$c$$a=\frac{c^3+6}{c^2-2}$$b=2a-2c$$a+b+c$
$2$$7$$10$$\mathbf{19}$ ← 최대
$-2$$-1$$2$$-1$
$4$$5$$2$$11$
$M=19$, $m=-1$ → $M+m=\mathbf{18}$
⚠️ $c=2$가 중근이어도 해는 $c=2$ 하나. $a, b$ 값이 모두 유일하게 결정됨.

검산
세 경우 검산

$c=4$, $a=5$, $b=2$: $(x+2)(x+4)^2$

$$(x+2)(x^2+8x+16)=x^3+10x^2+32x+32$$ $$\text{원다항식}(a=5): x^3+10x^2+32x+32 \checkmark$$

$c=-2$, $a=-1$, $b=2$: $(x+2)(x-2)^2$

$$(x+2)(x^2-4x+4)=x^3-4x^2+4x+2x^2-8x+8=x^3-2x^2-4x+8$$ $$\text{원다항식}(a=-1): x^3-2x^2-4x+8 \checkmark$$

Q20 — 한페이지 요약

공통수학1 · 논술형2 · 6.0점

정답
M+m = 18

핵심 풀이 흐름

① $(x+b)(x+c)^2$ 전개 → 계수비교 → (I)(II)(III) 세 방정식
② (I)로 $b=2a-2c$, (III)·(II)로 $a$를 $c$로 각각 표현
③ 두 $a$ 등치 → $c^4-6c^3+4c^2+24c-32=0$ → $(c-2)^2(c+2)(c-4)=0$
④ $c=2$: $a+b+c=19$, $c=-2$: $-1$, $c=4$: $11$
⑤ $M=19$, $m=-1$ → $M+m=18$
