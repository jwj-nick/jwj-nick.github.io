## 페치한 페이지 핵심 정리

### A. 중앙대 2028 시행계획 핵심 (DH뉴스 + 스마트경제 + 교수신문)

**전형 명칭 개편 = CAU FORMULA 2028**
- *2022 개정 교육과정* 대응 + *수험생 부담 경감*이 양대 축
- "수시의 정시화, 정시의 수시화" 슬로건

**수시 학생부교과 = 지역균형 전형 (1종)**
- 정량평가 100% 유지: **교과 90% + 출결 10%**
- 모든 등급 산출 과목 반영 → 변별력 강화
- 2022 개정 교육과정 이수자만 지원

**수시 학생부종합 = 3트랙 개편**
| 트랙 명 | 핵심 | 면접 | 수능최저 |
|---|---|---|---|
| **탐구하는 학종 (Core)** | 계열 관심 과목 집중 평가 | ○ (1단계 서류 → 2단계 면접) | △ |
| **모두의 학종 (All)** | 다양한 관심사 + **6 키워드 선택제 신규** | ✗ (서류 100%) | ✗ |
| **최저있는 학종 (Up)** | 학종 + 수능최저 | ✗ (서류 100%) | ○ |

→ 핵심 변화: **6 키워드 선택제** (수험생 자가 강점 6개 추출, 평가관이 강조)
→ 면접 비중을 *축소* (2026 융합/탐구 → 2028 Core만 면접)
→ Up 신설로 *내신 약 + 수능 강* 학생 새 진입로

**수시 논술 = 2종**
- **모두의 논술**: 지원 자격 제한 X, 수능최저 적용
- **재학생 논술**: 졸업예정자만, *수능 이전 시행* (수능 부담 분리)

**정시 = 2종 이원화**
| 전형 | 비율 | 모집인원 | 수능최저 |
|---|---|---|---|
| **수능89** | 수능 89% + 출결 11% | **1,459명** | 해당없음 (수능 자체가 평가) |
| **학종49** | 수능 51% + 학생부 서류 49% | **251명** | ○ (서울캠 일반 3합 6, 의학 4합 5) |

→ 학종49 = **정시인데 학생부 평가** → "정시의 수시화" 핵심
→ 학생부 평가는 내신·창체 *제외*, **학교 생활 기록 내용만** 검토

**기타 혁신**
- **CAU 수능 케어**: 수능 잘 나오면 *수시 합격 포기 → 정시 도전* 가능
- **영어 등급 조정**: 최상위 <20%일 때 *2등급 → 1등급* 변환
- **인문 지원자 과학 등급 조정**: 과학 2등급 → 1등급 변환 (즉 인문이 과학 잘하면 인센티브)
- **동점자 처리**: 최종 단계 *개근 학생* 우선

## 검증 한계 (꼭 기억)

1. **공식 2028 모집요강 PDF 직접 확보 실패** — JS 렌더링·바이너리 추출 실패. 2026 시행계획 PDF도 같은 문제.
2. 위 수치는 *언론 분석 기사 + 어디가 인용*. **2028 정확 모집인원 — 응통 학과 단위는 미확보**.
3. 중대 *학종49 251명 / 수능89 1,459명*은 **대학 전체** 총합이지 *응통 학과별 분배는 미확정*.
4. 응용통계학과 *수시 학종·논술 인원*은 2026 가이드북 PDF 직접 확인 필요.
5. 시행계획은 *큰 틀*. 실제 모집요강 = 2027 봄 (수시) / 2027 가을 (정시) 별도 공시.

## 메모 (NCC)

- *학종49* 발견이 가장 큰 수확. SKKU와 달리 *정시에서도 학생부 평가*. 아들 입시 전략 영향 큼.
- *6 키워드 선택제*는 자기소개서 폐지 흐름의 *대안*. 어떤 6개를 골라야 응통/통계 매력적인지 *시뮬레이션 가치 있음*.
- *CAU 수능 케어*는 *수능 잘 본 학생에게 매우 유리*. 아들이 수능 사수 가능하면 SKKU·CAU 둘 다 *수시+정시 보험 구조* 작동.
- 학생부교과 지역균형 *교과 90% + 출결 10%*는 **출결 변별력 거의 없음** → *내신만의 게임*. SKKU의 *정성평가 30%* 포함과 큰 차이.
- *PDF 직접 추출 실패*는 1호와 같은 패턴. Phase 2에서 *PDF 파싱 도구 필수*.
