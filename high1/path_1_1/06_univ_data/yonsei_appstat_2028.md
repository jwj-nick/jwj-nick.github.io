## 한 줄 요약

> 연세대 응용통계학과는 수시 25명 + 정시 12명 = **총 37명** 모집. 계열 분류는 *통합*(인문·자연 모두 응시). 정시는 전원 *'가군'* (유형III). 학생부교과 추천형이 5등급제를 직접 점수화(1등급=70점 기준)하는 구조.

## 1. 학과 진입 경로 — 학과 모집

- **소속**: 상경대학 (경제학부·응용통계학과·상경계열)
- **계열**: **통합** — 인문(유형I)·자연(유형II) 구분 없이 지원 가능
- **상경계열\*** 별도 존재: 15명 광역 모집 후 2학년에 경제학부·응용통계학과 중 택1

> 즉 연세대 응통 진입 경로는 **2가지**:
> 1. *응용통계학과 직접 모집* (37명) — 입학 즉시 통계
> 2. *상경계열 광역 모집* (15명) → 2학년에 응통 선택 가능

## 4. 학생부교과 추천형 — 5등급제 반영 방식 (2028 신)

> 2028 내신 5등급제를 연세대가 *직접 점수화*하는 구체적 방식.

**1단계 (100%)**: 전 과목 반영
- **반영과목 A** (공통·일반선택·진로선택): 등급점수(70%) + 성취도(30%) 이수단위 가중 평균
  - 등급점수: 1등급=70 / 2등급=63 / 3등급=45 / 4등급=25 / 5등급=0
  - 성취도: A=30 / B=27 / C=20 / D=10 / E=0
- **반영과목 B** (융합선택·체육예술·과학탐구실험): 성취도 기준 최대 10점 *감점*

**2단계 (100%)**: 1단계 80% + 서류평가 20% + 출결(감점)

> 핵심: 1등급=70+30=100점 만점, 2등급=63+27=90점. **1→2등급 격차 = 10점**. 5등급제에서도 충분한 변별력.

## 6. 정시 수능 반영 (유형III — 응통 해당)

| 영역 | 배점 |
|---|---|
| 국어 | 200 |
| 수학 | 200 |
| 영어 | 100 |
| 탐구(사회) | 100 |
| 탐구(과학) | 100 |
| **합계** | **700** |

> **유형III**: 통합·국제·체능 계열 = 국어·수학 *동등 비중* (인문=국어 300, 자연=수학 300과 다름)
> 응통이 유형III = *수학 강점이 결정적이지 않음*. 국어+수학 균형형 학생에게 유리.
