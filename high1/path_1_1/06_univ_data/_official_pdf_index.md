## 5. PDF 추출 한계 & Phase 2 우회 방안

### 공통 장애물

| 학교 패턴 | 장애물 |
|---|---|
| SKKU, CAU | `download.php` 세션·JS 필요 — WebFetch 실패 |
| 다수 | PDF 바이너리 추출 안됨 (10MB+ 또는 OCR 필요) |
| 진학사 | 403 Forbidden — User-Agent 검사 |
| 어디가 | 학과별 표 *로그인 세션* 필요 |

### Phase 2 우회 우선순위

1. **Nick 수동 다운로드** ★ 가장 확실 — Chrome으로 입학처 직접 방문
2. **playwright** (브라우저 자동화) — JS·세션 우회 가능
3. **pdfplumber + tesseract OCR** — PDF 텍스트 추출
4. **어디가 로그인 세션 활용** — 학과별 표 확보 (Nick 계정)
5. **언론 분석 기사 인용** — 차선책 (1·2호 패턴, 정확도 ↓)
