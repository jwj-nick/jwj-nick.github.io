## 2. 전형별 매트릭스 비교

### 2.1 학생부교과

| 항목 | SKKU 추천인재 | CAU 지역균형 |
|---|---|---|
| 평가방식 | 정량 70% + **정성 30%** | 정량 100% (교과 90 + 출결 10) |
| 모집인원 (대학 전체) | 415명 | 미확정 |
| 응통/경제 학과별 | (광역이라 직접 X) | 응통 **10명** (2026) |
| 수능최저 | 3합 7 | 3합 6 추정 |
| 핵심 차이 | *세특·창체 가중* → 균형 우수자 | *내신 절대 강자* 게임 |

→ **아들 적용**: 내신 *최상위* 가능하면 *CAU 유리*. *세특 풍부형*이면 SKKU 유리.

### 2.2 학생부종합

| 항목 | SKKU 학종 4트랙 | CAU 학종 3트랙 |
|---|---|---|
| 트랙 수 | 4개 (융합인재·탐구인재·면접형 성균인재·약학) | **3개** (Core·All·Up) |
| 면접 비중 | *증가* (면접형 성균인재 신설, 사회과학 포함) | *감소* (Core만 면접) |
| 신규 트랙 | 면접형 성균인재 222 (사회과학·공학 신규) | **Up — 최저있는 학종** (서류만 + 수능최저) |
| 자기서사 도구 | 일반 학종 양식 | **6 키워드 선택제** (지원자가 강점 6개 선택) |
| 수능최저 | 융합 3합 6 / 탐구 미적용 / 면접형 미확정 | Up만 적용 (서울캠 3합 6) |
| 모집인원 | 1,270명 (대학 전체) | 미확정 |

→ **아들 적용**: 
- SKKU = *면접 강세 + 사회과학 신규 트랙* (사회과학 면접형 222명 신설 = 기회)
- CAU = *6 키워드로 자기서사 명확화* + *Up으로 수능 강자 우회로*
- *김지호 페르소나*는 **CAU Up 또는 학종49가 가장 적합** (수능 + 활동)

### 2.3 논술

| 항목 | SKKU | CAU |
|---|---|---|
| 트랙 | 언어형 + 수리형 | 모두의 논술 + 재학생 논술 |
| 모집인원 | 376명 (언어 156 + 수리 220) | 미확정 |
| 수능최저 | 3합 6 (일부 이공 3합 5) | 3합 6 추정 |
| 시기 분리 | 일괄 | **재학생 논술 = 수능 전 시행** (수능 부담 분리) |

→ **CAU의 *수능 전 논술*은 독특**. 수능 망쳐도 논술이 *독립 변수*.

### 2.4 정시 — *가장 큰 차이*

| 항목 | SKKU | CAU |
|---|---|---|
| 트랙 | 일반형 + 특화형 | **수능89 + 학종49** |
| 일반형/수능89 | 일반형 1,307명 = 수능 100% | 수능89 1,459명 = 수능 89% + 출결 11% (사실상 수능 100%) |
| 특화형/학종49 | 특화형 150명 = *국어+탐구 가중* (사회과학) | **학종49 251명 = 수능 51% + 서류 49%** + 수능최저 |
| 학생부 정시 반영 | ✗ | **○ (학종49)** |
| 사회과학/응통 군 | 특화형에 사회과학 포함 | 정시 응통 = *인문 분류* |
| 수능 케어 제도 | 없음 | **CAU 수능 케어** (수시 합격 포기 → 정시 도전) |
| 응통 모집 (2026 기준) | (사회과학계열 광역) | **18명** |

→ **결정적 차**: 
- **SKKU는 정시도 *수능 중심* (특화형도 결국 수능 가중)**
- **CAU는 정시에 *학생부 49%* 도입** (정시의 수시화)
- CAU 수능 케어 = *수능 강자에게 강력한 안전망*

## 3. 입시 전략 축 — 3대 분기점

### 3.1 광역 vs 학과

| | 광역 (SKKU) | 학과 (CAU) |
|---|---|---|
| 입학 시 학과 결정 | ✗ (1년 후) | ○ |
| 적성 탐색 기회 | ◎ | ✗ (전과 필요) |
| 자기소개서 명확성 | ○ ("사회과학 일반") | ◎ ("응용통계학과") |
| 진로 확정도 | △ | ◎ |

→ **아들 진로 확정도**: 미정 (진로방향.md 미입력). 그러나 *축구분석가 path 검토 중*이면 *학과 단위 = CAU가 명확*.

### 3.2 수능 vs 학종 — *가중치 어디에 둘 것인가*

| 비중 | SKKU | CAU |
|---|---|---|
| 수능 비중 | 정시 43% × *수능 100%* = **순수 수능 의존** | 정시 40% × (수능89 *85%* + 학종49 *15%*) = **수능+서류 혼합** |
| 학종 비중 | 수시 학종 트랙 *세분화* | 수시 학종 *3트랙 + 정시 학종49* = **학종 영향 확산** |
| 권장 학생 유형 | *수능 사수* 가능, *세특 평이* | *수능 + 활동 모두* 가능, *세특 풍부* |

### 3.3 정성평가 비중

| | SKKU | CAU |
|---|---|---|
| 학생부교과 정성 | **30%** | 0% |
| 정시 학생부 평가 | ✗ | **49% (학종49)** |
| 평가 강화 방향 | *수시* 정성 ↑ | *정시* 정성 ↑ |

→ 두 학교 모두 *정성평가 확산* 방향이지만 *어디에 넣었느냐*가 다름.

## 4. 아들 입장 — 안전권/도전권 매핑

> *진로방향.md 미입력 상태이므로 아래는 시뮬레이션 가설*. 진로 확정 후 재평가.

### 4.1 가설 A: 수능 사수형 아들

| 학교 | 추천 트랙 | 안전도 |
|---|---|---|
| SKKU 사회과학 | 정시 일반형 (수능 100%) | ◎ |
| SKKU 사회과학 | 정시 특화형 (국어+탐구 가중) | ○ |
| CAU 응통 | 정시 수능89 | ◎ |
| CAU 응통 | 정시 학종49 (수능 + 서류) | ◎ + *세특 보너스* |
| CAU 응통 | CAU 수능 케어 → 수시 백업 + 정시 본진 | ◎◎ |

→ *수능 사수형이면* **CAU가 학종49·수능89·수능 케어로 안전망 3중**.

### 4.2 가설 B: 학종 풍부형 아들

| 학교 | 추천 트랙 | 안전도 |
|---|---|---|
| SKKU 사회과학 | 면접형 성균인재 (신규 사회과학) | ○ — 신규라 변수 |
| SKKU 사회과학 | 탐구인재 (수능최저 미적용) | ◎ |
| CAU 응통 | 학종 Core (수학·통계 탐구) | ◎ |
| CAU 응통 | 학종 All (6 키워드) | ◎ |

→ *학종 풍부형이면* 두 학교 *비슷한 매력*. *세특이 사회과학이면* SKKU, *세특이 통계·데이터*면 CAU.

### 4.3 가설 C: 축구분석가 path (R4 김지호)

| 학교 | 추천 트랙 | 적합도 |
|---|---|---|
| SKKU 사회과학 | 정시 특화형 (국어+탐구) | ○ — 그러나 *경제·통계 직진 안됨* |
| **CAU 응통** | **학종49 (수능 + 서류)** | **◎◎◎** |
| CAU 응통 | Up (수능최저 + 서류) | ◎ |
| CAU 응통 | 정시 수능89 + 수능 케어 | ◎ |

→ **CAU 압도적 우위**. 김지호 페르소나 = *학과 명칭 = 직업 명칭 근접* + *학종49 완전 매칭*.

## 7. NCC 결론 (Phase 3 입력)

1. **스키마 안정** — 1호·2호 패턴이 90% 호환. JSON 변환 가능.
