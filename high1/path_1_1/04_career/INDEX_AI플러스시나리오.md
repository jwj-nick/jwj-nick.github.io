## 시나리오 간 공통 패턴 (Cross-cutting Insights)

### 패턴 1: AI는 *형식·효율*에 강하지만 *의미·맥락*에는 약하다

| 분야 | AI가 잘하는 것 | 인간이 잘하는 것 |
|---|---|---|
| 의약 | 신약 후보 분자 1억개 시뮬레이션 | 환자의 *말 못한 이야기* 듣기 |
| 자연과학 | 거대 데이터에서 패턴 탐지 | *왜 그 패턴이 의미 있는가* 판단 |
| 인문·사회 | 50만 건 텍스트 자동 분석 | 한 사람의 *발화의 톤* 해석 |
| 경영·경제 | 시장 미시구조 통계 모델 | *오늘 같은 날* 사람의 행동 변화 직관 |
| 교육 | 28명에게 28가지 진도 자동 운영 | 한 학생의 *오늘 마음이 흔들리는 이유* 알아보기 |
| 예술 | 무한한 형식 생성 | *왜 이걸 만드는가*에 인생을 거는 것 |

→ **공통**: AI는 *평균과 통계*에 강하다. 인간은 *예외와 맥락*에 강하다. 진짜 가치는 *예외와 맥락*에서 자주 나온다.

---

### 패턴 2: 21세기 전문가의 새로운 기본기 — *코드를 읽을 줄 알아야 한다*

코드를 *짤* 필요는 없을 수도 있다. 그러나 코드가 *무엇을 하고 있는지 읽을 수 있는 능력*은 21세기 전문가의 *문해력*이다.

| 분야 | 새로운 기본기 |
|---|---|
| 의사 | 의료 AI의 학습 데이터 한계를 *읽을 수 있는 능력* |
| 과학자 | AI 모델이 *어떤 가설을 학습했는지* 검증할 수 있는 능력 |
| 법률가 | 알고리즘의 *대리 변수 차별*을 찾아낼 수 있는 능력 |
| 경영자 | 자동화 워크플로우를 *설계*할 수 있는 능력 |
| 교사 | 학습 데이터를 *통계적으로 해석*할 수 있는 능력 |
| 예술가 | 생성 모델의 *학습 편향*을 인식하고 다룰 수 있는 능력 |

→ **공통**: *기술자가 되라는 게 아니다*. *기술이 무엇을 하는지 모르면 그 기술 위에서 일을 못 한다*는 의미.

---

### 패턴 3: AI가 자동화될수록 *사람과 사람의 만남*이 더 무겁다

| 분야 | 시나리오에 등장한 "사람과 사람" 순간 |
|---|---|
| 의약 | 18세 환자가 "덕분이에요"라고 말하는 영상통화 |
| 자연과학 | 노 수학자가 "자네 추측이지"라며 미소 짓는 순간 |
| 인문·사회 | 70대 노인이 "신문 보고 그렇게 됐다고, 고마워요" |
| 경영·경제 | 어머니가 "매일 도서관 가" 라고 답하는 통화 |
| 교육 | 자폐 학생이 *친구를 친구의 의미*로 처음 부른 순간 |
| 예술 | 컬렉터가 "내가 사는 건 당신의 89시간이군요" |

→ **공통**: AI가 점점 더 많은 일을 자동화할 때, *한 사람의 시간을 다른 한 사람을 위해 들였다*는 사실이 점점 더 큰 가치를 가진다.

---

### 패턴 4: *질문의 능력*이 *답의 능력*보다 비싸진다

- 의사: *환자가 의사에게만 말하는 것*을 끌어내는 질문
- 과학자: *왜 이 현상이 의미 있는가*의 질문
- 법률가: *알고리즘이 *어떻게 차별을 자동화*했는가*의 질문
- 경영자: *어떤 일이 할 가치가 있는가*의 질문
- 교사: *이 학생이 왜 흔들리는가*의 질문
- 예술가: *아무도 안 묻는 질문*

→ AI는 *답*을 점점 더 잘 만든다. 그러나 *옳은 질문*은 인간이 만든다.

## 아이를 위한 가이드: 어떻게 이 시나리오를 *써먹을 것인가*

### 단계 1: 6개 다 읽지 마라. 끌리는 것부터.

처음부터 6개를 다 읽으려고 하면 압도된다. 제목과 *한 줄 요약*만 보고 *가장 끌리는 1개*부터 시작.

### 단계 2: 스토리 읽으면서 *주인공이 되어볼 것*

이가현 / 윤도현 / 정해린 / 최예진 / 임수영 / 강민호 — 이 사람들 중 *내가 가장 닮고 싶은 사람*은 누구인가? 닮고 싶지 *않은* 사람은? 그 이유는 무엇인가?

### 단계 3: 스토리에서 *내 마음에 남은 한 문장*을 메모

각 파일 끝의 "가장 강조하고 싶은 한 줄"이 NCC가 고른 거지만, 너는 *너만의 한 문장*을 골라보라. 그게 너의 출발점이다.

### 단계 4: 추천 검색어 1~2개 직접 검색해 볼 것

각 시나리오 파일 끝의 "추천 검색 시드"에서 *가장 호기심 가는 키워드 1개*만 검색해본다. 1시간만 빠져들어 보고, 다시 와서 일기에 적어보라 — *무엇이 새로웠나? 무엇이 이상했나?*

### 단계 5: 1개월 후 다시 와서 같은 시나리오를 읽어보라

지금 읽었을 때와 1개월 후 읽었을 때 느낌이 다르면 *너 안에서 무언가가 자라고 있다는 신호*다. 같으면 *지금이 그 시나리오와 맞다는 신호*일 수 있다.

## 시나리오에서 다루지 *못한 것* — 한계와 보완 가이드

이 6개 시나리오는 NCC의 *추론과 창작*이다. 실제 미래는 다를 수 있다. 특히:

| 못 다룬 것 | 보완 방법 |
|---|---|
| 한국 사회의 특수한 변수 (입시·병역·가족 구조) | 부모님과 직접 대화. 한국 진로 데이터 별도 조사. |
| 실패 시나리오 | 시나리오는 *성공한 페르소나*만 보여줌. 실제로는 80%가 그 길에서 다른 길로 빠짐. *플랜 B*를 함께 생각할 것. |
| 직업 외의 인생 (가족, 취미, 공동체) | 직업은 인생의 1/3. 나머지 2/3은 시나리오에 안 들어감. |
| 2050년대의 진짜 직업 | 지금 우리가 *상상조차 못 하는 직업*이 30년 후 가장 큰 부분일 수 있다. 시나리오는 *오늘 보이는 미래*만 다룸. |
| 윤리·환경·정치 위기로 인한 진로 자체의 변화 | 기후 위기, 전쟁, AGI 등으로 시나리오 전제 자체가 흔들릴 수 있음. |
