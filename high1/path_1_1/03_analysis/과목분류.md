## 선택 구조 요약 (전제 조건)

```
2-1  [막4] 4과목(12학점) + [막1] 1과목(3학점)   = 15학점
2-2  [막4] 4과목(12학점) + [막1] 1과목(3학점)   = 15학점
3-1  [막5] 5과목(15학점) + [막1]-A 1(3학점) + [막1]-B 1(2학점) = 20학점
3-2  [막3]-A 3과목(9학점) + [막3]-B 3과목(9학점)
     + [막1]-A 1(3학점) + [막1]-B 1(2학점)      = 23학점
```

**[막4] = 사회/인문/수학/과학 전 과목 통합 풀** (Nick 확인 완료)  
**소프트웨어와 생활 = 3-2 [막1]-A 소속** (Nick 확인 완료)

## 시나리오 1 — 공학 일반 (전기·전자·기계·화공)

**방향**: 물리+화학+수학 heavy. 수능 물리I/화학I 대비와 연계.

| 학기 | 막 | 추천 과목 | 핵심 이유 |
|---|---|---|---|
| 2-1 | [막4] | **기하, 물리학, 화학**, + 생명과학(화공) / 지구과학(환경공학) | 공학 3종 세트 + 세부전공 1 |
| 2-1 | [막1] | **정보** | 코딩 기초, 전 공학 계열 필수 |
| 2-2 | [막4] | **역학과 에너지, 물질과 에너지, 인공지능 수학**, + 세포와 물질대사(바이오화공) | 물리·화학 심화 + 수학 |
| 2-2 | [막1] | **인공지능 기초** | 소프트웨어·자동화 트렌드 대응 |
| 3-1 | [막5] | **미적분II, 전자기와 양자, 화학 반응의 세계, 영어 독해와 작문**, + 수학과제 탐구 | 공학수학·물리·화학 전공심화 |
| 3-1 | [막1]-A | **데이터 과학** | 공학 데이터 분석 |
| 3-1 | [막1]-B | **★미술과 매체** | CAD·설계 공간 감각 |
| 3-2 | [막3]-A | **실용 통계, 심화 영어 독해와 작문, 독서 토론과 글쓰기** | 통계·영어 논문·사고력 |
| 3-2 | [막3]-B | **★융합과학 탐구, ★기후변화와 환경생태, ★과학의 역사와 문화** | 과학 계열 심화 |
| 3-2 | [막1]-A | **소프트웨어와 생활** | 공학 SW 기초 |
| 3-2 | [막1]-B | **★미술 감상과 비평** | |

> 💡 화공: 생명과학 → 세포와 물질대사 / 기계: 지구과학 → 지구시스템과학 등 세부전공별 1과목 조정.

| 2-2 | [막4] | **인공지능 수학, 역학과 에너지, 물질과 에너지, 지구시스템과학** | AI 수학 + 물리 심화 |
| 2-2 | [막1] | **인공지능 기초** ★필수 | AI 개론 |

| 3-2 | [막3]-A | **실용 통계, 심화 영어 독해와 작문, 미디어 영어** | 데이터분석·영어 논문 |
