한 권에 1500원인 공책 $x$권을 살 때 총 가격을 $y$원이라 한다. $y$를 $x$의 식으로 나타내시오.
권 수가 2배, 3배 되면 가격은 어떻게 변하나요?
$y = 1500x$
1권 1500원, 2권 3000원, 3권 4500원...
$x$가 2배, 3배 되면 $y$도 2배, 3배 → 정비례.
비례상수 $a = 1500$ → $y = 1500x$

다음 중 $y$가 $x$에 정비례하는 식을 모두 고르시오.
(가) $y = -4x$   (나) $y = x + 2$   (다) $y = \dfrac{x}{5}$   (라) $y = \dfrac{3}{x}$
정비례는 $y = ax$ 형태. 상수항이 있거나 분수 형태($a/x$)면 정비례 아님.
(가), (다)
(가) $y = -4x$ → 정비례 ($a = -4$) ✓
(나) $y = x + 2$ → 상수항 있음, 정비례 X
(다) $y = \dfrac{x}{5} = \dfrac{1}{5}x$ → 정비례 ($a = 1/5$) ✓
(라) $y = \dfrac{3}{x}$ → 반비례

한 시간에 60 km를 달리는 자동차가 $x$시간 동안 달린 거리를 $y$ km라 한다. $y$를 $x$의 식으로 나타내고, 4시간 동안 달린 거리를 구하시오.
시간이 2배 되면 거리도 2배 → 정비례. 그 다음에 식에 $x = 4$를 대입.
식: $y = 60x$,   거리: 240 km
속도 일정 → 시간과 거리 정비례, $a = 60$.
$x = 4$ 대입: $y = 60 \times 4 = 240$ (km)

$y$가 $x$에 정비례하고 $\dfrac{y}{x} = -7$일 때, $y$를 $x$의 식으로 나타내시오.
$\dfrac{y}{x} = a$ 일 때 $y = ax$.
$y = -7x$
$\dfrac{y}{x} = -7$ 의 양변에 $x$를 곱하면 $y = -7x$.
비례상수 $a = -7$.

물탱크 1대가 1분에 25 L의 물을 채워요. $x$분 동안 채운 물의 양을 $y$ L라 할 때, $y$를 $x$의 식으로 나타내고, 물 200 L를 채우는 데 걸리는 시간을 구하시오.
시간이 2배 되면 물 양도 2배 → 정비례. 200 L 채우려면 식에 $y = 200$ 대입.

1분에 25 L → $y = 25x$.
$y = 200$ 대입: $200 = 25x \Rightarrow x = 8$ (분)

$y$가 $x$에 정비례할 때, 아래 표의 빈칸을 모두 채우시오.
$x$2?5?
$y$-812?-20
$x = 2, y = -8$ 짝으로 비례상수 $a$를 먼저 구해요.
$a = -4$, $y = -4x$
$\dfrac{-8}{2} = -4 \Rightarrow a = -4$, $y = -4x$.
① $y = 12$ 일 때: $12 = -4x \Rightarrow x = -3$
② $x = 5$ 일 때: $y = -4 \times 5 = -20$ ✓ (이미 표에 있음)
③ $y = -20$ 일 때: $-20 = -4x \Rightarrow x = 5$ ✓
표 완성: ($x: 2, -3, 5, 5$ — 마지막 칸이 5), ($y: -8, 12, -20, -20$).
※ 마지막 두 짝은 같은 점이에요.

두 양 $A$와 $B$ 사이에 $A$가 $B$에 정비례하고, $B = 4$일 때 $A = 6$이라 한다. $A = 15$일 때 $B$의 값을 구하시오.
$A$를 $y$, $B$를 $x$로 두고 식부터 세워요.
$B = 10$
$A = aB$ 라 하면 $6 = 4a \Rightarrow a = \dfrac{3}{2}$.
$A = \dfrac{3}{2} B$.
$A = 15$ 대입: $15 = \dfrac{3}{2} B \Rightarrow B = 15 \times \dfrac{2}{3} = 10$.

정비례 관계식 $y = ax$ 에서 $x$가 3 증가할 때 $y$가 -12 만큼 변한다. 비례상수 $a$의 값과 식을 구하시오.
$x = 0$일 때 $y = 0$이고, $x = 3$일 때는 $y$가 $-12$ 만큼 변한 값.
$a = -4$, $y = -4x$
$x$ 변화 3, $y$ 변화 $-12$ → $\dfrac{변화량 \; y}{변화량 \; x} = \dfrac{-12}{3} = -4$.
정비례에서 $\dfrac{y}{x} = a$ 이고, 변화량 비도 $a$ 와 같아요.
$a = -4$, $y = -4x$.
정비례에서 변화량의 비도 비례상수와 같아요. (직선의 기울기 개념과 직결)
