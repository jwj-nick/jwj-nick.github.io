문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어: gcd = 최대공약수, lcm = 최소공배수. 두 수의 GCD가 $g$이면 $A = gm$, $B = gn$ ($m, n$ 서로소). LCM $= gmn$.
유형 6 · H3

두 자연수 $A$, $B$에 대해 $\gcd(A, B) = 8$이고 $\text{lcm}(A, B) = 240$이다. $A + B = 104$일 때, $A$, $B$를 구하세요. ($A < B$)
🤔 잠깐, 손으로 생각해보세요 $A = 8m$, $B = 8n$으로 놓으면 LCM도, 합도 $m, n$의 식으로 바뀌어요. 그러면 $m+n$과 $mn$을 알 수 있어요.
1단계 · Page 1

$A, B$를 $\gcd$로 표현

왜 이 단계? — GCD가 $8$이라는 정보를 변수로 옮겨 식을 단순화.
표현
$A = 8m$, $B = 8n$  ($\gcd(m, n) = 1$, $m < n$)
LCM도 $m, n$으로

$\gcd(m,n) = 1$이면 $\text{lcm}(8m, 8n) = 8mn$ (공통 $8$ 한 번 + 각각의 고유 부분 곱):

$\text{lcm}(A, B) = 8mn = 240$
$mn = 30$

2단계 · Page 2

합 조건으로 $m + n$ 구하기

왜 이 단계? — $A + B$ 조건도 $m, n$의 식으로.
조건 적용
$A + B = 8m + 8n = 8(m + n) = 104$
$m + n = 13$
현재 정보 정리
$m + n = 13$
$m n = 30$
$\gcd(m, n) = 1$, $m < n$

3단계 · Page 3

$m, n$ 찾기

왜 이 단계? — "합과 곱"을 알면 두 수가 결정돼요. 직접 시도 or 이차방정식.
방법 1 — 직접 시도

$mn = 30$의 인수 쌍 중 합이 $13$인 것을 찾아요:

$30 = 1 \times 30$ → 합 $31$ ✗
$30 = 2 \times 15$ → 합 $17$ ✗
$30 = 3 \times 10$ → 합 $13$ ✅
$30 = 5 \times 6$ → 합 $11$ ✗

$(m, n) = (3, 10)$. $\gcd(3, 10) = 1$ ✅

방법 2 — 이차방정식

$m, n$은 $t^2 - 13t + 30 = 0$의 두 근:

$(t - 3)(t - 10) = 0$ → $t = 3$ 또는 $t = 10$

$m = 3$, $n = 10$ (단 $m < n$).

4단계 · Page 4

$A, B$ 계산 + 검산

왜 이 단계? — 원래 변수로 환원하고 모든 조건 확인.
$A, B$
$A = 8 \times 3 = 24$
$B = 8 \times 10 = 80$
검산
$\gcd(24, 80)$: $24 = 2^3 \times 3$, $80 = 2^4 \times 5$ → $\gcd = 2^3 = 8$ ✅
$\text{lcm}(24, 80) = 2^4 \times 3 \times 5 = 240$ ✅
$A + B = 24 + 80 = 104$ ✅
⚠️ 흔한 실수 $mn = 30$만 보고 $(5, 6)$이나 $(2, 15)$를 답으로 쓰면 합 조건을 어김! 그리고 $\gcd(m, n) = 1$ 확인도 잊으면 안 돼요.

정답 · Page ★

정답 + 풀이 흐름 요약

📖 핵심 공식: $A = gm, B = gn$ ($\gcd(m,n) = 1$) → $\text{lcm} = gmn$, $\gcd \cdot \text{lcm} = AB$.
$A = 24, \; B = 80$
전체 풀이 흐름 — 5단계
1
$A = 8m, B = 8n$, $\gcd(m, n) = 1$, $m < n$로 설정
2
$\text{lcm} = 8mn = 240$ → $mn = 30$
3
$A + B = 8(m + n) = 104$ → $m + n = 13$
4
$mn = 30, m + n = 13$ → $(m, n) = (3, 10)$ ($\gcd = 1$ ✅)
5
$A = 24, B = 80$ — 검산 모두 통과 ✅
⚠️ 이 유형의 핵심 함정 GCD·LCM·합/곱이 함께 주어지면 $A = gm, B = gn$ 변환으로 시작! 그러면 $m + n$, $mn$, 서로소 조건으로 자동 정리돼요.
