문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어·기호: 서로소 = 두 수의 gcd가 $1$. $d \mid n$ = "$d$는 $n$의 약수". 예: $3 \mid 12$.
유형 7 · H3

자연수 $n$에 대하여 $n$과 $n+1$은 항상 서로소임을 설명하세요. 또, $n = 12$일 때 확인하세요.
🤔 잠깐, 손으로 생각해보세요 $n$과 $n+1$의 공약수 $d$가 있다면, $d$는 두 수의 차도 나눠요. 그 차가 뭐죠?
1단계 · Page 1

핵심 성질 — 공약수는 차도 나눈다

왜 이 단계? — 증명의 출발점. 두 수 모두를 나누는 수는 둘의 차도 나눠요.
📖 핵심 정리: $d \mid a$이고 $d \mid b$이면 $d \mid (a - b)$. ($d$가 $a$도 나누고 $b$도 나누면, 차도 나눠요.)
왜 그런가

$a = d \times p$, $b = d \times q$이면:

$a - b = d \times p - d \times q = d \times (p - q)$

즉, $a - b$도 $d$로 나누어떨어져요.

← 이전
2단계 · Page 2

증명: $\gcd(n, n+1) = 1$

왜 이 단계? — 1단계 성질을 적용하면 한 줄에 결론이 나와요.
설정

$d = \gcd(n, n+1)$이라 둬요. 그러면:

$d \mid n$ ($d$는 $n$의 약수)
$d \mid (n+1)$ ($d$는 $n+1$의 약수)
차에 적용

두 수의 차는 $(n+1) - n = 1$. 1단계 성질에 따라:

$d \mid \big((n+1) - n\big) = d \mid 1$

$d$가 $1$의 약수 → $d = 1$ (자연수에서 $1$을 나누는 건 자기 자신뿐).

결론
$\gcd(n, n+1) = 1$ → $n$과 $n+1$은 항상 서로소 ∎
← 이전
3단계 · Page 3

$n = 12$일 때 확인

왜 이 단계? — 증명을 구체적인 수로 검증.
$\gcd(12, 13)$ 계산
$12 = 2^2 \times 3$
$13 = 13$ (소수)

공통 소인수 없음:

$\gcd(12, 13) = 1$ ✅
💡 보너스 관찰 $13$이 마침 소수라 더 명확하지만, $13$이 소수가 아니어도 결과는 같아요. 예: $\gcd(20, 21) = ?$ $20 = 2^2 \cdot 5$, $21 = 3 \cdot 7$ → 공통 없음 → $1$ ✅
⚠️ 흔한 실수 "$\gcd(12, 13) = 1$"이 "13이 소수라서" 라고만 설명하면 부족! 증명은 $n$이 어떤 자연수든 성립해야 해요. 그래서 "차 = 1"이 핵심 단서.

정답 · Page ★

정답 + 풀이 흐름 요약

📖 핵심 정리: $d \mid a$이고 $d \mid b$ → $d \mid (a - b)$. 이걸 "연속 자연수의 차 = 1"에 적용하면 끝.
$n$과 $n+1$은 항상 서로소 (증명 완료)
$n = 12$: $\gcd(12, 13) = 1$ ✅
전체 풀이 흐름 — 4단계
1
핵심 성질: $d \mid a$, $d \mid b$ → $d \mid (a - b)$
2
$d = \gcd(n, n+1)$이면 $d \mid n$, $d \mid (n+1)$
3
$d \mid \big((n+1) - n\big) = d \mid 1$ → $d = 1$
4
확인: $\gcd(12, 13) = 1$ ($12 = 2^2 \times 3$, $13 = 13$ 공통 없음)
⚠️ 이 유형의 핵심 함정 "구체적 수로 확인"과 "일반적 증명"은 달라요! 구체 예 하나로는 모든 $n$에서 성립함을 보장하지 못해요. "차의 약수" 논리가 진짜 증명이에요.
