소인수분해 완전 이해

어려운 문제가 어려운 이유와 핵심 아이디어를 탐구해요

🤔 이 유형이 어려운 이유

소인수분해 자체는 쉬운데, 심화 문제에서 막히는 이유가 있어요:

1. 역방향 사고 — 소인수분해 결과에서 원래 수를 역추론해야 할 때
2. 조건 조합 — "소인수분해가 $2^a \times 3^b$이고, 약수 개수가 n개"처럼 여러 조건이 동시에 붙을 때
3. 지수의 의미 — 지수를 "몇 번 곱하는지"가 아니라 "몇 가지 경우가 가능한지"로 읽어야 할 때

이 세 가지를 구별하면 어떤 문제도 풀 수 있어요.

💡 핵심 아이디어 — 클릭하여 단계 공개

문제: "$2^a \times 3^b$의 약수의 개수가 12개일 때, a+b의 최솟값은?"

$(a+1)(b+1) = 12$
12를 두 양의 정수의 곱으로 나타내는 방법:
$1\times12$, $2\times6$, $3\times4$, $4\times3$, $6\times2$, $12\times1$
$(a+1, b+1)$ → $(a, b)$:
$(1,12)$ → $(0,11)$: $a=0$은 $2^0=1$이므로 소인수 2가 없음
$(2,6)$ → $(1,5)$: a+b=6
$(3,4)$ → $(2,3)$: a+b=5
$(4,3)$ → $(3,2)$: a+b=5
$(6,2)$ → $(5,1)$: a+b=6
$(12,1)$ → $(11,0)$: $b=0$은 소인수 3이 없음
a, b ≥ 1 조건이 있다면: (1,5), (2,3), (3,2), (5,1)
a+b 최솟값 = 5 ($(a,b)=(2,3)$ 또는 $(3,2)$일 때)
이 때 수는: $2^2 \times 3^3 = 108$ 또는 $2^3 \times 3^2 = 72$
$72 = 2^3 \times 3^2$ → 약수 개수 = $(3+1)(2+1) = 12$ ✅
$108 = 2^2 \times 3^3$ → 약수 개수 = $(2+1)(3+1) = 12$ ✅
a+b = 5 ✅

⚠️ 자주 하는 실수 패턴

  • 실수 1: $(a+1)(b+1)=12$에서 $(a,b)=(12,1)$처럼 한 쪽에만 다 넣음 → 조건을 만족하지만 최솟값/최댓값이 아닐 수 있음
  • 실수 2: "소인수분해 결과에서 소인수 확인" 시 지수 대신 밑만 보기 → $2^3 \times 3^2$의 소인수는 2, 3 (3이나 9가 아님)
  • 실수 3: 소인수분해가 유일하다는 것을 활용 안 함 → 조건을 만족하는 수가 하나뿐이라고 확신할 수 있는 근거
  • 실수 4: 나눗셈 과정에서 나누다가 멈춤 → 몫이 1이 될 때까지 계속 나눠야 함

✏️ 추가 연습 문제

1$n^3 = 2^6 \times 3^3$일 때, $n$을 소인수분해하세요.
$n = 2^2 \times 3 = 12$

$n^3 = (2^2 \times 3)^3 = 2^6 \times 3^3$이므로
$n = 2^2 \times 3 = 12$
2어떤 자연수 $N$을 소인수분해하면 $2^a \times 5^b$ ($a, b \geq 1$) 꼴이 되고, 약수의 개수가 6개입니다. $N$이 될 수 있는 수를 모두 구하세요.
20, 50, 100, 500

$(a+1)(b+1)=6$, $a,b \geq 1$
$(a+1, b+1)$: $(2,3), (3,2)$
$(a,b)$: $(1,2), (2,1)$
$2^1 \times 5^2 = 50$, $2^2 \times 5^1 = 20$
※ 범위 제한 없으면 무한히 많음. 두 자리 이하: 20, 50
3$\dfrac{756}{n}$이 자연수의 제곱이 되도록 하는 가장 작은 자연수 $n$을 구하세요.
$n = 21$

$756 = 2^2 \times 3^3 \times 7$
$\dfrac{756}{n} = \dfrac{2^2 \times 3^3 \times 7}{n}$이 완전제곱수가 되려면
홀수 지수를 짝수로 만들어야 함: $3^3$ (지수 3), $7^1$ (지수 1)
$n = 3 \times 7 = 21$
$\dfrac{756}{21} = 36 = 6^2$ ✅
41부터 100까지의 자연수 중 소인수가 2뿐인 수의 개수를 구하세요.
6개

2의 거듭제곱만: $2^1=2$, $2^2=4$, $2^3=8$, $2^4=16$, $2^5=32$, $2^6=64$
$2^7=128>100$ → 6개
