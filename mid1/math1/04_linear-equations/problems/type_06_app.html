유형 6 — 복잡한 방정식 (괄호+분수 복합)

처리 순서: 괄호 → 분수 → 이항 → 동류항
← 단원 4 홈
1
L
$3x - 2 = x + 8$을 풀어라.
이항: $3x-x = 8+2$
$x = 5$
$2x=10$, $x=5$
검산: $15-2=13$, $5+8=13$ ✓
2
L
$4(x - 1) = 2(x + 3)$를 풀어라.
양쪽 괄호를 먼저 전개해요.
$x = 5$
$4x-4=2x+6$
$2x=10$, $x=5$
검산: $4(4)=16$, $2(8)=16$ ✓
3
L
$\dfrac{x+2}{3} = 2$를 풀어라.
양변에 3을 곱해 분수를 없애요.
$x = 4$
양변 × 3: $x+2=6$
$x=4$
검산: $(4+2)/3=2$ ✓
1
M
$\dfrac{2x-1}{3} = \dfrac{x+2}{2}$를 풀어라.
LCM(3,2) = 6. 양변에 6을 곱해요.
$x = 8$
양변 × 6: $2(2x-1)=3(x+2)$
$4x-2=3x+6$
$x=8$
검산: $(16-1)/3=5$, $(8+2)/2=5$ ✓
2
M
$\dfrac{2(x-3)}{5} = \dfrac{x}{2} - 1$을 풀어라.
LCM(5,2) = 10. 양변에 10을 곱해요.
$x = -2$
양변 × 10: $4(x-3)=5x-10$
$4x-12=5x-10$
$-x=2$, $x=-2$
검산: $2(-5)/5=-2$, $(-2)/2-1=-2$ ✓
3
M
$\dfrac{x}{6} + 1 = \dfrac{x}{4} - 1$을 풀어라.
LCM(6,4) = 12. 양변에 12를 곱해요.
$x = 24$
양변 × 12: $2x+12=3x-12$
$-x=-24$, $x=24$
검산: $24/6+1=5$, $24/4-1=5$ ✓
1
H
$\dfrac{x-1}{2} - \dfrac{x+2}{3} = 1$을 풀어라.
LCM(2,3) = 6. 양변에 6을 곱해요. 괄호 전개 시 부호에 주의!
$x = 13$
양변 × 6: $3(x-1)-2(x+2)=6$
$3x-3-2x-4=6$
$x-7=6$, $x=13$
검산: $12/2-15/3=6-5=1$ ✓
2
H
$\dfrac{3x+2}{4} - \dfrac{x-1}{6} = \dfrac{x+3}{2}$를 풀어라.
LCM(4,6,2) = 12. 양변에 12를 곱해요.
$x = 10$
양변 × 12: $3(3x+2)-2(x-1)=6(x+3)$
$9x+6-2x+2=6x+18$
$7x+8=6x+18$, $x=10$
검산: $32/4-9/6=8-1.5=6.5$, $13/2=6.5$ ✓

방정식 $\dfrac{2x+a}{3} = \dfrac{x}{2} + 1$의 해가 $x = 6$일 때, 상수 $a$의 값을 구하세요.
$x=6$을 방정식에 대입해 $a$에 대한 식을 만들어요.
$a = 0$
$x=6$ 대입: $\frac{12+a}{3}=\frac{6}{2}+1=4$
$12+a=12$, $a=0$
확인: $\frac{12}{3}=4=3+1$ ✓
