$x + 5 = 12$를 풀어라.
5를 우변으로 이항하면? 이항할 때 부호를 바꿔요.
$x = 7$
$x = 12 - 5 = 7$
검산: $7+5=12$ ✓
2
L
$3x = 18$을 풀어라.
양변을 $x$의 계수 3으로 나눠요.
$x = 6$
$x = 18 \div 3 = 6$
검산: $3(6)=18$ ✓

L
$4x - 1 = 11$을 풀어라.
−1을 이항해서 $4x = ?$로 만들어요.
$x = 3$
$4x = 11+1 = 12$
$x = 3$
검산: $4(3)-1=11$ ✓

$5x - 3 = 2x + 9$를 풀어라.
$x$항은 좌변으로, 숫자는 우변으로 이항하면 $5x-2x = 9+3$이 돼요.
$x = 4$
$5x-2x = 9+3$
$3x=12$, $x=4$
검산: $5(4)-3=17$, $2(4)+9=17$ ✓

$2x + 7 = x - 3$을 풀어라.
이항 결과: $2x - x = -3 - 7$. 음수가 나올 수 있어요!
$x = -10$
$2x-x = -3-7$
$x=-10$
검산: $2(-10)+7=-13$, $(-10)-3=-13$ ✓
$x$가 음수가 나와도 괜찮아요. 검산으로 확인하면 돼요!

$6 - x = 2x - 9$를 풀어라.
$-x$를 우변으로, $-9$를 좌변으로 이항해봐요.
$x = 5$
$6+9 = 2x+x$
$15=3x$, $x=5$
검산: $6-5=1$, $2(5)-9=1$ ✓

방정식 $ax + 3 = 15$의 해가 $x = 4$일 때, 상수 $a$의 값을 구하세요.
$x=4$를 방정식에 대입하면 $a$에 대한 방정식이 돼요.
$a = 3$
$x=4$ 대입: $4a+3=15$
$4a=12$, $a=3$
확인: $3(4)+3=15$ ✓

방정식 $2x - a = x + 5$의 해가 $x = -2$일 때, 상수 $a$의 값을 구하세요.
$x=-2$를 대입해 $a$에 대한 방정식을 만들어요.
$a = -7$
$x=-2$ 대입: $2(-2)-a=(-2)+5$
$-4-a=3$
$-a=7$, $a=-7$
확인: $2(-2)-(-7)=-4+7=3$, $(-2)+5=3$ ✓

방정식 $3x + 5 = x + k$의 해가 $x = 1$일 때, 상수 $k$의 값을 구하세요.
$x=1$을 대입하면 $k$에 대한 방정식이 나와요.
$k = 7$
$x=1$ 대입: $3(1)+5=1+k$
$8=1+k$, $k=7$
확인: $3(1)+5=8$, $1+7=8$ ✓
