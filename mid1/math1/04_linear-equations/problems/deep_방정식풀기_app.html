핵심 아이디어 — 이항의 원리
이항은 등식의 성질(덧셈·뺄셈)을 빠르게 쓰는 방법이에요.
$x + 3 = 7$ → 양변에서 3을 빼면 $x = 7 - 3$
→ "+3이 −3으로 부호가 바뀌어 우변으로" = 이항!
단계별 풀이 흐름
1
$x$항은 좌변으로, 숫자는 우변으로 이항 (부호 바꾸기)
2
동류항 계산 (양변 각각 더하고 빼기)
3
$x$의 계수로 양변 나누기
4
검산 — 구한 값 대입해서 양변 확인
자주 하는 실수
❌
이항 시 부호를 안 바꿈
$x + 3 = 7 \Rightarrow x = 7 + 3 = 10$ (틀림!)
이항하면 반드시 부호가 바뀌어요: $x = 7 - 3 = 4$
❌
$x$ 계수를 뺄셈으로 처리
$3x = 12$에서 "$3x - 3 = 9$"로 계산하는 실수. $x$의 계수로 나눠야 해요: $x = 12 \div 3 = 4$
추가 문제
문제 1
$7x - 4 = 3x + 12$를 풀어라.
$7x-3x = 12+4$
$4x=16$, $x=4$
검산: $7(4)-4=24$, $3(4)+12=24$ ✓
문제 2
$-3x + 8 = 2x - 7$을 풀어라.
$-3x-2x = -7-8$
$-5x=-15$, $x=3$
검산: $-9+8=-1$, $6-7=-1$ ✓
문제 3
방정식 $5x + 3 = 2x + k$의 해가 $x = -2$일 때, $k$를 구하세요.
$x=-2$ 대입: $5(-2)+3=2(-2)+k$
$-7=-4+k$, $k=-3$
확인: $-7=-4+(-3)$ ✓
연결 개념
기본 풀기를 완벽히 하면 쉬워지는 것
  • 분수·소수 방정식 (유형 4): 정수로 변환 후 동일한 이항 절차를 따라요.
  • 방정식 활용 (유형 7): 식을 세우는 게 핵심이고, 풀기는 이 유형과 동일해요.
  • 연립방정식 (중2): 두 방정식을 더하거나 빼서 변수를 소거한 뒤, 이 방법으로 풀어요.
핵심 아이디어
분수 → LCM 곱하기: 분모를 모두 없애서 정수 방정식으로 변환
소수 → 10의 거듭제곱 곱하기: 소수점을 없애서 정수로 변환
변환 후에는 기본 풀기(유형 3)와 동일해요!
LCM 구하는 법

분모가 2, 3이면 LCM = 6
분모가 3, 4이면 LCM = 12
분모가 2, 3, 6이면 LCM = 6
분모가 4, 6이면 LCM = 12

분모에 있는 모든 수의 LCM을 양변에 곱해야 해요. 하나라도 빠뜨리면 식이 틀려요!
자주 하는 실수
❌
정수항에 LCM을 곱하는 걸 빠뜨림
$\frac{x}{2}+1=3$에서 양변 ×2: $x+1=6$ (틀림!)
올바른 풀이: $x+2=6$ (상수항 1에도 2를 곱해야!)
❌
괄호 안에 LCM 곱하기 실수
$\frac{x+1}{3}$에서 ×3: $x+1$ (올바름)
하지만 $\frac{2(x+1)}{3}$에서 ×3: $2(x+1)$ (분자 전체에 적용!)
추가 문제
문제 1
$\dfrac{x}{4} + \dfrac{x}{2} = 6$을 풀어라.
양변 × 4: $x + 2x = 24$
$3x=24$, $x=8$
검산: $8/4+8/2=2+4=6$ ✓
문제 2
$0.2x + 1.4 = 0.5x - 1$을 풀어라.
양변 × 10: $2x+14=5x-10$
$-3x=-24$, $x=8$
검산: $0.2(8)+1.4=3$, $0.5(8)-1=3$ ✓
문제 3
$\dfrac{3x-1}{2} - \dfrac{x+3}{5} = 1$을 풀어라.
LCM(2,5) = 10. 양변 × 10:
$5(3x-1) - 2(x+3) = 10$
$15x-5-2x-6=10$
$13x=21$, $x=\dfrac{21}{13}$
검산: $\frac{63/13-1}{2}-\frac{21/13+3}{5}=\frac{50/13}{2}-\frac{60/13}{5}=\frac{25}{13}-\frac{12}{13}=\frac{13}{13}=1$ ✓
연결 개념
분수·소수 방정식을 잘 하면 쉬워지는 것
  • 복잡한 방정식 (유형 6): 괄호+분수 혼합 → 분수 처리가 기본이에요.
  • 방정식 활용 (유형 7) — 농도 문제: 비율(%)을 소수로 쓰면 소수 방정식이 돼요.
  • 비례식 (중2): $a:b = c:d$ 형태가 분수 방정식과 연결돼요.
핵심 아이디어
괄호 먼저, 이항은 나중
괄호를 풀지 않고 이항하면 틀려요. 반드시 분배법칙으로 괄호를 먼저 전개하세요.

음수 앞 괄호 주의:
$-(2x-3) = -2x+3$ (부호가 모두 바뀜!)
$-3(x+2) = -3x-6$ (모든 항에 −3을 곱함)
자주 하는 실수
❌
음수 앞 괄호 전개 오류
$-(2x-3)$을 $-2x-3$으로 전개하는 실수!
올바른 답: $-2x+3$. 빼기 앞에서는 모든 부호가 바뀌어요.
❌
괄호 안 분배를 일부만 적용
$3(x-2)$를 $3x-2$로 쓰는 실수!
올바른 답: $3x-6$. 괄호 안 모든 항에 3을 곱해야 해요.
추가 문제
문제 1
$3(2x+1) - 2(x-4) = 17$을 풀어라.
$6x+3-2x+8=17$
$4x+11=17$
$4x=6$, $x=\dfrac{3}{2}$
검산: $3\!\left(2 \cdot \dfrac{3}{2}+1\right)-2\!\left(\dfrac{3}{2}-4\right)=3(4)-2\!\left(-\dfrac{5}{2}\right)=12+5=17$ ✓
문제 2
$4 - 2(3x - 5) = x + 3$을 풀어라.
$4-6x+10=x+3$
$14-6x=x+3$
$-7x=-11$, $x=\dfrac{11}{7}$
검산: $4-2(3\cdot\frac{11}{7}-5)=4-2(\frac{33}{7}-\frac{35}{7})=4-2(-\frac{2}{7})=4+\frac{4}{7}=\frac{32}{7}$
$x+3=\frac{11}{7}+3=\frac{32}{7}$ ✓
문제 3
방정식 $a(x-3)+2 = 5x-4$의 해가 $x = 5$일 때, $a$를 구하세요.
$x=5$ 대입: $a(2)+2=21$
$2a=19$, $a=\dfrac{19}{2}$... 확인: $\frac{19}{2}(5-3)+2=19+2=21$, $5(5)-4=21$ ✓
연결 개념
괄호 처리를 잘 하면 쉬워지는 것
  • 복잡한 방정식 (유형 6): 괄호+분수 혼합에서 괄호 전개가 기본.
  • 다항식 곱셈 (고등): $(a+b)(c+d)$ 전개도 분배법칙의 확장이에요.
  • 인수분해 (중3): 분배법칙의 역과정. 괄호를 잘 알아야 인수분해를 잘해요.
