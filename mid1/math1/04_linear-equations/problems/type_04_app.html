유형 4 — 분수·소수 계수 방정식

분모의 LCM 곱하기 / 10의 거듭제곱 곱하기
← 단원 4 홈
1
L
$\dfrac{x}{3} = 4$를 풀어라.
양변에 분모 3을 곱해서 분수를 없애요.
$x = 12$
양변 × 3: $x = 12$
검산: $12/3=4$ ✓
2
L
$0.2x = 1.6$을 풀어라.
소수 한 자리 → 양변에 10을 곱해요.
$x = 8$
양변 × 10: $2x=16$
$x=8$
검산: $0.2(8)=1.6$ ✓
3
L
$\dfrac{x}{2} - 1 = 2$를 풀어라.
먼저 양변에 2를 곱하거나, 이항 후 계산해도 돼요.
$x = 6$
양변 × 2: $x - 2 = 4$
$x=6$
검산: $6/2-1=2$ ✓
1
M
$\dfrac{x}{3} + \dfrac{x}{6} = 2$를 풀어라.
분모 3과 6의 최소공배수(LCM) = 6. 양변에 6을 곱해요.
$x = 4$
양변 × 6: $2x + x = 12$
$3x=12$, $x=4$
검산: $4/3+4/6=8/6+4/6=12/6=2$ ✓
2
M
$0.5x - 0.2 = 1.3$을 풀어라.
양변에 10을 곱해서 정수로 바꿔요.
$x = 3$
양변 × 10: $5x-2=13$
$5x=15$, $x=3$
검산: $0.5(3)-0.2=1.3$ ✓
3
M
$\dfrac{x}{4} - \dfrac{x}{3} = -1$을 풀어라.
분모 4와 3의 LCM = 12. 양변에 12를 곱해요.
$x = 12$
양변 × 12: $3x - 4x = -12$
$-x=-12$, $x=12$
검산: $12/4-12/3=3-4=-1$ ✓
1
H
$\dfrac{x-1}{2} + \dfrac{x+3}{4} = 3$을 풀어라.
분모 2와 4의 LCM = 4. 양변에 4를 곱해요. 괄호에 주의!
$x = \dfrac{11}{3}$
양변 × 4: $2(x-1) + (x+3) = 12$
$2x-2+x+3=12$
$3x+1=12$, $3x=11$, $x=\dfrac{11}{3}$
검산: $\frac{8/3}{2}+\frac{20/3}{4}=\frac{4}{3}+\frac{5}{3}=\frac{9}{3}=3$ ✓
해가 분수로 나올 수 있어요. 분수 해도 검산으로 확인하면 돼요!
2
H
$0.3(x+2) = 0.1(2x+1) + 0.9$를 풀어라.
양변에 10을 곱해서 소수를 정수로 바꿔요. 괄호도 그대로 유지!
$x = 4$
양변 × 10: $3(x+2)=1(2x+1)+9$
$3x+6=2x+1+9$
$3x+6=2x+10$
$x=4$
검산: $0.3(6)=1.8$, $0.1(9)+0.9=1.8$ ✓
3
H
$\dfrac{x}{2} - \dfrac{1}{3} = \dfrac{x+1}{6}$을 풀어라.
분모 2, 3, 6의 LCM = 6. 양변에 6을 곱해요.
$x = \dfrac{3}{2}$
양변 × 6: $3x - 2 = x+1$
$2x=3$, $x=\dfrac{3}{2}$
검산: $\frac{3/2}{2}-\frac{1}{3}=\frac{3}{4}-\frac{1}{3}=\frac{9-4}{12}=\frac{5}{12}$
$\frac{3/2+1}{6}=\frac{5/2}{6}=\frac{5}{12}$ ✓
