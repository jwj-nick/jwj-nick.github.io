어떤 수의 3배보다 7이 큰 수가 31일 때, 어떤 수를 구하세요.
"어떤 수의 3배보다 7이 큰 수" = $3x + 7$
8
방정식 세우기
어떤 수를 $x$라 하면: $3x + 7 = 31$
풀기
$3x = 24$, $x = 8$
검토
$3(8)+7=31$ ✓

연속하는 세 자연수의 합이 36일 때, 가운데 수를 구하세요.
가운데 수를 $x$라 하면 세 수는 $x-1$, $x$, $x+1$이에요.
12

$(x-1)+x+(x+1)=36$
$3x=36$
풀기 + 검토
$x=12$
검토: $11+12+13=36$ ✓, 자연수 조건 ✓

한 개에 500원인 과자와 한 개에 300원인 사탕을 합쳐 10개 사고 4,200원을 냈습니다. 과자는 몇 개 샀나요?
과자 $x$개라 하면 사탕은 $(10-x)$개예요.
6개
방정식 세우기
$500x + 300(10-x) = 4200$
풀기
$500x+3000-300x=4200$
$200x=1200$, $x=6$
검토
$500(6)+300(4)=3000+1200=4200$ ✓

현재 아버지의 나이는 44세이고, 딸의 나이는 16세입니다. 몇 년 후에 아버지의 나이가 딸의 나이의 2배가 될까요?
$x$년 후 아버지: $44+x$, 딸: $16+x$. "아버지 = 딸의 2배"로 등식 세워요.
12년 후
방정식 세우기
$44+x = 2(16+x)$
풀기
$44+x=32+2x$
$-x=-12$, $x=12$
검토
12년 후: 아버지 56세, 딸 28세
$56 = 2 \times 28$ ✓

500원짜리 동전과 100원짜리 동전이 합쳐서 15개이고, 총 금액은 4,300원입니다. 500원짜리 동전은 몇 개인가요?
500원짜리 $x$개라 하면 100원짜리는 $(15-x)$개예요.
7개
방정식 세우기
$500x + 100(15-x) = 4300$
풀기
$500x+1500-100x=4300$
$400x=2800$, $x=7$
검토
$500(7)+100(8)=3500+800=4300$ ✓

갑은 A지점에서 출발해 시속 60km로, 을은 A지점에서 1시간 먼저 출발해 시속 40km로 같은 방향으로 이동합니다. 갑이 출발한 지 몇 시간 후에 갑이 을을 따라잡나요?
갑이 출발한 지 $x$시간 후 따라잡으면, 을은 $(x+1)$시간 이동. 두 사람의 이동 거리가 같아요.
2시간 후
방정식 세우기
갑의 거리 = 을의 거리
$60x = 40(x+1)$
풀기
$60x=40x+40$
$20x=40$, $x=2$
검토
갑: $60 \times 2 = 120$km
을: $40 \times 3 = 120$km ✓

12%의 소금물 300g에 소금을 더 녹여 20%의 소금물을 만들려고 합니다. 소금을 몇 g 더 넣어야 하나요?
소금 $x$g 추가 → 소금물 $(300+x)$g, 소금 $(36+x)$g. 농도 공식: 소금 ÷ 소금물 = 농도.
30g
현재 소금량
$0.12 \times 300 = 36$g
방정식 세우기
$\dfrac{36+x}{300+x} = 0.2$
풀기
$36+x = 0.2(300+x)$
$36+x = 60+0.2x$
$0.8x = 24$, $x=30$
검토
$\dfrac{66}{330} = 0.2 = 20\%$ ✓
농도 공식: $\text{농도} = \dfrac{\text{소금의 양}}{\text{소금물의 양}} \times 100\%$. 소금을 추가하면 소금물도 늘어요!

8%의 소금물 A와 14%의 소금물 B를 섞어서 12%의 소금물 300g을 만들려고 합니다. 소금물 A는 몇 g 필요한가요?
A를 $x$g 쓰면 B는 $(300-x)$g. 소금의 양: $0.08x + 0.14(300-x) = 0.12 \times 300$
100g
방정식 세우기
$0.08x + 0.14(300-x) = 36$
양변 × 100: $8x + 14(300-x) = 3600$
풀기
$8x+4200-14x=3600$
$-6x=-600$, $x=100$
검토
$0.08(100)+0.14(200)=8+28=36=0.12\times300$ ✓

둘레가 60cm인 직사각형이 있습니다. 가로의 길이가 세로의 길이보다 6cm 길다면, 가로와 세로는 각각 몇 cm인가요?
세로를 $x$cm라 하면 가로는 $(x+6)$cm. 둘레 = 2×(가로+세로)
가로 18cm, 세로 12cm
방정식 세우기
$2\{(x+6)+x\} = 60$
$2(2x+6)=60$
풀기
$4x+12=60$
$4x=48$, $x=12$
검토
가로 $18$cm, 세로 $12$cm
$2(18+12)=60$ ✓, 가로-세로 $=6$cm ✓
