깊이 탐구 — 방정식 기초

방정식·항등식 판별 + 등식의 성질

← 단원 4 홈
이 유형이 어려운 이유

방정식과 항등식은 둘 다 "등식"이에요. 겉모습이 비슷해서 헷갈려요.

핵심 차이는 딱 하나: 특정 값에서만 참이면 방정식, 모든 값에서 참이면 항등식.

핵심 아이디어
확인 방법:
$x$에 0, 1, 2를 대입해서 3번 모두 참 → 항등식일 가능성 높음
한 번이라도 거짓 → 방정식
단, 확실히 하려면 식을 전개해서 양변 비교!
항등식 미정계수 문제:
$ax+b = cx+d$가 항등식 → $a=c$이고 $b=d$
(같은 위치의 계수끼리 같아야 함)
단계별 풀이 흐름
1
등식을 확인: 좌변과 우변을 구분해요.
2
양변 전개: 분배법칙 등으로 전개해요.
3
비교: 양변이 같은 식이면 항등식, 아니면 방정식.
4
미정계수: 계수 비교로 $a$, $b$ 결정.
자주 하는 실수
❌
해가 있으면 방정식이라고 착각
항등식도 해가 있어요. 아니, 항등식은 모든 값이 해예요! "해"의 개수로 구분하지 마세요.
❌
$x=0$만 대입해서 확인
$x=0$에서만 참인 방정식도 있어요. 최소 2~3개 값을 대입해서 확인하세요.
❌
미정계수 문제에서 상수항을 빠뜨림
$ax+b=3x+c$에서 $a=3$만 구하고 $b=c$ 조건을 빠뜨리는 실수.
추가 문제 (5-c와 겹치지 않음)
문제 1
$2(3x - a) = 6x + 4$가 $x$에 관한 항등식이 되도록 하는 $a$의 값을 구하세요.
좌변 전개: $6x - 2a$
항등식: $6x-2a=6x+4$
계수 비교: $-2a=4 \Rightarrow a=-2$
문제 2
$x = -1$이 방정식 $3x + a = x - 5$의 해일 때, 상수 $a$의 값을 구하세요.
$x=-1$ 대입: $3(-1)+a=(-1)-5$
$-3+a=-6$
$a=-3$
확인: $3(-1)+(-3)=-6$, $(-1)-5=-6$ ✓
문제 3
다음 중 항등식인 것을 모두 고르세요.
① $x+2=2+x$   ② $x-x=0$   ③ $x^2=x$   ④ $2x=x+x$
① 교환법칙으로 항등식 ✓
② $x-x=0$은 어떤 $x$든 0=0 → 항등식 ✓
③ $x=0$ 또는 $x=1$일 때만 참 → 방정식
④ $x+x=2x$이므로 $2x=x+x$ → 항등식 ✓
답: ①②④
연결 개념
이 유형 이해 후 쉬워지는 것
  • 이차방정식 (중3): 방정식의 해 개념이 더 복잡해져요. 항등식 vs 방정식 구분이 기초가 돼요.
  • 함수 (중1~2): 등식 $y = 2x+1$은 방정식처럼 보이지만 "모든 $x$에 대해" 정의 → 항등식적 성질.
  • 증명 (고등): 등식 증명 문제에서 항등식 조건을 직접 사용해요.
이 유형이 어려운 이유

등식의 성질은 쉬워 보이지만, 나눗셈의 "$c \neq 0$ 조건"을 자꾸 잊어요.

또한 여러 단계로 연결해서 사용할 때 각 단계의 근거를 명확히 해야 해요.

핵심 아이디어 — 저울 원리
등식은 저울과 같아요.
양쪽에 같은 것을 더하거나 빼도 → 균형 유지
양쪽을 같은 수로 곱하거나 나눠도 → 균형 유지
단, 0으로 나누면 → 저울 자체가 망가짐 (정의 불가)
0으로 나누기 금지!
$\dfrac{a}{0}$은 어떤 값도 될 수 없어요. 수학에서 정의되지 않아요.
4가지 성질 한눈에
성질 수식 조건
덧셈$a=b \Rightarrow a+c=b+c$없음
뺄셈$a=b \Rightarrow a-c=b-c$없음
곱셈$a=b \Rightarrow ac=bc$없음
나눗셈$a=b \Rightarrow a/c=b/c$$c \neq 0$
자주 하는 실수
❌
나눗셈 성질에서 $c=0$ 대입
0으로 나누는 것은 항상 오류예요. 조건 "$c \neq 0$"을 잊으면 안 돼요.
❌
등식의 성질을 한 번에 여러 번 쓰는 실수
$x-3=5$에서 "$x$의 계수가 1이니까 바로 $x=8$"이라고 넘어가는 것은 나쁜 습관이에요. 한 단계씩 근거를 잡아요.
추가 문제 (5-c와 겹치지 않음)
문제 1
$a = b$일 때, 다음 중 항상 참인 것을 모두 고르세요.
① $a+5 = b+5$   ② $3a = 3b$   ③ $a/0 = b/0$   ④ $-a = -b$
① $a=b$ → 양변+5 → 참 ✓
② $a=b$ → 양변×3 → 참 ✓
③ 0으로 나누기 → 거짓 ✗
④ $a=b$ → 양변×(−1) → $-a=-b$ → 참 ✓
답: ①②④
문제 2
등식의 성질만 이용하여 $\dfrac{x}{5} = 3$을 단계별로 풀어보세요.
양변 × 5: $\dfrac{x}{5} \times 5 = 3 \times 5$
$x = 15$
검산: $15/5=3$ ✓
문제 3
$5a = 5b$이면 $a = b$가 성립하나요? 이유와 함께 설명하세요.
성립합니다.
$5a=5b$ → 양변을 5로 나눔 ($5 \neq 0$이므로 가능)
$a=b$ ✓
나눗셈 성질의 조건 $c \neq 0$을 만족하므로 사용 가능해요.
연결 개념
이 유형 이해 후 쉬워지는 것
  • 방정식 풀기 (유형 3~6): 이항의 원리가 등식의 성질에서 비롯돼요. 근거를 알면 실수가 줄어요.
  • 부등식 (중2): 등식 성질과 비슷하지만, 곱셈/나눗셈에서 음수이면 부등호 방향이 바뀌어요.
  • 연립방정식 (중2): 두 방정식에 같은 수를 곱한 뒤 더하거나 빼는 방법 = 등식 성질의 확장.
