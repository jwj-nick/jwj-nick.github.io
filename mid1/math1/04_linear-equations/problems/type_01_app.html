등식 $x + 2 = 5$는 방정식인가요, 항등식인가요? 그 이유도 설명하세요.
$x$에 1, 2, 3을 각각 대입해서 참이 되는지 확인해봐요.
방정식
$x=3$일 때만 참 ($3+2=5$). $x=1, 2$ 등을 넣으면 거짓.
특정 값에서만 참 → 방정식!
2
L
$x = 1$이 방정식 $2x + 1 = 3$의 해인지 확인하세요.
$x=1$을 대입해서 좌변과 우변이 같은지 봐요.
해이다
좌변: $2(1)+1=3$, 우변: $3$ → 같으므로 $x=1$은 이 방정식의 해예요.

L
$3(x-1) = 3x - 3$은 방정식인가요, 항등식인가요?
좌변을 분배법칙으로 전개해 우변과 비교해봐요.
항등식
좌변: $3x-3$ = 우변: $3x-3$ → 어떤 $x$를 넣어도 항상 참!
방정식처럼 보여도 양변이 같은 식이면 항등식이에요.

M
등식 $ax + 2 = 4x + b$가 $x$에 관한 항등식일 때, 상수 $a$, $b$의 값을 구하세요.
항등식이 되려면 양변의 $x$의 계수와 상수항이 각각 같아야 해요.
$a=4$, $b=2$
항등식 조건: $x$의 계수 → $a=4$, 상수항 → $2=b$
즉 $a=4$, $b=2$

$x + 3 = 3 + x$는 방정식인가요, 항등식인가요? 그 이유를 설명하세요.
$x$에 0, 1, 2, −1 등을 넣어 확인해봐요.
항등식
덧셈의 교환법칙: $x+3=3+x$는 어떤 $x$에도 참.
이처럼 교환법칙·결합법칙·분배법칙으로 만들어진 등식은 항등식!

방정식 $2x - 1 = 5$의 해를 구하고, 그 해를 대입해 검산하세요.
이항: $2x = 5+1$로 시작해요.
$x=3$
$2x=6$, $x=3$
검산: 좌변 $= 2(3)-1=5$ = 우변 ✓

$(a-1)x + 3b = 0$이 $x$에 관한 항등식이 되도록 하는 상수 $a$, $b$의 값을 구하세요.
항등식이 되려면 모든 항의 계수가 0이어야 해요. $x$의 계수 = 0, 상수항 = 0.
$a=1$, $b=0$
항등식 $\Leftrightarrow$ 모든 $x$에서 $0=0$이어야 함.
$x$의 계수: $a-1=0 \Rightarrow a=1$
상수항: $3b=0 \Rightarrow b=0$

H
$x=2$가 방정식 $ax + 6 = 2$의 해일 때, 상수 $a$의 값을 구하세요.
해라는 것은 "대입하면 등식이 성립한다"는 의미예요. $x=2$를 방정식에 대입해봐요.
$a = -2$
$x=2$ 대입: $2a+6=2$
$2a=-4$
$a=-2$
확인: $-2(2)+6=2$ ✓

H
등식 $ax + b = 3x + c$가 $x$에 관한 항등식일 때, $a$와 $b-c$의 값을 구하세요.
$x$의 계수끼리, 상수항끼리 같아야 항등식이 돼요.
$a=3$, $b-c=0$
$x$의 계수: $a=3$
상수항: $b=c \Rightarrow b-c=0$
항등식에서 $b$와 $c$는 같은 값이어야 해요. 각각의 구체적인 값은 주어지지 않으면 알 수 없어요.
