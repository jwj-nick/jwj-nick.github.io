Unit 04 · 개념 1/5

방정식 vs 항등식

등식이란?

등호 =를 사용하여 두 식이 같음을 나타낸 식을 등식이라고 해요.

= (등호) — "왼쪽과 오른쪽이 같다"는 뜻이에요. 수학에서 가장 중요한 기호 중 하나예요.
종류뜻예
방정식특정 값일 때만 참이 되는 등식$x + 3 = 5$
$x=2$일 때만 참
항등식어떤 값을 넣어도 항상 참인 등식$2(x+1) = 2x+2$
모든 $x$에 대해 참

핵심 용어

  • 미지수: 값을 모르는 문자 (보통 $x$)
  • 해(solution): 방정식을 참이 되게 하는 미지수의 값
  • 풀다(solve): 해를 구하는 것

$x + 3 = 5$ — 해가 $x = 2$인 이유

$x = 2$ 대입: 좌변 $= 2 + 3 = 5$, 우변 $= 5$ → 같으면 해!
$x = 1$ 대입: 좌변 $= 1 + 3 = 4$, 우변 $= 5$ → 다르면 해가 아님.
자주 하는 실수: "$x = 2$는 항상 맞는 답이겠지?"라고 생각하는 것. 방정식마다 해가 달라요!
Unit 04 · 개념 2/5

등식의 성질 — 저울처럼

방정식은 저울이에요

등식은 양쪽이 균형 잡힌 저울과 같아요. 양쪽에 같은 것을 더하거나 빼도 균형이 유지돼요.

$x + 3 = 7$ — 양쪽이 균형 잡힌 저울

등식의 4가지 성질

성질 표현
덧셈$a=b$ 이면 $a+c = b+c$
뺄셈$a=b$ 이면 $a-c = b-c$
곱셈$a=b$ 이면 $a \times c = b \times c$
나눗셈$a=b$ 이면 $\dfrac{a}{c} = \dfrac{b}{c}$ (단, $c \neq 0$)
$c \neq 0$ 조건 — 나눗셈에서 0으로 나누면 안 돼요. "0으로 나눈다"는 것은 수학에서 정의되지 않아요.
Unit 04 · 개념 3/5

이항 — 부호 바꿔 건너가기

이항이란?

등식의 한 항을 부호를 바꾸어 다른 변으로 옮기는 것을 이항(transposition)이라고 해요.

이항은 등식의 성질(덧셈·뺄셈)을 이용해요.
양변에서 같은 수를 빼면, 한쪽 항이 반대편으로 "부호가 바뀌어" 이동하는 것처럼 보여요.

이항 시각화

$x$ + 3 $= 7$ → 양변에서 3을 빼면
$x + 3 - 3 = 7 - 3$ → 정리하면
$x = 7$ $- 3$ $= 4$
+3이 −3으로 부호 바뀌어 우변으로 이동
자주 하는 실수: 이항할 때 부호를 안 바꾸는 것.
$x + 3 = 7$ → $x = 7 + 3 = 10$ (틀림!) — 반드시 $x = 7 - 3 = 4$

직접 바꿔보세요 — $x + a = b$

$a$ 값: 3
$b$ 값: 7
$$x + 3 = 7$$
$x = 7 - 3 = 4$
$a$와 $b$를 바꿔보세요. 이항할 때 부호가 어떻게 바뀌는지 관찰해봐요!
Unit 04 · 개념 4/5

일차방정식 — 단계별 풀기

일차방정식이란?

$ax + b = 0$ 꼴 (단, $a \neq 0$)로 변형할 수 있는 방정식이에요.

"일차"란 미지수의 최고 차수가 1이라는 뜻이에요. ($x^2$ 없음)

예제: 단계를 클릭해서 펼쳐보세요

$$2x - 3 = x + 5$$
1
이항: $x$항은 좌변, 숫자는 우변으로
$$2x - x = 5 + 3$$
$x$ → 이미 좌변 / $-3$은 우변으로 $+3$이 됨 / $x$는 우변에서 좌변으로 $-x$가 됨
2
동류항 계산
$$x = 8$$
좌변: $2x - x = x$ / 우변: $5 + 3 = 8$
3
검산
$x=8$ 대입: 좌변 $= 2(8) - 3 = 13$, 우변 $= 8 + 5 = 13$ ✓
검산은 귀찮아도 꼭 해야 해요! 시험에서 실수를 잡아주는 안전망이에요.

복잡한 경우 처리 순서

  • 괄호 있으면 → 먼저 분배법칙으로 풀기
  • 분수 있으면 → 양변에 분모의 LCM 곱하기
  • 소수 있으면 → 양변에 10, 100 등 곱하기
  • 그 다음 이항 → 동류항 계산 → 계수로 나누기

방정식 활용 + 직접 써보기

활용 문제 푸는 4단계

1
미지수 설정
구하려는 것을 $x$로 놓아요. ("어떤 수를 $x$라 하면…")
2
방정식 세우기
수량 관계를 등식으로 표현해요. (거리=속력×시간, 전체=단가×개수 등)
3
방정식 풀기
이항 → 동류항 → 계수로 나누기
4
검토
구한 $x$가 문제 조건에 맞는지 확인. ($x$가 자연수여야 하는 경우 등)

예제

어떤 수의 2배에서 5를 뺀 것은 어떤 수에 3을 더한 것과 같다. 어떤 수는?

어떤 수를 $x$라 하면: $2x - 5 = x + 3$
이항: $2x - x = 3 + 5$
정리: $x = 8$
검토: $2(8)-5=11$, $8+3=11$ ✓
