$y = -\dfrac{6}{x}$의 그래프에 대한 설명으로 옳지 않은 것은?
① 쌍곡선이다   ② 원점을 지난다   ③ 제2·4사분면을 지난다   ④ $x$축, $y$축에 점점 가까워진다
반비례 그래프와 원점의 관계는?
② 원점을 지난다 (틀린 설명)
반비례는 $x = 0$에서 정의되지 않음 → 원점을 지나지 않음.
① 쌍곡선 ✓   ③ $a = -6 < 0$이므로 2·4사분면 ✓   ④ 점근선 ✓

L
반비례 $y = \dfrac{a}{x}$의 그래프가 제2·4사분면에 있다. $a$의 부호는?
2·4사분면 → $x$와 $y$의 부호 관계는? $xy = a$ 사용.
$a < 0$ (음수)
2사분면: $x < 0, y > 0 \Rightarrow xy < 0$.
4사분면: $x > 0, y < 0 \Rightarrow xy < 0$.
$xy = a < 0$ → $a$는 음수.

두 반비례 그래프 $y = \dfrac{4}{x}$ 와 $y = \dfrac{-4}{x}$ 의 위치 관계를 설명하시오.
$|a|$가 같고 부호만 다름. $x = 1$ 대입해서 두 점 비교.
$x$축에 대해 대칭 (또는 $y$축에 대해 대칭)
$x = 1$일 때: $y = 4$ vs $y = -4$. → 같은 $x$에 대해 $y$가 부호 반대 → $x$축 대칭.
$y = 1$일 때: $x = 4$ vs $x = -4$. → 같은 $y$에 대해 $x$가 부호 반대 → $y$축 대칭.
두 그래프는 $x$축 대칭이면서 동시에 $y$축 대칭이에요. (또는 원점 대칭으로 해석해도 같음)

M
반비례 $y = \dfrac{a}{x}$ 의 그래프가 점 $(-4, 5)$ 를 지날 때, 이 그래프가 점 $(2, ?)$ 도 지난다. ?를 구하시오.
$xy = a$ 사용. 부호 신경.
$? = -10$
$a = (-4) \times 5 = -20$. 식: $y = \dfrac{-20}{x}$.
$x = 2$ 대입: $y = \dfrac{-20}{2} = -10$.

$|a|$가 클수록 원점에서 멀어요. 같은 $x = 2$ 에서 두 그래프의 $y$값을 비교.
$y = \dfrac{12}{x}$ 가 더 멀리 있음
$|a|$ 비교: $|12| = 12$ vs $|4| = 4$.
$|a|$가 클수록 원점에서 멀어요.
확인: $x = 2$에서 $y = 6$ vs $y = 2$ → $y = 12/x$ 가 더 위로 올라감 (1사분면에서).

반비례 $y = \dfrac{a}{x}$ 의 그래프가 제2·4사분면을 지나며 점 $(3, b)$를 지난다. $b$의 부호를 결정하고, $b$의 가능한 값 범위를 설명하시오.
제 몇 사분면? $x = 3 > 0$ 이므로 → ?
$b < 0$ (음수). 구체적 값은 $a$ 값에 따라 달라짐.
2·4사분면 그래프 → $a < 0$.
$x = 3 > 0$ 이면 점 $(3, b)$ 는 4사분면 → $b < 0$.
$b = a/3 < 0$ ($a < 0$이므로). 예: $a = -6$이면 $b = -2$.

H
반비례 $y = \dfrac{6}{x}$ 의 그래프 위에서 좌표가 모두 정수인 점은 몇 개인가? (모든 정수 점을 찾아 나열)
$xy = 6$ 인 정수 짝 $(x, y)$ 모두 찾기. 6의 약수 활용.
총 8개
$xy = 6$ 인 정수 짝:
양수: $(1,6), (2,3), (3,2), (6,1)$ → 4개
음수: $(-1,-6), (-2,-3), (-3,-2), (-6,-1)$ → 4개
총 8개. 1·3사분면에 4개씩 분포.
$x \neq 0$ 임을 잊지 말 것. (반비례는 $y$축 위에 점이 없음)

H
두 반비례 그래프 $y = \dfrac{6}{x}$ 와 $y = \dfrac{a}{x}$ 가 $x$축에 대해 대칭이려면 $a$의 값은? 또 같은 그래프가 되려면 $a$의 값은?
"$x$축 대칭" → 같은 $x$에 대해 $y$ 부호 반대. "같은 그래프" → 식이 같음.
$x$축 대칭: $a = -6$,   같은 그래프: $a = 6$
① $x$축 대칭: $y$값이 부호 반대 → $a = -6$.
② 같은 그래프: 식이 같음 → $a = 6$.
