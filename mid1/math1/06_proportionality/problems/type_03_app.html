길이 36 cm인 끈을 $x$도막으로 똑같이 자를 때, 한 도막의 길이를 $y$ cm라 한다. $y$를 $x$의 식으로 나타내시오.
한 도막 길이 = 전체 길이 ÷ 도막 수.
$y = \dfrac{36}{x}$
도막 수 $x$가 2배, 3배 되면 한 도막 길이 $y$는 1/2배, 1/3배 → 반비례.
비례상수 $a = 36$ → $y = \dfrac{36}{x}$.

다음 중 $y$가 $x$에 반비례하는 것을 모두 고르시오.
(가) $y = -\dfrac{8}{x}$   (나) $y = 8x$   (다) $xy = 12$   (라) $y = \dfrac{x}{4}$
반비례는 $y = a/x$ 또는 $xy = a$ 형태. $x$가 분모에 있어야 해요.
(가), (다)
(가) $y = -\dfrac{8}{x}$ → 반비례 ($a = -8$) ✓
(나) $y = 8x$ → 정비례
(다) $xy = 12 \Rightarrow y = \dfrac{12}{x}$ → 반비례 ($a = 12$) ✓
(라) $y = \dfrac{x}{4} = \dfrac{1}{4}x$ → 정비례 ($x$가 분자에 있음)

★ 학습자 핵심 함정
$y$가 $x$에 반비례하고 비례상수 $a = -2$이다. $y$를 $x$의 식으로 옳게 나타낸 것을 모두 고르시오.
(가) $y = \dfrac{-2}{x}$   (나) $y = -\dfrac{2}{x}$   (다) $y = \dfrac{2}{-x}$   (라) $xy = -2$   (마) $y = \dfrac{x}{-2}$
각 식을 가장 간단한 $y = a/x$ 또는 $xy = a$ 형태로 만들고, $a$ 값이 $-2$인지 확인. 또 $x$가 분모에 있는지도 확인.
(가), (나), (다), (라) — (마)는 정비례
(가) $y = \dfrac{-2}{x}$ → 분자에 부호. 표준 형태 ✓
(나) $y = -\dfrac{2}{x} = \dfrac{-2}{x}$ → (가)와 같은 식 ✓
(다) $y = \dfrac{2}{-x} = \dfrac{-2}{x}$ → 분자·분모 모두 부호 정리하면 같음 ✓
(라) $xy = -2 \Rightarrow y = \dfrac{-2}{x}$ → 같은 식 ✓
(마) $y = \dfrac{x}{-2} = -\dfrac{1}{2}x$ → $x$가 분자에 있으므로 정비례 X
★ 분자에 부호와 분수 앞 부호는 같은 것: $\dfrac{-2}{x} = -\dfrac{2}{x}$. 식이 달라 보여도 같은 그래프 (2·4사분면 쌍곡선).

한 변의 길이가 $x$ cm인 직사각형의 다른 변의 길이를 $y$ cm라 할 때, 넓이가 항상 24 cm²이다. $y$를 $x$의 식으로 나타내고, $x = 8$일 때 $y$값을 구하시오.
직사각형 넓이 = 가로 × 세로. 둘 곱이 일정 → 반비례.
$y = \dfrac{24}{x}$,   $y = 3$ cm
$xy = 24 \Rightarrow y = \dfrac{24}{x}$.
$x = 8$ 대입: $y = \dfrac{24}{8} = 3$ cm.

$xy = -15$ → $y = \dfrac{-15}{x}$. 비례상수 $a = -15$.
$x = 3$: $y = \dfrac{-15}{3} = -5$.

$y$가 $x$에 반비례하고 $x = -3$일 때 $y = 4$이다. 식을 구하고, $y = -6$일 때 $x$의 값을 구하시오.
$xy = a$ 사용해서 비례상수 $a$ 결정. 부호에 주의.
$y = \dfrac{-12}{x}$,   $x = 2$
$xy = a$ 에 대입: $(-3) \times 4 = -12 \Rightarrow a = -12$.
식: $y = \dfrac{-12}{x}$ (= $-\dfrac{12}{x}$).
$y = -6$ 대입: $-6 = \dfrac{-12}{x} \Rightarrow x = \dfrac{-12}{-6} = 2$.

$y$는 $x$에 반비례하고, $x$가 1만큼 증가할 때 $y$가 일정하게 변하지 않는다. 이 사실을 표 또는 식으로 설명하시오. ($a = 12$ 인 반비례 사용)
$y = 12/x$ 의 표를 만들어 $x$가 1, 2, 3, 4 일 때 $y$의 변화량을 직접 계산해 봐요.
$x$ 변화 1당 $y$ 변화량은 일정하지 않음
$x$1234
$y$12643
$y$ 변화량—$-6$$-2$$-1$
$x$ 가 1씩 늘어도 $y$ 의 변화량은 $-6, -2, -1$ 처럼 점점 작아져요.
이것이 반비례와 정비례(직선)의 큰 차이예요.
정비례에서는 $x$ 1 증가 시 $y$ 변화량이 항상 $a$로 일정.

반비례 $y = \dfrac{a}{x}$ 의 그래프가 점 $(2, -3)$ 을 지난다. 이 그래프가 점 $(b, 6)$도 지날 때 $a, b$의 값을 모두 구하시오.
먼저 $a$ 결정, 그 다음 $b$.
$a = -6$,   $b = -1$
① $a = xy = 2 \times (-3) = -6$. 식: $y = \dfrac{-6}{x}$.
② $(b, 6)$ 대입: $6 = \dfrac{-6}{b} \Rightarrow b = \dfrac{-6}{6} = -1$.
