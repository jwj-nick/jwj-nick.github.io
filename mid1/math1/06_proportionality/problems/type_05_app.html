유형 5 — 비례상수 결정

한 점 → 비례상수 $a$ → 식 → 다른 점
← 단원 6 홈
1
L
$y$는 $x$에 정비례하고 $x = 5$일 때 $y = 20$이다. 비례상수 $a$의 값과 식을 구하시오.
$y = ax$ 에 대입.
$a = 4$, $y = 4x$
$20 = a \times 5 \Rightarrow a = 4$. 식: $y = 4x$.
2
L
$y$는 $x$에 반비례하고 $x = 6$일 때 $y = 2$이다. 비례상수 $a$의 값과 식을 구하시오.
$xy = a$.
$a = 12$, $y = \dfrac{12}{x}$
$a = xy = 6 \times 2 = 12$. 식: $y = \dfrac{12}{x}$.

L
정비례 $y = ax$ 가 점 $(2, -10)$ 을 지난다. $a$의 값과 $x = 4$일 때 $y$의 값을 구하시오.
$a$ 먼저 → 식 → $x = 4$ 대입.
$a = -5$, $y = -20$
$-10 = a \times 2 \Rightarrow a = -5$. 식: $y = -5x$.
$x = 4$: $y = -20$.

M
$y$가 $x$에 정비례하고 $x = -3$일 때 $y = 12$이다. $x = 5$일 때 $y$의 값은?
$a$ 먼저 — 부호 주의.
$y = -20$
$12 = a \times (-3) \Rightarrow a = -4$. 식: $y = -4x$.
$x = 5$: $y = -4 \times 5 = -20$.
2
M
$y$가 $x$에 반비례하고 점 $(4, -3)$ 을 지난다. $x = -6$일 때 $y$의 값은?
$xy = a$ → $a$ 결정 → 식 → 대입.
$y = 2$
$a = 4 \times (-3) = -12$. 식: $y = \dfrac{-12}{x}$.
$x = -6$: $y = \dfrac{-12}{-6} = 2$.

M
$y$가 $x$에 정비례하고 $x = 2$일 때 $y = -8$ 이다. $y = 12$일 때 $x$의 값은?
$a$ 결정 → $y$를 알 때 $x$ 구하기 → 역방향 대입.
$x = -3$
$-8 = a \times 2 \Rightarrow a = -4$. 식: $y = -4x$.
$y = 12$ 대입: $12 = -4x \Rightarrow x = -3$.

H
$y$가 $x$에 정비례하고, $x$의 값이 $-2$에서 4로 변할 때 $y$의 값은 8에서 $-16$ 으로 변한다. 비례상수 $a$의 값을 구하시오.
$(-2, 8)$ 또는 $(4, -16)$ 어느 점이든 식에 대입. 두 점 모두 같은 $a$를 줘야 일관성 확인.
$a = -4$
점 $(-2, 8)$: $8 = a \times (-2) \Rightarrow a = -4$.
점 $(4, -16)$: $-16 = a \times 4 \Rightarrow a = -4$. ✓ (일관성 확인)
식: $y = -4x$.
2
H
$y$가 $x$에 반비례하고 두 점 $(3, k)$, $(k+5, 2)$ 가 모두 같은 그래프 위에 있을 때, 양수 $k$의 값과 비례상수 $a$를 구하시오.
같은 반비례 그래프 → $x \cdot y$ 값이 같음. 두 곱을 같다고 두면 $k$에 대한 일차방정식.
$k = 10$,   $a = 30$
같은 반비례 그래프 위 두 점 → $xy$ 값이 같음.
점 $(3, k)$: $xy = 3k$.
점 $(k+5, 2)$: $xy = 2(k+5) = 2k + 10$.
같다: $3k = 2k + 10 \Rightarrow k = 10$. ($k > 0$ ✓)
$a = 3 \times 10 = 30$, 식 $y = \dfrac{30}{x}$.
확인: $(15, 2)$ → $15 \times 2 = 30$ ✓.

H
$y$가 $x$에 반비례하고 $x = 2$일 때 $y$의 값과, $x = 8$일 때 $y$의 값을 합하면 5이다. 비례상수 $a$를 구하시오.
$y = a/x$ 사용해서 두 $y$값을 $a$로 표현.
$a = 8$
$x = 2$ 일 때 $y = a/2$.
$x = 8$ 일 때 $y = a/8$.
합 = 5: $\dfrac{a}{2} + \dfrac{a}{8} = 5$
공통분모 8: $\dfrac{4a}{8} + \dfrac{a}{8} = \dfrac{5a}{8} = 5 \Rightarrow a = 8$.
확인: $y = 8/2 = 4$, $y = 8/8 = 1$, 합 5 ✓.
