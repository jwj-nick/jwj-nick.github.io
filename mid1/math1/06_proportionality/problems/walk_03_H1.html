문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 | 정비례: $y=ax$ ($a\neq0$) 형태, $x$가 커지면 $y$도 같은 비율로 커짐  ·  반비례: $y=a/x$ ($a\neq0$) 형태, $x$가 커지면 $y$는 반대로 작아짐  ·  비례상수 $a$: $y=ax$ 또는 $y=a/x$에서 고정된 수 $a$  ·  $xy=a$: 반비례 관계를 곱으로 표현한 형태
유형 3 · H1

$y$가 $x$에 반비례하고 $x = -3$일 때 $y = 4$이다.
식을 구하고, $y = -6$일 때 $x$의 값을 구하시오.
🤔 잠깐, 손으로 풀어보세요 반비례 식 $y = a/x$에서 $a$를 구하는 방법은? $x = -3$, $y = 4$를 이용해 $a$를 구해보세요. 부호에 특히 주의해요.
1단계 · Page 1

비례상수 $a$ 결정

왜 이 단계? — 반비례 $y = a/x$에서 한 점이 주어지면 $xy = a$로 $a$를 바로 구할 수 있어요.
📖 용어 노트 | 반비례: $y=a/x$ · 비례상수 결정: $a = xy$
$xy = a$ 공식 사용

반비례 $y = \dfrac{a}{x}$에서: 양변에 $x$를 곱하면 $xy = a$.

$x = -3$, $y = 4$ 대입:
$a = xy = (-3) \times 4 = -12$
⚠️ ★ 핵심 — 음수 비례상수 대입 $y = a/x$에서 $a = -12$이면 $y = \dfrac{-12}{x}$ 이에요. "a 자리에 -12를 그대로 대입"하는 것! $y = \dfrac{12}{x}$로 쓰면 틀려요. 부호 포함해서 대입하세요.
식 완성
$y = \dfrac{-12}{x}$   (= $-\dfrac{12}{x}$ = $xy = -12$ — 모두 같은 식이에요)

2단계 · Page 2

$y = -6$일 때 $x$ 구하기

왜 이 단계? — 식이 완성됐으니, $y$ 값을 알 때 $x$를 역으로 구할 수 있어요.
📖 용어 노트 | 반비례: $y=a/x$ · $xy=a$: 반비례 곱 표현
방법 1 — 식에 대입
$y = \dfrac{-12}{x}$에 $y = -6$ 대입:
$-6 = \dfrac{-12}{x}$
$x = \dfrac{-12}{-6} = 2$
방법 2 — $xy = a$ 사용 (더 빠른 방법)
$xy = -12$에 $y = -6$ 대입:
$x \times (-6) = -12$
$x = \dfrac{-12}{-6} = 2$

두 방법 모두 같은 답이에요. 빠르게 풀 때는 방법 2가 편해요.

💡 확인해봐요 $x = 2$, $y = -6$ → $xy = 2 \times (-6) = -12$ ✓ (비례상수와 같아요!)

3단계 · Page 3

그래프 위치 확인 (사분면)

왜 이 단계? — $a = -12$인 반비례 그래프가 어느 사분면에 있는지 확인하면 풀이가 더 완전해져요.
📖 용어 노트 | 반비례 그래프: 쌍곡선 · $a>0$: 1·3사분면 · $a<0$: 2·4사분면
$a = -12$이면 어느 사분면?
$a = -12 < 0$  →  제2·4사분면의 쌍곡선

점 $(2, -6)$: $x > 0$, $y < 0$ → 4사분면 ✓
점 $(-3, 4)$: $x < 0$, $y > 0$ → 2사분면 ✓
두 점 모두 2·4사분면에 있어요.

정답 + 풀이 흐름 요약

📖 용어 노트 | 반비례: $y=a/x$ ($a\neq0$)  ·  비례상수 $a$: $a = xy$  ·  $a<0$: 2·4사분면 쌍곡선
식: $y = \dfrac{-12}{x}$
$y = -6$일 때 $x = 2$
반비례 식 세우기 · 유형 3 H1
전체 풀이 흐름 — 3단계
1
$a = xy = (-3) \times 4 = -12$ → 식 $y = \dfrac{-12}{x}$
2
$y = -6$ 대입: $-6 = -12/x$ → $x = 2$
3
$a = -12 < 0$ → 2·4사분면 쌍곡선, 두 점 확인 ✓
⚠️ ★ 이 단원 핵심 함정 $a = -12$이면 $y = \dfrac{-12}{x}$ 이에요. 절대로 $y = \dfrac{12}{x}$로 쓰지 마세요. 분자에 부호를 빠뜨리면 완전히 다른 그래프(1·3사분면)가 되어버려요. "-12를 a 자리에 그대로" 넣는 습관을 들여요.
