정비례와 반비례 — 시험 유형 7가지

시험에 자주 나오는 유형을 정리했어요

각 유형마다 L(쉬운) × 3 + M(중간) × 3 + H(어려운) × 3 = 9문제를 풀어볼 수 있어요.
유형을 클릭해서 들어가보세요. 깊이 탐구도 함께 활용하면 좋아요.
유형별 연습
1. 정비례 식 세우기
L M H
두 양의 관계가 정비례인지 식별 → $y = ax$ 형태로 표현
빈출: 일상 상황을 식으로 (가격, 거리 등) / 표에서 $y/x$ 일정 확인 / 함정: 덧셈 상수항, 제곱 식 구별
→ 연습하기
2. 정비례 그래프
L M H
$y = ax$ 그래프는 원점을 지나는 직선. $a$의 부호·크기로 모양 결정.
빈출: 식 → 그래프 사분면 / $a > 0$ vs $a < 0$ / $|a|$ 비교 (가팔기) / 그래프가 지나는 점
→ 연습하기
3. 반비례 식 세우기
L M H
두 양의 관계가 반비례인지 식별 → $y = \dfrac{a}{x}$ 형태로 표현
빈출: $xy$가 일정한지 확인 / 핵심: $a = -2$이면 $y = \dfrac{-2}{x}$ — 부호 포함 대입!
→ 연습하기
4. 반비례 그래프 (쌍곡선)
L M H
$y = \dfrac{a}{x}$ 그래프는 두 갈래 쌍곡선. $x$축, $y$축이 점근선.
빈출: $a > 0$: 1·3사분면 / $a < 0$: 2·4사분면 / 원점을 지나지 않음 / 점근선
→ 연습하기
5. 비례상수 결정
L M H
한 점이 주어졌을 때 정비례·반비례의 비례상수 $a$ 결정
빈출: 정비례 $a = y/x$ / 반비례 $a = xy$ / 비례상수 결정 후 다른 점의 좌표 구하기
→ 연습하기
6. 그래프에서 식 찾기
L M H
그래프 위의 한 점을 읽어 식을 결정. 식·그래프·표 세 형태 변환.
빈출: 정비례·반비례 그래프에서 점 → 식 / 두 그래프 교점 구하기
→ 연습하기
7. 비례 활용 (실생활 문제)
L M H
일상 현상을 정비례·반비례 식으로 모델링하고 문제 해결
빈출: 톱니바퀴 회전수 / 일·시간·사람 수 / 속도-시간-거리 / 단위 환산
→ 연습하기
