유형 6 (정비례 함정) + 유형 7 (도형 합·차)

← 단원 10 홈
⚠️ ★ 단원 시험 1순위 함정 — 이 유형은 거의 모든 시험에서 객관식 함정으로 등장해요. "호와 현 모두 중심각에 정비례"라는 보기는 거짓. 원리를 깊이 이해하면 절대 실수 안 해요.
이 유형이 어려운 이유

"중심각이 2배가 되면 모든 게 2배일까?"라는 직관이 — 호와 부채꼴 넓이엔 맞고, 현엔 틀려요. 이걸 시각적으로·수치적으로 깊이 알아야 함정에 안 빠져요.

핵심 아이디어 — 정비례 ⭕와 ❌
양중심각과 관계증명 직관
호의 길이정비례 ⭕각도 2배 → 호 곡선 2배
부채꼴 넓이정비례 ⭕각도 2배 → 부채꼴 영역 2배
현의 길이정비례 ❌각도와 현은 사인 함수 관계 (sin)
왜 호는 정비례인데 현은 아닐까? 호는 원의 곡선을 따라가는 길이고, 현은 두 점을 직선으로 잇는 길이예요. 각도가 커지면 두 점의 곡선 거리는 비례하지만, 직선 거리는 그렇지 않아요 (지름에 가까워질수록 증가율이 줄어듦).

★ 결정적 예 — $r = 1$ 원에서 직접 비교

중심각 $\theta$호의 길이현의 길이
$60°$$\dfrac{\pi}{3} \approx 1.05$$1$ (정삼각형)
$120°$ (60°×2)$\dfrac{2\pi}{3} \approx 2.09$ (2배 ✓)$\sqrt{3} \approx 1.73$ ($\ne 2$ ❌)
$180°$ (60°×3)$\pi \approx 3.14$ (3배 ✓)$2$ ($\ne 3$ ❌)
$240°$ (60°×4)$\dfrac{4\pi}{3} \approx 4.19$ (4배 ✓)$\sqrt{3}$ (다시 1.73, ❌)
중심각이 360°를 넘으면 다시 줄어드는 등 현은 매우 비선형. 호와 부채꼴 넓이는 일관되게 정비례.

비례식 활용 (호·부채꼴)

$\dfrac{l_1}{l_2} = \dfrac{\theta_1}{\theta_2} = \dfrac{S_1}{S_2}$

예: 같은 원에서 30°의 호 = 5cm 라면, 90°의 호는?
90°/30° = 3 → 호도 3배 = 15cm. (현은 이렇게 못 풀어요.)

자주 하는 실수
❌
"현의 길이 = 중심각에 정비례" 보기에 ✓
시험에서 가장 자주 나오는 함정. 이 보기는 거짓. 호·부채꼴 넓이는 정비례, 현은 아님.
❌
"같은 원에서"라는 조건 무시
정비례는 같은 원(같은 반지름)에서만 성립. 다른 원이면 안 통함.
❌
"호는 정비례"를 외워서 현에도 같은 식 적용
현의 길이 = $2r\sin(\theta/2)$ (중2 이상 sin 사용). 중1에서는 60°/90°/120° 같은 특수각으로만 처리.
추가 문제
★ 추가 문제 1
한 원에서 다음 중 옳은 것을 모두 고르세요.
① 같은 호에 대한 중심각의 크기는 같다.
② 호의 길이가 같으면 부채꼴의 넓이도 같다.
③ 현의 길이가 같으면 호의 길이도 같다.
④ 현의 길이가 두 배가 되면 호의 길이도 두 배가 된다.
⑤ 부채꼴의 넓이가 두 배가 되면 호의 길이도 두 배가 된다.
① ② ③ ⑤ (④만 거짓)

① 같은 호 → 같은 중심각 (일대일 대응) ✓
② 호 같음 → 중심각 같음 → 부채꼴 넓이도 같음 ✓
③ 현 같음 → 두 점 사이 직선 거리 같음. 같은 원에서 그런 두 점 → 짧은 호와 긴 호 두 가지가 가능하지만 짧은 호 기준으로는 같은 호 길이. ✓
④ 거짓: 현 두 배 ≠ 호 두 배. 핵심 함정.
⑤ 부채꼴 넓이 = $\dfrac{1}{2}rl$ 이므로 같은 $r$에서 넓이 ∝ $l$. ✓
★ 추가 문제 2
반지름 $5$ cm 인 원에서 두 부채꼴 A, B 가 있다. A의 중심각이 $40°$, B의 중심각이 $100°$ 일 때, 두 부채꼴의 넓이의 비 $S_A : S_B$ 와 호의 길이의 비 $l_A : l_B$ 를 가장 간단한 정수비로 표현하세요.
$S_A : S_B = 2 : 5$, $l_A : l_B = 2 : 5$ (둘 다 같음)
같은 원에서 호와 부채꼴 넓이는 모두 중심각에 정비례:
$40 : 100 = 2 : 5$ → 호도 부채꼴 넓이도 같은 비.

💡 두 비가 같다는 건 — 호와 부채꼴 넓이는 결국 같은 척도(중심각)를 따른다는 뜻.
★ 추가 문제 3
한 원에서 호 AB와 호 CD에 대해, $\overset{\frown}{AB}$ : $\overset{\frown}{CD} = 1 : 2$ 이다. 다음 중 옳은 것을 모두 고르세요.
① $\angle AOB : \angle COD = 1 : 2$
② $\overline{AB} : \overline{CD} = 1 : 2$
③ 부채꼴 OAB의 넓이 : 부채꼴 OCD의 넓이 = 1 : 2
④ 부채꼴 OAB와 OCD의 둘레의 비도 1 : 2
① 과 ③ (② ④ 거짓)

① 호 ∝ 중심각 → 비도 같음 ✓
② 현 비례 X. 현은 정비례하지 않으므로 거짓 ❌
③ 부채꼴 넓이 ∝ 중심각 → 비도 같음 ✓
④ 둘레 = 호 + 두 반지름 ($l + 2r$). 같은 $r$에서 호만 비례하지만 둘레 전체는 정비례 X.
예: $r=1$, 호 $1:2$ → 둘레 $1+2 : 2+2 = 3 : 4 \ne 1:2$ ❌
연결 개념
🔗 이 유형 이해 후 무엇이 쉬워질까?
호·현·중심각의 관계가 명확해지면:
• 중2 원주각: 원주각과 중심각 (2배 관계) — 이 유형의 발전형
• 중3 닮음·삼각비: 현의 길이 = $2r\sin(\theta/2)$ 의 정확한 식 등장
• 고1 호도법: $\theta$를 라디안으로 → 호 = $r\theta$ 더 깔끔. 부채꼴 넓이 = $\dfrac{1}{2}r^2\theta$.
🎨 도형 단원의 종합 사고력 — 복잡한 모양을 단순 도형의 합·차로 분해하는 능력. 아르키메데스가 부채꼴을 잘게 쪼개 원의 넓이를 구한 그 정신이 — 색칠한 부분 문제의 본질이에요.
이 유형이 어려운 이유

색칠한 모양이 복잡할수록 — 어떻게 분해할지 보이지 않아요. 또:

  • 같은 모양도 여러 방법으로 분해 가능 (어떤 방법이 빠른가?)
  • 겹치는 부분 처리 (포함-배제 원리)
  • 도형을 이동·회전해서 단순화하는 안목
핵심 아이디어 — 분해 전략 5가지

① 합·차 분해

색칠 = (큰 도형) − (제외할 영역) 또는 = (도형들의 합)

  • 도넛 = 큰 원 − 작은 원
  • 정사각형 안 사분원 → 정사각형 − 사분원
  • 두 반원의 차 → 색칠

② 포함-배제 (잎사귀)

잎(교집합) = (도형 A) + (도형 B) − (합집합)

예: 한 변 $a$ 정사각형의 두 마주보는 꼭짓점 사분원으로 만든 잎
= $2 \times \dfrac{\pi a^2}{4} - a^2 = \dfrac{\pi a^2}{2} - a^2$

③ 활꼴 = 부채꼴 − 삼각형

현과 호로 둘러싸인 영역 = 부채꼴에서 두 반지름 만든 삼각형 빼기.

④ 이동·회전으로 단순화

색칠한 부분을 잘라 다른 자리로 옮겨 더 단순한 도형으로 만들 수 있는 경우가 있어요. 이게 가장 우아한 방법.

⑤ 패턴 인식 — "사분원 4개 = 원 1개"

4개의 사분원이 서로 겹치지 않으면 합 = 원 한 개. 이런 패턴을 즉시 보면 시간 절약.

자주 하는 실수
❌
분해 잘못해서 영역 두 번 빼기
도형을 분해할 때 같은 영역을 두 번 빼면 답이 틀려요. 그림을 정확히 봐야 함.
❌
포함-배제에서 합 - 합집합을 잘못 계산
잎 = $A + B - A \cup B$. 합집합이 정확히 무엇인지 확인.
❌
단위 빠뜨리기
길이/넓이 모두 단위 ($cm, cm^2$) 정확히.
추가 문제
★ 추가 문제 1
한 변이 $4$ cm 인 정사각형 안에 정사각형의 한 대각선을 지름으로 하는 원이 그려져 있다. 정사각형 안에서 원 바깥의 부분의 넓이는?
$(16 - 8\pi)$ cm²? 음수가 나옴 — 잠깐. 다시 검토.

대각선 = $4\sqrt{2}$ → 원 반지름 = $2\sqrt{2}$
원 넓이 = $\pi(2\sqrt{2})^2 = 8\pi \approx 25.13$
정사각형 넓이 = $16$
하지만 — 대각선이 지름인 원이라면 원이 정사각형보다 큼! 따라서 "정사각형 안에서 원 바깥" 영역이 비어 있을 가능성.

📝 문제 정정: 이 케이스는 원이 정사각형을 덮어버려서 "정사각형 안에서 원 바깥"이 0. 따라서 의도가 다름. 표준 출제는 "한 변이 지름인 원" (반지름 = 한 변/2 = 2). 이 경우:
원 넓이 = $\pi(2)^2 = 4\pi$ — 단, 원이 정사각형 안에 다 들어오는지? 한 변=4, 원 지름=4 (= 한 변) → 원이 정사각형에 정확히 내접. 이 경우 답 = $16 - 4\pi$ cm².

💡 출제 시 주의: 원이 도형에 들어오는지 확인.
★ 추가 문제 2
한 변이 $a$ 인 정사각형의 두 마주보는 꼭짓점을 중심으로 반지름 $a$ 인 사분원 두 개를 그렸을 때, 두 사분원이 만드는 잎사귀 모양의 넓이를 $a$로 표현하세요. (★ 도형 단원의 가장 유명한 패턴 중 하나)
$\left(\dfrac{\pi}{2} - 1\right)a^2$
포함-배제로 풀기:
사분원 한 개 = $\dfrac{\pi a^2}{4}$
두 사분원 합 = $\dfrac{\pi a^2}{2}$
합집합 = 정사각형 (정확히 덮음, 사분원이 한 변 = 반지름이므로 두 사분원이 정사각형 전체를 덮음) = $a^2$
잎 = 합 - 합집합 = $\dfrac{\pi a^2}{2} - a^2 = \left(\dfrac{\pi}{2} - 1\right)a^2$

💡 또는: 잎 = 활꼴 두 개. 활꼴 한 개 = $\dfrac{\pi a^2}{4} - \dfrac{a^2}{2}$. 두 개 = $\dfrac{\pi a^2}{2} - a^2$ ✓
★ 추가 문제 3
반지름 $r$ 인 큰 원 안에, 큰 원의 지름을 지름으로 하는 두 개의 반원이 그려져 있다 (한 반원은 위, 다른 반원은 아래, 각 지름은 큰 원의 두 직각 반지름). 큰 원 영역에서 두 반원이 차지하지 않는 부분의 넓이는? (검산 필요)
📝 이 문제는 그림 의존도가 큰 문제예요. 단순화 버전으로:

(단순화) 반지름 $r$ 인 큰 원 안에, 반지름이 $r$ 의 절반인 작은 반원 두 개가 큰 원의 지름 위에 양쪽으로 그려져 있을 때:
큰 원 = $\pi r^2$
작은 반원 한 개 (반지름 $r/2$) = $\dfrac{1}{2}\pi(r/2)^2 = \dfrac{\pi r^2}{8}$
두 반원 합 = $\dfrac{\pi r^2}{4}$
나머지 = $\pi r^2 - \dfrac{\pi r^2}{4} = \dfrac{3\pi r^2}{4}$

💡 이런 문제는 그림이 명확해야 풀이가 정해져요. 시험에서는 항상 그림이 있으니 그림 보고 분해.

• 고2 적분: 적분 = "잘게 쪼개고 다시 합치기"의 일반화 — 아르키메데스의 부채꼴 분해 발상이 그대로 살아남.
