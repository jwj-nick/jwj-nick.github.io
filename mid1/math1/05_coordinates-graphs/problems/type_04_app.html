유형 4 — 대칭점·평행이동

$x$축·$y$축·원점 대칭, 좌표평면 평행이동
← 단원 5 홈

점 $\mathrm{A}(3, 4)$를 $x$축에 대하여 대칭이동한 점의 좌표는?
$x$축 대칭 → $y$좌표 부호만 반전.
$(3, -4)$
$x$축은 위·아래의 거울. $y$좌표 부호만 바뀌어요.
$(3, 4) \to (3, -4)$

점 $\mathrm{B}(-2, 5)$를 $y$축에 대하여 대칭이동한 점의 좌표는?
$y$축 대칭 → $x$좌표 부호만 반전.
$(2, 5)$
$y$축은 좌·우의 거울. $x$좌표 부호만 바뀌어요.
$(-2, 5) \to (2, 5)$

점 $\mathrm{C}(4, -1)$를 원점에 대하여 대칭이동한 점의 좌표는?
원점 대칭 → $x$, $y$ 좌표 모두 부호 반전.
$(-4, 1)$
원점 대칭 = $x$축 대칭 + $y$축 대칭 동시에
$(4, -1) \to (-4, 1)$

M
점 $\mathrm{P}(a, b)$를 $x$축에 대해 대칭한 점이 $(3, -2)$일 때, $a$와 $b$의 값은?
대칭 후 점이 $(3, -2)$ → 대칭 전 점은 $(?, ?)$. $x$축 대칭이므로 $y$좌표 부호 반전.
$a = 3, b = 2$
$\mathrm{P}(a, b)$ → $x$축 대칭 → $(a, -b) = (3, -2)$
$a = 3$, $-b = -2 \Rightarrow b = 2$

M
점 $\mathrm{A}(2, -3)$을 오른쪽으로 4, 위로 5만큼 평행이동한 점의 좌표는?
평행이동: 오른쪽 = $x$ +, 위 = $y$ +.
$(6, 2)$
$x$좌표: $2 + 4 = 6$
$y$좌표: $-3 + 5 = 2$
$\therefore (6, 2)$

M
점 $\mathrm{P}(4, -2)$를 $x$축, $y$축, 원점에 대해 대칭이동한 세 점을 각각 구하세요.
$x$축: $y$만 반전, $y$축: $x$만 반전, 원점: 둘 다 반전.
$x$축 대칭: $(4, 2)$   $y$축 대칭: $(-4, -2)$   원점 대칭: $(-4, 2)$
표 정리:
• $x$축: $(4, -2) \to (4, +2)$
• $y$축: $(4, -2) \to (-4, -2)$
• 원점: $(4, -2) \to (-4, +2)$
세 대칭은 자주 헷갈려요. "거울 어디에 두는가"로 떠올리면 좋아요. $x$축 대칭 = $x$축이 거울 = 위아래 뒤집힘.

점 $\mathrm{A}$를 평행이동했더니 $(3, 5)$가 되었어요. 이동 방향이 "왼쪽으로 2, 아래로 1"이었다면, 원래 점 $\mathrm{A}$의 좌표는?
"왼쪽 2 아래 1" 이동 후가 $(3, 5)$. 거꾸로 "오른쪽 2 위로 1"이 원래.
$\mathrm{A}(5, 6)$
이동: $\mathrm{A}(x, y) \to (x-2, y-1) = (3, 5)$
$x - 2 = 3 \Rightarrow x = 5$
$y - 1 = 5 \Rightarrow y = 6$
$\therefore \mathrm{A}(5, 6)$

H
점 $\mathrm{P}(a, 2a)$를 $x$축에 대하여 대칭이동한 점이 $(3, b)$일 때, $a + b$의 값을 구하세요.
$x$축 대칭: $(a, 2a) \to (a, -2a) = (3, b)$.
$a + b = -3$
$a = 3$
$-2a = b \Rightarrow b = -2(3) = -6$
$a + b = 3 + (-6) = -3$

H
점 $\mathrm{A}(-2, 3)$를 $y$축에 대해 대칭이동한 점을 $\mathrm{B}$, 점 $\mathrm{B}$를 다시 원점에 대해 대칭이동한 점을 $\mathrm{C}$라 할 때, 점 $\mathrm{C}$의 좌표는?
단계별로: A → B(y축 대칭) → C(원점 대칭).
$\mathrm{C}(-2, -3)$
A $(-2, 3)$
B = A의 $y$축 대칭 = $(2, 3)$
C = B의 원점 대칭 = $(-2, -3)$
(결과: A → C는 $x$축 대칭과 같음)
사실 $y$축 대칭 + 원점 대칭 = $x$축 대칭이에요! 직관적으로 두 거울을 합친 효과.
