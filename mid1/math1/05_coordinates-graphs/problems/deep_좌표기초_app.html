깊이 탐구 — 좌표 기초

순서쌍 + 좌표 읽기 + 사분면 판별 (3유형 통합)

← 단원 5 홈
이 유형이 어려운 이유

"순서쌍"이라는 이름이 낯설어요. 그냥 "두 개의 수가 묶여 있는 것"이라고 생각하면 돼요.

핵심 함정 두 가지:
① $(2, 3)$과 $(3, 2)$는 다른 점인데 같다고 헷갈림
② $x$좌표·$y$좌표 순서를 거꾸로 씀

핵심 아이디어

점은 "두 개의 거리"로 표현

좌표평면에서 한 점의 위치를 정확히 말하려면 두 가지 정보가 필요해요:

  • 가로 방향($x$축)으로 얼마나? → $x$좌표
  • 세로 방향($y$축)으로 얼마나? → $y$좌표
$\mathrm{P}(a, b)$의 의미:
원점에서 가로로 $a$, 세로로 $b$ 떨어진 점.
$a$가 양수면 오른쪽, 음수면 왼쪽.
$b$가 양수면 위, 음수면 아래.

같은 순서쌍 = 같은 자리 값이 같음

$(a, b) = (c, d)$ 일 때:
$\Rightarrow a = c$ 그리고 $b = d$

두 식을 따로 풀면 미지수 두 개를 구할 수 있어요.

자주 하는 실수
😵
$x$좌표·$y$좌표 순서 바꿔 쓰기
"오른쪽 3, 위로 5인 점" → $(5, 3)$ ❌ → $(3, 5)$ ✓
$x$좌표(가로)가 먼저, $y$좌표(세로)가 나중.
🤔
"왼쪽으로 2"를 $x = 2$로 잘못 쓰기
왼쪽 = 음수 방향. "왼쪽 2" → $x = -2$.
방향과 부호를 매칭하는 연습이 필요해요.
추가 문제 (3문제)
PROBLEM 1
두 순서쌍 $(2x - 1, 5)$와 $(3, y + 2)$가 같을 때, $x$와 $y$를 구하세요.
$2x - 1 = 3 \Rightarrow x = 2$
$5 = y + 2 \Rightarrow y = 3$
PROBLEM 2
점 $\mathrm{A}(a, 4)$와 점 $\mathrm{B}(-3, b)$가 같은 점일 때 $a + b$의 값은?
$a = -3, b = 4$
$a + b = -3 + 4 = 1$
PROBLEM 3
두 순서쌍 $(a + 2, 3b - 1)$과 $(5, 8)$이 같을 때 $a, b$를 구하세요.
$a + 2 = 5 \Rightarrow a = 3$
$3b - 1 = 8 \Rightarrow 3b = 9 \Rightarrow b = 3$
이 유형 이해하면…
두 순서쌍이 같다는 조건은 사실 두 개의 일차방정식이에요. 단원 4(일차방정식)를 좌표로 응용하는 출발점!
유형 2·3 (사분면 판별)도 이 좌표 표기 위에서 시작돼요.
이 유형이 어려운 이유

사분면이 4개나 되고, 번호 붙이는 순서가 익숙하지 않아요.
"왜 시계 방향이 아니라 반시계 방향?"이라고 의문이 들 수 있죠.

핵심 아이디어 — 한 장의 그림

사분면 4개 부호표

제2사분면
(−, +)
제1사분면
(+, +)
제3사분면
(−, −)
제4사분면
(+, −)

위 = $y$ 양수, 아래 = $y$ 음수
오른쪽 = $x$ 양수, 왼쪽 = $x$ 음수

외우는 팁: 1사분면(오른쪽 위)부터 반시계 방향으로 1→2→3→4.
$x$좌표 부호: 1·4는 +, 2·3은 −
$y$좌표 부호: 1·2는 +, 3·4는 −

축 위의 점 = 사분면 X

$x = 0$ 또는 $y = 0$인 점은 축 위에 있어 어느 사분면에도 속하지 않아요.

  • $(3, 0)$ → $x$축 위
  • $(0, -2)$ → $y$축 위
  • $(0, 0)$ → 원점 (둘 다 0)
자주 하는 실수
⚠️
축 위의 점을 사분면에 분류
$(3, 0)$ → "1사분면" ❌ → $x$축 위(어느 사분면에도 X)
$x$ 또는 $y$ 중 하나라도 0이면 축 위.
😵
2사분면과 3사분면 헷갈림
2사분면(왼쪽 위): $(-, +)$
3사분면(왼쪽 아래): $(-, -)$
$y$ 부호로 위/아래 구분!
추가 문제 (3문제)
PROBLEM 1
5개 점 $(2,3), (-1,4), (-2,-5), (0,3), (4,-1)$ 중 1사분면 점은 몇 개?
$(2,3)$ → 1 ✓
$(-1,4)$ → 2
$(-2,-5)$ → 3
$(0,3)$ → $y$축 (사분면 X)
$(4,-1)$ → 4
1사분면은 $(2,3)$ 1개만.
PROBLEM 2
점 $(0, -7)$은 어느 위치에 있나요?
$x = 0$이므로 $y$축 위.
$y = -7 < 0$이므로 $y$축의 음의 부분 (원점 아래쪽).
PROBLEM 3
점 $(-3, -8)$이 속하는 사분면을 구하고, 그 점에서 $x$축까지의 거리, $y$축까지의 거리를 각각 구하세요.
$(-, -)$ → 제3사분면
$x$축까지 거리 = $|-8| = 8$ (수직 거리)
$y$축까지 거리 = $|-3| = 3$ (수평 거리)
한 점에서 축까지 거리는 다른 좌표의 절댓값!
이 유형 이해하면…
유형 3(문자 부호 사분면)이 자연스럽게 풀려요. $a, b$의 부호 → $-a, ab$ 등의 부호 → 사분면 매칭.
그래프 단원에서 "1사분면만 그려진 그래프" 같은 표현도 잡힙니다.
이 유형이 어려운 이유

구체적인 수가 아니라 문자 $a, b$의 부호 조건으로 사분면을 판별해요. 두 단계 추론이 필요해서 헷갈려요.

예: "$a > 0, b < 0$일 때 점 $(-a, ab)$는?"
→ $-a$의 부호? $ab$의 부호? → 사분면.

핵심 아이디어

2단계 풀이법

  1. 1단계: 주어진 $a, b$의 부호를 표시 (예: $a > 0$, $b < 0$)
  2. 2단계: 변형된 좌표값 각각의 부호 결정 (예: $-a < 0$, $ab < 0$)
  3. 3단계: 부호 → 사분면 매칭

부호 변환 공식

  • $a > 0$ → $-a < 0$ (양수에 마이너스 → 음수)
  • $a < 0$ → $-a > 0$ (음수에 마이너스 → 양수)
  • $a > 0, b > 0$ → $ab > 0$ (양 × 양 = 양)
  • $a > 0, b < 0$ → $ab < 0$ (양 × 음 = 음)
  • $a < 0, b < 0$ → $ab > 0$ (음 × 음 = 양)
정수의 곱셈 규칙(단원 2)을 다시 한 번 떠올려요. 음 × 음 = 양이 가장 자주 헷갈려요.
자주 하는 실수
😵
$-a$가 항상 음수라고 착각
$-a$의 부호는 $a$의 부호 반대일 뿐.
$a < 0$이면 $-a > 0$. 마이너스 기호가 붙었다고 무조건 음수가 아님!
⚠️
$ab$의 부호 헷갈림
곱셈 부호 결정: 두 수 모두 같은 부호 → 양수, 다른 부호 → 음수.
$(+)(+) = +$, $(-)(-)= +$, $(+)(-) = -$.
추가 문제 (3문제)
PROBLEM 1
$a < 0, b > 0$일 때 점 $(a, -b)$는 어느 사분면?
$a < 0$ ($x$좌표 음수)
$b > 0 \Rightarrow -b < 0$ ($y$좌표 음수)
$(-, -)$ → 3사분면
PROBLEM 2
점 $\mathrm{P}(a, b)$가 1사분면에 있을 때 점 $\mathrm{Q}(-a, -b)$, 점 $\mathrm{R}(a, -b)$는 각각 어느 사분면?
P가 1사분면 → $a > 0, b > 0$
Q $(-a, -b) = (음, 음)$ → 3사분면 (P를 원점 대칭)
R $(a, -b) = (양, 음)$ → 4사분면 (P를 $x$축 대칭)
PROBLEM 3
$a, b$의 부호가 $a > 0, b < 0$일 때 다음 점이 어느 사분면인지 모두 답하세요. ① $(a, b)$ ② $(b, a)$ ③ $(ab, a-b)$
① $(a, b) = (+, -)$ → 4사분면
② $(b, a) = (-, +)$ → 2사분면 (자리 바뀌면 사분면도 바뀜)
③ $ab = + \times - = -$
$a - b = + - (-) = + + + = +$ (예: $a=2, b=-3 \Rightarrow a-b = 5$)
$(ab, a-b) = (-, +)$ → 2사분면
이 유형 이해하면…
대칭이동(유형 4)이 쉬워져요. $x$축 대칭 = $y$ 부호만 바꿈, $y$축 대칭 = $x$ 부호만 바꿈, 원점 대칭 = 둘 다 바꿈. 모두 사분면 변화로 시각화 가능!
