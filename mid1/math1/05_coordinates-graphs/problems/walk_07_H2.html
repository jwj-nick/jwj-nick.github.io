유형 7 · H2

두 그래프 교점의 실생활 의미

두 사람 A, B가 같은 출발점에서 걷기 시작한 시간-거리 그래프가 있어요. 두 그래프가 교점을 가질 때, 그 교점이 실생활에서 무엇을 의미하는지 설명하세요.

생각해보기

교점에서 $x$값과 $y$값이 두 그래프에서 같다는 것은 무엇을 뜻할까요?

📖 용어 노트 순서쌍 (a, b): x좌표 a, y좌표 b인 점 · 제1사분면(Ⅰ): x>0, y>0 · 제2사분면(Ⅱ): x<0, y>0 · 제3사분면(Ⅲ): x<0, y<0 · 제4사분면(Ⅳ): x>0, y<0 · 축 위의 점은 사분면에 속하지 않아요
단계 1 / 2

교점에서 두 값이 같다

교점은 두 그래프가 만나는 점 → 두 식이 동시에 성립하는 $x$, $y$ 값이에요.

교점의 좌표를 $(t, d)$라 하면:

A의 그래프: $(t, d)$ 위에 있다 → A가 시간 $t$에 거리 $d$에 있다

B의 그래프: $(t, d)$ 위에 있다 → B가 시간 $t$에 거리 $d$에 있다

같은 시간 $t$에 같은 거리 $d$ → A와 B가 같은 시각에 같은 위치에 있다
📖 용어 노트 순서쌍 (a, b): x좌표 a, y좌표 b인 점 · 제1사분면(Ⅰ): x>0, y>0 · 제2사분면(Ⅱ): x<0, y>0 · 제3사분면(Ⅲ): x<0, y<0 · 제4사분면(Ⅳ): x>0, y<0 · 축 위의 점은 사분면에 속하지 않아요
단계 2 / 2

실생활 언어로 표현하기

수학적 결론을 상황에 맞는 말로 바꾸는 것이 실생활 그래프 문제의 핵심이에요.

교점의 $x$좌표 = 두 사람이 만나는 시각

교점의 $y$좌표 = 만나는 지점까지의 거리

→ "A와 B는 출발 후 $t$분에 거리 $d$ km 지점에서 만난다"

교점 = 두 사람이 같은 시각, 같은 장소에 있는 순간
⚡ 교점이 없다면? 교점이 없으면 두 사람은 절대 만나지 않아요. 교점이 여러 개면 여러 번 만납니다.

★ 정답

두 사람이 같은 시각, 같은 장소에서 만난다

교점 = 두 사람이 만나는 시각과 장소

풀이 흐름 요약

1
교점 좌표 $(t, d)$ → 두 그래프 공통 점
2
$x=t$: 같은 시각 / $y=d$: 같은 거리(장소)
3
"두 사람이 시각 $t$에 거리 $d$ 지점에서 만남"
⚡ 그래프 해석의 핵심 좌표의 의미는 항상 축 이름에서 나와요. $x$축 = 시간이면 교점 $x$ = 만나는 시각, $y$축 = 거리이면 교점 $y$ = 만나는 위치!
