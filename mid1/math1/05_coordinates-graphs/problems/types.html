빈출: 두 순서쌍이 같은 조건에서 미지수 구하기 / 함정: 순서쌍은 순서가 중요 — $(a,b) \neq (b,a)$

2. 사분면 판별 (수치 좌표)
L M H

빈출: 여러 점 중 특정 사분면 선별 / 함정: 축 위의 점은 사분면에 속하지 않아요
→ 연습하기
3. 사분면 판별 (문자 부호 조건)
L M H
$a$, $b$의 부호 조건 → 변형 좌표 $(-a, b)$, $(ab, a/b)$ 등의 사분면 판단
빈출: $-a$의 부호 역전, $ab$·$a/b$ 부호 계산 / 함정: 음×음=양, 부호 두 번 뒤집기
→ 연습하기
4. 대칭점 · 평행이동
L M H
$x$축 대칭(y 부호 반전), $y$축 대칭(x 부호 반전), 원점 대칭(둘 다 반전), 평행이동
빈출: 대칭+평행이동 결합, 역방향(이동 후 좌표 → 원래 좌표) / 함정: 이동 방향 부호
→ 연습하기
5. 좌표로 도형 만들기
L M H

빈출: 삼각형 넓이, 평행사변형 4번째 점 (3가지 가능) / 함정: 격자 가로·세로 길이 혼동
→ 연습하기
6. 그래프에서 값 읽기
L M H
그래프 위 점에서 $x$값 ↔ $y$값 읽기, 표 ↔ 식 ↔ 그래프 변환
빈출: 두 점으로 직선 식 구하기, 표에서 $a$·$b$ 찾기 / 함정: 기울기 분자·분모 순서
→ 연습하기
7. 그래프 해석 (상황 그래프)
L M H
시간·거리·속도 실생활 그래프 해석, 기울기 = 속도, 교점 = 만나는 시각과 장소
빈출: 기울기 비교(속도 비교), 두 그래프 교점 의미, $y=ax$에 값 대입 / 함정: 그래프가 가파를수록 빠름
→ 연습하기
