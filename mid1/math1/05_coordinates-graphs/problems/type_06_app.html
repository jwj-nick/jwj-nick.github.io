식 $y = x + 3$일 때, $x = 2$를 대입하면 $y$의 값은?
$x$ 자리에 2를 넣어 계산해봐요.
$y = 5$
$y = x + 3$
$y = 2 + 3 = 5$
점으로 쓰면 $(2, 5)$가 그래프 위에 있어요.

아래 표는 어떤 식 $y = $ ? 를 나타내는 표예요. 빈칸의 식을 구하세요.
$x$01234
$y$036912
$x$가 1 늘 때마다 $y$가 얼마나 늘어요?
$y = 3x$
$x$가 1 증가할 때마다 $y$는 3 증가.
$x = 0$일 때 $y = 0$ → 원점 통과.
$\therefore y = 3x$
확인: $x=4 \Rightarrow y=3 \times 4 = 12$ ✓

L
식 $y = -2x + 1$일 때, $x = 3$을 대입하면 $y$값은?
$y = -2 \times 3 + 1$ 계산.
$y = -5$
$y = -2x + 1$
$y = -2(3) + 1 = -6 + 1 = -5$
점 $(3, -5)$가 그래프 위.

식 $y = 2x - 3$의 그래프 위에 있는 점은?
① $(0, 3)$   ② $(1, -1)$   ③ $(2, 0)$   ④ $(3, 4)$
각 점의 $x$값을 식에 대입해서 $y$값과 일치하는지 확인.
② $(1, -1)$
① $x=0 \Rightarrow y=-3$, 보기 3 → 거짓
② $x=1 \Rightarrow y=2-3=-1$ ✓
③ $x=2 \Rightarrow y=4-3=1$, 보기 0 → 거짓
④ $x=3 \Rightarrow y=6-3=3$, 보기 4 → 거짓

M
식 $y = x + 1$에서 $y$의 값이 $5$가 되는 $x$의 값은?
$y$ 자리에 5를 넣고 $x$를 구해요. (방정식 풀기와 같음)
$x = 4$
$5 = x + 1$
$x = 5 - 1 = 4$
$\therefore$ 점 $(4, 5)$가 그래프 위에 있어요.

식 $y = -x + 4$의 그래프에서 $x$절편 (그래프가 $x$축과 만나는 점의 $x$좌표)을 구하세요.
$x$축과 만나는 점은 $y = 0$인 점.
$x = 4$
$y = 0$ 대입: $0 = -x + 4$
$x = 4$
그래프는 점 $(4, 0)$에서 $x$축과 만나요.

식 $y = 3x + a$의 그래프가 점 $(2, 7)$을 지날 때, 상수 $a$의 값을 구하세요.
"그래프가 점 (2, 7)을 지난다" → $x=2, y=7$을 식에 대입해도 등식이 성립.
$a = 1$
$x = 2, y = 7$ 대입:
$7 = 3(2) + a$
$7 = 6 + a$
$a = 1$
$\therefore$ 식은 $y = 3x + 1$

아래 표는 어떤 식 $y = ax + b$ 형태의 관계예요. $a$와 $b$를 구하세요.
$x$0123
$y$420−2
$x$ 1 증가할 때 $y$ 변화 = $a$. $x = 0$일 때 $y$값 = $b$.
$a = -2, b = 4$
$x$가 1 증가할 때마다 $y$는 $-2$ → $a = -2$
$x = 0$일 때 $y = 4$ → $b = 4$
식: $y = -2x + 4$
확인: $x=3 \Rightarrow y = -6 + 4 = -2$ ✓
$y$ 변화량을 $x$ 변화량으로 나눈 값이 기울기 $a$. 음수일 수도 있음에 주의.

두 점 $(1, 5)$, $(3, 11)$을 모두 지나는 직선 $y = ax + b$의 식을 구하세요.
두 점을 식에 대입 → 두 식 → $a, b$.
$y = 3x + 2$
점 $(1, 5)$: $5 = a + b$ ……①
점 $(3, 11)$: $11 = 3a + b$ ……②
②$-$①: $6 = 2a \Rightarrow a = 3$
①에 대입: $5 = 3 + b \Rightarrow b = 2$
$\therefore y = 3x + 2$
검산: $(1, 5)$: $3(1)+2=5$ ✓   $(3, 11)$: $3(3)+2=11$ ✓
