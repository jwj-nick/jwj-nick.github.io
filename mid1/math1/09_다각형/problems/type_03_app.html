유형 3 — 삼각형 내각·외각

내각합 180° + 외각의 정리 (한 외각 = 이웃 안 하는 두 내각의 합)
← 단원 9 홈
1
L
삼각형의 두 각이 $35°$, $85°$일 때, 나머지 한 각의 크기를 구하시오.
세 내각의 합 = $180°$.
$60°$
$180° - 35° - 85° = 60°$
2
L
삼각형 ABC에서 $\angle A = 50°$, $\angle B = 70°$일 때, 꼭짓점 C에서의 외각의 크기를 구하시오.
외각 = 이웃하지 않는 두 내각의 합.
$120°$
외각의 정리: $\angle\text{외}_C = \angle A + \angle B = 50° + 70° = 120°$.
3
L
이등변삼각형의 두 밑각이 같고, 한 밑각이 $55°$이다. 꼭지각의 크기를 구하시오.
이등변삼각형의 두 밑각은 같음. 합은 $180°$.
$70°$
두 밑각 = $55° + 55° = 110°$. 꼭지각 = $180° - 110° = 70°$.
1
M
삼각형의 세 내각의 비가 $2 : 3 : 4$일 때, 가장 큰 각의 크기를 구하시오.
세 각을 $2x, 3x, 4x$로 두면 합이 $180°$.
$80°$
$2x + 3x + 4x = 180°$
$9x = 180°$ → $x = 20°$.
가장 큰 각 = $4x = 80°$.
2
M
삼각형의 한 외각이 $130°$일 때, 그 외각과 이웃한 내각의 크기를 구하시오.
한 꼭짓점에서 (내각) + (외각) = $180°$.
$50°$
(내각) + $130° = 180°$ → 내각 = $50°$.
3
M
삼각형 ABC에서 $\angle A = 30°$이고, $\angle B$의 외각이 $110°$이다. $\angle C$의 크기를 구하시오.
$\angle B$ 외각 = $\angle A + \angle C$ (외각의 정리).
$80°$
외각의 정리: $\angle B$의 외각 $= \angle A + \angle C$.
$110° = 30° + \angle C$
$\angle C = 80°$.
1
H
삼각형 ABC에서 $\angle A : \angle B : \angle C = 1 : 2 : 3$이고, $\angle C$의 외각의 크기를 구하시오.
먼저 세 내각을 비례로 구하고, $\angle C$의 외각 = $\angle A + \angle B$.
$90°$
$x + 2x + 3x = 180°$ → $x = 30°$.
$\angle A = 30°, \angle B = 60°, \angle C = 90°$.
$\angle C$의 외각 $= \angle A + \angle B = 30° + 60° = 90°$.
2
H
삼각형의 한 내각이 다른 두 내각의 합과 같을 때, 그 한 내각의 크기를 구하시오.
그 한 내각을 $x$, 다른 두 합을 $x$라 하면 전체 = $2x = 180°$.
$90°$ (직각)
그 내각 = $x$. 다른 두 내각의 합 = $x$ (조건).
전체 합 = $x + x = 2x = 180°$ → $x = 90°$.
이는 직각삼각형의 직각이에요.
3
H
삼각형 ABC에서 $\angle B$와 $\angle C$의 이등분선이 만나는 점을 I라 할 때, $\angle A = 70°$일 때 $\angle BIC$의 크기를 구하시오.
$\angle IBC + \angle ICB = \dfrac{\angle B + \angle C}{2}$. 그리고 삼각형 BIC에서 합 180°.
$\angle BIC = 125°$
$\angle B + \angle C = 180° - 70° = 110°$.
이등분선이므로 $\angle IBC + \angle ICB = \dfrac{110°}{2} = 55°$.
삼각형 BIC에서 $\angle BIC = 180° - 55° = 125°$.

(공식: $\angle BIC = 90° + \dfrac{\angle A}{2} = 90° + 35° = 125°$ ✓)
