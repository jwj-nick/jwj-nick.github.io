문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 $n$각형 내각의 합: $(n-2)\times180°$ · $n$각형 외각의 합: 항상 $360°$ ($n$에 무관) · 정$n$각형 한 내각: $(n-2)\times180°\div n$ · 대각선 수: $n(n-3)\div2$ · 대각선: 이웃하지 않는 두 꼭짓점을 잇는 선분
유형 3 · H3

삼각형 ABC에서 $\angle B$와 $\angle C$의 이등분선이 만나는 점을 I라 할 때,
$\angle A = 70°$이면 $\angle BIC$의 크기를 구하시오.
🤔 잠깐, 손으로 풀어보세요 삼각형 BIC에서 $\angle IBC + \angle ICB$를 먼저 구해보세요. 이등분선이니까 각의 절반이에요.
1단계 · Page 1

$\angle B + \angle C$ 먼저 구하기

왜 이 단계? — 이등분선으로 생긴 각 $\angle IBC$, $\angle ICB$를 알려면 먼저 원래 각의 합이 필요해요.
📖 용어 노트 삼각형 내각의 합: $180°$ · 이등분선: 각을 정확히 반으로 나누는 선
삼각형 ABC에서
$\angle A + \angle B + \angle C = 180°$
$70° + \angle B + \angle C = 180°$
$\angle B + \angle C = 110°$
← 이전
2단계 · Page 2

이등분선으로 생긴 각 구하기

왜 이 단계? — 이등분선은 각을 반으로 나눠요. 삼각형 BIC의 두 각을 알 수 있어요.
📖 용어 노트 이등분선: 각을 정확히 반으로 나누는 선
이등분이므로
$\angle IBC = \dfrac{\angle B}{2}$, $\quad \angle ICB = \dfrac{\angle C}{2}$

따라서: $\angle IBC + \angle ICB = \dfrac{\angle B + \angle C}{2} = \dfrac{110°}{2} = 55°$
← 이전
3단계 · Page 3

삼각형 BIC에서 $\angle BIC$ 구하기

왜 이 단계? — 삼각형 BIC도 삼각형이므로 내각의 합 = 180°예요.
📖 용어 노트 삼각형 내각의 합: $180°$
삼각형 BIC에서
$\angle BIC + \angle IBC + \angle ICB = 180°$
$\angle BIC + 55° = 180°$
$\angle BIC = 125°$
💡 공식으로도 확인! 이 유형의 공식: $\angle BIC = 90° + \dfrac{\angle A}{2} = 90° + 35° = 125°$ ✓
이 공식은 "내접원의 중심각" 패턴이에요. 기억해두면 빠르게 풀 수 있어요.

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 $n$각형 내각의 합: $(n-2)\times180°$ · $n$각형 외각의 합: 항상 $360°$ · 대각선 수: $n(n-3)\div2$ · 대각선: 이웃하지 않는 두 꼭짓점을 잇는 선분
$\angle BIC = 125°$
(공식: $90° + \dfrac{\angle A}{2}$)
전체 풀이 흐름
1
$\angle B + \angle C = 180° - 70° = 110°$
2
이등분선: $\angle IBC + \angle ICB = \dfrac{110°}{2} = 55°$
3
삼각형 BIC: $\angle BIC = 180° - 55° = 125°$
⚠️ 이 유형의 핵심 이등분선이 나오면 새로운 삼각형(BIC)을 만들어 그 삼각형의 내각의 합을 적용해요. 이 패턴이 이등분선 문제의 핵심 도구예요.
