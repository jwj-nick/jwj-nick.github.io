정사각형의 한 내각의 크기와 정삼각형의 한 내각의 크기의 합을 구하시오.
정사각형 90°, 정삼각형 60°.
$150°$
$90° + 60° = 150°$.
2
L
사각형 ABCD에서 $\angle A = 100°$, $\angle B = 80°$, $\angle C = 110°$일 때 $\angle D$를 구하시오.
사각형 내각의 합 = $360°$.
$70°$
$100 + 80 + 110 + \angle D = 360$
$290 + \angle D = 360$ → $\angle D = 70°$.

L
한 다각형의 외각이 모두 $40°$로 같다. 이 다각형의 변의 수를 구하시오.
외각이 모두 같으면 정$n$각형. 외각의 합 = 360°.
9개
외각의 합 = 360°이고, 모든 외각이 40°라면: 변의 수 = $\dfrac{360°}{40°} = 9$.

아래 별 모양(오각성)에서 다섯 끝점의 각의 합을 구하시오.
별의 다섯 끝각의 합은 별을 이루는 삼각형 5개와 안쪽 오각형의 관계로 구해요. 외각 정리 활용.
$180°$
오각성의 다섯 끝각 합은 항상 $180°$로 고정. (별 모양 = 오각형 + 5개 삼각형의 끝각, 외각 정리로 도출 가능.)
이는 정오각성, 일반 오각성 모두 동일.
2
M
한 다각형의 모든 내각이 같고 그 크기가 $150°$일 때, 이 다각형의 대각선의 수를 구하시오.
정$n$각형 한 내각 150° → $n$? 그 후 대각선 수.
54개
한 외각 = $180° - 150° = 30°$.
$\dfrac{360°}{n} = 30°$ → $n = 12$ (정십이각형).
대각선 = $\dfrac{12 \times 9}{2} = 54$.

M
한 정다각형의 한 내각이 한 외각보다 $108°$ 클 때, 이 정다각형의 한 내각의 크기와 변의 수를 구하시오.
내각 + 외각 = 180°이고 내각 - 외각 = 108°. 두 식 더하기.
한 내각 $144°$, 변 10개 (정십각형)
내각 + 외각 = $180°$, 내각 - 외각 = $108°$.
두 식 더하면: 2 × 내각 = $288°$ → 내각 = $144°$. 외각 = $36°$.
$\dfrac{360°}{n} = 36°$ → $n = 10$.

아래 그림에서 $\angle a + \angle b + \angle c + \angle d + \angle e$의 크기를 구하시오. (오각형 모양으로 다섯 점이 배치되고, 각 끝각을 표기)
이건 오각형의 내각의 합. $180°(5-2) = ?$
$540°$
$a, b, c, d, e$는 모두 오각형의 다섯 내각.
오각형 내각의 합 = $180° \times 3 = 540°$.

H
사각형 ABCD에서 $\angle A : \angle B : \angle C : \angle D = 1 : 2 : 3 : 4$일 때 가장 큰 각과 가장 작은 각의 차를 구하시오.
사각형 내각 합 = $360°$. $x + 2x + 3x + 4x = 10x$.
$108°$
$x + 2x + 3x + 4x = 10x = 360°$
$x = 36°$.
가장 큰 각 = $4x = 144°$, 가장 작은 각 = $x = 36°$.
차 = $144° - 36° = 108°$.
3
H
정$n$각형의 한 내각의 크기가 자연수가 되는 $n$ 중 $n = 3$ 다음으로 가장 작은 값을 구하시오.
한 내각 = $180° - \dfrac{360°}{n}$이 자연수 ⇔ $\dfrac{360°}{n}$이 자연수 ⇔ $n$이 360의 약수.
$n = 4$
한 내각이 자연수 = 한 외각이 자연수 ($\because$ 내각 + 외각 = 180°).
한 외각 = $\dfrac{360°}{n}$이 자연수가 되려면 $n$이 360의 약수.
360의 약수 중 $n \geq 3$: 3, 4, 5, 6, 8, 9, 10, 12, ...
$n=3$ 다음은 $n = 4$. (정사각형, 한 내각 90°)
