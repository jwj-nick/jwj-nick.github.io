유형 5 — 외각의 합·미지 외각

외각의 합 = 360° (변에 무관) + 미지 외각 찾기
← 단원 9 홈
1
L
육각형의 외각의 합은 몇 도인가?
다각형의 외각의 합은 변의 수에 관계없이 항상 ?
$360°$
모든 다각형의 외각의 합은 항상 $360°$. 육각형이든 백각형이든 같아요.
2
L
사각형의 네 외각이 $90°, 80°, 100°, x$일 때, $x$의 값은?
외각의 합 = 360°.
$x = 90°$
$90 + 80 + 100 + x = 360°$
$270 + x = 360°$ → $x = 90°$.
3
L
한 꼭짓점에서 내각이 $115°$일 때, 그 꼭짓점에서의 외각은?
한 꼭짓점에서 (내각) + (외각) = $180°$.
$65°$
외각 = $180° - 115° = 65°$.

M
오각형의 다섯 외각 중 네 외각이 $50°, 70°, 80°, 90°$일 때, 나머지 외각의 크기를 구하시오.
외각의 합 = $360°$.
$70°$
$50 + 70 + 80 + 90 + x = 360$
$290 + x = 360$ → $x = 70°$.
2
M
육각형의 여섯 외각의 비가 $1 : 2 : 3 : 4 : 5 : 5$일 때 가장 큰 외각의 크기를 구하시오.
$x + 2x + 3x + 4x + 5x + 5x = 360°$.
$90°$
$20x = 360°$ → $x = 18°$.
가장 큰 외각 = $5x = 90°$.
3
M
한 다각형의 외각의 합이 그 다각형의 한 외각의 5배일 때, 한 외각의 크기를 구하시오 (정다각형이라고 가정).
외각의 합 = $360°$. 한 외각의 5배 = $360°$.
$72°$
한 외각 × 5 = 360° → 한 외각 = $72°$.
(정오각형: 한 외각 $\dfrac{360°}{5} = 72°$ ✓)
1
H
팔각형에서 일곱 외각의 합이 $300°$이다. 나머지 한 꼭짓점에서의 내각의 크기를 구하시오.
전체 외각합 = 360°. 나머지 외각 = ?. 그 꼭짓점 내각 = $180° - $ 외각.
$120°$
나머지 외각 = $360° - 300° = 60°$.
그 꼭짓점의 내각 = $180° - 60° = 120°$.
2
H
한 다각형의 내각의 합과 외각의 합의 비가 $5 : 1$일 때, 이 다각형은 몇각형인가?
외각의 합 = 360°. 내각의 합 = 5 × 360° = 1800°.
십이각형 ($n = 12$)
내각의 합 : 외각의 합 = 5 : 1.
외각의 합 = $360°$ → 내각의 합 = $5 \times 360° = 1800°$.
$180°(n-2) = 1800°$ → $n - 2 = 10$ → $n = 12$.
3
H
사각형 ABCD에서 네 내각이 각각 $80°, 110°, 100°, 70°$이다. 네 외각의 합과 네 내각의 합을 구하고 두 합이 정확히 일치하는지 확인하시오. 또한 사각형은 어떤 종류인가?
사각형 내각합 = $360°$. 외각합도 $360°$. 둘이 일치하는 건 사각형의 특성!
내각합 $= 360°$, 외각합 $= 360°$. 일치함. (일반 사각형)
내각의 합: $80 + 110 + 100 + 70 = 360°$ ✓
각 외각: $100°, 70°, 80°, 110°$ → 외각합 $= 360°$ ✓
$\therefore$ 사각형은 유일하게 내각의 합과 외각의 합이 같은 다각형이에요. ($180°(4-2) = 360°$ = 외각합)
이 사각형은 정사각형도 아니고 평행사변형도 아닌 일반 사각형.
