깊이 탐구 — 회전체·복합 입체 (유형 6+7)

평면 → 회전 → 입체, 결합/제거로 부피 합성, ★ 결합면 처리
← 단원 12 홈

★핵심 전략 — 회전체

회전축 vs 도형 변의 관계
  • 축에 평행한 변 → 입체의 높이
  • 축에 수직인 변 → 입체의 반지름
  • 축에 비스듬한 변(빗변) → 모선 (원뿔의 경우)
★ 회전체 분류:
• 직사각형 회전 → 원기둥
• 직각삼각형 회전 (직각변 축) → 원뿔
• 반원 회전 (지름 축) → 구
• 사다리꼴 회전 (다리 축) → 원뿔대
• 축에 떨어진 도형 회전 → 도넛(토러스) 모양 (중1 부피 계산 X)
★ 회전축이 어디인가? — 같은 도형도 축이 바뀌면 다른 입체!
예: $3\times 5$ 직사각형 → 가로축 회전 시 $r=5, h=3$ ($V=75\pi$), 세로축 회전 시 $r=3, h=5$ ($V=45\pi$). 답이 다름.

📝대표 문제

M결합 도형 회전
직사각형($4\times 3$) 위에 직각삼각형(밑변 $4$, 높이 $2$)이 붙은 도형을 측면 수직변(축) 기준 회전. 부피?
원기둥($r=4, h=3$) + 원뿔($r=4, h=2$). $48\pi + \frac{32\pi}{3} = \frac{144 + 32}{3}\pi = \frac{176\pi}{3}$ cm³.
H빗변 축 회전 → 두 원뿔
직각삼각형($3, 4, 5$)을 빗변 $5$를 축으로 회전. 부피?
두 원뿔이 밑면(반지름 = 빗변에 내린 수선 $\frac{12}{5}$) 공유. 두 높이 합 = 빗변 $5$.
$V = \frac{1}{3}\pi \left(\frac{12}{5}\right)^2 \times 5 = \frac{1}{3}\pi \times \frac{144}{25} \times 5 = \frac{48\pi}{5}$ cm³.
HL자 도형 (큰 − 작은 원기둥)
L자 도형 회전 시 큰 원기둥 ($r=5, h=3$)에서 작은 원기둥 ($r=2, h=2$) 뺀 형태. 부피?
$75\pi - 8\pi = 67\pi$ cm³.

⚠️실수

• 회전축 잘못 잡아 다른 입체로 추론 (특히 도형 안에 있는 변 vs 도형 밖에 있는 축).
• "회전체 부피 = 단면 넓이 × 거리" 잘못된 일반화 (실제로는 카발리에리 적분 필요. 다만 회전축이 도형 한 변일 때는 단순화 가능).
• 빗변이 모선이 됨을 잊고 별도 변으로 계산.

★핵심 전략 — 복합 입체

부피: 단순 합산 또는 차감 ($V_{전체} = V_1 + V_2$ 또는 $V_1 - V_2$).
겉넓이: 단순 합산 후 결합면 두 번 빼기.
★ 결합면 처리가 가장 헷갈리는 함정:
• 두 입체를 붙이면, 만나는 면(평면 원, 사각형 등)은 내부로 사라짐.
• 부피 계산에는 영향 없음 (그냥 합산).
• 겉넓이 계산에서는 양쪽 면 모두 빼야 함 (즉 결합면의 넓이 $\times 2$ 빼기, 또는 처음부터 그 면을 더하지 않음).
★ 1:2:3 비율 (반지름 $r$ 같고 높이 $= 2r$):
$V_{원뿔} : V_{구} : V_{원기둥} = 1 : 2 : 3$.
→ 시험에서 한 값 주어지면 나머지 추론 가능. 자주 출제됨.

📝대표 문제

L캡슐 (원기둥 + 반구)
$r=3, h=5$ 원기둥 + 반지름 $3$ 반구.
부피: $45\pi + 18\pi = 63\pi$ cm³. 겉넓이(반구 위로 열림, 아래는 원기둥 밑면): 옆면 $30\pi$ + 원기둥 밑면 $9\pi$ + 반구 곡면 $18\pi$ = $57\pi$ cm². (결합면 평면 원은 빠짐.)
M구멍 뚫린 정육면체
한 모서리 $6$인 정육면체에 $r=2$ 원기둥 구멍 (길이 $6$).
부피 $= 6^3 - \pi \cdot 4 \cdot 6 = 216 - 24\pi$ cm³.
H반구 + 원뿔 겉넓이
$r=4$ 반구 + ($r=4, l=5$) 원뿔이 평면 원으로 결합. 겉넓이?
반구 곡면 $32\pi$ + 원뿔 옆면 $20\pi$ = $52\pi$ cm². ⚠️ 결합면 $16\pi$를 양쪽에서 빼야 하므로 처음부터 안 더한 결과. 모두 더하면 $32\pi+16\pi+16\pi+20\pi = 84\pi$ (오답).
H1:2:3 비율 응용
$r$ 같고 높이 $2r$. 구 $= 32\pi$일 때 원뿔, 원기둥은?
$1:2:3$에서 단위 = $32\pi/2 = 16\pi$. 원뿔 $= 16\pi$, 원기둥 $= 48\pi$. (자세한 검산은 type_07 참조.)

⚠️실수

• ★★★ 결합면 처리: 합쳐지는 평면 원을 겉넓이에 더해버림 → 과대 계산.
• 구멍 뚫린 입체에서 구멍 안쪽 옆면을 표면에 안 더함 (필요시 더해야 함).
• 1:2:3 비율을 거꾸로 ($3:2:1$로 외움) — 작은 쪽이 뿔.
• 분수 처리 실수 ($\frac{1}{3}, \frac{2}{3}, \frac{4}{3}$ 모두 등장).

🎯카발리에리 원리 — 단원의 심장

카발리에리 원리
높이가 같은 두 입체에서, 어떤 높이에서 자른 단면적이 같으면, 두 입체의 부피도 같다.
★ 직관: 같은 종이를 쌓아 똑바로 세운 책더미와 비스듬히 기울인 책더미의 부피는 같음 (페이지 수와 한 페이지 넓이가 같으니까).
★ 단원의 모든 공식이 이 한 원리에서 나옴:
• 각기둥 $V = Sh$ — 모든 단면이 합동인 $S$.
• 각뿔 $V = \frac{1}{3}Sh$ — 단면이 위로 갈수록 닮음 축소 → $S(1-t/h)^2$ → 적분 시 $\frac{1}{3}$.
• 구 $V = \frac{4}{3}\pi r^3$ — 단면이 원이고 반지름 $\sqrt{r^2 - y^2}$. 적분하면 $\frac{4}{3}\pi r^3$.
• 구 = 원기둥 − 원뿔 (같은 단면): 반지름 $r$ 원기둥(높이 $2r$)에서 두 원뿔을 빼면 단면이 정확히 구의 단면과 일치 → 부피 동일.

🧮1:2:3 비율의 카발리에리 증명 (직관)

높이가 $2r$인 원기둥에 안에서 똑같이 꼭 맞는 구가 있고, 같은 원기둥에 두 개의 원뿔이 위·아래로 꼭짓점이 가운데에 모이게 들어 있다고 상상.
같은 높이 $y$에서 자른 단면:
• 원기둥 단면 = $\pi r^2$ (일정).
• 구 단면 = $\pi(r^2 - y^2)$ (높이 $y$에서).
• 두 원뿔 합쳐진 단면 = $\pi y^2$ (가운데가 비어 있으므로 단면은 두 개의 작은 원이 합쳐진 모양, 합 = $\pi y^2$).
→ 구 단면 + 두 원뿔 단면 = $\pi(r^2-y^2) + \pi y^2 = \pi r^2$ = 원기둥 단면 ✓.
모든 높이에서 성립 → 부피도: $V_{구} + V_{두 원뿔} = V_{원기둥}$.
$\frac{4}{3}\pi r^3 + 2 \cdot \frac{1}{3}\pi r^2 \cdot r = \frac{4}{3}\pi r^3 + \frac{2}{3}\pi r^3 = 2\pi r^3 = \pi r^2 \cdot 2r$ ✓.
★ 이 한 그림으로 1:2:3이 자연스럽게 따라온다. 두 원뿔 합 = 원뿔 ($h=2r$ 1개) = 단위 1 → 구 = 단위 2 → 원기둥 = 단위 3.

• 뉴턴·라이프니츠(17세기 말): 미적분 정식 탄생 — 카발리에리의 indivisibilibus가 dx로 진화
• 고1 적분: 단면적 함수를 적분 → 부피 — 카발리에리의 직접 후계

💎이것만은 기억

1. 회전축 = 입체의 축. 축에 평행한 변 = 높이, 수직 변 = 반지름.
2. 복합 입체 부피 = 단순 합/차. 겉넓이 = 결합면 빼기.
3. 1:2:3 비율 (원뿔 : 구 : 원기둥, 같은 $r$, 높이 $=2r$).
4. 카발리에리: 같은 단면 → 같은 부피.
5. 빗변 회전 시 두 원뿔이 밑면 공유 (수선 = 반지름).
