유형 5 — 구의 겉넓이·부피

$S = 4\pi r^2$, $V = \frac{4}{3}\pi r^3$, 반구 $S = 3\pi r^2$, $V = \frac{2}{3}\pi r^3$
← 단원 12 홈
1
L
반지름 $3$ cm인 구의 부피는?
① $12\pi$ cm³   ② $27\pi$ cm³   ③ $36\pi$ cm³   ④ $108\pi$ cm³
$V = \frac{4}{3}\pi r^3$.
③ $36\pi$ cm³
$V = \frac{4}{3}\pi \times 3^3 = \frac{4}{3}\pi \times 27 = 4\pi \times 9 = 36\pi$ cm³.
⚠️ ④ $108\pi$는 $\frac{4}{3}$ 빼먹은 답 ($\pi r^3$). ② $27\pi$는 $\pi r^3$의 $\frac{4}{3}$를 $\frac{1}{1}$로 본 오답.
2
L
반지름 $5$ cm인 구의 겉넓이는?
① $25\pi$ cm²   ② $50\pi$ cm²   ③ $100\pi$ cm²   ④ $125\pi$ cm²
$S = 4\pi r^2$.
③ $100\pi$ cm²
$S = 4\pi \times 5^2 = 4\pi \times 25 = 100\pi$ cm².
⚠️ ① $25\pi$는 $\pi r^2$ (원 한 개의 넓이 = 큰 원). 구의 겉넓이는 그것의 4배.
3
L
반지름 $6$ cm인 반구의 부피는?
① $72\pi$ cm³   ② $144\pi$ cm³   ③ $216\pi$ cm³   ④ $288\pi$ cm³
반구 부피 = 구 부피 $\div 2$ = $\frac{2}{3}\pi r^3$.
② $144\pi$ cm³
구 부피 = $\frac{4}{3}\pi \times 6^3 = \frac{4}{3}\pi \times 216 = 288\pi$ cm³.
반구 = $288\pi \div 2 = 144\pi$ cm³.
또는 직접: $\frac{2}{3}\pi r^3 = \frac{2}{3}\pi \times 216 = 144\pi$ cm³.

M
반지름 $4$ cm인 반구의 겉넓이(곡면 + 평면 원)는?
① $16\pi$ cm²   ② $32\pi$ cm²   ③ $48\pi$ cm²   ④ $64\pi$ cm²
반구 겉넓이 = 곡면 $2\pi r^2$ + 평면 원 $\pi r^2$ = $3\pi r^2$.
③ $48\pi$ cm²
곡면 = $\frac{1}{2}\times 4\pi r^2 = 2\pi r^2 = 2\pi \times 16 = 32\pi$.
평면 원 = $\pi r^2 = 16\pi$.
합 = $32\pi + 16\pi = 48\pi$ cm² (= $3\pi r^2$).
⚠️ ★★★ ② $32\pi$는 곡면만 (평면 원 빠뜨림). 반구는 잘린 평면 원도 표면!
2
M
구 A의 반지름이 $r$, 구 B의 반지름이 $2r$일 때, B의 부피는 A의 몇 배?
① $2$배   ② $4$배   ③ $6$배   ④ $8$배
$V = \frac{4}{3}\pi r^3$. $r$이 $2$배면 $r^3$은 $8$배.
④ $8$배
$V_A = \frac{4}{3}\pi r^3$. $V_B = \frac{4}{3}\pi (2r)^3 = \frac{4}{3}\pi \times 8r^3 = 8 V_A$.
★ 일반화: 반지름 $k$배 → 겉넓이 $k^2$배, 부피 $k^3$배. (이 문제에서 겉넓이는 $4$배.)
3
M
구의 부피가 $36\pi$ cm³일 때, 반지름은?
① $2$ cm   ② $3$ cm   ③ $4$ cm   ④ $6$ cm
$\frac{4}{3}\pi r^3 = 36\pi$ → $r^3 = 27$.
② $3$ cm
$\frac{4}{3}\pi r^3 = 36\pi$ → $r^3 = 36 \times \frac{3}{4} = 27$ → $r = 3$ cm (왜냐하면 $3^3 = 27$).
검산: $\frac{4}{3}\pi \times 27 = 36\pi$ ✓.

H
반지름 $r$인 구가 원기둥 안에 꼭 맞게 들어갑니다 (원기둥 높이 = $2r$, 반지름 = $r$). 이때 구 부피와 원기둥 부피의 비 $V_{구} : V_{원기둥}$은?
① $1 : 2$   ② $2 : 3$   ③ $3 : 4$   ④ $1 : 3$
구 부피 = $\frac{4}{3}\pi r^3$, 원기둥 부피 = $\pi r^2 (2r) = 2\pi r^3$.
② $2 : 3$
$V_{구} = \frac{4}{3}\pi r^3$, $V_{원기둥} = \pi r^2 \times 2r = 2\pi r^3$.
비: $\dfrac{V_{구}}{V_{원기둥}} = \dfrac{\frac{4}{3}\pi r^3}{2\pi r^3} = \dfrac{4}{6} = \dfrac{2}{3}$.
★ 아르키메데스 묘비의 결과 (Unit 10). 카발리에리는 이를 단면 비교로 다시 증명.
2
H
같은 반지름 $r$이고, 원뿔 높이 = 구 지름 = 원기둥 높이 = $2r$일 때, 부피 비 $V_{원뿔} : V_{구} : V_{원기둥}$은?
① $1 : 2 : 3$   ② $1 : 3 : 2$   ③ $1 : 4 : 6$   ④ $2 : 4 : 3$
각각 $\frac{1}{3}\pi r^2 (2r), \frac{4}{3}\pi r^3, \pi r^2 (2r)$ 계산.
① $1 : 2 : 3$
$V_{원뿔} = \frac{1}{3}\pi r^2 \times 2r = \frac{2}{3}\pi r^3$.
$V_{구} = \frac{4}{3}\pi r^3$.
$V_{원기둥} = \pi r^2 \times 2r = 2\pi r^3 = \frac{6}{3}\pi r^3$.
비: $\frac{2}{3} : \frac{4}{3} : \frac{6}{3} = 2 : 4 : 6 = \boxed{1 : 2 : 3}$.
★ 아르키메데스의 1:2:3 비율 — 그가 가장 자랑스러워한 발견.
3
H
반지름 $3$ cm인 반구 위에 같은 반지름의 원기둥(높이 $4$ cm)이 얹어진 합성 입체(평평한 바닥은 원기둥의 다른 밑면). 이 입체의 부피는?
① $36\pi$ cm³   ② $48\pi$ cm³   ③ $54\pi$ cm³   ④ $72\pi$ cm³
반구 부피 + 원기둥 부피.
③ $54\pi$ cm³
반구 부피 = $\frac{2}{3}\pi \times 3^3 = \frac{2}{3}\pi \times 27 = 18\pi$ cm³.
원기둥 부피 = $\pi \times 3^2 \times 4 = 36\pi$ cm³.
합 = $18\pi + 36\pi = 54\pi$ cm³.
★ 결합면(평면 원)은 내부로 사라짐 — 부피 계산에는 영향 없음. 겉넓이 계산에서는 빼야 함.
