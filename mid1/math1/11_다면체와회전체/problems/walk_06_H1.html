유형 6 · H1

밑면의 반지름이 $r$, 모선의 길이가 $l$ 인 원뿔의 옆면 부채꼴의 중심각을 $\theta$ 라 할 때,
$\theta$ 를 $r$ 과 $l$ 로 나타내세요.
🤔 잠깐, 손으로 풀어보세요 원뿔을 펼치면 옆면은 부채꼴이 돼요. 그 부채꼴의 반지름은 무엇이고, 호의 길이는 무엇과 같아야 하나요?
1단계 · Page 1

원뿔 전개도의 연결 고리

왜 이 단계? — 원뿔을 펼치면 옆면이 부채꼴이 돼요. 부채꼴 호 = 밑면 둘레 관계를 파악해요.
📖 용어 노트 모선: 원뿔 꼭짓점에서 밑면 원 가장자리까지의 선  ·  부채꼴 호 $= \dfrac{\theta}{360} \times 2\pi l$
전개도 분석

원뿔을 펼치면:

옆면 = 부채꼴 (반지름 = 모선 $l$, 중심각 = $\theta$)
밑면 = 원 (반지름 $r$, 둘레 $= 2\pi r$)

연결 고리: 부채꼴의 호 = 밑면의 둘레

부채꼴 호 $= \dfrac{\theta}{360} \times 2\pi l$
이것이 밑면 둘레 $2\pi r$ 와 같아요!

2단계 · Page 2

방정식으로 $\theta$ 구하기

왜 이 단계? — 부채꼴 호 = 밑면 둘레 등식을 세워 $\theta$를 $r, l$로 나타내요.
📖 용어 노트 등식을 세운 뒤 $\theta$에 대해 풀어요.
방정식 풀기
$\dfrac{\theta}{360} \times 2\pi l = 2\pi r$

$2\pi$ 를 양변에서 나누면:
$\dfrac{\theta}{360} \times l = r$

$\dfrac{\theta}{360} = \dfrac{r}{l}$

$\theta = \dfrac{r}{l} \times 360°$
⚠️ 핵심 확인 부채꼴 반지름 = 원뿔 모선 $l$ (반지름 $r$이 아님!)
부채꼴 호 = 원뿔 밑면 둘레 $2\pi r$

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 원뿔 중심각 공식: $\theta = \dfrac{r}{l} \times 360°$
$\theta = \dfrac{r}{l} \times 360°$
예: $r=3, l=9$ → $\theta = \dfrac{3}{9} \times 360° = 120°$
전체 풀이 흐름 — 3단계
1
원뿔 전개: 옆면 = 부채꼴(반지름 $l$, 중심각 $\theta$)
2
핵심 관계: 부채꼴 호 = 밑면 둘레 → $\dfrac{\theta}{360} \times 2\pi l = 2\pi r$
3
$\theta$에 대해 풀기: $\theta = \dfrac{r}{l} \times 360°$
