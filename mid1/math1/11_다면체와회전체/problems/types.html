각 유형마다 L(쉬운) × 3 + M(중간) × 3 + H(어려운) × 3 = 9문제를 풀어볼 수 있어요.

빈출: 구성요소 세기, 오일러 검증 / 함정: 곡면 포함 입체를 다면체로 착각, 뒷면 빼먹기

L M H

빈출: $n$ 역산, 두 입체 비교 / 함정: 각기둥 모서리 $2n$ 오답 (정답 $3n$)

3. 정다면체 5종 ★
L M H

빈출: 5종 식별, 왜 5종뿐인가 / 함정: 정다각형 면만 확인하고 꼭짓점 조건 누락
→ 연습하기
4. 회전체 식별과 종류
L M H

빈출: 어떤 입체가 되나 / 함정: 각기둥을 회전체로 분류, 회전축 위치 무시
→ 연습하기
5. 회전체의 단면 ★
L M H
수직 단면 = 항상 원 / 축단면 = 좌우대칭 도형 (원기둥→직사각형, 원뿔→이등변삼각형)
빈출: 단면 모양, 둘레·넓이 계산 / 함정: 수직↔축단면 혼동, 원뿔 수직 단면 = 이등변삼각형 오답
→ 연습하기
6. 회전체의 전개도
L M H
원기둥 = 직사각형 + 원 2개 / 원뿔 = 부채꼴 + 원 / 부채꼴 호 = 밑면 둘레 $= 2\pi r$
빈출: 중심각 $\theta = \dfrac{r}{l} \times 360°$, 역산 / 함정: 원뿔 옆면 = 삼각형 오답, 가로 = 반지름 오답
→ 연습하기

L M H
전개도→입체, 입체→단면, 회전축 추론, 결합 입체의 $V, E, F$ 계산
빈출: 복합 입체 V·E·F / 함정: 결합 시 공유면 처리 누락 (면 2개 빼야 함)
→ 연습하기
