$2(x + 3) = 10$을 풀어라.
분배법칙: $2(x+3) = 2x + 6$. 먼저 괄호를 풀어요.
$x = 2$
$2x+6=10$
$2x=4$, $x=2$
검산: $2(2+3)=10$ ✓

$x = 2$
$6x-3=9$
$6x=12$, $x=2$
검산: $3(4-1)=9$ ✓

$x = 6$
$4x-8=2x+4$
$2x=12$, $x=6$

$x = \dfrac{14}{3}$
$6x-2=3x+12$
$3x=14$, $x=\dfrac{14}{3}$
검산: $2(3 \cdot \frac{14}{3}-1)=2(13)=26$, $3(\frac{14}{3}+4)=3 \cdot \frac{26}{3}=26$ ✓

$x = -17$
$5x-10-6x-3=4$
$-x-13=4$
$-x=17$, $x=-17$

$-3(2x+1) = -6x - 3$. 부호에 주의! $-3 \times 1 = -3$이에요.

$x=3$을 대입 → $a$에 대한 방정식이 돼요.

$a(x + 2) = 3x + 6$이 $x$에 관한 항등식이 되도록 하는 $a$의 값을 구하세요.
우변을 인수분해하면? $3x+6 = 3(x+2)$
$a = 3$
우변: $3x+6=3(x+2)$
따라서 $a(x+2)=3(x+2)$가 항등식이 되려면 $a=3$
확인: $3(x+2)=3x+6$ (모든 $x$에서 참) ✓

$3(x - 2) - 2(x + 1) = a(x - 1) + b$가 $x$에 관한 항등식일 때, $a + b$의 값을 구하세요.
좌변을 전개해서 $px + q$ 꼴로 만든 뒤, 우변과 계수 비교해요.
$a+b = 1 + (-7) = -6$
좌변: $3x-6-2x-2 = x-8$
우변: $ax-a+b$
$x$의 계수: $a=1$
상수항: $-a+b=-8 \Rightarrow -1+b=-8 \Rightarrow b=-7$
$a+b=1+(-7)=-6$
