일차방정식 풀기
L
$3x - 7 = 2x + 1$을 풀어라.

정답: $x = 8$
$3x - 2x = 1 + 7$
$x = 8$
검산: $3(8)-7=17$, $2(8)+1=17$ ✓

분수 계수
M
$\dfrac{x}{2} + 1 = 4$를 풀어라.

정답: $x = 6$
양변에 2를 곱하면: $x + 2 = 8$
이항: $x = 8 - 2 = 6$
검산: $6/2 + 1 = 4$ ✓

괄호 있는 방정식
M
$2(x - 3) = x + 1$을 풀어라.

정답: $x = 7$
괄호 풀기: $2x - 6 = x + 1$
이항: $2x - x = 1 + 6$
$x = 7$
검산: $2(7-3)=8$, $7+1=8$ ✓

소수 계수
M
$0.3x + 0.5 = 1.4$를 풀어라.

정답: $x = 3$
양변에 10을 곱하면: $3x + 5 = 14$
이항: $3x = 9$
$x = 3$
검산: $0.3(3)+0.5 = 1.4$ ✓

M

$2x + 1 = 25$

$x = 12$
