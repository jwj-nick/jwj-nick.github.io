유형 3 · H1

방정식 $ax + 3 = 15$의 해가 $x = 4$일 때, 상수 $a$의 값을 구하세요.
🤔 잠깐, 손으로 풀어보세요 "해가 $x = 4$"라는 말의 뜻을 생각해봐요. 막연하면 다음 페이지로 넘어가요.
1단계 · Page 1

"해가 $x=4$"의 의미

왜 이 단계? — 해의 정의를 정확히 알아야 풀이 방향이 보여요.
📖 용어 노트 해(근): 방정식을 참으로 만드는 미지수의 값
해의 정의

방정식의 해란 "대입하면 등식이 참이 되는 값"이에요.

"$x = 4$가 $ax + 3 = 15$의 해다"
⟺ $x = 4$를 대입하면 등식이 성립한다
⟺ $a \cdot 4 + 3 = 15$

따라서 $x = 4$를 방정식에 대입하면, $a$에 대한 방정식이 만들어져요!

💡 핵심 전략 $x = 4$를 대입 → $a$에 대한 방정식 → $a$를 구해요.

2단계 · Page 2

$x = 4$를 대입해요

왜 이 단계? — 대입하면 $a$에 대한 방정식이 나와요.
📖 용어 노트 이항: 항을 등호 반대쪽으로 옮기면서 부호가 반대로 바뀜
대입
$ax + 3 = 15$ 에 $x = 4$ 대입:
$a \cdot 4 + 3 = 15$
$4a + 3 = 15$

이제 $4a + 3 = 15$라는 $a$에 대한 방정식이 됐어요. 이항으로 풀어봐요.

3단계 · Page 3

$4a + 3 = 15$를 풀어요

왜 이 단계? — 이항으로 $a$를 구해요.
📖 용어 노트 이항: 항을 등호 반대쪽으로 옮기면서 부호가 반대로 바뀜
이항으로 풀기
$4a + 3 = 15$
$3$을 이항하면 → $4a = 15 - 3$
$4a = 12$
양변을 $4$로 나누면 → $a = 3$
⚠️ 이항 부호 반전 $+3$을 우변으로 이항하면 $-3$이 돼요. 이항할 때는 반드시 부호가 바뀌어요!

정답 + 풀이 흐름 요약

📖 용어 노트 방정식: 미지수의 값에 따라 참/거짓이 결정되는 등식  ·  해(근): 방정식을 참으로 만드는 미지수의 값  ·  이항: 항을 등호 반대쪽으로 옮기면서 부호가 반대로 바뀜  ·  일차방정식: 미지수의 최고 차수가 1인 방정식
$a = 3$
확인: $3(4)+3=15$ ✓
전체 풀이 흐름
1
"$x=4$가 해" → $x=4$를 대입하면 등식이 성립한다
2
$a \cdot 4 + 3 = 15$ → $4a + 3 = 15$
3
이항: $4a = 12$ → $a = 3$
4
확인: $3(4)+3=15$ ✓
⚠️ 이 유형의 핵심 전략 "해가 $x = k$이다"라는 조건은 항상 "$x = k$를 대입하면 등식이 성립한다"로 바꿔서 풀어요. 이렇게 하면 미지수(상수)에 대한 방정식이 만들어져요.
