문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
유형 2 · H2

수직선에서 $-\dfrac{5}{3}$과 $+\dfrac{7}{3}$ 사이에 있는 정수를 모두 쓰세요.
🤔 잠깐, 손으로 풀어보세요 분수를 먼저 소수나 혼수(대분수)로 바꿔봐요. 그러면 두 수가 수직선 위 어디쯤인지 보여요.
1단계 · Page 1

분수를 소수로 바꿔요

왜 이 단계? — 분수로는 어느 정수 사이에 있는지 바로 보이지 않아요. 소수로 바꾸면 위치가 명확해져요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수
분수 → 소수 변환
$-\dfrac{5}{3} = -1.666\ldots \approx -1.67$

$+\dfrac{7}{3} = +2.333\ldots \approx +2.33$

즉, 이 문제는 $-1.67$과 $+2.33$ 사이의 정수를 구하는 것이에요.

🤔 범위 부등식으로 표현 $-1.67 < x < 2.33$ (정수 $x$를 구하라)

2단계 · Page 2

범위 안의 정수를 나열해요

왜 이 단계? — 범위를 알았으니, 그 범위 안에 있는 정수를 하나씩 찾아요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수
$-1.67 < x < 2.33$ 인 정수

$-1.67$보다 큰 첫 번째 정수: $-1$
(왜냐면 $-2 < -1.67$이므로 $-2$는 범위 밖이에요)

$2.33$보다 작은 마지막 정수: $+2$
(왜냐면 $3 > 2.33$이므로 $+3$은 범위 밖이에요)

정수 나열: $-1, \; 0, \; +1, \; +2$  →  4개
⚠️ 경계 값 주의 $-\frac{5}{3}$과 $+\frac{7}{3}$ 자체는 정수가 아니에요.
만약 "이상·이하" 조건이면 경계도 포함하지만, "사이에 있는"(초과·미만)이므로 경계는 제외해요.
그런데 여기서 경계 자체가 이미 정수가 아니므로 결과는 같아요.

3단계 · Page 3

분수와 정수를 직접 비교하는 방법도 있어요

왜 이 단계? — 소수 변환 없이 분수와 정수를 직접 비교하는 방법을 알아봐요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수
분수 ↔ 정수 직접 비교

$-\dfrac{5}{3}$와 정수의 관계:

$-\dfrac{6}{3} = -2$    $-\dfrac{5}{3}$    $-\dfrac{3}{3} = -1$
즉, $-2 < -\dfrac{5}{3} < -1$

$+\dfrac{7}{3}$와 정수의 관계:

$+\dfrac{6}{3} = +2$    $+\dfrac{7}{3}$    $+\dfrac{9}{3} = +3$
즉, $+2 < +\dfrac{7}{3} < +3$

따라서 $-\dfrac{5}{3}$보다 크고 $+\dfrac{7}{3}$보다 작은 정수: $-1, 0, +1, +2$

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
$-1, \; 0, \; +1, \; +2$
(4개)
전체 풀이 흐름 — 3단계
1
분수를 소수로: $-\frac{5}{3} \approx -1.67$, $+\frac{7}{3} \approx 2.33$
2
$-1.67 < x < 2.33$인 정수 나열: $-1, 0, +1, +2$
3
분수 직접 비교로도 확인: $-2 < -\frac{5}{3} < -1$, $2 < \frac{7}{3} < 3$
⚠️ 핵심 함정 — 경계 처리 분수를 소수로 바꿀 때 반올림하면 실수할 수 있어요.
$-1.67$을 $-2$로 반올림하면 $-2$도 범위에 포함되는 것으로 착각할 수 있어요.
실제로 $-\frac{5}{3} = -1.666\ldots$이므로 $-2$는 범위 밖이에요.
