유형 04 — 대소 관계

부등호·대소 비교·범위 조건 · L×3 M×3 H×3

L-1 쉬움
빈칸에 $>$, $<$, $=$ 중 알맞은 것을 쓰세요.

$-7 \quad \square \quad -3$
음수끼리는 절댓값이 클수록 더 작아요. |−7|=7 > |−3|=3이므로 −7이 더 작아요.
L-2 쉬움
다음 수를 작은 것부터 순서대로 나열하세요.
$$+5, \quad -2, \quad 0, \quad -6, \quad +1$$
L-3 쉬움
다음 중 가장 큰 수를 고르세요.
$-10, \quad +2, \quad -0.5, \quad 0, \quad -100$
M-1 중간
$-4 \leq x \leq 2$를 만족하는 정수 $x$의 개수를 구하세요.
−4 이상 2 이하의 정수를 하나씩 나열해봐요.
M-2 중간
다음 조건을 모두 만족하는 정수 $x$를 모두 쓰세요.

• $x > -3$
• $x \leq 1$
• $x$는 음수
−3 < x ≤ 1이고 x < 0. 두 조건을 합치면 −3 < x < 0이에요.
M-3 중간
$a, b$가 정수일 때, $a < 0 < b$이면 다음 중 항상 옳은 것을 고르세요.
a<0 음수이므로 b > a, 즉 a < b는 항상 옳아요.
H-1 어려움
$-\dfrac{5}{2} < x < \dfrac{7}{3}$을 만족하는 정수 $x$의 개수를 구하세요.
분수를 소수로 바꿔봐요. −5/2=−2.5, +7/3≈2.33. 이 사이의 정수는?
H-2 어려움
절댓값이 $3$ 이하인 음의 정수 중 가장 큰 수와 절댓값이 $2$ 이상인 양의 정수 중 가장 작은 수의 합을 구하세요.
|a|≤3인 음의 정수: −1, −2, −3. 가장 큰 것은?
|b|≥2인 양의 정수: 2, 3, 4, …. 가장 작은 것은?
H-3 어려움
다음 세 수를 대소 관계 순서대로 나열하세요.
$$-\frac{3}{4}, \quad +\frac{2}{3}, \quad -\frac{5}{6}$$
통분해서 비교해봐요. −3/4=−9/12, −5/6=−10/12이에요.
