단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
유형 3 · H2

$a < 0$일 때, $|a|$와 $-a$의 관계를 설명하고, $a = -3$일 때 $|a| + (-a)$의 값을 구하세요.
🤔 잠깐, 손으로 풀어보세요 $a < 0$일 때 $-a$는 양수가 돼요. 그리고 $|a|$는? 두 값이 같은지 다른지 생각해봐요.
1단계 · Page 1

$a < 0$일 때 $|a|$와 $-a$ 이해하기

왜 이 단계? — 절댓값의 정의와 부호 반전의 의미를 함께 이해해야 해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
절댓값의 정의 (부호에 따른 경우)
$|a| = \begin{cases} a & (a \geq 0) \\ -a & (a < 0) \end{cases}$

$a$가 음수이면 $|a| = -a$예요.
예: $a = -3$이면 $|{-3}| = -(-3) = +3$

$a < 0$일 때 $|a|$와 $-a$ 비교

$a < 0$이면:

$|a| = -a$ (위 정의에 의해)

즉, $a$가 음수일 때 $|a|$와 $-a$는 같아요.

2단계 · Page 2

$a = -3$ 대입해서 계산해요

왜 이 단계? — 관계를 파악했으니 구체적인 값을 대입해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
$a = -3$ 대입
$|a| = |-3| = 3$
$-a = -(-3) = +3$

$|a|$와 $-a$ 모두 $3$이에요 — 위에서 확인한 관계($a < 0$이면 $|a| = -a$)와 일치해요.

$|a| + (-a)$ 계산
$|a| + (-a) = 3 + 3 = 6$

3단계 · Page 3

이중 부호 처리 핵심 확인

왜 이 단계? — $-a$를 "음수"로 오해하는 실수가 아주 흔해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
⚠️ 가장 흔한 실수 — "$-a$는 음수다?" $-a$는 "$a$에 $-1$을 곱한 것"이지 "음수"가 아니에요.
$a = -3$이면 $-a = -(-3) = +3$으로 양수예요.
$-a$의 부호는 $a$의 부호에 따라 달라져요:
• $a > 0$이면 $-a < 0$ (음수)
• $a < 0$이면 $-a > 0$ (양수)
• $a = 0$이면 $-a = 0$
⚠️ 이중 부호 정리 $-(-3) = +3$ (음수 두 개 → 양수)
$-(-a) = a$ (일반적으로)
이 규칙은 수 계산의 가장 기본이에요 — 확실히 기억해요!

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
$|a| + (-a) = 6$
($a < 0$일 때 $|a| = -a$, $a=-3$이면 $3+3=6$)
전체 풀이 흐름 — 3단계
1
$a < 0$일 때 절댓값 정의: $|a| = -a$ → $|a|$와 $-a$는 같아요
2
$a = -3$ 대입: $|{-3}| = 3$, $-(-3) = +3$
3
$|a| + (-a) = 3 + 3 = 6$
⚠️ 핵심 정리 $a < 0$이면: $|a| = -a$ (절댓값 = 부호 반전)
$-a$는 "음수"가 아니라 "$a$의 부호 반전"이에요. $a$가 음수면 $-a$는 양수!
