문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
유형 3 · H3

두 수 $a = -\dfrac{7}{3}$, $b = +\dfrac{5}{3}$에 대해 $|a| - |b|$의 값을 구하세요.
🤔 잠깐, 손으로 풀어보세요 $|a|$와 $|b|$를 각각 구한 다음 빼봐요. 절댓값은 항상 양수(또는 0)예요.
1단계 · Page 1

$|a|$와 $|b|$ 각각 계산해요

왜 이 단계? — 절댓값은 부호를 제거한 거리예요. 음수의 절댓값도 양수가 돼요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
절댓값 계산
$|a| = \left|-\dfrac{7}{3}\right| = +\dfrac{7}{3}$

$|b| = \left|+\dfrac{5}{3}\right| = +\dfrac{5}{3}$

음수의 절댓값 → 부호만 제거, 양수의 절댓값 → 그대로예요.

← 이전
2단계 · Page 2

$|a| - |b|$ 계산해요

왜 이 단계? — 분모가 같은 분수 뺄셈이에요. 직접 계산해봐요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
계산
$|a| - |b| = \dfrac{7}{3} - \dfrac{5}{3} = \dfrac{7-5}{3} = \dfrac{2}{3}$

분모가 같으므로 분자끼리 빼면 돼요.

⚠️ 흔한 실수 $|a| - |b|$를 $|a - b|$로 잘못 계산하면 안 돼요.
$|a - b| = \left|-\frac{7}{3} - \frac{5}{3}\right| = \left|-4\right| = 4$ — 다른 값이 나와요.
절댓값을 먼저 계산한 다음 연산을 적용해야 해요.

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
$|a| - |b| = \dfrac{2}{3}$
($\frac{7}{3} - \frac{5}{3} = \frac{2}{3}$)
전체 풀이 흐름 — 2단계
1
절댓값 계산: $|{-\frac{7}{3}}| = \frac{7}{3}$, $|{+\frac{5}{3}}| = \frac{5}{3}$
2
뺄셈: $\frac{7}{3} - \frac{5}{3} = \frac{2}{3}$
⚠️ 핵심 주의 $|a| - |b| \neq |a - b|$ — 순서가 달라요!
절댓값을 먼저 구한 후 뺄셈: $\frac{7}{3} - \frac{5}{3} = \frac{2}{3}$
뺄셈 후 절댓값: $\left|-\frac{7}{3} - \frac{5}{3}\right| = \frac{12}{3} = 4$ — 다른 값!
