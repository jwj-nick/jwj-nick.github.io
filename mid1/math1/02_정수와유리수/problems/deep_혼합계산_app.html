Unit 02 · 깊이 탐구

사칙혼합계산

연산 순서와 괄호 처리를 완벽하게

이 유형이 어려운 이유

흔한 실수 패턴
• 덧셈·뺄셈을 곱셈·나눗셈보다 먼저 계산하는 실수
• 괄호 안 음수 처리 실수: $-(−3)^2$에서 거듭제곱이 먼저라 $(-3)^2=9$, 그 다음 $-9$
• 분수 나눗셈을 곱셈으로 바꾸는 과정에서 부호 실수
• 중괄호{ }와 소괄호( )를 동시에 쓸 때 안쪽부터 계산 안 하는 경우

연산 순서 — 3단계

①
괄호 안
( ) 안 먼저
②
곱셈·나눗셈
× ÷ (왼쪽부터)
③
덧셈·뺄셈
+ − (왼쪽부터)
괄호가 여러 개: 소괄호( ) → 중괄호{ } → 대괄호[ ] 순서로 안쪽부터
거듭제곱이 있으면: 괄호 다음, 곱셈 전에 계산
$(-2)^3 = -8$ (홀수 지수 → 음수)
$(-2)^2 = +4$ (짝수 지수 → 양수)
$-(-2)^2 = -(+4) = -4$ (부호 앞 마이너스는 거듭제곱 후 적용!)

함께 풀어봐요 — 단계별 공개

M 계산하세요.
$$4 - (-3)^2 \times 2 + (-6) \div 3$$
① 거듭제곱: $(-3)^2 = +9$
→ $4 - 9 \times 2 + (-6) \div 3$
② 곱셈·나눗셈: $9 \times 2 = 18$, $(-6) \div 3 = -2$
→ $4 - 18 + (-2)$
③ 뺄셈→덧셈: $4 + (-18) + (-2)$
④ 합산: $4 + (-20) = -16$
정답: −16
M 계산하세요.
$$(-3) \times \left\{(-4) + (-6) \div (+2)\right\}$$
① 중괄호 안 먼저:
  나눗셈: $(-6) \div (+2) = -3$
  덧셈: $(-4) + (-3) = -7$
② 곱셈: $(-3) \times (-7) = +21$
정답: +21
H 계산하세요.
$$(-2)^3 \div \left(-\frac{1}{4}\right) + \left(-\frac{3}{2}\right)^2 \times (-4)$$
① 거듭제곱:
$(-2)^3 = -8$ (홀수 지수 → 음수)
$(-\dfrac{3}{2})^2 = +\dfrac{9}{4}$ (짝수 지수 → 양수)
② 나눗셈 → 역수:
$(-8) \div (-\dfrac{1}{4}) = (-8) \times (-4) = +32$
③ 곱셈:
$(+\dfrac{9}{4}) \times (-4) = -9$
④ 덧셈:
$(+32) + (-9) = +23$
정답: +23
H $[(-2)^2 - \{5 - (-3)\}] \times \dfrac{1}{2}$를 계산하세요.
① 소괄호 안: $5 - (-3) = 5 + 3 = 8$
→ $[(-2)^2 - 8] \times \dfrac{1}{2}$
② 거듭제곱: $(-2)^2 = +4$
→ $[4 - 8] \times \dfrac{1}{2}$
③ 중괄호(대괄호 안): $4 - 8 = -4$
④ 곱셈: $(-4) \times \dfrac{1}{2} = -2$
정답: −2
H $a = +3$, $b = -2$일 때, $a^2 - 2ab + b^2$의 값을 구하세요.
① 각 항 계산:
$a^2 = (+3)^2 = 9$
$2ab = 2 \times (+3) \times (-2) = -12$
$b^2 = (-2)^2 = 4$
② 대입: $9 - (-12) + 4 = 9 + 12 + 4 = 25$
(참고: $a^2 - 2ab + b^2 = (a-b)^2 = (3-(-2))^2 = 5^2 = 25$)
정답: 25

실수 패턴 총정리

거듭제곱 앞 마이너스 처리:
$-(-3)^2$ ≠ $(-3)^2$ ← 다른 식이에요!
$-(-3)^2 = -(+9) = -9$
$(-3)^2 = +9$
분수 나눗셈:
$\div \dfrac{a}{b} = \times \dfrac{b}{a}$ (역수 곱하기)

이 유형을 이해하면 무엇이 쉬워지나요?

사칙혼합 순서는 → 중1 문자와 식(단원 3)에서 대입 계산의 필수 기초예요.
거듭제곱 처리는 → 중2, 3의 다항식 연산에서 계속 등장해요.
괄호 처리는 → 중1 일차방정식의 전개와 직접 연결돼요.
