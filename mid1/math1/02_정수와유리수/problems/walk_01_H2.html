2단계 · Page 2

절댓값 조건으로 범위를 좁혀요

왜 이 단계? — 절댓값(|a|)은 원점까지의 거리예요. 이 조건이 숫자 범위를 정해줘요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
조건 4 — 절댓값 ≤ 2

$|a| \leq 2$이면 원점에서 거리가 2 이하예요.

$|a| \leq 2$  ⟺  $-2 \leq a \leq 2$

지금까지 후보는 음의 정수였어요. 여기에 $-2 \leq a$를 추가하면:

음의 정수 중 $|a| \leq 2$  →  $a = -1$ 또는 $a = -2$
💡 확인해봐요 $-3$은 음의 정수이지만 $|-3| = 3 > 2$이므로 조건 4를 만족하지 않아요.
$-1$: $|-1| = 1 \leq 2$ → 조건 충족
$-2$: $|-2| = 2 \leq 2$ → 조건 충족

최종 후보를 검증해요

왜 이 단계? — 구한 답이 모든 조건을 만족하는지 하나씩 확인해봐요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수
$a = -1$ 검증
①유리수? $-1 = \frac{-1}{1}$ → 유리수 O
②정수? → O
③0보다 작다? $-1 < 0$ → O
④절댓값 $\leq 2$? $|-1|=1 \leq 2$ → O
$a = -2$ 검증
①유리수? → O
②정수? → O
③0보다 작다? $-2 < 0$ → O
④절댓값 $\leq 2$? $|-2|=2 \leq 2$ → O
두 수 모두 4가지 조건을 만족해요 → 정답: $-1$, $-2$

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
$-1$, $-2$
(두 수 모두 4가지 조건 충족)
전체 풀이 흐름 — 3단계
1
조건 1+2+3 결합: 유리수이고 정수이고 0보다 작다 → 음의 정수 $(-1, -2, -3, \ldots)$
2
조건 4 적용: $|a| \leq 2$ → $-2 \leq a \leq 2$ → 음의 정수 중 $-1, -2$
3
검증: $-1$, $-2$ 모두 4가지 조건 충족 확인
⚠️ 이 유형의 핵심 함정 절댓값이 2 이하라고 해서 $-2 < a < 2$로 쓰면 안 돼요.
$|a| \leq 2$ → $-2 \leq a \leq 2$ (등호 포함!). $-2$도 절댓값이 2이므로 포함해요.
