문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
유형 2 · H1

수직선에서 $-3$과 $+1$의 한가운데에 있는 점이 나타내는 수를 구하세요.
🤔 잠깐, 손으로 풀어보세요 두 수를 수직선에 표시하고, 그 사이의 정확한 가운데를 찾아봐요. 두 수의 평균을 구하는 방법도 있어요.
1단계 · Page 1

"한가운데"가 뭘 의미하는지 알아봐요

왜 이 단계? — 수직선에서 두 점의 한가운데는 두 수의 평균이에요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리
한가운데 = 두 수의 평균

수직선에서 두 수 $a$와 $b$ 사이의 한가운데 수는:

한가운데 수 $= \dfrac{a + b}{2}$

이유: 두 수로부터 같은 거리에 있는 수는 두 수를 더해 2로 나눈 값이에요.
(이것이 바로 두 수의 평균이에요.)

🤔 직관적으로 생각해봐요 예시: $0$과 $4$ 사이의 한가운데는? → $\frac{0+4}{2} = 2$
수직선에서 2는 0으로부터 2칸, 4로부터 2칸 — 정확히 가운데예요.

2단계 · Page 2

실제로 계산해봐요

왜 이 단계? — 음수를 포함한 평균 계산은 부호에 주의해야 해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
계산 과정

$a = -3$, $b = +1$을 공식에 대입해요.

한가운데 수 $= \dfrac{(-3) + (+1)}{2} = \dfrac{-3 + 1}{2} = \dfrac{-2}{2} = -1$
수직선으로 확인

$-1$에서 $-3$까지: 2칸 / $-1$에서 $+1$까지: 2칸 → 정확히 가운데예요.

3단계 · Page 3

흔한 실수 짚어봐요

왜 이 단계? — 음수 덧셈에서 실수가 자주 일어나요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
⚠️ 흔한 실수 1 — 음수 덧셈 부호 오류 $(-3) + (+1) = -2$임을 잊지 마세요.
"부호가 다르면 큰 절댓값의 부호 따르기 + 절댓값 차이":
$|-3| = 3$, $|+1| = 1$, 차 $= 2$, 큰 절댓값은 $-3$ 쪽 → 결과는 $-2$
⚠️ 흔한 실수 2 — 두 수를 더하지 않고 빼기 "한가운데니까 $\frac{|-3 - (+1)|}{2} = \frac{4}{2} = 2$?"
거리를 2로 나눈 것은 반거리예요. 출발점에 더해야 해요.
$-3 + 2 = -1$ 또는 $+1 - 2 = -1$ — 어느 방향에서도 $-1$이 나와요.

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
$-1$
($-3$과 $+1$의 한가운데)
전체 풀이 흐름 — 3단계
1
한가운데 수 = 두 수의 평균 = $\frac{a+b}{2}$
2
$\frac{(-3)+(+1)}{2} = \frac{-2}{2} = -1$
3
수직선 확인: $-1$에서 $-3$까지 2칸, $-1$에서 $+1$까지 2칸 → 정확히 가운데
⚠️ 핵심 공식 수직선에서 두 수 $a$, $b$ 사이의 한가운데 수 $= \dfrac{a+b}{2}$
이 공식은 두 수가 음수여도, 분수여도 똑같이 적용돼요.
