깊이 탐구 — 식의 값 & 단항식·다항식

같은 식을 두 관점으로 보기: 값을 계산하는 관점 vs 구조를 읽는 관점

식의 값 계산 전략

1
대입 전 준비: 음수 값은 괄호로 묶어요. $x = -3$이면 $x$ 자리에 $(-3)$ 그대로 씀.
2
거듭제곱 주의: $(-3)^2 = 9$ (양수), $-3^2 = -9$ (음수). 괄호 위치가 결과를 바꿔요.
3
분수 대입: $x = \dfrac{1}{2}$ 같은 분수를 대입할 때도 괄호로! $3 \times \left(\dfrac{1}{2}\right) = \dfrac{3}{2}$
4
두 변수: $x$, $y$ 모두 있으면 각각 따로 대입하고 계산해요.

단항식·다항식 구조 읽기 전략

1
항 나누기: '+', '−'로 나뉘는 각 덩어리가 항이에요. (괄호 안은 하나의 항!)
2
계수 읽기: 문자 앞 숫자가 계수. 보이지 않으면 $1$ 또는 $-1$이에요.
3
차수 세기: 하나의 항에서 문자가 곱해진 총 횟수가 그 항의 차수. ($x^2y$의 차수 = $3$)
4
다항식 차수: 모든 항 중 가장 높은 차수가 그 다항식의 차수.
M — 중간 난이도
M1
$a = -2$, $b = 3$일 때, $a^2 - ab + b$의 값을 구하세요.
$a^2 = (-2)^2 = 4$, $ab = (-2)(3) = -6$이에요. 각 항을 따로 계산!
정답: $13$
$a^2 = (-2)^2 = 4$
$ab = (-2)(3) = -6$
$a^2 - ab + b = 4 - (-6) + 3 = 4 + 6 + 3 = 13$
M2
$3x^2y - 2xy + x - 5$에서 다음을 구하세요.
① $xy$의 계수    ② 다항식의 차수    ③ 상수항
$3x^2y$의 차수 = $2+1 = 3$, $-2xy$의 차수 = $1+1 = 2$이에요.
① $xy$의 계수: $-2$
② 다항식의 차수: $3$ ($3x^2y$의 차수가 가장 높음)
③ 상수항: $-5$
M3
$x = \dfrac{1}{3}$일 때, $9x^2 - 3x + 1$의 값을 구하세요.
$x = \dfrac{1}{3}$을 대입: $x^2 = \left(\dfrac{1}{3}\right)^2 = \dfrac{1}{9}$이에요.
정답: $1$
$9x^2 = 9 \times \left(\dfrac{1}{3}\right)^2 = 9 \times \dfrac{1}{9} = 1$
$3x = 3 \times \dfrac{1}{3} = 1$
$1 - 1 + 1 = 1$
H — 심화 난이도
H1
$x + y = 5$, $xy = 4$일 때, $\dfrac{x}{y} + \dfrac{y}{x}$의 값을 구하세요.
$\dfrac{x}{y} + \dfrac{y}{x}$를 통분하면 $\dfrac{x^2 + y^2}{xy}$가 돼요. $x^2 + y^2 = (x+y)^2 - 2xy$를 이용!
정답: $\dfrac{17}{4}$
$\dfrac{x}{y} + \dfrac{y}{x} = \dfrac{x^2 + y^2}{xy}$
$x^2 + y^2 = (x+y)^2 - 2xy = 25 - 8 = 17$
$xy = 4$
$\dfrac{17}{4}$
H2
$(a^2 + 1)x^3 + (b - 3)x^2 + cx + 2$가 일차식이 되려면 $a$, $b$, $c$에 어떤 조건이 필요한가요?
일차식이 되려면 $x^3$, $x^2$의 계수가 $0$이어야 해요. $x$의 계수는 $0$이 아니어야 해요.
$x^3$ 계수: $a^2 + 1 = 0$ → $a^2 = -1$인데 실수에서 불가능하므로 조건 없음
잠깐 — $a^2 + 1 \geq 1 > 0$이므로 $x^3$ 계수가 절대로 $0$이 될 수 없어요!
따라서 이 식은 어떤 실수 $a$에 대해서도 일차식이 될 수 없어요.
$a^2 \geq 0$이므로 $a^2 + 1 \geq 1 > 0$ — 항상 양수!
H3
$f(x) = 2x^2 - 3x + 1$이라 할 때, $f(2) + f(-1)$의 값을 구하세요.
$f(2)$는 $x = 2$를 대입한 식의 값, $f(-1)$은 $x = -1$을 대입한 값이에요.
정답: $9$
$f(2) = 2(4) - 3(2) + 1 = 8 - 6 + 1 = 3$
$f(-1) = 2(1) - 3(-1) + 1 = 2 + 3 + 1 = 6$
$f(2) + f(-1) = 3 + 6 = 9$

식의 값에서 자주 하는 실수

✗
$(-2)^2 = -4$라고 계산 (음수 제곱을 음수로 착각)
✓
$(-2)^2 = (-2) \times (-2) = 4$ (음수 × 음수 = 양수)
✗
$-2^2 = 4$라고 계산 ($-$를 괄호 안으로 넣음)
✓
$-2^2 = -(2^2) = -4$ (지수는 $2$에만 적용)
✗
음수 대입 시 괄호 생략: $3x$에 $x=-2$ 대입할 때 $3 \times -2$
✓
괄호 필수: $3 \times (-2) = -6$

단항식·다항식 구조에서 자주 하는 실수

✗
$-x$의 계수를 $1$이라고 함
✓
$-x = (-1) \times x$이므로 계수는 $-1$
✗
$2x^2$의 차수를 $4$라고 함 (계수와 지수를 곱함)
✓
차수는 지수만! $2x^2$의 차수 = $2$ (계수 $2$는 관계없음)
✗
$\dfrac{1}{x}$을 다항식이라고 분류
✓
$\dfrac{1}{x} = x^{-1}$: 중1 범위의 다항식이 아니에요 (분모에 문자 불가)

식의 값과 다항식 구조의 관계

같은 식 $2x^2 - 3x + 1$을 두 관점에서 봐요.


구조 관점: 이 식은 3개의 항으로 된 이차 다항식. $x^2$의 계수는 $2$, $x$의 계수는 $-3$, 상수항은 $1$.


계산 관점: $x = 2$를 넣으면 $2(4) - 3(2) + 1 = 3$. 구체적인 수가 나와요.


구조를 알면 → 대입 전에 어떤 항이 클지 예측할 수 있어요!

다음 단원과의 연결

4단원 일차방정식: "이차항의 계수가 $0$인 식" = 일차식 → 방정식이 돼요.


함수 $f(x)$ 표기: 고등학교에서 배울 함수 표기법의 기초. $f(2)$는 "식에 $2$를 대입한 값"이에요.


수학 언어 관점: 알콰리즈미 이전에는 "어떤 수의 제곱에서 그 수를 빼고 1을 더한 것"을 말로 설명해야 했어요. 지금은 $x^2 - x + 1$로 깔끔하게 쓸 수 있죠!
