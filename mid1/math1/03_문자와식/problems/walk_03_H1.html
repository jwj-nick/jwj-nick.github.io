문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 항: 수나 문자의 곱으로 이루어진 식의 단위 (예: 3x, −2y, 5)  ·  계수: 항에서 문자 앞에 곱해진 수 (3x에서 3)  ·  상수항: 문자가 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수가 1인 식
유형 3 · H1

$x + y = 4$이고 $xy = 3$일 때, $x^2 + y^2$의 값을 구하세요.
🤔 잠깐, 손으로 풀어보세요 $x$와 $y$를 직접 구하지 않고 $x^2 + y^2$을 계산할 수 있을까요?
$(x + y)^2$을 전개해보면 힌트가 보여요.
1단계 · Page 1

$(x+y)^2$ 전개 — 핵심 아이디어

왜 이 단계? — $x^2 + y^2$을 직접 계산하기 어려울 때, 이미 아는 값으로 표현하는 방법이에요.
📖 용어 노트 대입: 문자 자리에 수를 넣어 계산하기  ·  식의 값: 대입 후 계산한 결과
$(x+y)^2$ 전개

$(x + y)^2$을 전개하면:

$(x + y)^2 = x^2 + 2xy + y^2$

여기서 $x^2 + y^2$을 꺼내면:

$x^2 + y^2 = (x + y)^2 - 2xy$
💡 아이디어 $(x+y)$와 $xy$는 문제에서 이미 알려줬어요!
$x + y = 4$, $xy = 3$이니까, 바로 대입할 수 있어요.

2단계 · Page 2

알려진 값을 대입해요

왜 이 단계? — 관계식을 발견했으니 이제 대입해서 계산해요. 음수 대입 시 괄호 주의!
📖 용어 노트 대입: 문자 자리에 수를 넣기  ·  사칙연산 순서: 거듭제곱 → 곱셈·나눗셈 → 덧셈·뺄셈
대입
$x^2 + y^2 = (x + y)^2 - 2xy$
$= 4^2 - 2 \times 3$
$= 16 - 6$
$= 10$
⚠️ 사칙연산 순서 주의 $4^2 - 2 \times 3$에서: 거듭제곱 $4^2 = 16$ 먼저, 그 다음 곱셈 $2 \times 3 = 6$, 마지막에 뺄셈!
$4^2 - 2 \times 3 = 16 - 6 = 10$ (O)
$4^2 - 2 \times 3 = (4^2 - 2) \times 3 = 42$ (×)

3단계 · Page 3

실제 값으로 확인해봐요

왜 이 단계? — 공식으로 풀었으니 실제 $x$, $y$ 값을 찾아서 직접 검산해요.
📖 용어 노트 대입: 문자 자리에 수를 넣기  ·  식의 값: 대입 후 계산한 결과
$x + y = 4$, $xy = 3$을 만족하는 수

$x = 1, y = 3$ (또는 $x = 3, y = 1$)이에요.

검증: $x^2 + y^2 = 1^2 + 3^2 = 1 + 9 = 10$ ✓
💡 이 풀이법의 강점 $x = 1, y = 3$처럼 구체적인 값을 찾지 않아도 답을 구할 수 있었어요.
$(x+y)$와 $xy$만 알면 $x^2 + y^2$을 계산할 수 있는 공식 덕분이에요!

정답 + 풀이 흐름 요약

📖 용어 노트 항: 수나 문자의 곱으로 이루어진 단위  ·  계수: 항에서 문자 앞에 곱해진 수  ·  상수항: 문자가 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수가 1인 식
$x^2 + y^2 = 10$
($= 4^2 - 2 \times 3 = 16 - 6$)
전체 풀이 흐름 — 3단계
1
$(x+y)^2 = x^2 + 2xy + y^2$ 전개 → $x^2 + y^2 = (x+y)^2 - 2xy$
2
$x+y = 4$, $xy = 3$ 대입 → $4^2 - 2 \times 3 = 16 - 6 = 10$
3
검증: $x=1, y=3$ → $1 + 9 = 10$ ✓
⚠️ 이 유형의 핵심 함정 $x$와 $y$를 개별적으로 구하려고 하면 시간이 많이 걸려요.
$(x+y)^2 - 2xy$ 관계식을 이용하면 훨씬 빠르게 풀 수 있어요!
