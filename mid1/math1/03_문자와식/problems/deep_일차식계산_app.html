깊이 탐구 — 일차식의 계산

동류항 정리 → 덧셈·뺄셈 → 곱셈·나눗셈. 세 가지 계산이 하나의 흐름으로 연결돼요.

통합 계산 흐름 — 이 순서로!

괄호 전개→동류항 모으기→계수끼리 계산
괄호 전개 시 핵심 규칙:
· 덧셈 괄호 $(+)$: 부호 그대로
· 뺄셈 괄호 $(-)$: 안의 부호 모두 반전
· 계수 곱셈 $k(\cdots)$: 분배법칙으로 각 항에 $k$ 곱하기

동류항 정리 핵심

1
동류항 = 문자와 차수 모두 같은 항 ($3x$와 $5x$ → 같음, $3x$와 $3x^2$ → 다름)
2
동류항끼리는 계수만 더하고 빼요 — $3x + 5x = (3+5)x = 8x$
3
다른 종류의 항은 절대 합칠 수 없어요 — $2x + 3$은 $5x$가 아니에요!

분수 계수 계산 핵심

1
$\div k$는 $\times \dfrac{1}{k}$로 바꿔 분배법칙 적용
2
분모가 다른 분수 계수의 합은 통분 후 계산
3
$\dfrac{1}{2}(4x - 6) = 2x - 3$ (약분되는 경우 깔끔하게)
M — 중간 난이도 (동류항 + 덧뺄셈 + 곱나눗셈 통합)
M1
$2(3x - 1) - 3(x + 2) + 4$를 계산하세요.
각 괄호에 앞 계수를 분배하고, 상수 $4$는 상수항으로 동류항 정리에 포함해요.
정답: $3x - 4$
$6x - 2 - 3x - 6 + 4$
$= (6-3)x + (-2 - 6 + 4)$
$= 3x - 4$
M2
$\dfrac{x + 2}{3} - \dfrac{2x - 1}{6}$을 계산하세요.
분모 3과 6의 최소공배수는 6이에요. $\dfrac{x+2}{3} = \dfrac{2(x+2)}{6}$으로 변환해요.
정답: $\dfrac{5}{6}$
$\dfrac{2(x+2)}{6} - \dfrac{2x-1}{6} = \dfrac{2x+4-2x+1}{6} = \dfrac{5}{6}$
$x$가 사라지고 상수만 남아요!
M3
$A = 3x - 2$, $B = x + 4$일 때, $\dfrac{1}{2}(4A - 2B)$를 계산하세요.
먼저 $4A - 2B$를 계산하고, 그 결과에 $\dfrac{1}{2}$를 곱해요.
정답: $5x - 8$
$4A - 2B = 4(3x-2) - 2(x+4) = 12x - 8 - 2x - 8 = 10x - 16$
$\dfrac{1}{2}(10x - 16) = 5x - 8$
H — 심화 난이도 (복합 계산)
H1
$3\{2x - (x - 3)\} - 4(x + 2)$를 계산하세요.
안쪽 소괄호부터: $(x - 3)$을 먼저 처리 → 중괄호 전개 → 분배 → 동류항 정리
정답: $-x + 1$
$3\{2x - x + 3\} - 4x - 8$
$= 3\{x + 3\} - 4x - 8$
$= 3x + 9 - 4x - 8$
$= -x + 1$
H2
어떤 일차식에 $2$를 곱하면 $6x - 4$가 되고, 다른 일차식에서 $x + 1$을 빼면 $3x - 2$가 돼요.
두 일차식의 합을 구하세요.
첫 번째 일차식: $6x - 4$를 $2$로 나눠요. 두 번째 일차식: $3x - 2$에 $x + 1$을 더해요.
정답: $7x - 3$
첫 번째 일차식: $(6x - 4) \div 2 = 3x - 2$
두 번째 일차식: $(3x - 2) + (x + 1) = 4x - 1$
합: $(3x - 2) + (4x - 1) = 7x - 3$
H3
$\dfrac{4x - 3}{2} - \dfrac{3x + 1}{4} = ax + b$일 때, $4a - b$의 값을 구하세요.
먼저 좌변을 통분해서 계산하고 $ax + b$ 형태로 만들어요. 분모 2와 4의 최소공배수는 4.
정답: $4a - b = 4 \times \dfrac{5}{4} - \left(-\dfrac{7}{4}\right) = 5 + \dfrac{7}{4} = \dfrac{27}{4}$

계산 과정:
$\dfrac{2(4x-3)}{4} - \dfrac{3x+1}{4} = \dfrac{8x - 6 - 3x - 1}{4} = \dfrac{5x - 7}{4}$
$a = \dfrac{5}{4}$, $b = -\dfrac{7}{4}$
$4a - b = 4 \times \dfrac{5}{4} - \left(-\dfrac{7}{4}\right) = 5 + \dfrac{7}{4} = \dfrac{27}{4}$

동류항 정리에서 자주 하는 실수

✗
$2x + 3 = 5x$로 계산 (다른 종류의 항을 더함)
✓
$2x + 3$은 더 간단히 할 수 없어요. $x$항과 상수항은 다른 종류!
✗
$x^2 + x = 2x^2$으로 계산 (차수가 다른 항을 더함)
✓
$x^2$과 $x$는 차수가 달라서 합칠 수 없어요.

괄호 전개에서 자주 하는 실수

✗
$(4x + 2) - (x - 3) = 4x + 2 - x - 3 = 3x - 1$
✓
$-(x - 3) = -x + 3$ → $4x + 2 - x + 3 = 3x + 5$
✗
$-2(x + 1) = -2x + 2$ (마이너스 분배 오류)
✓
$-2(x + 1) = -2x - 2$ (모든 항에 $-2$ 곱하기)

분수 계산에서 자주 하는 실수

✗
$\dfrac{x+1}{2} + \dfrac{x+3}{4} = \dfrac{2x+4}{6}$ (분모를 더함)
✓
통분해서 분모를 같게 만들고 분자를 더해요.
✗
$\dfrac{2x+4}{4} - \dfrac{x+1}{4} = \dfrac{2x+4-x+1}{4}$ (빼기 분자 처리 오류)
✓
$= \dfrac{(2x+4)-(x+1)}{4} = \dfrac{x+3}{4}$ (분자 전체를 빼야 해요!)

동류항 → 덧뺄셈 → 곱나눗셈의 연결

동류항 정리는 모든 계산의 마지막 단계예요.
덧셈·뺄셈을 하면 괄호를 풀고 → 동류항 모아서 정리.
곱셈·나눗셈을 하면 분배법칙 → 동류항 모아서 정리.
결국 모든 계산의 끝은 동류항 정리예요!

4단원 일차방정식으로 연결

방정식 $2(x - 1) = 3x + 5$를 풀 때,
좌변 전개: $2x - 2 = 3x + 5$
이항하면: $2x - 3x = 5 + 2$
동류항 정리: $-x = 7$
이 모든 과정이 이 단원에서 배운 계산이에요!
