유형 7 — 일차식 × 수 / ÷ 수

분배법칙 + 분수 계수 통분까지
핵심 공식
곱셈: $k(ax + b) = kax + kb$ (분배법칙)
나눗셈: $(ax + b) \div k = \dfrac{ax + b}{k} = \dfrac{a}{k}x + \dfrac{b}{k}$
             또는 역수 곱하기: $(ax + b) \times \dfrac{1}{k}$
음수 곱: $-(ax + b) = -ax - b$ (안의 부호 모두 반전)
1
L
$4(3x - 2)$를 전개하세요.
$4$를 괄호 안 각 항에 곱해요.
정답: $12x - 8$
$4 \times 3x + 4 \times (-2) = 12x - 8$
2
L
$-3(2x - 5)$를 전개하세요.
$-3$을 각 항에 곱해요. 마이너스 × 마이너스는 플러스!
정답: $-6x + 15$
$(-3) \times 2x + (-3) \times (-5) = -6x + 15$
$(-3) \times (-5) = +15$. 음수 × 음수 = 양수!
3
L
$(8x - 4) \div 2$를 계산하세요.
$\div 2$는 $\times \dfrac{1}{2}$와 같아요. 각 항을 $2$로 나눠요.
정답: $4x - 2$
$(8x - 4) \div 2 = \dfrac{8x}{2} - \dfrac{4}{2} = 4x - 2$
1
M
$\dfrac{1}{2}(6x - 4) - 3(x + 1)$을 계산하세요.
두 부분을 각각 분배하고 나서 동류항을 정리해요.
정답: $-5$
$\dfrac{1}{2}(6x-4) = 3x - 2$
$3(x+1) = 3x + 3$
$(3x - 2) - (3x + 3) = 3x - 2 - 3x - 3 = -5$
2
M
$\dfrac{2x + 1}{3} + \dfrac{x - 2}{3}$를 계산하세요.
분모가 같으면 분자를 그대로 더할 수 있어요.
정답: $x - \dfrac{1}{3}$
$\dfrac{(2x+1)+(x-2)}{3} = \dfrac{3x - 1}{3} = x - \dfrac{1}{3}$
3
M
$\dfrac{a}{2} + \dfrac{a}{3}$을 계산하세요.
분모가 다를 때는 통분해요. 2와 3의 최소공배수는 6이에요.
정답: $\dfrac{5a}{6}$
$\dfrac{a}{2} + \dfrac{a}{3} = \dfrac{3a}{6} + \dfrac{2a}{6} = \dfrac{5a}{6}$
1
H
$\dfrac{3x - 1}{4} - \dfrac{2x + 3}{6}$을 계산하세요.
4와 6의 최소공배수는 12예요. 두 분수를 분모 12로 통분해요.
정답: $\dfrac{5x - 9}{12}$
$= \dfrac{3(3x-1)}{12} - \dfrac{2(2x+3)}{12}$
$= \dfrac{9x - 3 - (4x + 6)}{12}$
$= \dfrac{9x - 3 - 4x - 6}{12}$
$= \dfrac{5x - 9}{12}$
2
H
$2(3x - 1) - 3(x - 2) = ax + b$ 일 때, $a + b$의 값을 구하세요.
먼저 좌변을 전개하고 정리해서 $ax + b$ 형태로 만들어요.
정답: $a + b = 3 + 4 = 7$
$2(3x-1) - 3(x-2)$
$= 6x - 2 - 3x + 6$
$= 3x + 4$
$a = 3$, $b = 4$ → $a + b = 7$
3
H
$4(2x - 1) - k(x - 3) = 5x + 5$를 만족하는 상수 $k$를 구하세요.
좌변을 전개해서 $x$의 계수와 상수항을 우변과 맞춰요.
정답: $k = 3$
$8x - 4 - kx + 3k = 5x + 5$
$x$항: $(8 - k) = 5$ → $k = 3$
확인 — 상수항: $-4 + 3k = -4 + 9 = 5$ ✓
