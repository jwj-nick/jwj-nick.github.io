문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
유형 4 · H1

$-\dfrac{5}{2} < x < \dfrac{7}{3}$을 만족하는 정수 $x$의 개수를 구하세요.
🤔 잠깐, 손으로 풀어보세요 분수를 소수로 바꾸면 범위가 어디서 어디까지인지 보여요. 그 범위 안 정수를 찾아봐요.
1단계 · Page 1

분수를 소수로 바꿔서 범위 파악해요

왜 이 단계? — 분수로는 정수와 비교가 어려워요. 소수나 분수 직접 비교로 범위를 명확히 해요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수
분수 → 소수 변환
$-\dfrac{5}{2} = -2.5$

$+\dfrac{7}{3} = +2.333\ldots \approx +2.33$

범위: $-2.5 < x < 2.33$ (정수 $x$)

경계 정수 확인

왼쪽 경계 $-2.5$보다 큰 첫 번째 정수: $-2$
($-3 < -2.5$이므로 $-3$은 범위 밖)

오른쪽 경계 $2.33$보다 작은 마지막 정수: $+2$
($3 > 2.33$이므로 $+3$은 범위 밖)

2단계 · Page 2

정수 나열하고 개수 세어요

왜 이 단계? — 범위 안의 정수를 모두 나열하고 빠짐없이 세어요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수
$-2.5 < x < 2.33$인 정수 나열
$-2, \; -1, \; 0, \; +1, \; +2$

개수: 5개

⚠️ 경계 처리 주의 $-\frac{5}{2} = -2.5$이고, 부등호가 $<$(초과)이므로 $-2.5$ 자체는 포함 안 해요.
그런데 $-2.5$는 정수가 아니므로 어차피 정수 중 경계는 $-2$예요.
만약 $-\frac{5}{2} = -2.5$ 대신 $-2$가 경계였다면 $-2 < x$이므로 $-2$ 제외 주의!

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
5개
($-2, -1, 0, +1, +2$)
전체 풀이 흐름 — 2단계
1
분수 → 소수: $-\frac{5}{2} = -2.5$, $\frac{7}{3} \approx 2.33$ → 범위: $-2.5 < x < 2.33$
2
범위 내 정수: $-2, -1, 0, +1, +2$ → 5개
⚠️ 핵심 전략 분수 범위의 정수 개수 문제는:
① 분수를 소수로 변환 (또는 인근 정수와 직접 비교)
② 범위 내 정수를 순서대로 나열
③ 개수 확인
