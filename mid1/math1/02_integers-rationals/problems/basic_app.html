Q1 수 분류
다음 수 $-7$에 대한 설명으로 옳지 않은 것을 고르세요.
자연수는 1, 2, 3, … 양의 정수예요. −7은 0보다 작으므로 자연수가 아니에요.
Q2 수직선
아래 수직선에서 점 A가 나타내는 수를 쓰세요.
수직선에서 0을 기준으로 왼쪽으로 몇 칸인지 세어봐요.
Q3 유리수 판별
다음 중 정수가 아닌 유리수를 고르세요.
정수가 아닌 유리수는 분수나 소수처럼 정수와 정수 사이에 있는 수예요. $\frac{3}{4}$은 0과 1 사이에 있어요.
Q4 절댓값
$|-5|$ 의 값을 구하세요.
절댓값은 수직선에서 0까지의 거리예요. 부호와 관계없이 양수예요.
Q5 절댓값 조건
절댓값이 $3$인 수를 모두 쓰세요. (쉼표로 구분)
수직선에서 0으로부터 거리가 3인 수는 몇 개일까요?
Q6 대소 관계
다음 수를 작은 것부터 차례로 나열하세요.

$$+3, \quad 0, \quad -5, \quad -2$$
수직선에서 왼쪽에 있을수록 작아요. 음수끼리는 절댓값이 클수록 작아요.
Q7 덧셈
다음을 계산하세요.
$$( +3 ) + ( -5 )$$
다른 부호끼리 더할 때: 절댓값의 차를 구하고, 절댓값이 더 큰 쪽의 부호를 붙여요.
Q8 뺄셈→덧셈
다음을 계산하세요.
$$( -3 ) - ( +4 )$$
뺄셈 → 덧셈: $A - B = A + (-B)$. 빼는 수 +4의 부호를 −4로 바꿔 더해봐요.
Q9 곱셈
다음을 계산하세요.
$$( -3 ) \times ( -4 )$$
음수 × 음수 = 양수. 절댓값끼리 곱하면 3×4=12, 부호는 +.
Q10 혼합계산
다음을 계산하세요.
$$(-3) + (+2) \times (-2) - (-1)$$
순서: 곱셈 먼저 → $(+2) \times (-2) = -4$
그러면: $(-3) + (-4) - (-1)$
뺄셈 → 덧셈: $(-3) + (-4) + (+1)$
마지막으로 더해봐요!
