Unit 02 · 깊이 탐구

절댓값 & 대소 관계

어려운 이유를 알고, 핵심 전략으로 정복해요

이 유형이 어려운 이유

절댓값 기호 $|\cdot|$은 중1에서 처음 만나는 기호예요. 기호가 낯설어서 실수하기 쉬워요.

흔한 실수 패턴
• $|{-5}| = -5$라고 쓰는 경우 (절댓값은 항상 0 이상!)
• $a < 0$일 때 $|a| = a$라고 혼동 (실제로는 $|a| = -a$)
• $|a| = k$인 수가 1개라고 생각 (실제로는 $+k$와 $-k$ 두 개)

핵심 아이디어

절댓값 = 수직선에서 0까지의 거리
거리는 방향이 없어요. 항상 0 이상!

$|a| = \begin{cases} a & (a \geq 0) \\ -a & (a < 0) \end{cases}$
$|a| = k$ (k > 0)이면 a = +k 또는 a = −k
0에서 오른쪽으로 k, 왼쪽으로 k — 두 개!
$|a| < k$이면 $-k < a < k$
절댓값 부등식은 범위로 변환해요.

단계별 풀이 흐름

1
숫자 부호 확인: a가 양수인지 음수인지 먼저 판단
2
규칙 적용: 양수/0이면 그대로, 음수면 부호 반대
3
조건 문제: |a|=k이면 a=±k 두 개 나열
4
부등식 변환: |a|<k → -k<a<k 범위로 변환 후 정수 나열

함께 풀어봐요 (클릭으로 단계 공개)

M $|a| \leq 3$을 만족하는 정수 $a$를 모두 구하고, 그 합을 구하세요.
풀이:
|a| ≤ 3이면 −3 ≤ a ≤ 3으로 범위를 변환해요.
이 범위의 정수: −3, −2, −1, 0, +1, +2, +3 (7개)
합: (−3)+(−2)+(−1)+0+1+2+3 = 0
(음수와 양수가 짝을 이뤄 상쇄돼요!)
정답: 합 = 0
H 절댓값이 $2$ 이상이고 $5$ 미만인 정수의 개수를 구하세요.
풀이:
조건: $2 \leq |a| < 5$
이를 범위로: $a \leq -2$ 또는 $a \geq 2$, 그리고 $-5 < a < 5$
결합: $-5 < a \leq -2$ 또는 $2 \leq a < 5$
정수 목록:
• $-5 < a \leq -2$: $-4, -3, -2$ (3개)
• $2 \leq a < 5$: $2, 3, 4$ (3개)
총 6개
정답: 6개
H $a < 0 < b$이고 $|a| > |b|$일 때, $a + b$의 부호를 결정하세요.
풀이:
$a < 0$: a는 음수, $b > 0$: b는 양수
$|a| > |b|$: a의 절댓값이 b의 절댓값보다 커요
→ 음수 쪽이 더 크므로 $a + b < 0$ (음수)

예: a = −5, b = +3이면 |a|=5 > |b|=3. a+b = −2 < 0
정답: a + b는 음수 (부호: −)

이 유형을 이해하면 무엇이 쉬워지나요?

절댓값 개념은 → 중2 방정식에서 $|x-a|=b$ 형태 문제의 기초예요.
또한 실생활의 "오차 범위"(±0.5cm 등)를 수학으로 표현하는 방법이기도 해요.

이 유형이 어려운 이유

음수끼리 비교할 때 직관과 반대가 돼서 헷갈려요.

흔한 실수 패턴
• $-5$와 $-3$ 비교 시 "5가 크니까 −5가 크다"는 실수
• 분수끼리 비교 시 통분 없이 숫자만 비교하는 실수
• $-\frac{3}{4}$과 $-\frac{2}{3}$: "분자만 보면 −3이 작아 보이지만 실제로는 −3/4 < −2/3"

핵심 아이디어

수직선에서 오른쪽 = 큰 수
양수 > 0 > 음수 (항상!)
음수끼리 비교: 절댓값이 클수록 더 작다
$-5 < -3$ (왜냐면 −5가 더 왼쪽)
분수 비교: 통분해서 분자 비교
$-\frac{3}{4}$과 $-\frac{2}{3}$: 통분하면 $-\frac{9}{12}$, $-\frac{8}{12}$ → $-\frac{9}{12} < -\frac{8}{12}$

단계별 풀이 흐름

1
부호 먼저: 양수 vs 음수 → 양수가 무조건 큼
2
같은 부호끼리: 절댓값 비교. 음수는 반대!
양수: 절댓값 큰 것이 큼. 음수: 절댓값 큰 것이 작음.
3
분수: 통분 후 분자 비교
4
범위 조건: $a \leq x \leq b$ → 정수 하나씩 나열

함께 풀어봐요 (클릭으로 단계 공개)

M 다음 세 수를 작은 것부터 순서대로 나열하세요.
$-\dfrac{2}{3}, \quad +\dfrac{1}{2}, \quad -\dfrac{3}{4}$
풀이:
+1/2은 양수이므로 두 음수보다 무조건 커요.
음수끼리 비교: −2/3과 −3/4
통분(12분모): −2/3 = −8/12, −3/4 = −9/12
분자 비교(음수이므로 분자 작은 것이 더 작다): −9/12 < −8/12
→ −3/4 < −2/3
최종 순서: $-\dfrac{3}{4} < -\dfrac{2}{3} < +\dfrac{1}{2}$
정답: $-\dfrac{3}{4} < -\dfrac{2}{3} < +\dfrac{1}{2}$
M $-3 \leq x < 2$이면서 $|x| > 1$을 동시에 만족하는 정수 $x$를 모두 구하세요.
풀이:
조건 1: $-3 \leq x < 2$ → 정수: −3, −2, −1, 0, 1
조건 2: $|x| > 1$ → $x < -1$ 또는 $x > 1$
두 조건 모두 만족: −3, −2 (x < -1이고 x ≥ -3)
정답: −3, −2
H $a, b$가 정수이고 $|a| = 3$, $|b| = 5$, $a > b$일 때, $a - b$의 값을 구하세요. (단, a, b는 같은 부호가 아님)
풀이:
|a|=3이면 a = +3 또는 a = −3
|b|=5이면 b = +5 또는 b = −5
조건: a > b이고 다른 부호
경우의 수: (a=+3, b=−5) → a>b ✓, 다른부호 ✓
(a=−3, b=+5) → a < b ✗
따라서 a = +3, b = −5
$a - b = (+3) - (-5) = (+3) + (+5) = +8$
정답: a − b = +8

이 유형을 이해하면 무엇이 쉬워지나요?

대소 관계는 → 중1 후반 방정식의 해 범위 파악, 중2 부등식 단원의 직접적 기초예요.
또한 온도 비교, 고도 비교, 재정 비교 등 실생활 문제에서 매우 자주 등장해요.
