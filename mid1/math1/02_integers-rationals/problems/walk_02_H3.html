문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
유형 2 · H3

수직선 위의 두 점 A, B에 대해 A는 $-\dfrac{3}{2}$, B는 $+\dfrac{5}{2}$에 있어요.
A, B 사이의 거리를 구하세요.
🤔 잠깐, 손으로 풀어보세요 두 점 사이의 거리 공식을 떠올려봐요. 분수가 나와도 공식은 똑같이 적용해요.
1단계 · Page 1

두 점 사이의 거리 공식을 복습해요

왜 이 단계? — 수직선 위 두 점의 거리는 절댓값으로 계산해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
두 점 사이의 거리 공식

수직선 위에서 좌표 $a$와 $b$인 두 점 사이의 거리:

거리 $= |b - a|$  (또는 $|a - b|$, 절댓값이니 순서 무관)

이유: 큰 수에서 작은 수를 빼면 양수가 나오고, 그게 곧 거리예요.
어느 쪽이 큰지 모를 때는 절댓값을 씌우면 안전해요.

💡 확인 $a = -3$, $b = +2$이면 거리 $= |(+2) - (-3)| = |2 + 3| = |5| = 5$

2단계 · Page 2

대입해서 계산해요

왜 이 단계? — 분수 뺄셈과 절댓값 처리가 이 문제의 핵심이에요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
계산 과정

A $= -\dfrac{3}{2}$, B $= +\dfrac{5}{2}$

거리 $= \left|B - A\right| = \left|\frac{5}{2} - \left(-\frac{3}{2}\right)\right|$

뺄셈을 덧셈으로 변환해요:

$= \left|\frac{5}{2} + \frac{3}{2}\right| = \left|\frac{5+3}{2}\right| = \left|\frac{8}{2}\right| = |4| = 4$

3단계 · Page 3

검증하고 함정 확인해요

왜 이 단계? — 음수를 빼면 덧셈이 된다는 것, 확실히 기억해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
수직선으로 직관 확인

$-\dfrac{3}{2} = -1.5$, $+\dfrac{5}{2} = +2.5$

$-1.5$부터 $+2.5$까지의 거리: $2.5 - (-1.5) = 2.5 + 1.5 = 4$ ✓

⚠️ 흔한 실수 — 음수 빼기 처리 $-\left(-\dfrac{3}{2}\right) = +\dfrac{3}{2}$임을 꼭 기억해요.
음수 부호가 두 개 겹치면 양수가 돼요: $-(-3) = +3$
마찬가지로 $\frac{5}{2} - (-\frac{3}{2}) = \frac{5}{2} + \frac{3}{2}$예요.

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
거리 $= 4$
($-\frac{3}{2}$에서 $+\frac{5}{2}$까지)
전체 풀이 흐름 — 3단계
1
두 점 거리 공식: 거리 $= |B - A|$
2
$\left|\frac{5}{2} - \left(-\frac{3}{2}\right)\right| = \left|\frac{5}{2} + \frac{3}{2}\right| = \left|\frac{8}{2}\right| = 4$
3
수직선 직관 확인: $2.5 - (-1.5) = 4$ ✓
⚠️ 핵심 정리 수직선 거리 = $|큰 수 - 작은 수|$ = $|작은 수 - 큰 수|$ (어느 쪽이든 절댓값으로 같아요)
음수 빼기 법칙: $a - (-b) = a + b$ — 이중 부호 처리가 핵심이에요.
