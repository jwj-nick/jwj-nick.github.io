문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
유형 5 · H1

계산하세요. $$\left(-\frac{5}{6}\right) - \left(-\frac{2}{3}\right) + \left(+\frac{1}{2}\right)$$
🤔 잠깐, 손으로 풀어보세요 뺄셈을 덧셈으로 바꾸고, 통분해서 계산해봐요. 공통 분모를 먼저 찾아봐요.
1단계 · Page 1

뺄셈을 덧셈으로 변환해요

왜 이 단계? — 뺄셈은 "빼는 수의 부호를 바꿔서 더하기"로 변환해요. 그래야 계산이 편해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
뺄셈 → 덧셈 변환 규칙

$A - B = A + (-B)$  →  빼는 수의 부호를 반전하여 더해요.

$\left(-\dfrac{5}{6}\right) - \left(-\dfrac{2}{3}\right) + \left(+\dfrac{1}{2}\right)$

$= \left(-\dfrac{5}{6}\right) + \left(+\dfrac{2}{3}\right) + \left(+\dfrac{1}{2}\right)$

$-\left(-\dfrac{2}{3}\right) = +\dfrac{2}{3}$으로 변환됐어요.

⚠️ 이중 부호 처리 $-(-\frac{2}{3}) = +\frac{2}{3}$ — 음수 부호 두 개가 만나면 양수예요!
이 변환이 이 문제의 핵심이에요.

2단계 · Page 2

통분해서 분모를 맞춰요

왜 이 단계? — 분모가 다른 분수들의 덧셈은 통분해야 계산할 수 있어요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
공통 분모 찾기

분모: 6, 3, 2 → 최소공배수 = 6

$-\dfrac{5}{6}$ (분모이미 6)  →   $-\dfrac{5}{6}$

$+\dfrac{2}{3} = +\dfrac{4}{6}$

$+\dfrac{1}{2} = +\dfrac{3}{6}$
계산
$-\dfrac{5}{6} + \dfrac{4}{6} + \dfrac{3}{6} = \dfrac{-5 + 4 + 3}{6} = \dfrac{2}{6} = \dfrac{1}{3}$

3단계 · Page 3

검증하고 함정 확인해요

왜 이 단계? — 분자 계산 $-5+4+3$에서 실수가 흔해요. 단계별로 확인해요.
📖 용어 노트 절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)
분자 계산 단계별 확인
$-5 + 4 = -1$
$-1 + 3 = +2$
$\Rightarrow \dfrac{2}{6} = \dfrac{1}{3}$
⚠️ 흔한 실수 — 분자 계산 순서 $-5 + 4 + 3$을 한 번에 계산하다 부호 실수가 날 수 있어요.
양수끼리 먼저: $4 + 3 = 7$. 그 다음: $-5 + 7 = 2$. 이 방법도 좋아요.

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
$\dfrac{1}{3}$
($-\frac{5}{6} + \frac{4}{6} + \frac{3}{6} = \frac{2}{6} = \frac{1}{3}$)
전체 풀이 흐름 — 3단계
1
뺄셈→덧셈: $-(-\frac{2}{3}) = +\frac{2}{3}$으로 변환
2
통분 (분모 6): $-\frac{5}{6} + \frac{4}{6} + \frac{3}{6}$
3
분자 계산: $-5+4+3=2$ → $\frac{2}{6} = \frac{1}{3}$
⚠️ 핵심 2가지 1. 뺄셈 → 덧셈 변환 시 이중 부호 처리: $-(-a) = +a$
2. 통분 후 분자끼리만 계산 (분모는 건드리지 않아요)
