유형 03 — 절댓값

|a| 계산·조건·부등식 · L×3 M×3 H×3

L-1 쉬움
$|{-8}|$의 값을 구하세요.
L-2 쉬움
절댓값이 $\dfrac{5}{2}$인 수를 모두 쓰세요. (쉼표 구분)
0에서 5/2만큼 떨어진 수는 오른쪽과 왼쪽 두 개예요.
L-3 쉬움
다음 중 절댓값이 가장 큰 수를 고르세요.
$+2, \quad -5, \quad 0, \quad +4, \quad -3$
각 수의 절댓값: |+2|=2, |−5|=5, |0|=0, |+4|=4, |−3|=3
M-1 중간
$|a| = 4$일 때, $a$의 값을 모두 구하고 두 수의 합을 구하세요.
|a|=k이면 a=+k 또는 a=−k예요. 두 수는 크기가 같고 부호가 반대예요.
M-2 중간
$|a| < 3$을 만족하는 정수 $a$를 모두 구하세요. (쉼표 구분)
|a|<3이면 −3 < a < 3이에요. 이 범위의 정수를 나열해봐요.
M-3 중간
다음 중 옳지 않은 것을 고르세요.
|−a|는 항상 양수예요. −|a|는 음수예요. 둘은 같을 수 없어요 (a≠0일 때).
H-1 어려움
$|a| \leq 4$이고 $a$가 음의 정수일 때, 모든 $a$의 합을 구하세요.
|a|≤4이면 −4≤a≤4. 이 중 음의 정수만 선택해봐요.
H-2 어려움
$a < 0$일 때, $|a|$와 $-a$의 관계를 설명하고, $a = -3$일 때 $|a| + (-a)$의 값을 구하세요.
a<0일 때 |a|=−a예요. 즉 |a|와 −a는 같아요. a=−3이면 −a=+3.
H-3 어려움
두 수 $a = -\dfrac{7}{3}$, $b = +\dfrac{5}{3}$에 대해 $|a| - |b|$의 값을 구하세요.
$|a| = \frac{7}{3}$, $|b| = \frac{5}{3}$. 이제 빼봐요.
