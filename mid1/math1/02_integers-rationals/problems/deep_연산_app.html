Unit 02 · 깊이 탐구

덧셈·뺄셈 & 곱셈·나눗셈

부호 처리의 핵심 전략을 익혀요

이 유형이 어려운 이유

흔한 실수 패턴
• 뺄셈을 덧셈으로 바꿀 때 부호 실수: $(-3)-(+4)$를 $(-3)+4$로 쓰는 경우 (올바른 변환: $(-3)+(-4)$)
• 다른 부호 덧셈에서 "큰 쪽" 판단 실수: $(+3)+(-5)$에서 절댓값 차 = 2, 큰 쪽(−5)의 부호 = −
• 혼합 계산에서 음수끼리 합산 빠뜨리기

핵심 아이디어

뺄셈 → 덧셈 변환이 핵심!
$A - B = A + (-B)$ : 빼는 수의 부호를 반대로 바꿔요
덧셈 두 가지 경우
같은 부호: 절댓값의 합, 공통 부호
다른 부호: 절댓값의 차, 절댓값 큰 쪽의 부호
혼합 계산 전략
① 모든 뺄셈 → 덧셈 변환
② 양수끼리 합산, 음수끼리 합산
③ 두 합을 최종 계산

함께 풀어봐요

M 계산하세요.
$$(-5) + (+8) - (-3) + (-6)$$
① 뺄셈 → 덧셈: $(-5) + (+8) + (+3) + (-6)$
② 양수끼리: $+8 + (+3) = +11$
③ 음수끼리: $(-5) + (-6) = -11$
④ 최종: $(+11) + (-11) = 0$
정답: 0
M $a = -\dfrac{1}{2}$, $b = +\dfrac{3}{4}$일 때, $a + b - (-a)$의 값을 구하세요.
① $-(-a) = +a = -\dfrac{1}{2}$
② $a + b + a = 2a + b = 2 \times (-\dfrac{1}{2}) + \dfrac{3}{4} = -1 + \dfrac{3}{4} = -\dfrac{1}{4}$
정답: $-\dfrac{1}{4}$
H $x - (-3) = -2$일 때, $x + 5$의 값을 구하세요.
① $x - (-3) = x + 3 = -2$
② $x = -2 - 3 = -5$
③ $x + 5 = (-5) + 5 = 0$
정답: 0

이 유형을 이해하면 무엇이 쉬워지나요?

정수 덧셈·뺄셈은 → 중1 일차방정식의 이항(이항 = 뺄셈을 덧셈으로 바꿔 이동)의 직접 기초예요.
또한 유리수 연산은 중2 연립방정식에서도 핵심이에요.

이 유형이 어려운 이유

흔한 실수 패턴
• 음수 개수 홀짝 판단 실수: $(−2)×(−3)×(−1)$에서 음수 3개(홀수) → 음수임을 놓침
• 역수 계산 오류: $-\dfrac{3}{4}$의 역수를 $\dfrac{4}{3}$(부호 빠뜨림)으로 쓰는 경우
• 나눗셈 → 역수 변환 미적용: $6 \div \dfrac{2}{3}$을 $6 × \dfrac{2}{3} = 4$로 계산하는 오류

핵심 아이디어

부호 결정 규칙 (음수 개수)
음수 개수 짝수 → 결과 양수 (+)
음수 개수 홀수 → 결과 음수 (−)
역수: 분자↔분모 교환, 부호 유지
$-\dfrac{3}{4}$의 역수 = $-\dfrac{4}{3}$
나눗셈 → 역수 곱셈
$A \div B = A \times \dfrac{1}{B}$
분배법칙 역으로 활용
$a \times b + a \times c = a \times (b + c)$
공통인수를 묶으면 계산이 빨라져요!

함께 풀어봐요

M 계산하세요. $$\left(-\frac{3}{4}\right) \div \left(-\frac{9}{8}\right)$$
① 역수 변환: $(-\dfrac{3}{4}) \times (-\dfrac{8}{9})$
② 부호: (−) × (−) = (+)
③ 절댓값: $\dfrac{3}{4} \times \dfrac{8}{9} = \dfrac{24}{36} = \dfrac{2}{3}$
④ 결과: $+\dfrac{2}{3}$
정답: $+\dfrac{2}{3}$
M 분배법칙을 이용해 계산하세요.
$$99 \times (-12) + (-12)$$
공통인수 묶기: $(-12) \times 99 + (-12) \times 1 = (-12) \times (99 + 1) = (-12) \times 100 = -1200$
직접 계산보다 훨씬 빠르죠!
정답: −1200
H $(-2)^2 \times (-3) \div \left(-\dfrac{1}{2}\right)^3$을 계산하세요.
① 거듭제곱 먼저:
$(-2)^2 = +4$ (짝수 지수 → 양수)
$(-\dfrac{1}{2})^3 = -\dfrac{1}{8}$ (홀수 지수 → 음수)
② 나눗셈 → 역수:
$+4 \times (-3) \times (-8)$
(왜냐면 $\div (-\dfrac{1}{8}) = \times (-8)$)
③ 부호: 음수 개수: (−3), (−8) → 2개(짝수) → 양수
④ 절댓값: $4 \times 3 \times 8 = 96$
⑤ 결과: $+96$
정답: +96

이 유형을 이해하면 무엇이 쉬워지나요?

곱셈·나눗셈 부호 규칙은 → 중1 문자와 식(단원 3)에서 계수 계산의 직접 기초예요.
분배법칙은 → 일차식 전개와 인수분해의 핵심이에요.
