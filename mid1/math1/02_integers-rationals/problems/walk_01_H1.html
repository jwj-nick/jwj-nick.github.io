문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수 ($a$의 역수 $= \frac{1}{a}$)
유형 1 · H1

다음 설명 중 옳은 것의 개수를 구하세요.

(ㄱ) 양수와 음수는 모두 유리수이다.
(ㄴ) 0은 양수도 음수도 아니므로 유리수가 아니다.
(ㄷ) 자연수는 모두 정수이다.
(ㄹ) 음의 유리수 중에 정수인 것이 있다.
(ㅁ) 모든 정수는 자연수이다.
🤔 잠깐, 손으로 풀어보세요 각 설명을 참(O)·거짓(X)으로 표시해봐요. 헷갈리는 것이 있으면 다음 단계로 넘어가요.
1단계 · Page 1

수의 분류 체계부터 정리해요

왜 이 단계? — 각 설명의 참·거짓은 수의 포함 관계를 알면 쉽게 판단할 수 있어요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리
수의 포함 관계 (큰 집합 → 작은 집합)
유리수 ⊃ 정수 ⊃ 자연수 (양의 정수)
유리수 = 양의 유리수 + 0 + 음의 유리수
정수 = 양의 정수(자연수) + 0 + 음의 정수

핵심: 자연수 → 정수 → 유리수 순서로 포함되고,
0도 유리수예요 ($0 = \frac{0}{1}$).

🤔 확인해봐요 $-1$은 음의 정수이면서 동시에 유리수예요. 음의 유리수 중 정수인 것이 있을까요? (ㄹ)을 다시 봐요.

2단계 · Page 2

각 설명을 하나씩 판단해요

왜 이 단계? — 포함 관계를 기억하면서 각 명제의 참·거짓을 직접 판단해봐요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수
각 설명 판단
(ㄱ) 양수와 음수는 모두 유리수이다.  → 참 (O)
(ㄴ) 0은 양수도 음수도 아니므로 유리수가 아니다.  → 거짓 (X)
0도 유리수예요: $0 = \frac{0}{1}$. 0은 양수도 음수도 아니지만 유리수에는 속해요.
(ㄷ) 자연수는 모두 정수이다.  → 참 (O)
자연수 ⊂ 정수 — 자연수는 정수의 부분집합이에요.
(ㄹ) 음의 유리수 중에 정수인 것이 있다.  → 참 (O)
$-1, -2, -3, \ldots$는 음의 정수인 동시에 음의 유리수예요.
(ㅁ) 모든 정수는 자연수이다.  → 거짓 (X)
0과 음의 정수(−1, −2, …)는 정수이지만 자연수가 아니에요.
참(O): (ㄱ), (ㄷ), (ㄹ)  →  3개
거짓(X): (ㄴ), (ㅁ)  →  2개

3단계 · Page 3

핵심 함정 2가지 짚어봐요

왜 이 단계? — (ㄴ)과 (ㅁ)은 시험에서 자주 틀리는 함정이에요. 확실히 기억해요.
📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수
⚠️ 함정 1 — 0은 유리수예요 "0은 양수도 음수도 아니니까 유리수도 아니다" → 틀린 생각이에요.
$0 = \frac{0}{1}$이므로 분수 형태로 나타낼 수 있어요 → 유리수입니다.
유리수는 양의 유리수 + 0 + 음의 유리수로 구성돼요.
⚠️ 함정 2 — 정수가 자연수는 아니에요 "모든 정수 ⊂ 자연수" → 틀린 방향이에요.
올바른 방향: 자연수 ⊂ 정수 ⊂ 유리수
큰 집합에서 작은 집합으로 포함되고, 반대 방향은 성립하지 않아요.
정수 중 0과 음의 정수는 자연수가 아니에요.

정답 + 풀이 흐름 요약

📖 용어 노트 정수: 양의 정수(자연수), 0, 음의 정수를 모두 합친 수  ·  유리수: 분수 $\frac{a}{b}$($b \neq 0$)로 나타낼 수 있는 수  ·  절댓값 $|a|$: 수직선에서 원점까지의 거리 (항상 0 이상)  ·  역수: 곱해서 1이 되는 수
옳은 것의 개수 = 3개
((ㄱ), (ㄷ), (ㄹ)이 참)
전체 풀이 흐름 — 3단계
1
수의 포함 관계 정리: 자연수 ⊂ 정수 ⊂ 유리수. 0도 유리수($0=\frac{0}{1}$)
2
각 설명 판단: (ㄱ)O, (ㄴ)X(0도 유리수), (ㄷ)O, (ㄹ)O, (ㅁ)X(0·음정수는 자연수 아님)
3
참인 것: (ㄱ)(ㄷ)(ㄹ) → 3개
⚠️ 이 유형의 핵심 함정 0의 위치: 0은 정수이고 유리수예요. "0은 특별하니까 어디에도 안 속한다"는 생각은 틀렸어요.
방향 주의: "자연수 ⊂ 정수"는 참. "정수 ⊂ 자연수"는 거짓.
