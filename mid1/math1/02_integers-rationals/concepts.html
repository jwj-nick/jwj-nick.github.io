수의 세계 확장

양수·음수·정수·유리수를 한눈에 정리해요

수의 분류 구조

중1에서 처음으로 음수를 만나요. 수의 세계가 이렇게 확장돼요.

유리수
├── 정수
│   ├── 양의 정수(자연수): 1, 2, 3, …
│   ├── 0
│   └── 음의 정수: −1, −2, −3, …
└── 정수가 아닌 유리수
    ├── 양의 유리수: 1/2, 0.3, …
    └── 음의 유리수: −1/2, −0.3, …

유리수는 $\dfrac{a}{b}$ (단, $a$는 정수, $b$는 0이 아닌 정수) 꼴로 나타낼 수 있는 수예요.

주의! 0은 양수도 음수도 아니에요. 하지만 유리수이자 정수예요.

직접 분류해봐요

아래 수들이 정수인지, 유리수인지, 양수인지 음수인지 생각해봐요.

−3
정수 ✓  |  유리수 ✓  |  음의 정수 ✓
수직선에서: 0의 왼쪽 세 번째 칸
+2.5
정수 ✗  |  유리수 ✓  |  양의 유리수 ✓
$2.5 = \dfrac{5}{2}$ 이므로 유리수예요
0
정수 ✓  |  유리수 ✓  |  양수도 음수도 아님
$0 = \dfrac{0}{1}$ 이므로 유리수예요
−1/2
정수 ✗  |  유리수 ✓  |  음의 유리수 ✓
이미 $\dfrac{a}{b}$ 꼴이에요

개념 3

절댓값

0으로부터의 거리예요

기호 안내: |a| (세로 막대 두 개)
이걸 "절댓값"이라고 읽어요. 수직선에서 0으로부터 a까지의 거리를 나타내요.
거리는 항상 0 이상이에요. (음수가 될 수 없어요!)
처음 볼 때 헷갈릴 수 있으니 꼭 기억해두세요.

절댓값 직접 확인

−4
$|{-4}| = 4$  ·  $|{+4}| = 4$
−4와 +4는 절댓값이 같아요 (0으로부터 거리가 같음)

절댓값 규칙 정리

경우규칙예
$a > 0$$|a| = a$$|+3| = 3$
$a = 0$$|0| = 0$$|0| = 0$
$a < 0$$|a| = -a$ (부호 반대)$|-3| = 3$
흔한 실수: $|{-3}| = -3$이라고 쓰는 경우. 절댓값은 항상 0 이상이에요!
개념 4

대소 관계

수직선에서 오른쪽이 크다

대소 관계 규칙

규칙예
양수 > 0 > 음수$+1 > 0 > -1$
양수끼리: 절댓값 클수록 크다$+5 > +3$
음수끼리: 절댓값 클수록 작다$-5 < -3$
자주 하는 실수: −5와 −3을 비교할 때 "5가 크니까 −5가 크다"고 생각하는 경우.
수직선에서 왼쪽에 있는 수가 더 작아요. −5는 −3보다 왼쪽에 있어요!

두 수 비교해봐요

−5
−3
−5 < −3
−5는 수직선에서 −3보다 왼쪽에 있어요.

직접 써봐요

개념 5

덧셈과 뺄셈

같은 부호, 다른 부호, 뺄셈 변환

덧셈 규칙

경우방법예
양수 + 양수 절댓값의 합, 부호 + $(+3)+(+5) = +8$
음수 + 음수 절댓값의 합, 부호 − $(-3)+(-5) = -8$
양수 + 음수 절댓값의 차, 큰 쪽 부호 $(+5)+(-3) = +2$
음수 + 양수 절댓값의 차, 큰 쪽 부호 $(-5)+(+3) = -2$

덧셈 직접 계산

−5
+3
−5 + 3 = −2
다른 부호: 절댓값의 차 = 5−3 = 2, 큰 쪽(−5)의 부호 → −2

뺄셈 → 덧셈으로

뺄셈은 항상 덧셈으로 바꿀 수 있어요:

$A - B = A + (-B)$

빼는 수의 부호를 바꾸어 더해요.

뺄셈→ 덧셈으로결과
$(+5) - (+3)$$(+5) + (-3)$$+2$
$(-2) - (-5)$$(-2) + (+5)$$+3$
$(-3) - (+4)$$(-3) + (-4)$$-7$
개념 6

곱셈과 나눗셈

부호 규칙, 역수, 분배법칙

곱셈 부호 규칙

부호 조합결과예
(+) × (+)+$(+3) \times (+4) = +12$
(−) × (−)+$(-3) \times (-4) = +12$
(+) × (−)−$(+3) \times (-4) = -12$
(−) × (+)−$(-3) \times (+4) = -12$
외우는 법: 음수의 개수가 짝수 개 → 결과 +, 홀수 개 → 결과 −
예: $(-1) \times (-1) \times (-1) = -1$ (음수 3개 = 홀수 → 음수)

곱셈 계산기

−3
−4
−3 × −4 = +12
음수 2개 (짝수) → 결과: 양수. 절댓값의 곱 = 3×4 = 12

나눗셈 → 역수 곱하기

$A \div B = A \times \dfrac{1}{B}$
나눗셈→ 곱셈으로결과
$(-6) \div (+2)$$(-6) \times \frac{1}{2}$$-3$
$(-6) \div (-2)$$(-6) \times (-\frac{1}{2})$$+3$
$(+8) \div (-4)$$(+8) \times (-\frac{1}{4})$$-2$

부호 규칙은 곱셈과 동일해요!

분배법칙

$a \times (b + c) = a \times b + a \times c$

예: $(-2) \times (3 + (-5)) = (-2) \times 3 + (-2) \times (-5) = -6 + 10 = 4$
