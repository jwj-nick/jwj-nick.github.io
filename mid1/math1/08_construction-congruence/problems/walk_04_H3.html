유형 4 · H3

자와 컴퍼스로 60°를 작도한 후 이등분 작도를 사용해 30°와 15°를 만들었다. 같은 방법으로 만들 수 없는 각은?

(가) 7.5°   (나) 22.5° (= 90°의 4분의 1)   (다) 20° (= 60°의 3분의 1)   (라) 45°
🤔 잠깐, 손으로 풀어보세요 "이등분 작도" = 어떤 각을 2로 나누는 것. 이등분을 반복하면 $60° \div 2^n$ 형태의 각을 만들 수 있어요. 20°는 60°를 몇 등분하는 건가요?
1단계 · Page 1

이등분 반복으로 만들 수 있는 각

왜 이 단계? — 이등분 작도를 몇 번 반복하면 어떤 각이 나오는지 체계적으로 정리해요.
📖 용어 노트 이등분 작도: 각을 정확히 2로 나누는 작도
60°에서 이등분 반복
시작: 60°
1번 이등분: 30°
2번 이등분: 15°
3번 이등분: 7.5°
4번 이등분: 3.75°
→ $60° \div 2^n$ 형태의 각 모두 가능
90°에서 이등분 반복
시작: 90° (직각 작도 가능)
1번 이등분: 45°
2번 이등분: 22.5°
3번 이등분: 11.25°
→ $90° \div 2^n$ 형태의 각 모두 가능

2단계 · Page 2

각 선택지 확인

왜 이 단계? — (가)~(라) 각각이 이등분 반복으로 만들 수 있는지 확인해요.
📖 용어 노트 삼등분 불가: 자와 컴퍼스만으로는 임의의 각을 3등분 못함 (19세기 증명)
각 선택지 분석
(가) 7.5° = 60° → 30° → 15° → 7.5° (이등분 3번) ✓

(나) 22.5° = 90° → 45° → 22.5° (이등분 2번) ✓

(다) 20° = 60° ÷ 3 → "60°를 삼등분"해야 함
    자와 컴퍼스로 임의의 각 삼등분 = 불가능 (19세기 Wantzel 증명)

(라) 45° = 90° → 45° (이등분 1번) ✓
⚠️ 핵심 이등분은 무한히 반복 가능. 그러나 삼등분은 일반적으로 불가능.
특수한 각(예: 90°를 삼등분한 30°)은 다른 방법으로 작도 가능하지만, "이등분 반복"으로는 60° → 20°를 만들 수 없어요.

3단계 · Page 3

도구의 한계 — 수학사적 의미

왜 이 단계? — 이 문제가 단순 계산이 아니라 수학의 깊은 질문과 연결된다는 것을 느껴봐요.
📖 용어 노트 그리스 3대 작도 불가능 문제: 자와 컴퍼스만으로 해결이 불가능한 고대 3가지 문제
각의 삼등분 문제

고대 그리스 때부터 "임의의 각을 자와 컴퍼스로 삼등분할 수 있는가?"라는 질문이 있었어요.

2000년 이상 수학자들이 시도했고 19세기(1837년, Wantzel)에 불가능함이 증명됐어요.

자와 컴퍼스 작도 → 유리수 계수 다항식의 근만 만들 수 있음
각의 삼등분 → 3차 방정식의 근이 필요 (일반적으로 작도 불가)
도구 = 가능성의 경계

같은 사람이 같은 지식을 가지고 있어도 어떤 도구를 쓰느냐에 따라 풀 수 있는 문제가 달라져요.

자·컴퍼스: 이등분 ✓, 삼등분 ✗ (일반)
종이접기: 삼등분 ✓ (더 강력한 도구)
컴퓨터 수치 계산: 정확한 삼등분점 계산 가능
