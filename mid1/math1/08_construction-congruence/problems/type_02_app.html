L
컴퍼스로 같은 길이를 옮길 때, "컴퍼스 벌림"을 바꾸지 않는 이유는?
컴퍼스 벌림 = 길이.
벌림이 곧 옮기는 길이이기 때문
컴퍼스 벌림은 두 다리 끝의 거리. 이 거리가 그대로 새 위치에 옮겨져요.
벌림이 바뀌면 옮긴 길이도 달라져 작도가 무너져요.
1
M
∠XOY와 크기가 같은 각을 작도했을 때, 두 각이 같음을 보장하는 핵심 근거는?
① 각도기로 같은 값을 두 번 잰다   ② 컴퍼스 두 번의 같은 반지름이 두 삼각형을 SSS 합동으로 만든다   ③ 자로 같은 길이를 직접 측정해 옮긴다   ④ 우연히 같은 각이 나온다
각 옮기기 절차에서 컴퍼스를 어떤 두 길이로 같게 잡았나요?
②
각 옮기기 절차:
1) O 중심 호 → A, B (∴ $\overline{OA} = \overline{OB}$)
2) 같은 반지름으로 O' 중심 호 → A', B' (∴ $\overline{O'A'} = \overline{O'B'} = \overline{OA}$)
3) AB 길이로 A'에서 호 → B' (∴ $\overline{A'B'} = \overline{AB}$)
→ 삼각형 OAB ≡ O'A'B' (SSS) → ∠O = ∠O' ✓
→ 작도의 정확성은 합동에서 나와요.

③ 임의 두 점은 길이 정보가 없음 X

M
두 선분 $\overline{AB}$와 $\overline{CD}$가 주어져 있다 ($\overline{AB} > \overline{CD}$). 직선 $l$ 위에 $\overline{AB} - \overline{CD}$ 길이의 선분을 작도하려고 한다. 가장 적절한 절차는?
먼저 $\overline{AB}$를 옮기고, 그 안쪽에서 $\overline{CD}$ 만큼을 빼요.
① 직선 $l$ 위에 점 P 잡기 → ② P에서 호 (반지름 $\overline{AB}$) → 교점 Q ($\overline{PQ} = \overline{AB}$) → ③ Q에서 호 (반지름 $\overline{CD}$, P 방향) → 교점 R ($\overline{QR} = \overline{CD}$) → 결과: $\overline{PR} = \overline{AB} - \overline{CD}$
컴퍼스를 두 번 사용. 직접 빼지 않고 "긴 거리에서 짧은 거리만큼 안쪽으로 들어가기"로 차이를 작도.

H
∠XOY를 옮길 때 사용한 컴퍼스 절차에서 다음 중 변경 불가한 단계를 모두 고르시오.
(가) 1단계 호의 반지름 (O 중심)   (나) 2단계 새 직선 위 호의 반지름 (O' 중심)   (다) 3단계 새 호의 반지름 (A' 중심, AB 길이)   (라) 호를 그리는 시계/반시계 방향
SSS 합동에서 어떤 길이들이 같아야 했나요?
(가) = (나), (다)는 AB 길이와 같아야
(가)와 (나): 둘이 같은 반지름이어야 $\overline{OA} = \overline{O'A'}$ 보장
(다): AB 길이와 같아야 $\overline{A'B'} = \overline{AB}$ 보장 → SSS 합동 성립
(라): 시계/반시계 방향은 결과에 영향 없음. 단, 일반적으로 같은 방향으로 통일해요.
→ 핵심: (가)와 (나)는 서로 같아야 하고, (다)는 AB와 같아야 함.
2
H
선분 $\overline{AB}$가 주어져 있다. $\overline{AB}$의 2배 길이의 선분을 직선 $l$ 위에 작도하는 방법을 단계별로 설명하시오.
$\overline{AB}$만큼을 두 번 연속으로 직선 위에 옮겨요.
① 직선 $l$ 위 점 P 잡기
② P에 컴퍼스 꽂고 $\overline{AB}$ 길이로 벌려 호 → 교점 Q ($\overline{PQ} = \overline{AB}$)
③ 컴퍼스 같은 벌림으로 Q에 꽂아 호 (P 반대 방향) → 교점 R ($\overline{QR} = \overline{AB}$)
④ 결과: $\overline{PR} = \overline{PQ} + \overline{QR} = 2\overline{AB}$
같은 컴퍼스 벌림을 두 번 연속 사용 = 같은 길이 두 개 연결.
3
H
∠AOB가 주어져 있다 (예각). 한 직선 $l$ 위에 ∠AOB의 2배 크기의 각을 작도하려고 할 때, 핵심 아이디어를 설명하시오.
각 옮기기를 두 번 사용해서 두 각을 이어 붙여요.
각 옮기기를 연달아 두 번
① 직선 $l$ 위 점 O' 잡기
② $\overrightarrow{O'X}$를 한 변으로 하여 ∠AOB와 같은 각을 작도 → 새 변 $\overrightarrow{O'P}$ ($\angle XO'P = \angle AOB$)
③ 이번엔 $\overrightarrow{O'P}$를 한 변으로 하여 또 ∠AOB와 같은 각을 같은 방향으로 작도 → 새 변 $\overrightarrow{O'Q}$
④ 결과: $\angle XO'Q = 2 \angle AOB$
→ "옮기기"는 반복 가능. 각의 합·뺄셈도 같은 원리로 작도 가능.
★ 핵심 통찰: 자와 컴퍼스의 기본 작도(같은 길이·같은 각)를 반복하면 더 복잡한 도형이 만들어져요. 그래서 「원론」은 단순 도구로 수백 정리를 쌓아 올릴 수 있었던 거예요.
