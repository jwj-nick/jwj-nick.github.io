유형 2 · H2

선분 $\overline{AB}$가 주어져 있다. $\overline{AB}$의 2배 길이의 선분을 직선 $l$ 위에 작도하는 방법을 단계별로 설명하시오.
🤔 잠깐, 손으로 풀어보세요 길이를 "두 배"로 만들려면 같은 길이를 두 번 이어 붙이면 되겠죠? 컴퍼스로 어떻게 "같은 길이를 두 번" 옮길 수 있을까요?
1단계 · Page 1

핵심 아이디어 — 같은 길이 두 번 연결

왜 이 단계? — "2배"는 같은 단위를 두 번 잇는 것이에요. 컴퍼스 벌림을 유지하면 같은 길이를 반복해서 옮길 수 있어요.
📖 용어 노트 컴퍼스 벌림: 두 다리 끝의 거리 = 옮길 길이
핵심 원리

컴퍼스 벌림을 바꾸지 않으면 같은 길이를 여러 번 옮길 수 있어요.

벌림 = $\overline{AB}$ 고정
→ 한 번 옮기면 $\overline{AB}$ 길이 1개
→ 두 번 이어 옮기면 $\overline{AB} + \overline{AB} = 2\overline{AB}$

핵심은 두 번째 옮길 때 첫 번째 옮긴 끝점에서 시작하는 것이에요.

💡 힌트 직선 $l$ 위에 점 P를 먼저 잡아요. P에서 $\overline{AB}$ 길이만큼 옮겨서 Q를 만들고, Q에서 다시 같은 길이만큼 옮겨서 R을 만들면 어떻게 될까요?

2단계 · Page 2

구체적 작도 절차

왜 이 단계? — 아이디어를 실제 컴퍼스·자 조작으로 번역해요.
📖 용어 노트 컴퍼스 벌림: 두 다리 끝의 거리
단계별 작도
① 직선 $l$ 위 임의의 점 P를 잡는다
② 컴퍼스를 원본 A에 꽂고 B까지 벌린다 (벌림 = $\overline{AB}$)
③ 벌림을 유지한 채 P에 꽂고 호를 그어 직선 $l$과 만나는 점 Q를 찾는다
   ($\overline{PQ} = \overline{AB}$)
④ 벌림을 그대로 유지한 채 Q에 꽂고 P 반대 방향으로 호를 그어 직선 $l$과 만나는 점 R을 찾는다
   ($\overline{QR} = \overline{AB}$)
⑤ $\overline{PR} = \overline{PQ} + \overline{QR} = \overline{AB} + \overline{AB} = 2\overline{AB}$
왜 "P 반대 방향"인가?

Q에서 호를 그릴 때 P 방향으로 가면 P 근처 점이 R이 되어버려요.
P 반대 방향으로 가야 Q 너머에 R이 생겨 $\overline{PQR}$이 한 직선 위에 순서대로 놓여요.

P ← Q → R (P 반대 방향으로 R)
$\overline{PR} = \overline{PQ} + \overline{QR} = 2\overline{AB}$ ✓

3단계 · Page 3

확인 — 왜 2배인가?

왜 이 단계? — 작도 결과를 수식으로 확인해요.
📖 용어 노트 컴퍼스 벌림: 두 다리 끝의 거리 = 옮길 길이
확인

②~③에서: 컴퍼스 벌림 = $\overline{AB}$이고, P에 꽂아 호 → 교점 Q

$\overline{PQ}$ = 컴퍼스 벌림 = $\overline{AB}$

④에서: 같은 벌림 유지, Q에 꽂아 P 반대 방향 호 → 교점 R

$\overline{QR}$ = 컴퍼스 벌림 = $\overline{AB}$

따라서:

$\overline{PR} = \overline{PQ} + \overline{QR} = \overline{AB} + \overline{AB} = 2\overline{AB}$ ✓
⚠️ 흔한 실수 Q에서 두 번째 호를 그릴 때 방향을 실수하면 P와 R이 같은 쪽에 있게 돼요. 반드시 P의 반대 방향으로 호를 그려야 해요.

정답 + 풀이 흐름 요약

📖 용어 노트 합동(≅): 두 도형 모양·크기 완전히 같음  ·  컴퍼스 벌림: 두 다리 끝의 거리 = 옮길 길이
$\overline{PR} = 2\overline{AB}$
직선 $l$ 위에 P → Q → R 순서로 같은 벌림 두 번 적용
전체 풀이 흐름
1
직선 $l$ 위 점 P 잡기
2
컴퍼스를 A→B까지 벌려 $\overline{AB}$ 길이 기억
3
P에 꽂아 호 → 교점 Q: $\overline{PQ} = \overline{AB}$
4
벌림 유지, Q에 꽂아 P 반대 방향 호 → 교점 R: $\overline{QR} = \overline{AB}$
5
$\overline{PR} = \overline{PQ} + \overline{QR} = 2\overline{AB}$ ✓
⚠️ 핵심 통찰 "같은 길이 n배"는 컴퍼스를 n번 연속으로 같은 방향으로 옮기면 돼요. 3배, 4배도 같은 방법으로 가능해요.
