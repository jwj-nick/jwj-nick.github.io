유형 4 · H1

두 직선 $l$, $m$이 점 O에서 만나 90°가 아닌 각을 이룬다. 두 직선에서 같은 거리에 있는 점들의 자취는?
🤔 잠깐, 손으로 풀어보세요 두 직선이 만나면 4개의 각이 생겨요. 각의 이등분선 위 점은 두 변에서 같은 거리였어요. 그렇다면 두 직선에서 같은 거리인 점은 어디에 있을까요?
1단계 · Page 1

각의 이등분선 위 점의 성질

왜 이 단계? — "두 변에서 같은 거리" 성질을 두 직선에 적용해요.
📖 용어 노트 점에서 직선까지의 거리: 점에서 직선에 내린 수선의 길이
각의 이등분선 성질 복습

∠XOY의 이등분선 위 점 P는 두 변 OX, OY에서 같은 거리를 가져요.

P가 이등분선 위 → P에서 OX까지 거리 = P에서 OY까지 거리

역도 성립: 두 변에서 같은 거리인 점은 각의 이등분선 위에 있어요.

두 직선의 경우 — 4개의 각

두 직선 $l$, $m$이 O에서 만나면 4개의 각이 생겨요 (두 쌍의 맞꼭지각).

각 1: ∠AOC, 각 2: ∠COB, 각 3: ∠BOD, 각 4: ∠DOA
(∠AOC = ∠BOD 맞꼭지각, ∠COB = ∠DOA 맞꼭지각)

"두 직선에서 같은 거리"인 점은 l에서의 거리 = m에서의 거리를 의미해요.
이는 두 직선이 이루는 어떤 각의 이등분선 위 점과 같아요.

2단계 · Page 2

이등분선이 두 개 — 서로 수직

왜 이 단계? — 4개의 각에 대해 이등분선이 몇 개인지, 그리고 그 이등분선들의 관계를 찾아요.
📖 용어 노트 맞꼭지각: 두 직선의 교점에서 서로 마주 보는 각 (크기가 같음)
4개의 각 → 이등분선 2개

4개의 각 중 맞꼭지각끼리는 이등분선을 공유해요.

∠AOC와 ∠BOD (맞꼭지각) → 같은 이등분선 $p$
∠COB와 ∠DOA (맞꼭지각) → 같은 이등분선 $q$

따라서 이등분선은 총 2개: $p$와 $q$.

두 이등분선은 서로 수직

∠AOC = $\alpha$라 하면 이웃한 각 ∠COB = 180° − $\alpha$ (직선 위).
이등분선 $p$는 ∠AOC를 $\alpha/2$로 나누고, $q$는 ∠COB를 $(180°-\alpha)/2$로 나눠요.

$p$와 $q$ 사이 각 = $\alpha/2 + (180°-\alpha)/2 = \alpha/2 + 90° - \alpha/2 = 90°$

두 이등분선은 항상 서로 수직이에요.

3단계 · Page 3

결론 — 자취는 두 이등분선

왜 이 단계? — 조건을 만족하는 모든 점을 하나의 문장으로 정리해요.
📖 용어 노트 자취: 조건을 만족하는 모든 점들의 집합 (도형)
자취 = 두 이등분선

"두 직선 $l$, $m$에서 같은 거리에 있는 점들의 집합"은 O에서 만나는 두 개의 각의 이등분선이에요.

자취 = 이등분선 $p$ ∪ 이등분선 $q$
(두 직선이 이루는 두 쌍의 각을 각각 이등분하는 직선들)
이 두 직선은 서로 수직으로 만남
⚠️ "두 직선에서 같은 거리" vs "두 점에서 같은 거리" 두 점에서 같은 거리 → 수직이등분선 (1개)
두 직선에서 같은 거리 → 각의 이등분선 2개 (서로 수직)
대상이 "점"이냐 "직선"이냐에 따라 자취가 달라요.
