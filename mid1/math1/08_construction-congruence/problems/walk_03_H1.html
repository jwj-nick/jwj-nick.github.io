유형 3 · H1

수직이등분선 작도가 정확하다는 것을 합동을 사용해 설명하시오.

(수직이등분선이 PQ, 중점 M일 때, 왜 ∠AMP = 90°인가?)
🤔 잠깐, 손으로 풀어보세요 두 삼각형 APM과 BPM을 그려보세요. 세 변의 길이를 각각 비교하면 어떤 합동 조건이 보이나요?
1단계 · Page 1

수직이등분선 작도 복습

왜 이 단계? — 증명에 사용할 사실들이 작도 절차에서 나오므로 먼저 정리해요.
📖 용어 노트 수직이등분선: 선분의 중점을 지나며 수직인 직선  ·  SSS 합동: 세 변 같음
수직이등분선 작도 절차
A 중심 호 (반지름 $r$ > $\overline{AB}/2$) → 호
B 중심 호 (같은 반지름 $r$) → 호
두 호의 교점: 위쪽 P, 아래쪽 Q (또는 한쪽만)
자로 PQ 연결 → 수직이등분선
작도에서 얻어지는 사실

작도 절차에서 직접 읽어낼 수 있는 등변 관계:

$\overline{PA} = \overline{PB}$ (두 호의 반지름이 같으므로: P는 A 중심 호 위이면서 B 중심 호 위)
$\overline{QA} = \overline{QB}$ (같은 이유로 Q도)

M은 PQ와 $\overline{AB}$의 교점이에요. M에서도 $\overline{MA}$와 $\overline{MB}$를 비교할 수 있어요.

2단계 · Page 2

두 삼각형 비교 — APM과 BPM

왜 이 단계? — ∠AMP = 90°를 보이려면 두 삼각형이 합동임을 먼저 보여야 해요.
📖 용어 노트 SSS 합동: 세 변 같음 → 대응각 같음  ·  공통변: 두 삼각형이 공유하는 변 (자동으로 같음)
삼각형 APM과 BPM의 세 변 비교
$\overline{PA} = \overline{PB}$ (1단계에서: 두 호 같은 반지름)
$\overline{MA} = \overline{MB}$ (M이 중점이므로 — 아래 확인)
$\overline{PM}$ 공통
→ 세 변이 모두 같음 → SSS 합동
잠깐 — MA = MB는 어떻게?

M이 중점이라는 것을 어떻게 알까요? 사실 M이 중점임도 이 합동에서 나와요.

P와 Q 모두 $\overline{PA}=\overline{PB}$, $\overline{QA}=\overline{QB}$이므로 PQ 위 모든 점은 A·B에서 같은 거리를 가져요. 특히 M도 마찬가지: $\overline{MA} = \overline{MB}$.

PQ 위 점 M → $\overline{MA} = \overline{MB}$ (PQ 성질)

3단계 · Page 3

합동에서 90° 도출

왜 이 단계? — SSS 합동이 성립하므로 대응각이 같고, 그 대응각들이 평각을 이루면 90°가 나와요.
📖 용어 노트 평각: 180°가 되는 각 (반직선)  ·  대응각: SSS 합동인 두 삼각형에서 같은 위치의 각
합동 → 대응각 같음
삼각형 APM ≅ 삼각형 BPM (SSS)
→ ∠AMP = ∠BMP (대응각)
평각 활용 → 90°

∠AMP와 ∠BMP는 직선 $\overline{AB}$ 위에서 같은 점 M에서 만나므로 평각을 이뤄요.

∠AMP + ∠BMP = 180° (평각)
∠AMP = ∠BMP (합동에서)
→ $2 \times \angle AMP = 180°$
→ $\angle AMP = 90°$ ✓

따라서 PQ는 $\overline{AB}$에 수직! 그리고 M은 $\overline{AB}$의 중점. PQ는 수직이등분선이 맞아요.

4단계 · Page 4

M이 중점임도 같은 합동으로!

왜 이 단계? — M = 중점임도 따로 가정한 것이 아니라 합동에서 도출돼요.
📖 용어 노트 중점: 선분을 정확히 반으로 나누는 점
M = 중점 증명

삼각형 APM ≅ 삼각형 BPM (SSS)이므로 대응변도 같아요.

M이 대응변: $\overline{AM} \leftrightarrow \overline{BM}$
→ $\overline{AM} = \overline{BM}$ ✓
→ M은 $\overline{AB}$의 중점

이처럼 수직(∠AMP = 90°)과 이등분(AM = BM) 두 성질이 동시에 합동 하나에서 나와요.

💡 큰 그림 작도가 "정확하다"는 것을 수학적으로 증명하는 방법은 항상 합동이에요. 자와 컴퍼스로 만들어진 두 삼각형의 세 변이 같음(SSS)을 보이면, 모든 각도 같고 모든 결론이 따라와요.

정답 + 풀이 흐름 요약

📖 용어 노트 합동(≅): 두 도형 모양·크기 완전히 같음  ·  SSS 합동: 세 변 같음 → 대응각 같음
삼각형 APM ≅ 삼각형 BPM (SSS) → ∠AMP = ∠BMP
평각 180° → ∠AMP = 90°
전체 풀이 흐름
1
작도에서: $\overline{PA} = \overline{PB}$ (두 호 같은 반지름)
2
PQ 위 M은 $\overline{MA} = \overline{MB}$ (PQ의 성질)
3
$\overline{PM}$ 공통 → 삼각형 APM ≅ BPM (SSS)
4
대응각: ∠AMP = ∠BMP
5
평각: ∠AMP + ∠BMP = 180° → ∠AMP = 90° ✓
⚠️ 핵심 통찰 수직(90°)과 이등분(AM=BM) 두 성질 모두 같은 SSS 합동 하나에서 나와요. 작도의 정확성 증명 = SSS 합동 1개면 끝!
