유형 1 — 직선·반직선·선분 표기 구분

$\overleftrightarrow{AB}, \overrightarrow{AB}, \overline{AB}$ 의 차이를 정확히 알기
← 단원 7 홈
1
L
"두 점 A, B를 지나며 양쪽으로 끝없이 뻗은 도형"의 표기는?
화살표가 양쪽으로 가는 표기를 떠올려요.
$\overleftrightarrow{AB}$ — 직선
양쪽 무한 → 직선 → 양방향 화살표 표기 $\overleftrightarrow{AB}$.
2
L
$\overrightarrow{AB}$ 가 나타내는 도형을 모두 고르시오.
(가) 직선 (나) 반직선 (다) 선분 (라) 점 A에서 출발 (마) 양쪽 무한
$\overrightarrow{AB}$ 표기에서 화살표는 한쪽만 향해요. 시작점은 첫 글자.
(나), (라)
$\overrightarrow{AB}$ : 반직선, A에서 출발해 B 방향으로 무한.
(가) 직선은 양쪽 무한 X → $\overleftrightarrow{AB}$
(다) 선분은 양 끝 있음 X → $\overline{AB}$
(마) 한쪽만 무한.
3
L
한 직선 위에 점 A, B, C, D가 이 순서대로 있을 때, 다음 중 표기가 같은 것끼리 짝지으시오.
(가) $\overline{AB}$   (나) $\overline{BA}$   (다) $\overrightarrow{AC}$   (라) $\overrightarrow{CA}$
선분은 방향이 없고, 반직선은 방향이 있어요.
(가) = (나),   (다) ≠ (라)
(가)(나) 선분 → 방향 없으니 $\overline{AB} = \overline{BA}$ ✓
(다)(라) 반직선 → A에서 출발 vs C에서 출발은 다른 도형. 같지 않음.
1
M
한 직선 위에 세 점 A, B, C가 이 순서대로 있다. $\overrightarrow{AB}$ 와 같은 도형은?
(가) $\overrightarrow{BA}$   (나) $\overrightarrow{AC}$   (다) $\overrightarrow{BC}$   (라) $\overleftrightarrow{AB}$
반직선은 (1) 시작점이 같고 (2) 같은 방향이면 같은 도형이에요. 한 직선 위에 점 순서가 A→B→C이면 A에서 시작해 B 방향과 C 방향은 같은 방향이에요.
(나) $\overrightarrow{AC}$
$\overrightarrow{AB}$ : A에서 시작, B 방향으로 무한.
점 순서가 A→B→C이므로 B 방향 = C 방향. 또 시작점 A로 같음.
$\overrightarrow{AB} = \overrightarrow{AC}$ ✓
(가) 시작점 다름 X
(다) 시작점 B로 다름 X
(라) 직선은 양쪽 무한, 반직선과 다른 도형 X
★ 반직선 등식 조건: 시작점 같고 + 방향 같음. 한 직선 위 점 순서로 방향 판단.
2
M
한 직선 위에 점 A, B, C가 이 순서대로 있다. 다음 중 옳은 것은?
① $\overline{AB} = \overline{AC}$   ② $\overrightarrow{AB} = \overrightarrow{BA}$   ③ $\overrightarrow{AC} = \overrightarrow{BC}$   ④ $\overleftrightarrow{AB} = \overleftrightarrow{BC}$   ⑤ $\overline{AB} = \overrightarrow{AB}$
선분 길이, 반직선 시작점, 직선의 양쪽 무한성 ─ 각 도형의 결정 요소를 떠올려요.
④ $\overleftrightarrow{AB} = \overleftrightarrow{BC}$
① 길이 다름. $\overline{AC} > \overline{AB}$ X
② 반직선 방향 다름 X
③ 시작점 A vs B로 다름 X
④ A, B, C가 한 직선 위에 있으면 그 직선은 한 개. $\overleftrightarrow{AB} = \overleftrightarrow{BC} = \overleftrightarrow{AC}$ ✓
⑤ 선분과 반직선은 끝 있음/없음 다름 X
3
M
두 점 A, B를 지나는 다음 5가지 표기 중, 서로 다른 도형은 모두 몇 종류인가?
$\overline{AB},\; \overline{BA},\; \overrightarrow{AB},\; \overrightarrow{BA},\; \overleftrightarrow{AB}$
같은 도형끼리 묶어보고, 서로 다른 종류만 세요.
4종류
$\overline{AB} = \overline{BA}$ → 1종 (선분, 방향 없음)
$\overrightarrow{AB}$ → 1종 (반직선, A→B 방향)
$\overrightarrow{BA}$ → 1종 (반직선, B→A 방향, 위와 다름!)
$\overleftrightarrow{AB}$ → 1종 (직선, 양방향 무한)
총 4종류의 다른 도형.
★ 5표기 중 $\overline{AB}=\overline{BA}$ 만 같음. 나머지는 모두 다름. 도형 종류 = 4.

H
한 직선 위에 점 4개 A, B, C, D가 이 순서대로 있다. $\overrightarrow{BC}$ 와 같은 도형은? (모두 고르시오)
① $\overrightarrow{BD}$   ② $\overrightarrow{CD}$   ③ $\overrightarrow{AC}$   ④ $\overrightarrow{BA}$
$\overrightarrow{BC}$ : B에서 시작, C 방향. C 방향 = D 방향 (점 순서가 B→C→D이므로).
① $\overrightarrow{BD}$
$\overrightarrow{BC}$ : 시작점 B, 방향 = C(=D) 쪽.
① $\overrightarrow{BD}$ : 시작점 B, 방향 D(=C 같은 방향) → 같음 ✓
② $\overrightarrow{CD}$ : 시작점 C로 다름 X
③ $\overrightarrow{AC}$ : 시작점 A로 다름 X
④ $\overrightarrow{BA}$ : 방향 반대 X
2
H
한 직선 위에 서로 다른 점 5개가 있다. 이 점들로 만들 수 있는 서로 다른 직선의 개수와 서로 다른 선분의 개수, 서로 다른 반직선의 개수는 각각 몇 개?
한 직선 위 점들이라 직선은 1개. 선분은 두 점 선택 ${}_5C_2$. 반직선은 (시작점 결정 + 방향 결정).
직선: 1개,   선분: 10개,   반직선: 8개
① 직선: 한 직선 위니까 모두 같은 직선. → 1
② 선분: 5점 중 2점 고르는 조합 = $\dfrac{5 \times 4}{2} = 10$. → 10
③ 반직선: 양 끝점은 한 방향만 가능, 안의 3개 점은 두 방향 가능.
$2 \times 1 + 3 \times 2 = 2 + 6 = $ 8
★ 반직선 카운트: 끝점은 1방향, 안쪽 점은 2방향. 점 5개 → 끝 2 × 1 + 안쪽 3 × 2 = 8.
3
H
한 평면 위에 어느 세 점도 한 직선 위에 있지 않은 점 4개 A, B, C, D가 있다. 이 점들로 만들 수 있는 서로 다른 직선의 개수는?
두 점이 정해지면 직선이 하나 결정돼요. 어느 세 점도 한 직선 위에 없으면 두 점 묶음마다 서로 다른 직선.
6개
두 점으로 직선 하나가 정해짐. 4점 중 2점 고르는 경우의 수: $\dfrac{4 \times 3}{2} = 6$.
"어느 세 점도 한 직선 위에 없다" → 모든 두 점 쌍이 서로 다른 직선을 만든다.
★ 만약 점이 한 직선 위에 있다면 직선 수가 줄어요. 조건 "어느 셋도 한 직선 위에 없다"가 핵심!
