문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$ (→ 방향): A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$ (−AB): A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
유형 3 · H3

$\angle AOB$가 직각이다. $\angle AOB$의 내부에 점 C, D가 있어
$\overrightarrow{OC}$가 $\angle AOB$의 이등분선이고, $\overrightarrow{OD}$가 $\angle COB$의 이등분선이다.
$\angle AOD$의 크기는?
🤔 잠깐, 손으로 풀어보세요 이등분선이 두 번 나와요. 한 번에 하나씩 이등분해서 각도를 구해보세요.
1단계 · Page 1

첫 번째 이등분선으로 $\angle AOC$ 구하기

왜 이 단계? — 이등분선은 각을 절반으로 나눠요. 먼저 C의 위치를 구해요.
📖 용어 노트 이등분선: 한 각을 똑같이 두 부분으로 나누는 반직선
$\overrightarrow{OC}$가 $\angle AOB$의 이등분선
$\angle AOB = 90°$
이등분 → $\angle AOC = \angle COB = \dfrac{90°}{2} = 45°$
← 이전
2단계 · Page 2

두 번째 이등분선으로 $\angle COD$ 구하기

왜 이 단계? — D는 $\angle COB$를 이등분하는 점. $\angle COB = 45°$를 다시 절반으로 나눠요.
📖 용어 노트 이등분선: 한 각을 똑같이 두 부분으로 나누는 반직선
$\overrightarrow{OD}$가 $\angle COB$의 이등분선
$\angle COB = 45°$
이등분 → $\angle COD = \angle DOB = \dfrac{45°}{2} = 22.5°$
← 이전
3단계 · Page 3

$\angle AOD$ 계산

왜 이 단계? — A와 D 사이 각 = $\angle AOC$ + $\angle COD$.
📖 용어 노트 $\angle AOD$: 꼭짓점 O에서 OA와 OD가 만드는 각
$\angle AOD = \angle AOC + \angle COD$

점 순서가 A, C, D, B 이므로 (C, D가 모두 $\angle AOB$ 내부):

$\angle AOD = \angle AOC + \angle COD = 45° + 22.5° = 67.5°$

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$: A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$: A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
$\angle AOD = 67.5°$
(이등분 2번: $90° \to 45° \to 22.5°$)
전체 풀이 흐름 — 3단계
1
$\overrightarrow{OC}$가 $\angle AOB = 90°$ 이등분 → $\angle AOC = \angle COB = 45°$
2
$\overrightarrow{OD}$가 $\angle COB = 45°$ 이등분 → $\angle COD = 22.5°$
3
$\angle AOD = \angle AOC + \angle COD = 45° + 22.5° = 67.5°$
⚠️ 이 유형의 핵심 이등분선이 연속으로 나올 때는 한 번에 하나씩 순서대로 적용해요. 이등분 = 절반이라는 것만 기억하면 돼요!
