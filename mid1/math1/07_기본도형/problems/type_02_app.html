유형 2 — 선분 길이의 합·중점

중점, 길이의 합·차로 미지의 길이 구하기
← 단원 7 홈
1
L
$\overline{AB} = 12$ cm 이고 점 M이 $\overline{AB}$의 중점일 때, $\overline{AM}$의 길이는?
중점 M은 $\overline{AB}$를 똑같은 두 부분으로 나눠요.
$\overline{AM} = 6$ cm
$\overline{AM} = \overline{MB} = \dfrac{1}{2}\overline{AB} = \dfrac{12}{2} = 6$ (cm)
2
L
한 직선 위에 점 A, B, C가 이 순서대로 있고 $\overline{AB} = 5$ cm, $\overline{BC} = 8$ cm 일 때, $\overline{AC}$의 길이는?
점 B가 A와 C 사이에 있어요. 두 부분의 길이를 더해요.
$\overline{AC} = 13$ cm
$\overline{AC} = \overline{AB} + \overline{BC} = 5 + 8 = 13$ (cm)
3
L
$\overline{AB} = 18$ cm 이고 점 M이 $\overline{AB}$의 중점, 점 N이 $\overline{AM}$의 중점일 때, $\overline{AN}$의 길이는?
$\overline{AB}$의 절반이 $\overline{AM}$. 그 절반이 $\overline{AN}$이에요.
$\overline{AN} = 4.5$ cm (= $\dfrac{9}{2}$ cm)
$\overline{AM} = \dfrac{1}{2} \times 18 = 9$
$\overline{AN} = \dfrac{1}{2} \times 9 = 4.5$ (cm)
1
M
한 직선 위에 점 A, B, C, D가 이 순서대로 있고, $\overline{AB} = 4$ cm, $\overline{AC} = 10$ cm, $\overline{AD} = 16$ cm 일 때, $\overline{BD}$의 길이는?
$\overline{BD} = \overline{AD} - \overline{AB}$.
$\overline{BD} = 12$ cm
$\overline{BD} = \overline{AD} - \overline{AB} = 16 - 4 = 12$ (cm)
2
M
한 직선 위에 점 A, M, B가 이 순서대로 있다. M이 $\overline{AB}$의 중점이고 $\overline{AM} = 7$ cm 일 때, $\overline{AB}$의 길이는?
중점은 두 부분이 같다 → $\overline{AB} = 2 \times \overline{AM}$.
$\overline{AB} = 14$ cm
$M$이 중점 → $\overline{AB} = 2\overline{AM} = 2 \times 7 = 14$ (cm)
3
M
한 직선 위에 점 A, M, N, B 가 이 순서대로 있다. M은 $\overline{AB}$의 중점, N은 $\overline{MB}$의 중점이다. $\overline{AB} = 24$ cm 일 때, $\overline{AN}$의 길이는?
AB = 24 → AM = 12. MB = 12 → MN = 6. AN = AM + MN.
$\overline{AN} = 18$ cm
$\overline{AM} = \dfrac{1}{2}\overline{AB} = 12$
$\overline{MB} = \overline{AB} - \overline{AM} = 24 - 12 = 12$
N은 $\overline{MB}$의 중점 → $\overline{MN} = \dfrac{1}{2} \cdot 12 = 6$
$\overline{AN} = \overline{AM} + \overline{MN} = 12 + 6 = 18$ (cm)
1
H
한 직선 위에 점 A, B, C가 이 순서대로 있고 $\overline{AC} = 24$ cm 이다. M은 $\overline{AB}$의 중점, N은 $\overline{BC}$의 중점일 때, $\overline{MN}$의 길이는?
$\overline{AB} = a$, $\overline{BC} = b$ 라 두면 $a + b = 24$. $\overline{MN} = \overline{MB} + \overline{BN} = \dfrac{a}{2} + \dfrac{b}{2} = \dfrac{a+b}{2}$.
$\overline{MN} = 12$ cm
$\overline{AB} = a$, $\overline{BC} = b$, $a+b = \overline{AC} = 24$
$\overline{MB} = \dfrac{a}{2}$, $\overline{BN} = \dfrac{b}{2}$
$\overline{MN} = \overline{MB} + \overline{BN} = \dfrac{a + b}{2} = \dfrac{24}{2} = 12$ (cm)
★ 멋진 결과: 두 중점 사이의 거리는 항상 $\overline{AC}$의 절반이에요. $a, b$가 무엇이든!
2
H
한 직선 위에 점 A, B, C, D가 이 순서대로 있고 $\overline{AB} : \overline{BC} : \overline{CD} = 1 : 2 : 3$ 이다. $\overline{AD} = 30$ cm 일 때, $\overline{BC}$의 길이는?
비를 한 단위로 묶고 단위 길이를 $k$라 두어 $\overline{AB} = k, \overline{BC} = 2k, \overline{CD} = 3k$. 합은 6k.
$\overline{BC} = 10$ cm
$\overline{AB} = k$, $\overline{BC} = 2k$, $\overline{CD} = 3k$ 라 하면
$\overline{AD} = k + 2k + 3k = 6k = 30$
$k = 5$
$\overline{BC} = 2k = 10$ (cm)
3
H
한 직선 위에 점 A, B, C가 있고 $\overline{AB} = 16$ cm, $\overline{BC} = 8$ cm 이다. M이 $\overline{AC}$의 중점일 때, $\overline{BM}$의 길이를 모두 구하시오. (B가 A와 C 사이에 있을 때 / B가 A와 C의 같은 쪽 끝에 있을 때 두 경우)
경우 1: A — B — C (B가 사이) → $\overline{AC} = 24$, M은 A에서 12 떨어진 점.
경우 2: A — C — B (C가 A, B 사이) → $\overline{AC} = 8$, M은 A에서 4 떨어진 점.
$\overline{BM} = 4$ cm 또는 $\overline{BM} = 12$ cm

경우 1: A — B — C 순 (B가 A, C 사이)
$\overline{AC} = \overline{AB} + \overline{BC} = 16 + 8 = 24$
M은 $\overline{AC}$의 중점이므로 A에서 12 떨어진 점.
B는 A에서 16 떨어진 점이므로 M보다 오른쪽.
$\overline{BM} = 16 - 12 = 4$ cm

경우 2: A — C — B 순 (C가 A, B 사이)
$\overline{AB} = \overline{AC} + \overline{CB} = \overline{AC} + 8 = 16$ → $\overline{AC} = 8$
M은 $\overline{AC}$의 중점 → A에서 4 떨어진 점.
B는 A에서 16 떨어진 점.
$\overline{BM} = 16 - 4 = 12$ cm

※ 두 경우 모두 가능한 답.
★ "한 직선 위에" 라는 조건만 있으면 점 순서가 정해지지 않을 수 있어요. 경우를 나눠서 생각!
