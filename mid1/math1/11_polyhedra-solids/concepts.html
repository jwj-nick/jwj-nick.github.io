⬢ 개념 탐구 — 다면체와 회전체

1. 다면체란?

다면체(polyhedron) = 다각형 면들로만 둘러싸인 입체도형.

💚 다면체의 어원: 다(多, poly-) + 면(-hedron) — "여러 면을 가진 것" (그리스어).
⚠️ 모든 면이 평면 다각형이어야 함. 곡면이 하나라도 있으면 다면체 아님 (원기둥은 다면체 ❌).

구성요소

정육면체 — 면 6, 모서리 12, 꼭짓점 8
요소정의표기
면 (face)평평한 다각형 부분$F$
모서리 (edge)두 면이 만나는 선분$E$
꼭짓점 (vertex)모서리들이 만나는 점$V$
🔍 정육면체에서: $V = 8, E = 12, F = 6$.
$V - E + F = 8 - 12 + 6 = \mathbf{2}$.
이 식이 오일러 공식! Page 4에서 다시 만나요.

2. 각기둥 · 각뿔 · 각뿔대

밑면이 $n$각형일 때, 세 종류 다면체의 $V, E, F$는?

현재: $n=$3 ($n$각기둥 / $n$각뿔 / $n$각뿔대)
각기둥각뿔각뿔대
밑면2개 (합동, 평행)1개2개 (닮음)
옆면 모양직사각형삼각형사다리꼴
$V$6 = $2n$4 = $n+1$6 = $2n$
$E$9 = $3n$6 = $2n$9 = $3n$
$F$5 = $n+2$4 = $n+1$5 = $n+2$
🔍 각기둥과 각뿔대는 $V, E, F$가 같다! 차이는 옆면 모양 (직사각형 vs 사다리꼴).
다 외울 필요 없어요. $n$각기둥의 모서리 = $3n$ 만 외우면 나머지는 비교로 풀 수 있어요.
⚠️ 자주 하는 실수: $n$각기둥 모서리 = $2n$ 으로 적기 (오답). 정답은 $3n$ — 위 $n$ + 아래 $n$ + 옆 $n$.

4. 왜 정다면체는 5종뿐일까?

한 꼭짓점에 모이는 정다각형들의 내각의 합 < 360° 이어야 입체가 닫혀요.

  • 360° 면 합 → 평면 (입체 안 됨)
  • 360° 초과 → 닫히지 않음
면 모양한 내각꼭짓점에 모인 수각 합입체?
정삼각형60°3180°✅ 정사면체
정삼각형60°4240°✅ 정팔면체
정삼각형60°5300°✅ 정이십면체
정삼각형60°6360°❌ 평면
정사각형90°3270°✅ 정육면체
정사각형90°4360°❌ 평면
정오각형108°3324°✅ 정십이면체
정오각형108°4432°❌ 360° 초과
정육각형120°3360°❌ 평면
🌟 그래서 정확히 3+1+1 = 5종!
• 정삼각형으로 만든 정다면체 3종 (사·팔·이십)
• 정사각형으로 만든 정다면체 1종 (육)
• 정오각형으로 만든 정다면체 1종 (십이)
• 정육각형 이상은 평면이 되어버림 — 안 됨.

5. 회전체 4종

회전체 = 평면도형을 한 직선(회전축) 둘레로 1회전 시킬 때 생기는 입체.

💚 회전축: 회전의 중심이 되는 직선. 평면도형의 변일 수도, 외부 직선일 수도.
⚠️ 각기둥은 회전체가 아니다 — 각기둥은 다면체. 곡면 없음.
회전 전 도형회전축회전체
직사각형한 변원기둥
직각삼각형직각낀 한 변원뿔
사다리꼴평행하지 않은 한 변원뿔대
반원지름구

모선 vs 높이 (★)

원뿔에서:

  • 높이: 꼭짓점 → 밑면 (수직 거리)
  • 모선 $l$: 꼭짓점 → 밑면 둘레 (옆면 위)
  • $l^2 = h^2 + r^2$ (피타고라스)

6. 회전체의 단면 (★ 핵심)

같은 회전체라도 어떻게 자르냐에 따라 단면이 다르다!

① 회전축에 수직인 단면 — 항상 원

② 회전축을 포함하는 단면 (= 축단면) — 좌우대칭 도형

회전체수직 단면축단면
원기둥원직사각형
원뿔원이등변삼각형
원뿔대원(등변)사다리꼴
구원원 (지름이 단면의 지름)
⚠️ 자주 하는 실수:
❌ "원뿔의 수직 단면 = 이등변삼각형" — 이등변삼각형은 축단면, 수직은 원.
❌ "원기둥의 축단면 = 원" — 축단면은 직사각형, 수직이 원.
💡 외우는 법: 수직 = 원, 축단면 = 좌우대칭 도형.

7. 회전체 전개도

전개도: 입체 표면을 평면에 펼친 그림.

원기둥의 전개도

★ 옆면 직사각형의 가로 = 밑면 둘레 $2\pi r$

원뿔의 전개도

★ 옆면 부채꼴의 반지름 = 모선 $l$, 호 = 밑면 둘레 $2\pi r$
💡 구는 전개도가 없음 — 평면에 완전히 펼칠 수 없어요. 그래서 지구본을 평면 지도로 만들면 왜곡이 생긴답니다 (메르카토르 도법).

직접 풀어보세요 ✏️

밑면 반지름이 $3$인 원기둥의 옆면을 펼치면 직사각형이 돼요. 직사각형의 가로의 길이는?

✅ 정답: $\boxed{6\pi}$
풀이: 밑면 둘레 = $2\pi r = 2\pi(3) = 6\pi$. 이게 옆면 직사각형의 가로!

밑면 반지름이 $4$, 모선이 $10$인 원뿔의 옆면(부채꼴) 호의 길이는?

✅ 정답: $\boxed{8\pi}$
풀이: 부채꼴 호의 길이 = 밑면 둘레 = $2\pi r = 2\pi(4) = 8\pi$.
