유형 3 · H1

한 꼭짓점에 정삼각형 6개가 모이면 정다면체가 만들어질까요? 그 이유와 함께 답하세요.
🤔 잠깐, 손으로 풀어보세요 정삼각형 한 내각은 몇 도인가요? 6개가 모이면 각의 합이 어떻게 될지 계산해봐요.
1단계 · Page 1

입체가 만들어지는 조건

왜 이 단계? — 정다면체가 만들어지려면 꼭짓점에서 각의 합이 얼마여야 하는지 알아야 해요.
📖 용어 노트 꼭짓점 각의 합 조건: 입체가 되려면 꼭짓점에 모인 각의 합이 360°보다 작아야 해요
왜 360°보다 작아야 하나요?

여러 면이 한 꼭짓점에서 만나 입체로 "솟아오르려면" 각의 합이 완전히 평면($360°$)이 되지 않아야 해요.

각의 합 < $360°$ → 입체 가능
각의 합 $= 360°$ → 평면 (입체 불가)
각의 합 > $360°$ → 불가능 (면이 겹침)
정삼각형 정다면체 3종
정삼각형 3개: $3 \times 60° = 180°$ → 정사면체 ✓
정삼각형 4개: $4 \times 60° = 240°$ → 정팔면체 ✓
정삼각형 5개: $5 \times 60° = 300°$ → 정이십면체 ✓

2단계 · Page 2

정삼각형 6개 분석

왜 이 단계? — 6개가 모일 때 각의 합을 직접 계산해요.
📖 용어 노트 정삼각형 한 내각 = $60°$
각의 합 계산
$6 \times 60° = 360°$

360°! 이는 정확히 평면이에요.

결론

각의 합이 $360°$이면 평면이 되어 입체가 만들어지지 않아요.

$6 \times 60° = 360°$ → 평면 (벌집 모양) → 정다면체 불가

실제로 정삼각형 6개를 한 점에 모아보면 육각형 타일 모양(벌집)이 되어 평면에 깔려요.

3단계 · Page 3

정다면체가 5종뿐인 이유

왜 이 단계? — 이 문제의 더 큰 맥락, 왜 정다면체가 정확히 5종인지 이해해요.
📖 용어 노트 5가지 정다면체: 정사면체(4), 정육면체(6), 정팔면체(8), 정십이면체(12), 정이십면체(20)
각 정다각형별 가능 개수
정삼각형(60°): 3·4·5개 가능 (3종)
정사각형(90°): 3개만 (3×90°=270°) — 4개 = 360° 평면
정오각형(108°): 3개만 (3×108°=324°) — 4개 = 432° 불가
정육각형(120°): 3개 = 360° → 평면 → 불가

→ 총 5종의 정다면체만 가능해요.
