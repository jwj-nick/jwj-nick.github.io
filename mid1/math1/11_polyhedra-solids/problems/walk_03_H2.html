단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 다면체: 다각형으로만 둘러싸인 입체도형  ·  꼭짓점(V)·모서리(E)·면(F)  ·  오일러 공식: $V-E+F=2$  ·  정다면체: ①모든 면이 합동인 정다각형 ②각 꼭짓점에 모인 면의 수가 같음  ·  회전체: 평면도형을 직선(축) 주위로 1회전시켜 만든 입체
유형 3 · H2

정육각형 정다면체가 존재하지 않는 이유는? (한 꼭짓점에 모이는 정육각형 수에 대해 분석하세요.)
🤔 잠깐, 손으로 풀어보세요 정육각형의 한 내각은 몇 도인가요? 입체 꼭짓점이 만들어지려면 최소 몇 개의 면이 모여야 하나요? 그렇게 모일 때 각의 합은 얼마인지 계산해봐요.
1단계 · Page 1

정육각형의 내각과 입체 조건

왜 이 단계? — 정육각형 내각과 꼭짓점 각의 합 조건을 확인해요.
📖 용어 노트 정육각형 한 내각 = $\dfrac{(6-2) \times 180°}{6} = 120°$  ·  입체 조건: 꼭짓점 각의 합 < $360°$
정육각형 내각 계산
정육각형 한 내각 $= \dfrac{(6-2) \times 180°}{6} = \dfrac{720°}{6} = 120°$
꼭짓점에 최소 3개 이상 모여야 입체

다면체 꼭짓점이 되려면 면이 최소 3개 이상 모여야 해요 (2개면 평평함).

정육각형 3개 모이면: $3 \times 120° = 360°$

$360°$는 평면 — 입체 불가!

2단계 · Page 2

4개 이상은 어떨까요?

왜 이 단계? — 3개도 안 되는데 4개 이상은 더 안 되는 이유를 확인해요.
📖 용어 노트 정육각형 내각 $= 120°$
4개 이상 분석
정육각형 4개: $4 \times 120° = 480°$ → $360°$ 초과 → 불가

4개 이상은 각의 합이 360°를 넘어 면들이 겹쳐요.

결론

정육각형은 3개 모이면 평면, 4개 이상 모이면 불가. 따라서 정육각형 정다면체는 절대 존재할 수 없어요.

벌집 무늬(정육각형 타일링)가 바로 3개가 360° 딱 맞아 평면을 덮는 예시!

정답 + 풀이 흐름 요약

📖 용어 노트 정육각형 내각 $= 120°$  ·  입체 조건: 꼭짓점 각의 합 < $360°$
정육각형 3개 모이면 $3 \times 120° = 360°$ → 평면
3개 미만으로는 닫히지 않고, 3개는 평면, 4개는 초과 → 정다면체 불가
전체 풀이 흐름 — 3단계
1
정육각형 한 내각 $= 120°$ 확인
2
최소 3개 모였을 때: $3 \times 120° = 360°$ → 평면 (입체 불가)
3
4개 이상: $4 \times 120° = 480° > 360°$ → 불가 → 정육각형 정다면체 존재 불가
💡 큰 그림 정다면체는 정삼각형(60°)·정사각형(90°)·정오각형(108°) 면만 가능해요. 정육각형(120°)부터는 3개만 모여도 360°이므로 불가!
