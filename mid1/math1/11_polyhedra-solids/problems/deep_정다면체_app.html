이 유형이 어려운 이유

정다면체는 평생 처음 만나는 도형들이에요. 그리고 가장 큰 함정은 두 가지:

  1. "정"이라는 글자만 보고 정다면체로 착각 — 정사각뿔, 정삼각기둥은 정다면체 ❌
  2. "정다면체는 더 많을 것 같다"는 직관 — 정확히 5종뿐

이 단원의 핵심은 왜 5종뿐인지 직접 이해하는 것이에요.

핵심 아이디어 ① — 정다면체의 두 조건

정다면체는 다음 두 조건을 모두 만족하는 다면체:

  1. 모든 면이 합동인 정다각형
  2. 각 꼭짓점에 모이는 면의 수가 모두 같음
⚠️ 둘 다 만족해야 정다면체. 하나만 만족하면 정다면체 아니에요.

★ 단계별 풀이 — 왜 정확히 5종뿐인가?

Step 1. 한 꼭짓점 조건

다면체의 한 꼭짓점에 모인 다각형들의 내각의 합 < 360° 이어야 입체가 닫혀요.

  • 360° 정확히 → 평면이 됨 (입체 안 됨)
  • 360° 초과 → 닫히지 않고 평평해지지도 못함

Step 2. 면 종류별 분석

Step 3. 결론

위 표에서 ✅ 표시 = 정확히 5종:

정사면체 + 정팔면체 + 정이십면체 + 정육면체 + 정십이면체

정삼각형 면으로 3종 (3·4·5 모임), 정사각형 면으로 1종, 정오각형 면으로 1종. 정육각형 이상은 만들 수 없음.

★ 쌍대 (Dual) — 신기한 짝

5종 정다면체에는 짝이 있어요:

입체 A입체 B$V \leftrightarrow F$ 관계
정사면체정사면체 (자기 자신)$V=F=4$
정육면체 ($V=8, F=6$)정팔면체 ($V=6, F=8$)$V$ ↔ $F$ 교환
정십이면체 ($V=20, F=12$)정이십면체 ($V=12, F=20$)$V$ ↔ $F$ 교환

두 입체에서 한 입체의 각 면 중심을 꼭짓점으로 하면 다른 입체가 만들어져요. 이걸 쌍대(dual) 관계라고 해요.

자주 하는 실수
❌
정사각뿔, 정삼각기둥을 정다면체로 착각
"정"자가 붙어도 정다면체 아닐 수 있음. 정사각뿔: 옆면(정삼각형)과 밑면(정사각형) 합동 X. 정삼각기둥: 옆면(정사각형)과 밑면(정삼각형) 합동 X.
❌
"정육각형 정다면체가 있을 것 같다"
정육각형 한 내각 = 120°. 한 꼭짓점에 3개만 모여도 360° → 평면. 입체 못 만듦.
❌
정다면체가 5개보다 많다고 착각
위 분석으로 정확히 5종. 더는 없어요. (수학에서 가장 우아한 사실 중 하나!)
추가 문제 (5-c와 다름)
L
정십이면체의 면의 수, 꼭짓점의 수, 모서리의 수의 합 ($V+E+F$)은?
정십이면체: $V=20, E=30, F=12$. 합 = $62$.
M
한 꼭짓점에 정삼각형 4개가 모인 정다면체가 있어요. 한 꼭짓점에 모인 각의 합과 이 정다면체 이름은?
한 꼭짓점 각 합 = $4 \times 60° = 240°$. $< 360°$이므로 입체 가능. 정다면체 = 정팔면체.
M
정육면체의 모든 면이 정사각형이고, 한 꼭짓점에 모인 각 합이 $270°$ 인 이유를 설명하세요.
정사각형 한 내각 = $90°$. 한 꼭짓점에 정사각형 3개 모임 → 각 합 = $3 \times 90° = 270° < 360°$ → 입체 가능 → 정육면체.
H
정사면체와 정팔면체에서 모서리 수의 비는?
정사면체 $E = 6$, 정팔면체 $E = 12$. 비 = $6 : 12 = 1 : 2$.
H
"각 꼭짓점에 모이는 면의 수"가 모든 정다면체에서 같은가요? 답하고 5종 각각의 값을 적어보세요.
정다면체마다 다름.
정사면체 → 3개
정육면체 → 3개
정팔면체 → 4개
정십이면체 → 3개
정이십면체 → 5개
💡 한 정다면체 안에서는 모든 꼭짓점 모임 수가 같아야 정의 만족 (그게 정다면체 둘째 조건). 그러나 다른 정다면체끼리는 서로 다름.
🪐 케플러의 우주 모델 (1596) — 25살 케플러는 5개 정다면체로 6개 행성 궤도를 설명하려 했어요. 가장 안쪽 수성 → 정팔면체 → 금성 → 정이십면체 → 지구 → 정십이면체 → 화성 → 정사면체 → 목성 → 정육면체 → 토성. 모델은 결국 틀렸어요. 그러나 그 시도가 케플러 3법칙 → 뉴턴 만유인력 → 현대 물리학으로 이어졌어요.

🔗 연결 개념

위로: 오일러 공식 $V-E+F=2$는 정다면체뿐 아니라 모든 (구멍 없는) 다면체에서 성립 — Unit 12에서 다시 만남.

옆으로: 정다면체의 면이 정다각형이라는 조건은 Unit 09(다각형) 정다각형 단원으로 직접 이어짐. 정육각형이 정다면체를 못 만든다는 사실은 정육각형의 한 내각이 120°이기 때문 — 다각형 단원에서 배운 공식.

아래로: 케플러-푸앵소 다면체(별 모양 정다면체) 4종은 정다면체 5종의 확장 버전. 면이 면을 가로지르는 신기한 입체. 연구심화 자료로!
