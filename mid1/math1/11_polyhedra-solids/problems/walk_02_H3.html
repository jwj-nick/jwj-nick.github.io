유형 2 · H3

$n$각뿔과 $n$각기둥의 면 수의 차는 항상 얼마인가요? ($n \geq 3$)
🤔 잠깐, 손으로 풀어보세요 $n$각기둥의 면 수와 $n$각뿔의 면 수 공식을 각각 적고, 빼봐요. $n$이 사라질까요?
1단계 · Page 1

면 수 공식 정리

왜 이 단계? — 각기둥과 각뿔의 면 수 공식을 나란히 써봐요.
📖 용어 노트 $n$각기둥: $F = n+2$  ·  $n$각뿔: $F = n+1$
면 수 공식
$n$각기둥: $F = n + 2$ (밑면 2 + 옆면 $n$)
$n$각뿔: $F = n + 1$ (밑면 1 + 옆면 $n$)
차 계산
$(n+2) - (n+1) = 1$

$n$이 어떤 값이어도 항상 차 = 1이에요!

2단계 · Page 2

직관적으로 이해하기

왜 이 단계? — 왜 차가 1인지 구조적으로 이해해요.
📖 용어 노트 각기둥 = 위아래 밑면 2개 + 옆면 $n$개  ·  각뿔 = 밑면 1개 + 옆면 $n$개
구조 비교
$n$각기둥 = 밑면 2개 + 직사각형 옆면 $n$개 → $F = n + 2$
$n$각뿔 = 밑면 1개 + 삼각형 옆면 $n$개 → $F = n + 1$

차이는 딱 하나 — 각기둥에는 윗 밑면이 1개 더 있다는 것.

각뿔은 위가 점으로 모이므로 윗 밑면이 없어요. 그래서 면이 1개 적어요.

💡 구체적인 예 삼각기둥($F=5$) vs 삼각뿔($F=4$) → 차 = 1
사각기둥($F=6$) vs 사각뿔($F=5$) → 차 = 1
오각기둥($F=7$) vs 오각뿔($F=6$) → 차 = 1

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 $n$각기둥: $F=n+2$  ·  $n$각뿔: $F=n+1$
$n$각기둥과 $n$각뿔의 면 수 차 = $\mathbf{1}$
$n$에 관계없이 항상 1
전체 풀이 흐름 — 2단계
1
$n$각기둥 $F = n+2$, $n$각뿔 $F = n+1$
2
차 $= (n+2)-(n+1) = 1$ ($n$에 무관)
💡 직관 각기둥은 위가 닫혀있고 (윗 밑면 있음), 각뿔은 위가 점 하나로 끝나요 (윗 밑면 없음). 그래서 면이 딱 1개 차이!
