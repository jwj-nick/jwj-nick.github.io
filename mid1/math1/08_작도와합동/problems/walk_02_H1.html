문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 합동(≅): 두 도형이 모양과 크기가 완전히 같음 (≡ 기호 쓰기도 함)  ·  대응: 합동인 두 도형에서 서로 겹치는 꼭짓점·변·각  ·  SSS 합동: 세 변의 길이가 모두 같음  ·  SAS 합동: 두 변의 길이와 그 사이 각이 같음  ·  ASA 합동: 두 각의 크기와 그 사이 변의 길이가 같음
유형 2 · H1

∠XOY를 옮길 때 사용한 컴퍼스 절차에서 다음 중 변경 불가한 단계를 모두 고르시오.

(가) 1단계 호의 반지름 (O 중심)
(나) 2단계 새 직선 위 호의 반지름 (O' 중심)
(다) 3단계 새 호의 반지름 (A' 중심, AB 길이)
(라) 호를 그리는 시계/반시계 방향
🤔 잠깐, 손으로 풀어보세요 각 옮기기 절차를 떠올려 봐요. 어떤 단계의 길이가 바뀌면 두 삼각형이 SSS 합동이 깨질까요?
1단계 · Page 1

각 옮기기 절차 복습

왜 이 단계? — 어떤 단계가 변경 불가인지 알려면 먼저 절차 전체를 명확히 정리해야 해요.
📖 용어 노트 합동(≅): 두 도형이 모양과 크기 완전히 같음  ·  SSS 합동: 세 변 같음
각 옮기기 4단계
1단계: O 중심으로 호 → OX 위 점 A, OY 위 점 B 잡기
2단계: 새 직선 위 O' 중심으로 같은 반지름으로 호 → 새 직선 위 점 A', B' 후보 잡기
3단계: A 중심 컴퍼스를 B까지 벌려 AB 길이 잡기
4단계: A' 중심으로 AB 길이만큼 호 → 교점 B'
5단계: O'B' 잇기 → 각 옮기기 완료
왜 이 절차가 작동하는가?

삼각형 OAB와 O'A'B'가 SSS 합동이기 때문이에요.

$\overline{OA} = \overline{O'A'}$ (1·2단계 같은 반지름)
$\overline{OB} = \overline{O'B'}$ (1·2단계 같은 반지름)
$\overline{AB} = \overline{A'B'}$ (3·4단계)
→ SSS 합동 → ∠AOB = ∠A'O'B' (대응각)
🤔 핵심 질문 세 변 중 어느 하나라도 달라지면 SSS 합동이 깨져요. 그렇다면 "변경 불가"인 조건은 무엇일까요?

2단계 · Page 2

(가)와 (나): 두 반지름의 관계

왜 이 단계? — (가)와 (나) 각각을 독립적으로 바꿀 수는 없어요. 둘의 관계가 핵심이에요.
📖 용어 노트 SSS 합동: 세 변 같음  ·  대응각: 합동인 두 삼각형에서 같은 위치의 각
(가) = (나) 조건

1단계와 2단계의 반지름이 서로 같아야 해요.

(가) 반지름 $r_1$ (O 중심)
(나) 반지름 $r_2$ (O' 중심)
필요 조건: $r_1 = r_2$
→ $\overline{OA} = \overline{OB} = \overline{O'A'} = \overline{O'B'}$ 보장

만약 $r_1 \neq r_2$이면 삼각형 두 변의 길이가 달라져서 SSS 합동이 깨져요.

단, (가) 자체의 값은 자유

$r_1$의 크기 자체(예: 3cm냐 5cm냐)는 자유예요.
중요한 건 (가)와 (나)가 같아야 한다는 것.

$r_1 = 3\text{cm}$, $r_2 = 3\text{cm}$ → OK
$r_1 = 5\text{cm}$, $r_2 = 5\text{cm}$ → OK
$r_1 = 3\text{cm}$, $r_2 = 5\text{cm}$ → 합동 깨짐 ✗

3단계 · Page 3

(다)와 (라): AB 길이와 방향

왜 이 단계? — (다)의 길이는 반드시 AB여야 하고, (라) 방향은 결과에 영향 없어요.
📖 용어 노트 SSS 합동: 세 변 같음  ·  합동(≅): 모양·크기 완전히 같음
(다): AB 길이 — 변경 불가

3·4단계에서 A' 중심 호의 반지름은 반드시 원본의 $\overline{AB}$ 길이여야 해요.

원본: $\overline{AB}$ (A에서 B까지 거리)
3단계: 컴퍼스를 A에서 B까지 벌림 → 반지름 = $\overline{AB}$
4단계: A' 중심으로 같은 벌림으로 호 → $\overline{A'B'} = \overline{AB}$ 보장
만약 다른 길이 사용 → $\overline{A'B'} \neq \overline{AB}$ → SSS 합동 깨짐 ✗
(라): 방향 — 변경 가능

호를 시계 방향으로 그리든 반시계 방향으로 그리든 교점의 위치는 달라지지 않아요.

방향은 어느 쪽으로 호를 그리느냐일 뿐
교점(B')의 위치와 삼각형의 모양은 동일
→ 방향은 결과에 영향 없음 → 변경 가능
⚠️ 주의 방향(라)은 자유이지만, 보통 원본과 같은 방향으로 통일하는 것이 관행이에요. 결과의 정확성과는 무관해요.

정답 + 풀이 흐름 요약

📖 용어 노트 합동(≅): 두 도형 모양·크기 완전히 같음  ·  SSS 합동: 세 변 같음
(가) = (나) 서로 같아야 하고, (다)는 AB 길이와 같아야 함
(라) 방향은 변경 가능
전체 풀이 흐름
1
각 옮기기의 정확성은 삼각형 OAB ≡ O'A'B' (SSS 합동)에서 나옴
2
(가)와 (나): 두 반지름이 서로 같아야 $\overline{OA}=\overline{O'A'}$, $\overline{OB}=\overline{O'B'}$ 보장
3
(다): A' 중심 호의 반지름은 반드시 AB 길이여야 $\overline{A'B'}=\overline{AB}$ 보장
4
(라): 방향은 교점 위치에 영향 없음 → 자유
⚠️ 핵심 통찰 작도의 정확성은 "어떤 길이가 어떤 길이와 같아야 하는가"라는 합동 조건에서 나와요. 어디에서 벌림이 바뀌면 합동이 깨지는지 항상 의식해요.
