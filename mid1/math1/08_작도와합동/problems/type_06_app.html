유형 6 — 합동 식별·대응 ★

합동 표기 순서가 곧 대응이에요
← 단원 8 홈
1
L
$\triangle ABC \equiv \triangle DEF$일 때, 점 A의 대응점은?
표기 순서가 곧 대응이에요. ABC의 첫째는 DEF의 첫째.
점 D
$\triangle ABC \equiv \triangle DEF$ → A↔D, B↔E, C↔F
(같은 자리)
2
L
$\triangle ABC \equiv \triangle DEF$일 때, ∠C의 대응각은?
C는 ABC에서 3번째.
∠F
C ↔ F (둘 다 3번째 자리) → ∠C ↔ ∠F
3
L
$\triangle ABC \equiv \triangle DEF$일 때, 변 $\overline{AB}$의 대응변은?
A의 대응 + B의 대응. 두 글자 묶음으로.
$\overline{DE}$
A↔D, B↔E → $\overline{AB}$의 대응변은 $\overline{DE}$.
1
M
$\triangle ABC \equiv \triangle DEF$이고 $\overline{AB} = 8$ cm, $\overline{BC} = 5$ cm, $\overline{CA} = 7$ cm이다. $\overline{DF}$의 길이는?
$\overline{DF}$의 대응변을 찾아요. D ↔ ?, F ↔ ?
7 cm
D↔A, F↔C → $\overline{DF}$의 대응변은 $\overline{AC} = \overline{CA} = 7$ cm.
($\overline{AC}$와 $\overline{CA}$는 같은 선분)
따라서 $\overline{DF} = 7$ cm.
2
M
$\triangle ABC \equiv \triangle DEF$이고 $\angle A = 70°, \angle B = 60°$이다. $\angle F$의 크기는?
∠F의 대응각을 먼저 찾고, 삼각형 내각의 합을 사용해요.
∠F = 50°
F ↔ C → ∠F = ∠C
$\angle C = 180° - \angle A - \angle B = 180° - 70° - 60° = 50°$
따라서 ∠F = 50°.
3
M
두 삼각형이 합동이고 $\triangle PQR \equiv \triangle XYZ$일 때 다음 중 옳은 것을 모두 고르시오.
(가) $\overline{PQ} = \overline{XY}$   (나) $\overline{QR} = \overline{YZ}$   (다) ∠R = ∠X   (라) ∠Q = ∠Y   (마) $\overline{PR} = \overline{YZ}$
표기 순서: P↔X, Q↔Y, R↔Z. 같은 자리끼리 대응.
(가), (나), (라)
P↔X (1), Q↔Y (2), R↔Z (3)
(가) $\overline{PQ} \leftrightarrow \overline{XY}$ ✓
(나) $\overline{QR} \leftrightarrow \overline{YZ}$ ✓
(다) ∠R ↔ ∠Z (∠X 아님) X
(라) ∠Q ↔ ∠Y ✓
(마) $\overline{PR} \leftrightarrow \overline{XZ}$ (YZ 아님) X
★ 표기 순서가 어긋나면 대응도 어긋나요. 글자 위치(1·2·3번째)를 정확히 봐요.
1
H
아래 그림에서 두 삼각형이 합동이라고 한다. 적절한 합동 표기를 다음 중 모두 고르시오.

(왼쪽 삼각형 꼭짓점: A, B, C로 표시. 오른쪽 삼각형: A에 대응하는 점은 P, B에 대응하는 점은 Q, C에 대응하는 점은 R로 그려져 있음)
(가) $\triangle ABC \equiv \triangle PQR$   (나) $\triangle ACB \equiv \triangle PRQ$   (다) $\triangle BAC \equiv \triangle QPR$   (라) $\triangle CBA \equiv \triangle PQR$
대응이 맞는 표기는 여러 가지예요. A↔P, B↔Q, C↔R 만 일관되게 유지하면 OK.
(가), (나), (다)
A↔P, B↔Q, C↔R 만 유지하면 됨.
(가) ABC ↔ PQR ✓ (1↔P, 2↔Q, 3↔R)
(나) ACB ↔ PRQ ✓ (A↔P, C↔R, B↔Q)
(다) BAC ↔ QPR ✓ (B↔Q, A↔P, C↔R)
(라) CBA ↔ PQR ✗ (C↔P 안 됨, C↔R이어야)
→ 두 삼각형의 대응 자체가 정해져 있을 때, 그 대응을 일관되게 따른 표기는 모두 옳음.
2
H
$\triangle ABC \equiv \triangle DEF$이다. 다음 중 항상 참인 것을 모두 고르시오.
(가) 둘레의 길이가 같다   (나) 넓이가 같다   (다) 가장 긴 변의 길이가 같다   (라) 위치(좌표)가 같다   (마) 모양과 크기가 같다
합동 = 포개진다. 이동시켜서 겹쳐지면 됨. 위치는 보통 다름.
(가), (나), (다), (마)
합동이면 모든 대응변·대응각이 같으므로:
(가) 둘레 = 세 변의 합 → 같음 ✓
(나) 넓이도 같음 (모양·크기 같으므로) ✓
(다) 가장 긴 변도 대응변이 있어 같음 ✓
(라) 위치는 다른 자리에 있어도 합동 ✗ (위치는 무관)
(마) 합동 = 모양과 크기가 같음 ✓
3
H
두 사각형 ABCD와 EFGH가 합동이다 ($\square ABCD \equiv \square EFGH$). $\overline{AB} = 5, \overline{BC} = 7, \overline{CD} = 4, \overline{DA} = 6$일 때 $\overline{GH}$와 $\overline{HE}$의 길이는?
사각형도 같은 원리. A↔E, B↔F, C↔G, D↔H. 변은 두 글자 묶음으로 대응.
$\overline{GH} = 4, \overline{HE} = 6$
대응: A↔E, B↔F, C↔G, D↔H
$\overline{GH}$의 대응변: G↔C, H↔D → $\overline{CD} = 4$
$\overline{HE}$의 대응변: H↔D, E↔A → $\overline{DA} = 6$
→ 사각형 합동도 표기 순서가 대응을 정함. 삼각형과 같은 원리.
★ 합동 표기 순서 — 도형의 종류(삼각형·사각형·다각형)와 무관하게 항상 적용되는 규칙.
