문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 합동(≅): 두 도형이 모양과 크기가 완전히 같음  ·  대응: 합동인 두 도형에서 서로 겹치는 꼭짓점·변·각  ·  SSS 합동: 세 변의 길이가 모두 같음  ·  SAS 합동: 두 변의 길이와 그 사이 각이 같음  ·  ASA 합동: 두 각의 크기와 그 사이 변의 길이가 같음
유형 5 · H1

"두 변 7cm, 5cm와 5cm 변의 끝 각 30°가 주어졌다. 5cm 변의 다른 끝과 7cm 변의 다른 끝을 연결한다."

이 작도 조건이 SSA(모호한 경우)에 해당하는지 확인하고, 가능한 삼각형의 개수를 구하시오.
🤔 잠깐, 손으로 풀어보세요 30°각이 있는 꼭짓점 O에서 출발해봐요. 한 변 OA = 5cm. 다른 변 방향은 30° 기울어진 방향. A에서 7cm 떨어진 점 B가 그 다른 변 위에 있어야 해요. A에서 반지름 7cm의 호를 그리면 다른 변과 몇 번 만날까요?
1단계 · Page 1

SSA가 왜 모호한가?

왜 이 단계? — 끼인각이 아닌 각이 주어지면 왜 삼각형이 하나로 정해지지 않는지 원리를 파악해요.
📖 용어 노트 끼인각: 두 변 사이의 각 (SAS에서 필수)  ·  SSA: 두 변과 끼인각이 아닌 한 각 → 합동 보장 안 됨
SSA 모호성의 원리

꼭짓점 O에 30° 각, 한 변 $\overrightarrow{OX}$ 위에 A (OA = 5cm). 다른 변 $\overrightarrow{OY}$는 30° 방향.

B는 $\overrightarrow{OY}$ 위의 점이고, AB = 7cm 조건이 있어요.

A에서 반지름 7cm의 호를 그림 → $\overrightarrow{OY}$와의 교점이 B
호가 $\overrightarrow{OY}$와 몇 번 만나느냐 = 가능한 삼각형의 수
이 문제의 경우 — 2번 교차

AB = 7cm > OA sin(30°) = 5 × 0.5 = 2.5cm이고 AB = 7cm < OA = 5cm… 아니, AB = 7 > 5 이므로:

A에서 그린 호 (반지름 7cm)가 $\overrightarrow{OY}$와
한 점은 O 가까이 (날카로운 삼각형)
한 점은 O 멀리 (둔각 삼각형)
→ 두 교점 → 두 가지 모양의 삼각형 가능

2단계 · Page 2

두 가능한 삼각형 확인

왜 이 단계? — 실제로 두 삼각형이 가능함을 구체적으로 확인해요.
📖 용어 노트 SSA: 두 변과 끼인각 아닌 각 → 1개 또는 2개 또는 0개 삼각형
삼각형 1: B₁ (O 가까이)
O 중심, OA = 5cm, 30° 각
A에서 AB₁ = 7cm, B₁이 $\overrightarrow{OY}$ 위의 O 가까운 쪽
→ 삼각형 OAB₁: ∠O = 30°, OA = 5, AB₁ = 7 (날카로운 모양)
삼각형 2: B₂ (O 멀리)
A에서 AB₂ = 7cm, B₂가 $\overrightarrow{OY}$ 위의 O 먼 쪽
→ 삼각형 OAB₂: ∠O = 30°, OA = 5, AB₂ = 7 (둔각 모양)
(∠OAB₂는 둔각)
두 삼각형 모두 SSA 조건 만족

두 삼각형 모두 "∠O = 30°, OA = 5cm, AB = 7cm"을 공유해요. 하지만 모양이 달라요.

→ SSA는 합동 조건이 아님 확인 ✓
→ "두 변과 한 각"만으로는 삼각형이 하나로 정해지지 않음

3단계 · Page 3

SSA의 세 가지 경우

왜 이 단계? — 항상 2개가 아니에요. 호가 어떻게 만나느냐에 따라 0, 1, 2개가 될 수 있어요.
📖 용어 노트 SSA 모호성: 끼인각이 아닌 각 + 두 변 → 삼각형 개수가 0, 1, 2개 중 하나
SSA의 세 가지 경우
0개: A에서 그린 호가 $\overrightarrow{OY}$에 전혀 닿지 않을 때
   (AB가 너무 짧아 호가 다른 변까지 못 미침)

1개: A에서 그린 호가 $\overrightarrow{OY}$에 정확히 접할 때
   (AB = A에서 $\overrightarrow{OY}$까지의 수직 거리 → 직각삼각형)

2개: A에서 그린 호가 $\overrightarrow{OY}$를 두 점에서 만날 때 (이 문제의 경우)
⚠️ SSA ≠ 합동 조건 "두 변과 한 각이 같다"고 해서 두 삼각형이 합동이라고 할 수 없어요. 반드시 그 각이 끼인각인지 확인해야 해요.

SSA에 해당. 가능한 삼각형 2개
(A에서 반지름 7cm 호가 $\overrightarrow{OY}$를 두 점에서 만남)
전체 풀이 흐름
1
조건 분석: ∠O = 30° (끼인각 아님), OA = 5cm, AB = 7cm → SSA

3
교점 B₁ (O 가까이), B₂ (O 멀리) → 두 가지 삼각형
4
두 삼각형 모두 "∠O = 30°, OA = 5, AB = 7" 만족 → SSA는 합동 조건 아님 ✓
