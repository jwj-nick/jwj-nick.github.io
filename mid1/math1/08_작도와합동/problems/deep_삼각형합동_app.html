★ 시험 빈출 1순위
← 단원 8 홈
이 유형이 어려운 이유

SSS·SAS·ASA가 비슷해 보여 헷갈리고, "끼인각" 개념이 추상적이에요. 또 "AAA는 안 되는 이유", "SSA의 모호성"을 이해해야 해요.

핵심 함정:
① "두 변과 한 각" → 그 한 각이 끼인각인지 확인 안 함 (SSA 함정)
② AAA = 닮음 ≠ 합동 (크기 정보 없음)
③ "각 두 개의 합 < 180°" 같은 조건 만족 여부 확인 안 함

핵심 아이디어 — 삼각형이 "유일하게" 정해지는 세 조건
조건의미예
SSS세 변 길이4, 5, 6
SAS두 변 + 끼인각4, 6, 끼인각 60°
ASA한 변 + 양 끝 두 각5, 50°, 70°
★ SAS의 "끼인각": 두 변 사이의 각. 두 변이 만나는 꼭짓점에 있어야 끼인각.

왜 AAA가 합동 조건이 아닌가?

한 변 1cm 정삼각형과 한 변 100cm 정삼각형: 둘 다 세 각이 60°·60°·60°.
→ 모양은 같지만 크기가 다름 → 합동 X (포개도 안 겹침).
→ AAA는 닮음(similar) 조건이지만 합동 조건이 아니에요.

합동 = 모양 + 크기 모두 같음

왜 SSA(끼인각 아닌)가 합동 조건이 아닌가?

두 변과 한 각만 같다고 해서 모양이 정해지지 않아요.
예: 한 각 30° + 한 변 5cm + 다른 변 4cm (끼인각 X) → 두 가지 모양 가능.
이를 "ambiguous case(모호한 경우)"라고 해요.

★ 끼인각은 두 변 사이에 있는 각. 두 변과 인접한 다른 각이면 SSA.
자주 하는 실수
❌
"두 변 + 한 각이면 무조건 SAS"
그 각이 두 변 사이에 있는지 확인 필수. 아니면 SSA → 합동 보장 X.
❌
"세 각이 같으니 합동"
AAA는 닮음만. 크기 정보(길이) 적어도 한 개가 필요해요.
❌
조건은 만족하지만 실제 삼각형 불가능 모름
2+3=5 < 6은 삼각형 부등식 위반. 100°+90°=190°는 내각 합 모순. 조건 충족 + 실현 가능성도 봐야.
추가 문제 (점진적 난이도)
PROBLEM 1 (L)
"두 변 5, 7과 끼인각 50°"는 어떤 작도 조건?
SAS — 두 변 + 끼인각 → SAS. 삼각형이 하나로 정해짐.
PROBLEM 2 (M)
한 변 8cm와 한 끝의 각 60°가 주어졌다. 추가로 어떤 정보가 있으면 SAS·ASA·SSS 중 어느 조건이 성립하나?
여러 가능성:
• 같은 끝의 또 다른 각(60°와 인접) → ASA
• 60° 각의 다른 변 길이 → SAS (60°가 끼인각)
• 다른 두 변의 길이 → SSS
→ 한 변과 한 각만으로는 부족. 어떤 정보를 추가하느냐에 따라 합동 조건이 결정.
PROBLEM 3 (H)
SAS 조건으로 작도한 삼각형이 SSS·ASA로도 같은 삼각형임을 보일 수 있는가? 그 의미는?
가능. 같은 삼각형은 어느 합동 조건으로도 식별됨.
SAS로 작도된 삼각형: 두 변 a, b + 끼인각 ∠C가 정해짐.
→ 삼각형이 유일하게 결정 → 세 변·세 각 모두 정해짐 → SSS·ASA로도 같은 결론 가능.
의미: 합동 조건 SSS·SAS·ASA는 같은 사실을 다른 정보로 표현한 것. 어느 조건이든 충분히 한 삼각형을 결정.
→ 「원론」에서 유클리드는 SAS(명제 4)로 시작해 다른 합동들을 차례로 증명.
이 유형 이해 후 무엇이 쉬워지나
→ 합동 식별·활용 (유형 6, 7)
작도 조건은 합동 조건과 같음. SSS·SAS·ASA를 이해하면 두 도형이 합동인지 판단할 때 어떤 조건을 사용할지 자연스럽게 보임.
이 유형이 어려운 이유

합동 표기 $\triangle ABC \equiv \triangle DEF$의 순서가 곧 대응이라는 약속이 학습자에게 어색해요. 표기를 마음대로 쓰는 실수가 자주 발생.

핵심 함정:
① $\triangle ABC \equiv \triangle DEF$를 보고 A↔E라고 착각
② $\overline{AB}$의 대응변을 $\overline{EF}$ 또는 $\overline{FD}$로 잘못 답함
③ 도형이 회전·뒤집힌 그림에서 대응을 못 찾음

핵심 아이디어 — 표기 순서 = 대응

대응 규칙

$\triangle ABC \equiv \triangle DEF$
→ 첫째끼리: A ↔ D
→ 둘째끼리: B ↔ E
→ 셋째끼리: C ↔ F

대응 종류찾는 법
대응점같은 자리(1번째, 2번째, 3번째)
대응변두 글자 묶음, 같은 자리 묶음과 짝
대응각꼭짓점이 같은 자리인 각
$\overline{AB}$의 대응변: A→D, B→E → $\overline{DE}$

같은 합동을 여러 표기로

$\triangle ABC \equiv \triangle DEF$ (A↔D, B↔E, C↔F)는 다음과 같이도 쓸 수 있어요:
$\triangle BAC \equiv \triangle EDF$ (B↔E, A↔D, C↔F)
$\triangle CBA \equiv \triangle FED$ (C↔F, B↔E, A↔D)
→ 대응만 일관되게 유지되면 여러 표기 모두 옳음.

자주 하는 실수
❌
표기 순서를 무시하고 대응 결정
$\triangle ABC \equiv \triangle EFD$를 ABC와 EFD가 합동이라고 표기하면 A↔E, B↔F, C↔D로 다른 도형 의미.
❌
대응변 찾을 때 글자 순서 무시
$\overline{AB}$의 대응변은 A↔D, B↔E이니 $\overline{DE}$. 임의로 $\overline{EF}$나 $\overline{FD}$ 답하면 X.
추가 문제 (점진적 난이도)
PROBLEM 1 (L)
$\triangle PQR \equiv \triangle ABC$이고 $\overline{QR} = 6$cm. $\overline{BC}$는?
6 cm
Q↔B, R↔C → $\overline{QR}$의 대응변은 $\overline{BC}$ → 6 cm.
PROBLEM 2 (M)
$\triangle ABC \equiv \triangle DEF$이고 $\angle A = 50°, \angle B = 60°, \overline{AB} = 8, \overline{BC} = 6, \overline{CA} = 5$. 다음을 모두 구하시오.
(1) $\angle F$   (2) $\overline{DE}$   (3) $\overline{EF}$   (4) $\overline{DF}$
(1) 70°   (2) 8 cm   (3) 6 cm   (4) 5 cm
(1) $\angle C = 180° - 50° - 60° = 70°$. F↔C → $\angle F = 70°$
(2) D↔A, E↔B → $\overline{DE}$의 대응변 = $\overline{AB}$ = 8
(3) E↔B, F↔C → $\overline{EF}$의 대응변 = $\overline{BC}$ = 6
(4) D↔A, F↔C → $\overline{DF}$의 대응변 = $\overline{AC}$ = 5
PROBLEM 3 (H)
두 삼각형이 합동이고, A↔Y, B↔Z, C↔X 대응이 성립한다. 가장 자연스러운 합동 표기를 적으시오. (3가지 방법으로)
$\triangle ABC \equiv \triangle YZX$ (가장 자연스럽게 ABC 순서로)
다른 표현:
$\triangle BCA \equiv \triangle ZXY$
$\triangle CAB \equiv \triangle XYZ$
→ 대응만 유지되면 글자 순서를 회전(rotate)해도 OK.
주의: $\triangle XYZ \equiv \triangle CAB$ 도 같은 의미. 어느 쪽을 먼저 쓸지는 자유.
이 유형 이해 후 무엇이 쉬워지나
→ 합동 활용 문제 (유형 7)
합동을 보인 후 "그러므로 어떤 변·각이 같다"는 결론 도출이 합동 활용의 핵심. 대응 규칙을 단단히 알면 결론을 빠르게 적을 수 있어요.
이 유형이 어려운 이유

그림에서 어떤 두 삼각형이 합동인지 직접 발견해야 하고, 어느 합동 조건(SSS·SAS·ASA)으로 보일지 판단해야 해요. 또한 "합동 → 대응 정보 활용"이라는 패턴에 익숙해져야 해요.

핵심 함정:
① 공통변·맞꼭지각·중점 같은 정보를 활용 못 함
② 합동 조건 식별 후 결론 도출 누락
③ 두 변과 한 각만 같을 때 끼인각 여부 확인 안 함

핵심 아이디어 — 합동의 활용 패턴

1단계: 같은 정보 모으기

  • 가정에서 직접 주어진 같은 변·각
  • 공통변: 두 삼각형이 공유하는 변 (자동으로 같음)
  • 맞꼭지각: 교차하는 두 직선의 마주보는 각
  • 중점: $\overline{AM} = \overline{MB}$
  • 이등분선: 같은 두 각
  • 이등변삼각형: 두 변 같음
  • 평행 + 엇각/동위각: 같은 두 각

2단계: 합동 조건 식별

모은 정보가 SSS·SAS·ASA 중 어느 조건을 만족하는지 확인.
특히 SAS는 끼인각 여부를 반드시 점검.

★ "두 변과 한 각" 발견 시: 그 각이 두 변 사이의 끼인각인지 확인! 끼인각이 아니면 SSA(합동 X).

3단계: 합동 결론 → 대응 정보 활용

$\triangle ABC \equiv \triangle DEF$를 보였으면:
→ 모든 대응변·대응각이 같음
→ 문제에서 묻는 변·각의 길이/크기 결론.

합동 보이기 → 대응 정보 추출 → 결론
자주 하는 실수
❌
"공통변" 활용 누락
두 삼각형이 한 변을 공유하면 그 변이 자동으로 두 삼각형 모두에 포함. 이게 SAS 합동의 한 변이 됨.
❌
합동을 보였지만 결론 도출 안 함
"합동이다"로 멈추면 안 됨. 합동의 결과로 어떤 변·각이 같은지가 답이 되는 경우가 많음.
❌
SSA를 SAS로 착각
두 변과 한 각이 있으면 무조건 SAS라고 단정. 그 각이 끼인각이 아니면 합동 보장 X.
추가 문제 (점진적 난이도)
PROBLEM 1 (L)
두 삼각형 ABC와 ABD가 변 $\overline{AB}$를 공유하고 $\overline{AC} = \overline{AD}, \angle BAC = \angle BAD$이다. 합동인가? 어느 조건?
합동, SAS
$\overline{AC} = \overline{AD}$ (가정)
$\angle BAC = \angle BAD$ (가정, 끼인각)
$\overline{AB}$ 공통
→ SAS 합동.
PROBLEM 2 (M)
정삼각형 ABC가 있다. 변 BC 위에 점 D, 변 CA 위에 점 E를 잡아 $\overline{BD} = \overline{CE}$가 되도록 했다. $\triangle ABD \equiv \triangle BCE$임을 보이시오.
SAS 합동
$\overline{AB} = \overline{BC}$ (정삼각형)
$\angle ABD = \angle BCE = 60°$ (정삼각형 내각, 끼인각)
$\overline{BD} = \overline{CE}$ (가정)
→ SAS 합동 ✓
$\therefore \triangle ABD \equiv \triangle BCE$
→ 결과로 $\overline{AD} = \overline{BE}$ 등도 알 수 있음.
PROBLEM 3 (H)
평행사변형 ABCD에서 대각선 AC와 BD의 교점을 O라 하자. $\triangle AOB \equiv \triangle COD$ 임을 보이고, 평행사변형의 대각선이 서로를 이등분함을 결론지으시오.
ASA 합동
$\overline{AB} = \overline{CD}$ (평행사변형의 대변)
$\angle OAB = \angle OCD$ ($\overline{AB} \parallel \overline{CD}$, 엇각)
$\angle OBA = \angle ODC$ (엇각)
→ 한 변($\overline{AB} = \overline{CD}$)과 양 끝의 두 각 같음 → ASA 합동 ✓
$\therefore \triangle AOB \equiv \triangle COD$
대응변에서:
$\overline{AO} = \overline{CO}$ (O가 AC의 중점)
$\overline{BO} = \overline{DO}$ (O가 BD의 중점)
→ 평행사변형의 대각선은 서로를 이등분한다는 결론!
→ 합동을 활용한 평면도형 증명의 대표 패턴.
이 유형 이해 후 무엇이 쉬워지나
→ 평면도형 증명 (중2~3, 고등)
합동을 활용한 증명은 평면도형 단원의 가장 기본적 도구예요. 이등변삼각형·평행사변형·다각형의 성질, 외심·내심까지 모두 합동 증명에서 출발해요. 이번 단원에서 단단히 잡으면 다음 단원이 쉬워져요.
