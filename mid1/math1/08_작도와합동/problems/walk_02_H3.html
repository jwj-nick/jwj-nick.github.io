단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 합동(≅): 두 도형이 모양과 크기가 완전히 같음  ·  대응: 합동인 두 도형에서 서로 겹치는 꼭짓점·변·각  ·  SSS 합동: 세 변의 길이가 모두 같음  ·  SAS 합동: 두 변의 길이와 그 사이 각이 같음  ·  ASA 합동: 두 각의 크기와 그 사이 변의 길이가 같음
유형 2 · H3

∠AOB가 주어져 있다 (예각). 한 직선 $l$ 위에 ∠AOB의 2배 크기의 각을 작도하려고 할 때, 핵심 아이디어를 설명하시오.
🤔 잠깐, 손으로 풀어보세요 길이의 2배가 "같은 길이를 두 번 이어 붙이기"였다면, 각의 2배는 어떻게 할까요? 각 옮기기를 두 번 하면 어떻게 될까요?
1단계 · Page 1

핵심 아이디어 — 각 옮기기를 두 번

왜 이 단계? — 길이의 2배와 완전히 같은 원리예요. "같은 각을 같은 방향으로 두 번 이어 붙이기"가 2배예요.
📖 용어 노트 각 옮기기: 원본 각과 같은 크기의 각을 다른 위치에 작도하는 절차
비교 — 길이의 2배 vs 각의 2배
길이 2배: P에서 $\overline{AB}$ 옮기기 → Q, Q에서 또 $\overline{AB}$ 옮기기 → R
결과: $\overline{PR} = 2\overline{AB}$

각의 2배: O'에서 ∠AOB 옮기기 → 새 변 OP, OP를 기준으로 다시 ∠AOB 옮기기 → 새 변 OQ
결과: $\angle XO'Q = 2\angle AOB$

핵심: 첫 번째 옮긴 각의 새 변(OP)을 두 번째 옮기기의 기준 변으로 사용해요.

💡 시각화 $\overrightarrow{O'X}$에서 시작 → ∠AOB만큼 열면 $\overrightarrow{O'P}$
$\overrightarrow{O'P}$에서 시작 → ∠AOB만큼 더 열면 $\overrightarrow{O'Q}$
$\angle XO'Q$ = 두 번 쌓인 각 = $2\angle AOB$

2단계 · Page 2

구체적 작도 절차

왜 이 단계? — 아이디어를 컴퍼스·자 조작으로 번역해요.
📖 용어 노트 각 옮기기 절차: O 중심 호 → 새 직선 위 같은 반지름 호 → AB 길이로 교점 잡기
전체 절차
① 직선 $l$ 위 점 O' 잡기
② $\overrightarrow{O'X}$를 한 변으로 설정 (직선 $l$의 한쪽 방향)
③ 각 옮기기 절차로 ∠AOB를 O'에서 작도:
  — O' 중심 호 (원본과 같은 반지름) → A', B' 잡기
  — AB 길이로 A'에서 호 → 교점 P'
  — $\overrightarrow{O'P'}$ 그리기 → $\angle XO'P' = \angle AOB$
④ 이번엔 $\overrightarrow{O'P'}$를 기준 변으로 하여
  같은 각 옮기기 절차를 한 번 더 → $\overrightarrow{O'Q}$
⑤ $\angle XO'Q = \angle XO'P' + \angle P'O'Q = \angle AOB + \angle AOB = 2\angle AOB$
⚠️ 주의 두 번째 옮기기는 첫 번째 새 변인 $\overrightarrow{O'P'}$에서 시작해야 해요. $\overrightarrow{O'X}$에서 다시 시작하면 그냥 같은 각을 두 번 그리는 것이 됩니다.

3단계 · Page 3

왜 정확한가? — 합동으로 보장

왜 이 단계? — 작도의 정확성을 항상 합동으로 근거를 댈 수 있어야 해요.
📖 용어 노트 SSS 합동: 세 변 같음 → 대응각 같음
각 옮기기가 정확한 이유

각 옮기기 1회에서 삼각형 OAB ≡ O'A'B' (SSS)이므로 ∠AOB = ∠A'O'B'.

1회 옮기기: $\angle XO'P' = \angle AOB$ — SSS 합동으로 보장
2회 옮기기: $\angle P'O'Q = \angle AOB$ — 또 다른 SSS 합동으로 보장
합산: $\angle XO'Q = \angle XO'P' + \angle P'O'Q = 2\angle AOB$ ✓
일반화 — n배도 가능

각 옮기기를 n번 연속 반복하면 $n \times \angle AOB$를 작도할 수 있어요.

3배: 3회 연속 옮기기 → $3\angle AOB$
주의: 합산 각이 180°를 넘으면 직선 각도 = 평각이 돼서 "각"이 사라짐

정답 + 풀이 흐름 요약

📖 용어 노트 합동(≅): 두 도형 모양·크기 완전히 같음  ·  SSS 합동: 세 변 같음 → 대응각 같음
각 옮기기를 연달아 두 번 — $\angle XO'Q = 2\angle AOB$
첫 번째 새 변을 두 번째 기준 변으로 사용
전체 풀이 흐름
1
직선 $l$ 위 O' 잡기. $\overrightarrow{O'X}$를 기준 변으로 설정
2
각 옮기기 1회: ∠AOB를 옮겨 $\overrightarrow{O'P'}$ 작도. $\angle XO'P' = \angle AOB$
3
$\overrightarrow{O'P'}$를 새 기준 변으로 하여 각 옮기기 2회: $\overrightarrow{O'Q}$ 작도
4
$\angle XO'Q = \angle XO'P' + \angle P'O'Q = \angle AOB + \angle AOB = 2\angle AOB$ ✓
⚠️ 핵심 통찰 자와 컴퍼스의 기본 작도(길이 옮기기·각 옮기기)를 반복하면 더 복잡한 도형을 만들 수 있어요. 「원론」이 단순 두 도구로 수백 정리를 쌓아 올린 방법이 바로 이것이에요.
