유형 6 · H2

표에서 $a$, $b$ 구하기

아래 표는 $x$와 $y$ 사이의 관계를 $y = ax + b$ 형태로 나타낸 것이에요.

$x$0123
$y$420$-2$

$a$와 $b$의 값을 각각 구하세요.

생각해보기

$x$가 1씩 늘 때 $y$는 어떻게 변하나요? 그 변화량이 $a$와 어떤 관계일까요?

단계 1 / 3

$b$는 $x=0$일 때 $y$값

$y = ax + b$에서 $x=0$을 대입하면 $y = b$가 돼요.

표에서 $x = 0$일 때 $y = 4$예요.

$y = ax + b$에 $x=0$을 넣으면:

$$y = a \cdot 0 + b = b$$ $$\therefore\; b = 4$$
⚡ 포인트 $b$를 구할 때 굳이 연립방정식을 세울 필요 없어요. $x=0$ 행을 바로 읽으면 됩니다.
📖 용어 노트 순서쌍 (a, b): x좌표 a, y좌표 b인 점 · 제1사분면(Ⅰ): x>0, y>0 · 제2사분면(Ⅱ): x<0, y>0 · 제3사분면(Ⅲ): x<0, y<0 · 제4사분면(Ⅳ): x>0, y<0 · 축 위의 점은 사분면에 속하지 않아요
단계 2 / 3

$a$는 $x$ 한 칸당 $y$ 변화량

$y = ax + b$에서 $x$가 1 늘면 $y$는 정확히 $a$만큼 변해요.

표를 다시 보면:

$x: 0 \to 1$ → $y: 4 \to 2$ (변화 $= -2$)

$x: 1 \to 2$ → $y: 2 \to 0$ (변화 $= -2$)

$x: 2 \to 3$ → $y: 0 \to -2$ (변화 $= -2$)

$x$가 1 증가할 때마다 $y$는 일정하게 $-2$ 감소 $$\therefore\; a = -2$$
⚡ 음수 기울기 주의 $a < 0$이면 그래프가 오른쪽으로 갈수록 내려가요. 표에서 $y$가 줄어들고 있다면 $a$는 음수!

단계 3 / 3

식을 세워 검증하기

$a=-2, b=4$가 맞는지 표의 다른 점으로 확인해요.

구한 식: $y = -2x + 4$

$x=2$ 대입: $y = -2(2) + 4 = 0$ ✓

$x=3$ 대입: $y = -2(3) + 4 = -2$ ✓

표의 모든 점이 $y = -2x + 4$를 만족해요.
