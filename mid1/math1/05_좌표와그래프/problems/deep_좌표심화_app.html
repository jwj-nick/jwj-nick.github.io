깊이 탐구 — 좌표 심화

대칭이동·평행이동 + 좌표로 도형 만들기 (2유형 통합)

← 단원 5 홈
이 유형이 어려운 이유

$x$축 대칭, $y$축 대칭, 원점 대칭. 셋이 비슷해 보여서 헷갈려요.

거울로 떠올리세요: 거울이 어느 축에 놓이는지에 따라 좌표가 어떻게 변하는지 보면 한 번에 구분돼요.

핵심 아이디어 — 표 한 장으로 정리

대칭이동 공식 표

대칭 종류$x$좌표$y$좌표예: $(3, -2)$ →
$x$축 대칭그대로부호 반전$(3, 2)$
$y$축 대칭부호 반전그대로$(-3, -2)$
원점 대칭부호 반전부호 반전$(-3, 2)$
외우기 팁:
$x$축 대칭 → "$y$만 바꿈" (이름의 반대축 좌표만 변함)
$y$축 대칭 → "$x$만 바꿈"
원점 대칭 → "둘 다 바꿈" (= $x$축 + $y$축 동시)

평행이동: 점을 그냥 옮기기

점 $(x, y)$를 오른쪽으로 $a$, 위로 $b$ 평행이동:
$\Rightarrow (x + a, y + b)$

  • 오른쪽 = $x$ 더하기, 왼쪽 = $x$ 빼기
  • 위 = $y$ 더하기, 아래 = $y$ 빼기
자주 하는 실수
😵
$x$축 대칭에서 $x$좌표 바꿈
"$x$축 대칭"이란 이름 때문에 $x$좌표를 바꾼다고 착각.
실제로는 $y$좌표를 바꿔요! 거울이 $x$축에 놓이면 위·아래만 뒤집힘.
⚠️
평행이동에서 부호 헷갈림
"왼쪽으로 3 이동" → $x - 3$ ✓ (음수 더하기와 같음)
"위로 5 이동" → $y + 5$ ✓
방향과 부호를 또박또박 매칭하세요.
🤔
"이동 후 점이 (3, 5)인데 원래는?" 역방향 헷갈림
이동 후 → 원래로 돌아가려면 반대로 움직이세요.
"오른쪽 2, 위 1" 후 (3, 5) → 원래는 (3-2, 5-1) = (1, 4).
추가 문제 (3문제)
PROBLEM 1
점 $\mathrm{A}(-4, 7)$의 $y$축 대칭점을 $\mathrm{B}$, $\mathrm{B}$의 원점 대칭점을 $\mathrm{C}$라 하면 $\mathrm{C}$의 좌표는?
A $(-4, 7)$
B = A의 $y$축 대칭 = $(4, 7)$
C = B의 원점 대칭 = $(-4, -7)$
(결과적으로 A → C는 $x$축 대칭과 같아요)
PROBLEM 2
점 $\mathrm{P}(a, 5)$를 $x$축 대칭한 점이 $(3, b)$이고, 그 후 다시 $y$축 대칭한 점이 $(c, -5)$일 때 $a + b + c$를 구하세요.
P $(a, 5)$ → $x$축 대칭 → $(a, -5) = (3, b)$
$\Rightarrow a = 3, b = -5$
$(3, -5) \to y$축 대칭 → $(-3, -5) = (c, -5)$
$\Rightarrow c = -3$
$a + b + c = 3 + (-5) + (-3) = -5$
PROBLEM 3
점 $\mathrm{A}(2, -3)$를 왼쪽으로 5, 위로 7만큼 평행이동한 점의 좌표는? 또 그 점이 어느 사분면에 있나요?
$x$좌표: $2 - 5 = -3$
$y$좌표: $-3 + 7 = 4$
이동 후: $(-3, 4)$
$(-, +)$ → 제2사분면
이 유형 이해하면…
함수의 평행이동/대칭이동(2학년 일차함수, 3학년 이차함수)이 같은 원리로 설명돼요. 이 단원이 사실 함수 그래프 변형의 시작점!
이 유형이 어려운 이유

점 좌표만 보고 머릿속에 도형을 그려야 해요. 좌표 → 점 위치 → 도형 모양 → 넓이까지 추론하는 길이 길어서 어려워요.

해결책: 격자 종이를 떠올리고, 좌표를 점으로 찍어보는 습관.

핵심 아이디어

두 점 사이 거리 (수평·수직만)

  • $y$좌표 같음 (가로 평행) → $x$좌표 차이가 거리
  • $x$좌표 같음 (세로 평행) → $y$좌표 차이가 거리

예: $(2, 3)$과 $(7, 3)$ → $7 - 2 = 5$ (거리)

대각선 거리는 중1에서 다루지 않아요. 직사각형 둘레/넓이 등은 가로·세로만 알면 충분.

도형 종류 판별 (사각형)

  • 정사각형: 네 변 모두 같은 길이
  • 직사각형: 마주보는 변 같음, 네 각이 직각 (가로 ≠ 세로)
  • 평행사변형: 마주보는 변이 평행이고 같음 (직각이 아닐 수 있음)

좌표가 격자 위에 깔끔하게 놓이면 대부분 직사각형 또는 정사각형!

넓이 계산 핵심

  • 직사각형: 가로 × 세로
  • 직각삼각형 (한 점이 원점, 두 다리가 축에 있는 경우): $\frac{1}{2}$ × 밑변 × 높이
  • 일반 삼각형: 한 변을 밑변으로, 그 변과 다른 한 점 사이 수직 거리를 높이로
자주 하는 실수
😵
정사각형/직사각형 구분 실패
"가로·세로 같음" → 정사각형, "다름" → 직사각형. 가로와 세로 길이를 각각 계산해서 비교 필요.
⚠️
평행사변형의 4번째 점이 1개라고 착각
세 점만 주어졌을 때 평행사변형의 4번째 점은 3가지 가능!
(어느 두 점이 마주보는 변인지에 따라 달라짐)
🤔
두 점 거리에서 빼기 방향 실수
$(7, 0)$와 $(2, 0)$ 사이 거리: $|7 - 2| = 5$ ✓
음수가 끼면: $(3, 0)$와 $(-2, 0)$ → $3 - (-2) = 5$ (절댓값으로 계산하면 안전).
추가 문제 (3문제)
PROBLEM 1
두 점 $(-3, 2)$와 $(5, 2)$ 사이의 거리를 구하세요.
$y$좌표가 모두 2로 같음 → 가로 거리.
$|5 - (-3)| = |5 + 3| = 8$
PROBLEM 2
네 점 $\mathrm{A}(0, 0)$, $\mathrm{B}(4, 0)$, $\mathrm{C}(4, 4)$, $\mathrm{D}(0, 4)$로 만든 도형의 모양과 넓이는?
가로 AB = 4, 세로 BC = 4 → 가로=세로 → 정사각형
넓이 = $4 \times 4 = 16$
PROBLEM 3
세 점 $\mathrm{A}(-2, 0)$, $\mathrm{B}(4, 0)$, $\mathrm{C}(0, 5)$로 만든 삼각형의 넓이는?
A, B 모두 $x$축 위 → AB가 밑변
밑변 AB = $4 - (-2) = 6$
C의 $y$좌표 = 5 → 높이 = 5
넓이 = $\dfrac{1}{2} \times 6 \times 5 = 15$
이 유형 이해하면…
도형 단원(7~12)에서 좌표를 활용한 풀이가 가능해져요. 그리고 데카르트의 진짜 발견 — "도형이 식으로, 식이 도형으로" — 의 첫 단계예요!
