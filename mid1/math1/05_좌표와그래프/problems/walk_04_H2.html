📖 용어 노트 순서쌍 $(a, b)$: $x$좌표 $a$, $y$좌표 $b$인 점  ·  제1사분면(Ⅰ): $x>0, y>0$  ·  제2사분면(Ⅱ): $x<0, y>0$  ·  제3사분면(Ⅲ): $x<0, y<0$  ·  제4사분면(Ⅳ): $x>0, y<0$  ·  축 위의 점은 사분면에 속하지 않아요
유형 4 · H2

점 $\mathrm{P}(a,\,2a)$를 $x$축에 대하여 대칭이동한 점이 $(3,\,b)$일 때,
$a + b$의 값을 구하세요.
1단계 · Page 1

$x$축 대칭 규칙 적용

왜 이 단계? — $x$축 대칭은 $y$좌표만 부호가 반전돼요. 이걸 적용하면 $a, b$를 구할 수 있어요.
📖 용어 노트 순서쌍 $(a, b)$: $x$좌표 $a$, $y$좌표 $b$인 점  ·  제4사분면(Ⅳ): $x>0, y<0$  ·  축 위의 점은 사분면에 속하지 않아요
$x$축 대칭 규칙
점 $(x, y)$를 $x$축 대칭 → $(x, -y)$
$x$좌표 변화 없음, $y$좌표만 부호 반전

점 $\mathrm{P}(a, 2a)$를 $x$축 대칭:

$\mathrm{P}(a,\; 2a) \xrightarrow{x\text{축 대칭}} (a,\; -2a)$
대칭 결과와 $(3, b)$ 비교

대칭 결과 $(a, -2a) = (3, b)$이므로:

$x$좌표: $a = 3$
$y$좌표: $-2a = b$ → $-2(3) = b$ → $b = -6$
⚠️ 좌표 순서 주의 좌표 순서: P($x$좌표, $y$좌표) — $x$가 먼저예요.
$x$축 대칭에서 $x$좌표는 그대로, $y$좌표만 부호가 바뀌어요.

$a + b$ 계산

왜 이 단계? — $a, b$ 값을 구했으니 합을 계산해요.

최종 계산
$a = 3$, $b = -6$
$a + b = 3 + (-6) = -3$

정답 + 풀이 흐름 요약

📖 용어 노트 순서쌍 $(a, b)$: $x$좌표 $a$, $y$좌표 $b$인 점  ·  제1사분면(Ⅰ): $x>0, y>0$  ·  제2사분면(Ⅱ): $x<0, y>0$  ·  제3사분면(Ⅲ): $x<0, y<0$  ·  제4사분면(Ⅳ): $x>0, y<0$  ·  축 위의 점은 사분면에 속하지 않아요
$a + b = \mathbf{-3}$
$a = 3,\; b = -6$
전체 풀이 흐름 — 2단계
1
$x$축 대칭 규칙: $\mathrm{P}(a, 2a) \to (a, -2a)$ = $(3, b)$ → $a = 3$, $b = -2(3) = -6$
2
$a + b = 3 + (-6) = -3$
⚠️ 이 유형의 핵심 패턴 좌표에 미지수가 포함된 대칭 문제 → 대칭 결과를 식으로 쓴 다음 주어진 좌표와 일치 조건으로 방정식 세우기.
