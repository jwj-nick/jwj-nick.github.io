유형 1 — 순서쌍과 좌표

순서쌍의 순서, 좌표 표기, 점 위치 — 좌표의 기초

L
두 순서쌍 $(2, 5)$와 $(5, 2)$는 같은 점을 나타내나요? 이유와 함께 설명하세요.
"순서"쌍이라는 이름에 답이 있어요. $x$좌표와 $y$좌표를 떠올려봐요.
다른 점이에요
$(2, 5)$는 오른쪽 2, 위로 5인 점.
$(5, 2)$는 오른쪽 5, 위로 2인 점.
$x$좌표와 $y$좌표가 다르므로 완전히 다른 위치예요.
2
L
두 순서쌍 $(a, 3)$과 $(2, b)$가 같을 때, $a$와 $b$의 값을 각각 구하세요.
두 순서쌍이 같으려면 같은 자리(첫 번째 자리끼리, 두 번째 자리끼리)의 값이 같아야 해요.
$a = 2$, $b = 3$
$x$좌표: $a = 2$
$y$좌표: $3 = b \Rightarrow b = 3$
3
L
점 $\mathrm{P}(4, -3)$의 $x$좌표와 $y$좌표를 각각 쓰세요.
$\mathrm{P}(a, b)$ 표기에서 앞은 $x$좌표, 뒤는 $y$좌표예요.
$x$좌표 = 4, $y$좌표 = $-3$
좌표 표기 규칙: $(x, y)$ 순서.
$\mathrm{P}(4, -3)$ → 오른쪽으로 4, 아래로 3 위치.

M
두 순서쌍 $(3, a)$와 $(b, 5)$가 서로 같을 때, $a + b$의 값은?
$x$좌표끼리, $y$좌표끼리 같다는 식을 세워봐요.
$a + b = 8$
$3 = b \Rightarrow b = 3$
$a = 5$
$\therefore a + b = 5 + 3 = 8$

M
점 $\mathrm{A}$가 원점에서 왼쪽으로 3, 아래로 2 떨어진 위치에 있어요. 점 $\mathrm{A}$의 좌표를 구하세요.
왼쪽 = $x$ 음수, 아래 = $y$ 음수예요.
$\mathrm{A}(-3, -2)$
왼쪽으로 3 → $x$좌표 $= -3$
아래로 2 → $y$좌표 $= -2$
3
M
두 순서쌍 $(2a + 1, 3)$과 $(7, b - 4)$가 서로 같을 때, $a, b$의 값을 구하세요.
두 순서쌍이 같다 → 두 개의 일차방정식이 생겨요.
$a = 3$, $b = 7$
$x$좌표: $2a + 1 = 7 \Rightarrow 2a = 6 \Rightarrow a = 3$
$y$좌표: $3 = b - 4 \Rightarrow b = 7$
1
H
두 순서쌍 $(a + b, 2)$와 $(5, a - b)$가 서로 같을 때, $a$와 $b$의 값을 구하세요.
$a + b = 5$, $a - b = 2$ 두 식을 나란히 풀어요. (연립의 맛보기)
$a = \dfrac{7}{2}$, $b = \dfrac{3}{2}$
식 1: $a + b = 5$
식 2: $a - b = 2$
두 식을 더하면: $2a = 7 \Rightarrow a = 7/2$
식 1에 대입: $b = 5 - 7/2 = 3/2$
2
H
점 $\mathrm{P}(a, 5)$가 점 $\mathrm{Q}(7, b)$의 오른쪽으로 4만큼 떨어져 있고, 같은 $y$좌표를 가질 때, $a + b$의 값은?
"오른쪽으로 4만큼" → $x$좌표 차이가 4. "같은 $y$좌표" → $5 = b$.
$a + b = 16$
$y$좌표 같음: $5 = b \Rightarrow b = 5$
$\mathrm{P}$가 $\mathrm{Q}$ 오른쪽으로 4: $a = 7 + 4 = 11$
$\therefore a + b = 11 + 5 = 16$
3
H
두 점 $\mathrm{A}(2a, b - 1)$, $\mathrm{B}(6, 3)$이 같은 점일 때, 점 $\mathrm{C}(a, b)$의 $x$좌표와 $y$좌표의 합을 구하세요.
먼저 $a, b$를 구하고, 그 다음에 점 $\mathrm{C}$의 좌표를 적어보세요.
$a + b = 7$
$2a = 6 \Rightarrow a = 3$
$b - 1 = 3 \Rightarrow b = 4$
점 $\mathrm{C}(3, 4)$ → $x$좌표 + $y$좌표 = $3 + 4 = 7$
단계별로 풀어요. ① $a, b$ 구하기 → ② 점 $\mathrm{C}$ 좌표 → ③ 합. 한 번에 바로 식을 세우려 하지 마세요.
