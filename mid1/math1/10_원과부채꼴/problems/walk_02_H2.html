🤔 잠깐, 손으로 풀어보세요 반원의 "둘레"는 무엇으로 이루어져 있나요? 호만인가요, 아니면 다른 선분도 있나요?
1단계 · Page 1

반원의 둘레가 뭐였더라?

왜 이 단계? — 이 문제에서 가장 흔한 실수가 바로 "반원 둘레 = 호의 길이만"이라고 착각하는 거예요.
📖 용어 노트 원의 둘레(원주): $2\pi r$  ·  원의 넓이: $\pi r^2$  ·  부채꼴 호의 길이: $2\pi r \times \dfrac{\theta}{360°}$  ·  부채꼴 넓이: $\pi r^2 \times \dfrac{\theta}{360°} = \dfrac{1}{2}rl$  ·  $\pi$ ≈ 3.14…
반원의 둘레 — 두 부분
반원의 둘레 = 반원 호 + 지름
$= \pi r + 2r$
$= r(\pi + 2)$

반원 호의 길이: 원 둘레 $2\pi r$의 절반 → $\pi r$
지름: $2r$ (반원을 닫는 직선 부분)

⚠️ 핵심 함정 반원 둘레를 "$\pi r$"이라고만 쓰면 틀려요! 지름 $2r$를 빠뜨리면 오답이에요.

2단계 · Page 2

방정식 세우기

왜 이 단계? — 공식과 주어진 값을 연결해서 $r$에 대한 방정식을 만들어요.
📖 용어 노트 원의 둘레(원주): $2\pi r$  ·  원의 넓이: $\pi r^2$  ·  부채꼴 호의 길이: $2\pi r \times \dfrac{\theta}{360°}$  ·  부채꼴 넓이: $\pi r^2 \times \dfrac{\theta}{360°} = \dfrac{1}{2}rl$  ·  $\pi$ ≈ 3.14…
방정식 설정
$\pi r + 2r = 10\pi + 20$
$r(\pi + 2) = 10(\pi + 2)$

양변에 $(\pi + 2)$가 공통 인수로 묶여요!

3단계 · Page 3

$r$ 구하기

왜 이 단계? — 양변을 $(\pi + 2)$로 나누면 $r$이 나와요.
📖 용어 노트 원의 둘레(원주): $2\pi r$  ·  원의 넓이: $\pi r^2$  ·  부채꼴 호의 길이: $2\pi r \times \dfrac{\theta}{360°}$  ·  부채꼴 넓이: $\pi r^2 \times \dfrac{\theta}{360°} = \dfrac{1}{2}rl$  ·  $\pi$ ≈ 3.14…
계산
$r(\pi + 2) = 10(\pi + 2)$
양변을 $(\pi + 2)$로 나누면:
$r = 10$ cm
💡 검산 $r = 10$일 때: 반원 호 $= \pi \times 10 = 10\pi$, 지름 $= 20$
둘레 $= 10\pi + 20$ ✓

정답 + 풀이 흐름 요약

📖 용어 노트 원의 둘레(원주): $2\pi r$  ·  원의 넓이: $\pi r^2$  ·  부채꼴 호의 길이: $2\pi r \times \dfrac{\theta}{360°}$  ·  부채꼴 넓이: $\pi r^2 \times \dfrac{\theta}{360°} = \dfrac{1}{2}rl$  ·  $\pi$ ≈ 3.14…
$r = 10$ cm
전체 풀이 흐름
1
반원 둘레 = 호 + 지름 = $\pi r + 2r$ (지름 빠뜨리지 않기!)
2
$\pi r + 2r = 10\pi + 20$ 방정식 설정
3
$r(\pi + 2) = 10(\pi + 2)$ → $r = 10$ cm
⚠️ 이 유형의 핵심 함정 반원 둘레 = 호만? 아니에요! 반원 둘레 = 호($\pi r$) + 지름($2r$).
호만 쓰면 방정식이 $\pi r = 10\pi + 20$이 되어 $r$이 깔끔하게 나오지 않아요.
