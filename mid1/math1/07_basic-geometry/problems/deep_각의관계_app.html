깊이 탐구 — 각의 관계 ★

맞꼭지각 + 동위각·엇각·평행선 (시험 빈출 1순위)

← 단원 7 홈
이 유형이 어려운 이유

"맞꼭지각" 자체는 쉬워요. 하지만 시험에서는 미지수 $x$ 를 포함한 식 두 개로 나오는 경우가 많아요.

핵심 함정:
① 맞꼭지각 vs 이웃한 각 헷갈림 (전자는 같음, 후자는 합 = 180°)
② 두 직선 외에 세 직선이 한 점에서 만나면 6개 각, 식 세우기 까다로움

핵심 아이디어 — 두 가지 도구

도구 1 — 맞꼭지각 같음

맞꼭지각 (마주보는) → 같다

두 직선 교차 시 4개 각 = 두 쌍의 맞꼭지각.

도구 2 — 이웃한 각의 합 = 180°

이웃한 두 각 합 = 평각 = $180°$

이 두 도구만으로 4개 각 모두 결정됨.

식 세우기 패턴

  • 두 식이 맞꼭지각이면 → 식 = 식 (같음)
  • 두 식이 이웃한 각이면 → 식 + 식 = 180°
예: 한 각이 $(2x+10)°$, 맞꼭지각이 $(x+30)°$ → $2x + 10 = x + 30$ → $x = 20$.
예: 한 각이 $(2x+10)°$, 이웃한 각이 $(3x-5)°$ → $(2x+10) + (3x-5) = 180$ → $x = 35$.
자주 하는 실수
⚠️
맞꼭지각 vs 이웃한 각 식 잘못 세움
맞꼭지각이면 = 식, 이웃이면 합 = 180°. 그림의 위치를 보고 결정.
🤔
3직선 교차 시 6각의 합을 잊음
세 직선이 한 점에서 만나면 6개 각의 합 = 360°. 한쪽 평각 안의 3개 = 180°.
추가 연습
PROBLEM 1 (M)
두 직선이 만나서 한 각이 $(5x - 20)°$, 그 맞꼭지각이 $(2x + 40)°$. $x$ 값과 그 각의 크기는?
$x = 20$,   각 $= 80°$
맞꼭지각 같음: $5x - 20 = 2x + 40$
$3x = 60$, $x = 20$
각: $5 \times 20 - 20 = 80°$ ($2 \times 20 + 40 = 80°$ ✓)
PROBLEM 2 (M)
세 직선이 한 점 O에서 만난다. 한쪽 평각의 3개 각이 차례로 $\angle a, \angle b, \angle c$ 이고 $\angle a = 50°$, $\angle c = 70°$. $\angle b$ 는?
$\angle b = 60°$
한쪽 평각 = 180°. $50° + \angle b + 70° = 180°$ → $\angle b = 60°$.
PROBLEM 3 (H)
두 직선이 만나서 4각의 비가 $\angle a : \angle b : \angle c : \angle d = 1 : 4 : 1 : 4$ 라면 가장 큰 각의 크기는?
$144°$
4개 각의 합 $= 360°$.
$1 + 4 + 1 + 4 = 10$ → 단위 $k = 36°$.
가장 큰 각 = $4k = 144°$.
(검산: 1 : 4 : 1 : 4 → 36, 144, 36, 144 — 맞꼭지각 같음, 이웃한 각 합 36 + 144 = 180° ✓)

• 평행선과 동위각 (옆 탭): 맞꼭지각 + 동위각 결합 문제 단골
• 다각형 내각·외각: 한 점에서 갈라진 각의 합 = 360°
이 유형이 어려운 이유 ★

두 직선과 한 직선이 만나서 8개의 각이 생기는데, 그 중 어느 것이 동위각인지 엇각인지 동측내각인지 식별하는 게 첫 관문.

핵심 함정:
① 이름(동위각/엇각)과 크기 같음이 같은 게 아니라는 점 (평행일 때만 같음)
② 평행선 사이 꺾인 점이 있을 때 보조선 그어 풀어야 함

핵심 아이디어 — 위치로 식별

① 동위각: "같은 위치"

두 교점에서, 횡단선 기준 같은 쪽 + 평행선 기준 같은 쪽.

위↔위, 아래↔아래 같은 위치. 4쌍.

② 엇각: "엇갈린 위치"

평행선 사이(내부) + 횡단선 기준 반대쪽. 2쌍.

③ 평행이면 같다 — 핵심 정리

$l \parallel m \iff$ 동위각 같음 $\iff$ 엇각 같음

동측내각 (같은 쪽 내각) 의 합 = 180° (역도 성립).

★ 꺾인 점 보조선 기법 (단골 풀이)

$l \parallel m$ 이고 두 평행선 사이에 꺾인 점 P가 있다면:

  1. P를 지나며 $l$, $m$과 평행한 보조선을 그어요.
  2. 꺾인 각 $\angle x$ 가 위·아래 두 부분으로 분할.
  3. 위 부분 = $l$ 의 한 각과 엇각.
  4. 아래 부분 = $m$ 의 한 각과 엇각.
  5. $\angle x = $ 위 + 아래.
예: 위 $30°$, 아래 $40°$ → $\angle x = 70°$
자주 하는 실수
⚠️
평행이 아닌데 동위각이 같다고 함
"동위각"은 위치 이름. 평행이 아니면 이름은 같아도 크기가 다름.
🤔
동측내각의 합이 180°인 걸 잊음
평행선 + 같은 쪽 내각 = 180°. 동위각·엇각과 다르지만 똑같이 자주 출제!
📐
꺾인 점에서 보조선을 안 그림
꺾인 각은 보조선 그어 두 엇각의 합으로 분할 → 풀이의 표준 절차.
추가 연습
PROBLEM 1 (M)
$l \parallel m$ 이고 한 동측내각이 $(3x + 20)°$, 다른 동측내각이 $(2x + 40)°$. $x$ 값은?
$x = 24$
동측내각 합 $= 180°$:
$(3x + 20) + (2x + 40) = 180$
$5x + 60 = 180$
$5x = 120$, $x = 24$.
PROBLEM 2 (M)
$l \parallel m$ 일 때, $l$ 위 한 각이 $\angle A = 110°$. 그 동위각의 맞꼭지각의 크기는?
$110°$
$l \parallel m$ → 동위각 같음 → $110°$.
맞꼭지각 같음 → $110°$.
(동위각의 맞꼭지각 = 원래 각의 엇각이기도 — 평행 시 모두 같음!)
PROBLEM 3 (H) — ★ 꺾인 점
$l \parallel m$ 이고 두 평행선 사이에 꺾인 점 P가 있다. $l$ 의 한 각이 $40°$, $m$ 의 한 각이 $55°$ (보조선 기준). 꺾인 각 $\angle x$ 의 크기는?
$\angle x = 95°$
P를 지나며 $l$, $m$과 평행한 보조선을 그음.
$\angle x$ = 위 부분 + 아래 부분
위 부분 = $l$의 $40°$ 와 엇각 = $40°$
아래 부분 = $m$의 $55°$ 와 엇각 = $55°$
$\angle x = 40° + 55° = 95°$.
