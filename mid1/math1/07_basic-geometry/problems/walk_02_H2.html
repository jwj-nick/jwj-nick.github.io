문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$ (→ 방향): A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$ (−AB): A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
유형 2 · H2

한 직선 위에 점 A, B, C, D가 이 순서대로 있고
$\overline{AB} : \overline{BC} : \overline{CD} = 1 : 2 : 3$ 이다.
$\overline{AD} = 30$ cm 일 때, $\overline{BC}$의 길이는?
🤔 잠깐, 손으로 풀어보세요 비 1 : 2 : 3 이 주어질 때, "단위 $k$"를 이용해 각 길이를 $k$로 나타내보세요.
1단계 · Page 1

비를 단위 $k$로 나타내요

왜 이 단계? — 비가 주어질 때는 공통 단위 $k$를 두면 식을 세울 수 있어요.
📖 용어 노트 비(ratio): 두 양의 크기 관계. $a : b = 1 : 2$이면 $a$는 $b$의 절반
단위 $k$ 설정

$\overline{AB} : \overline{BC} : \overline{CD} = 1 : 2 : 3$ 이므로

$\overline{AB} = k$
$\overline{BC} = 2k$
$\overline{CD} = 3k$

($k$는 길이의 한 단위, 양수)

AD 전체 길이

A, B, C, D가 이 순서이므로:

$\overline{AD} = \overline{AB} + \overline{BC} + \overline{CD} = k + 2k + 3k = 6k$

2단계 · Page 2

$k$의 값을 구해요

왜 이 단계? — AD = 30 cm 조건을 이용해 k를 구하면 BC도 바로 나와요.
📖 용어 노트 선분 $\overline{AB}$: A와 B를 잇는 유한한 선
방정식으로 $k$ 구하기
$\overline{AD} = 6k = 30$ cm
$k = 5$ cm
BC 계산
$\overline{BC} = 2k = 2 \times 5 = 10$ cm
💡 검산해봐요 $\overline{AB} = 5$, $\overline{BC} = 10$, $\overline{CD} = 15$
합: $5 + 10 + 15 = 30 = \overline{AD}$ ✓
비: $5 : 10 : 15 = 1 : 2 : 3$ ✓

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$: A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$: A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
$\overline{BC} = 10$ cm
($k = 5$ cm, $\overline{BC} = 2k$)
전체 풀이 흐름 — 3단계
1
비 $1 : 2 : 3$ → $\overline{AB} = k$, $\overline{BC} = 2k$, $\overline{CD} = 3k$ 설정
2
$\overline{AD} = k + 2k + 3k = 6k = 30$ → $k = 5$ cm
3
$\overline{BC} = 2k = 10$ cm
⚠️ 이 유형의 핵심 비가 주어질 때는 공통 단위 $k$를 설정해서 식을 세우는 게 핵심 전략이에요. 직접 길이를 구하려 하면 막혀요.
