③, ⑤

② 90° → 직각 (둔각 아님)

2
L
$\angle AOB$ 가 평각일 때 $\angle AOB$의 크기는?
평각은 두 반직선이 정반대 방향으로 펴진 각이에요.
$180°$
평각 = $180°$. (두 반직선이 일직선을 이룸)
3
L
한 점 O에서 세 반직선 OA, OB, OC가 갈라진다. $\angle AOB = 50°$, $\angle BOC = 70°$ 이고 점 B가 $\angle AOC$의 내부에 있을 때, $\angle AOC$의 크기는?
B가 ∠AOC 내부 → $\angle AOC = \angle AOB + \angle BOC$.
$\angle AOC = 120°$
$\angle AOC = \angle AOB + \angle BOC = 50° + 70° = 120°$
1
M
$\angle AOB = 90°$ 이고 점 C가 $\angle AOB$의 내부에 있다. $\angle BOC = 27°$ 일 때, $\angle AOC$의 크기는?
$\angle AOB = \angle AOC + \angle COB$. 90° = ?° + 27°.
$\angle AOC = 63°$
$\angle AOC = \angle AOB - \angle BOC = 90° - 27° = 63°$
2
M
$\angle AOB = 180°$ (평각), 점 C, D가 $\angle AOB$의 내부에 차례로 있다. $\angle AOC = 50°$, $\angle DOB = 70°$ 일 때, $\angle COD$의 크기는?
$\angle AOC + \angle COD + \angle DOB = 180°$.
$\angle COD = 60°$
$\angle AOC + \angle COD + \angle DOB = 180°$
$50° + \angle COD + 70° = 180°$
$\angle COD = 180° - 120° = 60°$
3
M
시계의 시침과 분침이 정확히 3시 정각을 가리킬 때, 두 바늘이 이루는 각의 크기 (작은 쪽)는?
시계 한 바퀴 = 360°. 시계는 12등분으로 나뉘어요.
$90°$ (직각)
한 바퀴 360°를 12로 나누면 한 칸 = 30°.
3시: 분침 12시 방향, 시침 3시 방향 → 칸 차이 3
$30° \times 3 = 90°$ → 직각
1
H
$\angle AOB = 90°$ 이고 점 C가 $\angle AOB$의 내부에 있다. $\angle AOC : \angle COB = 2 : 7$ 일 때, $\angle AOC$의 크기는?
$\angle AOC = 2k$, $\angle COB = 7k$. 합이 90°.
$\angle AOC = 20°$
$\angle AOC = 2k$, $\angle COB = 7k$ 라 하면
$2k + 7k = 9k = 90°$
$k = 10°$
$\angle AOC = 2k = 20°$
2
H
$\angle AOB = 180°$ (평각), 점 C가 $\angle AOB$의 내부에 있다. $\angle AOC$ 의 크기는 $\angle COB$ 의 크기보다 $40°$ 만큼 더 크다. $\angle COB$의 크기는?
$\angle COB = x$, $\angle AOC = x + 40°$. 합 180°.
$\angle COB = 70°$
$\angle COB = x$ 라 하면 $\angle AOC = x + 40°$.
$x + (x + 40°) = 180°$
$2x = 140°$
$x = 70°$
(검산: $\angle AOC = 110°$, $110° + 70° = 180°$ ✓)
3
H
$\angle AOB$ 가 직각이다. $\angle AOB$의 내부에 점 C, D 가 있어 $\overrightarrow{OC}$ 가 $\angle AOB$ 의 이등분선이고, $\overrightarrow{OD}$ 가 $\angle COB$ 의 이등분선이다. $\angle AOD$ 의 크기는?
∠AOC = ∠COB = 45°. ∠COD = (∠COB)/2 = 22.5°. ∠AOD = ∠AOC + ∠COD.
$\angle AOD = 67.5°$
$\angle AOB = 90°$
$\overrightarrow{OC}$가 이등분 → $\angle AOC = \angle COB = 45°$
$\overrightarrow{OD}$가 $\angle COB$의 이등분 → $\angle COD = \angle DOB = \dfrac{45°}{2} = 22.5°$
$\angle AOD = \angle AOC + \angle COD = 45° + 22.5° = 67.5°$
★ "이등분선"은 한 각을 똑같이 두 부분으로 나누는 반직선. 자주 쓰이는 도구!
