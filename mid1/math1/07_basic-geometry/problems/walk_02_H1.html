문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$ (→ 방향): A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$ (−AB): A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
유형 2 · H1

한 직선 위에 점 A, B, C가 이 순서대로 있고 $\overline{AC} = 24$ cm 이다.
M은 $\overline{AB}$의 중점, N은 $\overline{BC}$의 중점일 때, $\overline{MN}$의 길이는?
🤔 잠깐, 손으로 풀어보세요 직선 위에 A, B, C를 표시하고 M, N을 각 선분의 중점으로 표시해 보세요. $\overline{AB} = a$, $\overline{BC} = b$로 두면 어떻게 식이 만들어질까요?
1단계 · Page 1

중점의 정의를 확인해요

왜 이 단계? — 중점이 어디에 있는지 정확히 이해하면 문제가 풀려요.
📖 용어 노트 중점: 선분을 정확히 반으로 나누는 점
중점의 정의

점 M이 $\overline{AB}$의 중점이면:

$\overline{AM} = \overline{MB} = \dfrac{1}{2}\overline{AB}$

마찬가지로 N이 $\overline{BC}$의 중점이면:

$\overline{BN} = \overline{NC} = \dfrac{1}{2}\overline{BC}$
변수 설정

$\overline{AB} = a$, $\overline{BC} = b$ 라 두면:

$a + b = \overline{AC} = 24$ cm
$\overline{MB} = \dfrac{a}{2}$ (M이 AB의 중점이므로 MB = a/2)
$\overline{BN} = \dfrac{b}{2}$ (N이 BC의 중점이므로 BN = b/2)

2단계 · Page 2

$\overline{MN}$을 식으로 나타내요

왜 이 단계? — M과 N 사이 거리를 a, b로 표현하면 계산할 수 있어요.
📖 용어 노트 중점: 선분을 정확히 반으로 나누는 점
MN = MB + BN

점 순서가 A, M, B, N, C 이므로 (M은 AB의 중점, N은 BC의 중점):

$\overline{MN} = \overline{MB} + \overline{BN}$
$= \dfrac{a}{2} + \dfrac{b}{2}$
$= \dfrac{a + b}{2}$
대입

$a + b = \overline{AC} = 24$ 를 대입하면:

$\overline{MN} = \dfrac{a + b}{2} = \dfrac{24}{2} = 12$ cm
💡 놀라운 결과! $a$와 $b$가 각각 얼마든 관계없이, $\overline{MN}$은 항상 $\overline{AC}$의 절반이에요. AB와 BC의 비율이 어떻게 바뀌어도 MN은 언제나 12 cm!

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$: A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$: A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
$\overline{MN} = 12$ cm
(AB와 BC의 비율에 무관하게 항상 AC의 절반)
전체 풀이 흐름 — 3단계
1
$\overline{AB} = a$, $\overline{BC} = b$, $a + b = 24$ cm 설정
2
중점 정의: $\overline{MB} = \dfrac{a}{2}$, $\overline{BN} = \dfrac{b}{2}$
3
$\overline{MN} = \overline{MB} + \overline{BN} = \dfrac{a+b}{2} = \dfrac{24}{2} = 12$ cm
⚠️ 이 유형의 핵심 두 중점 M, N 사이의 거리는 항상 전체 길이 AC의 절반이에요. a, b 각각이 얼마인지 알 필요가 없어요!
