깊이 탐구 — 점·선·각

표기 + 길이 + 각의 종류 (3유형 통합)

← 단원 7 홈
이 유형이 어려운 이유

$\overleftrightarrow{AB}, \overrightarrow{AB}, \overline{AB}$ 처럼 비슷해 보이는 기호 3가지가 한꺼번에 등장해요.

핵심 함정:
① 화살표 모양만 보고 "비슷한 거 아닐까?" 헷갈리는 순간
② 방향이 있는 것(반직선)과 없는 것(선분, 직선)을 같이 취급

핵심 아이디어 — "끝"과 "방향"으로 구분
표기이름끝방향성
$\overleftrightarrow{AB}$직선둘 다 무한없음
$\overrightarrow{AB}$반직선한쪽만 무한있음 (A에서 B 방향)
$\overline{AB}$선분둘 다 끝없음
결정적 차이: 반직선만 방향이 있다.
$\overrightarrow{AB} \neq \overrightarrow{BA}$ ← 이게 평생 잊으면 안 되는 한 줄

등식 판별 규칙

  • $\overline{AB} = \overline{BA}$ ✓ (선분, 방향 없음)
  • $\overleftrightarrow{AB} = \overleftrightarrow{BA}$ ✓ (직선, 방향 없음)
  • $\overrightarrow{AB} = \overrightarrow{BA}$ ✗ (반직선, 방향 다름)

한 직선 위 점 여러 개일 때 반직선 등식

한 직선 위에 점 A, B, C 가 이 순서대로 있다면:

$\overrightarrow{AB} = \overrightarrow{AC}$

이유: 시작점 같음 (A) + 방향 같음 (B 방향 = C 방향, 점 순서가 A→B→C이므로).

단, $\overrightarrow{AB} \neq \overrightarrow{BC}$ — 시작점 다름!
자주 하는 실수
⚠️
$\overrightarrow{AB} = \overrightarrow{BA}$ 라고 답함
"두 점이 같으니까 같은 도형 아닐까?" — 함정. 반직선은 시작점이 핵심.
🤔
선분과 반직선을 같다고 함
$\overline{AB}$ 는 끝이 둘 다 있고, $\overrightarrow{AB}$ 는 한쪽이 무한. 길이도 다르고 모양도 다름.
📍
한 직선 위 점들로 만든 직선이 여러 개라고 답
한 직선 위에 점이 5개 있어도 그 직선은 딱 하나. ($\overleftrightarrow{AB} = \overleftrightarrow{BC} = ... = $ 모두 같은 직선)
추가 연습 (난이도 ↑)
PROBLEM 1 (M)
한 직선 위에 점 A, B, C, D 가 이 순서대로 있다. $\overrightarrow{BC}$ 와 같은 도형은? (모두 고르시오)
① $\overrightarrow{BD}$   ② $\overrightarrow{CD}$   ③ $\overrightarrow{AB}$
① 만 같음
$\overrightarrow{BC}$: B 에서 시작, C 방향 (= D 방향).
① $\overrightarrow{BD}$: B 시작, D 방향 → 같음 ✓
② $\overrightarrow{CD}$: C 시작 → 다름 (시작점이 B가 아님)
③ $\overrightarrow{AB}$: A 시작 → 다름
PROBLEM 2 (M)
서로 다른 5개의 점이 한 직선 위에 있다. 이 점들로 만들 수 있는 서로 다른 반직선의 개수는?
8개
각 점은 시작점이 될 수 있고, 두 방향이 있음.
하지만 양 끝점은 한 방향만 가능 (다른 쪽은 점이 없음).
점 5개 중: 양 끝 2개 × 1방향 + 안쪽 3개 × 2방향 = 2 + 6 = 8
PROBLEM 3 (H)
평면에 점 A, B, C, D, E 5개가 있고 어느 세 점도 한 직선 위에 있지 않다. 이 점들로 만들 수 있는 서로 다른 직선의 개수는?
10개
"어느 세 점도 한 직선 위에 없다" → 모든 두 점 쌍이 다른 직선 만듦.
5점 중 2점 고르는 조합: $\dfrac{5 \times 4}{2} = 10$.
이 유형을 이해하면
▸ 다음이 쉬워져요
• 각의 표기: $\angle AOB$ 의 가운데 글자가 꼭짓점 — 이런 약속의 중요성 체득
• 도형 합동·증명: 어느 변(선분)이 어느 변과 대응하는지 명확히 표기
• 벡터 (고등): 반직선의 방향성 개념이 그대로 화살표 벡터로 연결
이 유형이 어려운 이유

한 직선 위 여러 점이 있을 때, 점의 순서를 정확히 파악해야 길이의 합·차를 풀 수 있어요.

핵심 함정: "한 직선 위에 있다"고만 하면 점 순서가 모호해져 답이 두 개일 수도.

핵심 아이디어 — 그림 그리기 → 식 세우기

① 중점 공식

M이 $\overline{AB}$의 중점일 때:

$\overline{AM} = \overline{MB} = \dfrac{1}{2}\overline{AB}$

② 한 점이 다른 두 점 사이에 있을 때

점 B가 A와 C 사이에 있다면:

$\overline{AC} = \overline{AB} + \overline{BC}$

③ 두 중점 사이의 거리 (중요!)

한 직선 위에 점 A, B, C 가 이 순서. M은 $\overline{AB}$의 중점, N은 $\overline{BC}$의 중점.

$\overline{MN} = \dfrac{1}{2}\overline{AC}$

증명: $\overline{MB} = \dfrac{a}{2}$, $\overline{BN} = \dfrac{b}{2}$ → $\overline{MN} = \dfrac{a+b}{2} = \dfrac{\overline{AC}}{2}$

이 결과가 $a, b$ 길이와 무관하게 항상 성립한다는 게 멋진 점!
자주 하는 실수
⚠️
점 순서를 가정하지 않고 풀이
"한 직선 위 점 A, B, C" 라고만 하면 순서가 ABC, ACB, BAC 등 여러 가지일 수 있어요. 그림으로 확인!
🤔
중점의 중점에서 분수가 헷갈림
M이 AB 중점이면 AM = AB/2. N이 AM 중점이면 AN = AM/2 = AB/4. 분수 단계로 차근차근 계산.
추가 연습
PROBLEM 1 (M)
$\overline{AB} = 36$ cm. M은 $\overline{AB}$의 중점. N은 $\overline{AM}$의 중점. K는 $\overline{NB}$의 중점. $\overline{MK}$의 길이는?
$\overline{MK} = 4.5$ cm
각 점의 A로부터의 거리를 차례로 구해요:
• $\overline{AM} = 18$ (M은 AB 중점)
• $\overline{AN} = 9$ (N은 AM 중점)
• $\overline{NB} = 36 - 9 = 27$
• $\overline{NK} = 27/2 = 13.5$ (K는 NB 중점)
• $\overline{AK} = 9 + 13.5 = 22.5$
$\overline{MK} = \overline{AK} - \overline{AM} = 22.5 - 18 = $ 4.5 cm
PROBLEM 2 (M)
한 직선 위 점 A, B, C, D 가 이 순서. $\overline{AB} : \overline{BC} : \overline{CD} = 2 : 3 : 5$, $\overline{AD} = 40$ cm. $\overline{AC}$의 길이는?
$\overline{AC} = 20$ cm
$\overline{AB} = 2k$, $\overline{BC} = 3k$, $\overline{CD} = 5k$, 합 $10k = 40$ → $k = 4$.
$\overline{AC} = \overline{AB} + \overline{BC} = 2k + 3k = 5k = 20$ cm.
PROBLEM 3 (H)
한 직선 위 점 A, B, C 가 있고 $\overline{AB} = 12$, $\overline{BC} = 4$. M은 $\overline{AC}$ 의 중점. $\overline{BM}$의 가능한 길이를 모두 구하시오.
$\overline{BM} = 4$ cm 또는 $\overline{BM} = 8$ cm
경우 1: A — B — C → $\overline{AC} = 16$, $\overline{AM} = 8$, $\overline{BM} = |12 - 8| = 4$.
경우 2: A — C — B → $\overline{AC} = 8$, $\overline{AM} = 4$, $\overline{BM} = 12 - 4 = 8$.
이 유형을 이해하면
▸ 다음이 쉬워져요
• 좌표평면 위 거리 (고1): 두 점 사이 거리, 중점 좌표 공식
• 도형 닮음·합동: 변의 비율, 길이 합 활용 증명
이 유형이 어려운 이유

각의 종류 (예각·직각·둔각·평각·우각) 5가지 분류는 단순해 보여요. 하지만 시험 문제에서는 분할각의 합·차로 미지각을 구하는 경우가 많아요.

핵심 함정: 90°와 180°가 둔각·우각의 범위에 들지 않음을 잊는 것.

핵심 아이디어 — 분류 + 합·차 식

분류 (등호 위치 주의!)

이름범위예
예각$0° < \angle < 90°$30°
직각$= 90°$90°
둔각$90° < \angle < 180°$120°
평각$= 180°$180°
우각$180° < \angle < 360°$270°
$90°$ 은 직각, $180°$ 은 평각 — 둔각·우각의 범위에 들지 않아요.

분할각의 합·차

점 C가 $\angle AOB$ 의 내부에 있을 때:

$\angle AOB = \angle AOC + \angle COB$

→ 미지각: $\angle AOC = \angle AOB - \angle COB$.

이등분선·각의 비

$\overrightarrow{OC}$ 가 $\angle AOB$ 의 이등분선 → $\angle AOC = \angle COB = \dfrac{1}{2}\angle AOB$.

비율 $\angle AOC : \angle COB = m : n$ → 단위 $k$ 두기. $\angle AOC = mk, \angle COB = nk$.

자주 하는 실수
⚠️
$90°$ 를 둔각이라고 답함
$90°$ 는 직각이에요. 둔각은 $90° < \angle < 180°$ — 90도는 빠집니다.
🤔
분할각 식 세울 때 점 위치 안 그림
"C가 ∠AOB 내부에" 라는 조건을 안 쓰면 식이 맞지 않을 수 있어요. 그림 그리기 필수.
추가 연습
PROBLEM 1 (M)
$\angle AOB = 60°$, $\overrightarrow{OC}$ 가 $\angle AOB$ 의 이등분선이다. $\angle AOC$ 의 종류는?
$\angle AOC = 30°$ → 예각
이등분 → $\angle AOC = 60°/2 = 30°$. 0 < 30 < 90 → 예각.
PROBLEM 2 (M)
$\angle AOB = 90°$, 점 C, D 가 $\angle AOB$ 의 내부에. $\angle AOC = 30°$, $\angle BOD = 25°$. $\angle COD$ 의 크기는?
$\angle COD = 35°$
$\angle AOC + \angle COD + \angle DOB = 90°$
$30° + \angle COD + 25° = 90°$
$\angle COD = 35°$.
(점 순서 A — C — D — B 가정)
PROBLEM 3 (H)
평각 $\angle AOB$ 의 내부에 점 C, D 가 있다. $\angle AOC : \angle COD : \angle DOB = 1 : 2 : 3$. $\angle AOC + \angle DOB$ 의 크기는?
$120°$
$\angle AOC = k, \angle COD = 2k, \angle DOB = 3k$. 합 $6k = 180°$ → $k = 30°$.
$\angle AOC + \angle DOB = k + 3k = 4k = 120°$.
