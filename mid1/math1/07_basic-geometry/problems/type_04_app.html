유형 4 — 맞꼭지각

두 직선이 만나면 4개 각이 생긴다 — 맞꼭지각은 같다
← 단원 7 홈
1
L
두 직선이 한 점에서 만나서 한 각이 $65°$ 다. 그 맞꼭지각의 크기는?
맞꼭지각은 서로 같다.
$65°$
맞꼭지각은 서로 같으므로 마주보는 각도 $65°$.
2
L
두 직선이 만나서 한 각이 $48°$ 일 때, 이웃한 각의 크기는?
두 직선이 만나서 생긴 두 이웃 각의 합은 평각 $180°$.
$132°$
이웃한 두 각의 합은 평각이므로
$180° - 48° = 132°$
3
L
두 직선이 만나서 생긴 4개의 각 중에 한 각이 $40°$ 일 때, 4개 각의 크기는 차례로?
맞꼭지각이 같고, 이웃한 두 각의 합이 180°.
$40°, 140°, 40°, 140°$
두 직선 교차 시 4각: 마주보는 두 쌍이 각각 같음.
한 각 $40°$ → 맞꼭지각 $40°$
이웃한 각 $180° - 40° = 140°$ → 그 맞꼭지각 $140°$
합 검산: $40 + 140 + 40 + 140 = 360°$ ✓

M
두 직선이 만나서 한 각이 $(2x + 10)°$ 이고, 이웃한 각이 $(3x - 5)°$ 다. $x$의 값은?
이웃한 두 각의 합 $= 180°$.
$x = 35$
이웃한 두 각: $(2x+10) + (3x-5) = 180°$
$5x + 5 = 180$
$5x = 175$
$x = 35$
2
M
두 직선이 만나서 한 각이 $(4x - 20)°$ 이고, 그 맞꼭지각이 $(2x + 30)°$ 다. $x$의 값과 그 각의 크기는?
맞꼭지각은 서로 같다 → 식 = 식.
$x = 25$,   각의 크기 $= 80°$
맞꼭지각 같음: $4x - 20 = 2x + 30$
$2x = 50$
$x = 25$
각의 크기: $4 \times 25 - 20 = 80°$ (또는 $2 \times 25 + 30 = 80°$ ✓)
3
M
세 직선이 한 점에서 만나서 6개의 각이 생긴다. 이웃한 세 각의 크기가 각각 $30°, 70°, ?°$ 일 때, ?에 들어갈 각의 크기는?
한 직선의 한쪽에 있는 각들의 합 = 평각 $180°$.
$80°$
한 직선의 한쪽: 평각 $180°$.
$30° + 70° + ?° = 180°$
$?° = 80°$

H
두 직선이 만나서 한 각의 크기가 다른 한 각의 크기의 5배다. 두 각 중 큰 각의 크기는? (서로 이웃한 두 각이라고 가정)
작은 각 $x$, 큰 각 $5x$. 이웃 → 합 180°.
$150°$
작은 각 $x$, 큰 각 $5x$.
이웃 → $x + 5x = 180°$ → $6x = 180°$ → $x = 30°$
큰 각: $5x = 150°$
2
H
세 직선이 한 점에서 만난다. 이웃한 세 각이 차례로 $\angle a, \angle b, \angle c$ 이고 $\angle a : \angle b : \angle c = 2 : 3 : 4$ 일 때, $\angle b$ 의 크기는?
$\angle a + \angle b + \angle c = 180°$. 비를 활용해 단위 $k$로 두기.
$\angle b = 60°$
$\angle a = 2k$, $\angle b = 3k$, $\angle c = 4k$
$2k + 3k + 4k = 9k = 180°$
$k = 20°$
$\angle b = 3k = 60°$

H
두 직선이 한 점에서 만나서 4개의 각이 생긴다. 그 중 한 각의 크기가 다른 한 이웃한 각의 크기보다 $30°$ 더 크다. 4개 각 중 가장 큰 각의 크기는?
작은 각 $x$, 큰 각 $x + 30°$. 이웃 → 합 $180°$.
가장 큰 각 $= 105°$
이웃한 두 각: 작은 $x$, 큰 $x + 30°$.
$x + (x + 30°) = 180°$
$2x = 150°$
$x = 75°$
큰 각 = $75° + 30° = 105°$.
4개 각: $75°, 105°, 75°, 105°$ (맞꼭지각이 서로 같음). 가장 큰 각 $= 105°$.
★ 두 직선 교차 시 4각의 종류는 항상 단 2개. 그 두 종류는 합이 180°.
