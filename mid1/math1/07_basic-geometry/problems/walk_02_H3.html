문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$ (→ 방향): A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$ (−AB): A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
유형 2 · H3

한 직선 위에 점 A, B, C가 있고 $\overline{AB} = 16$ cm, $\overline{BC} = 8$ cm 이다.
M이 $\overline{AC}$의 중점일 때, $\overline{BM}$의 길이를 모두 구하시오.
(B의 위치에 따라 경우를 나눠 구하기)
🤔 잠깐, 손으로 풀어보세요 "한 직선 위에 A, B, C" 라고 했는데 B가 A와 C 사이에 있는 경우와 A, C의 같은 쪽에 있는 경우가 있어요. 두 경우를 그림으로 그려보세요.
1단계 · Page 1

왜 경우를 나눠야 할까요?

왜 이 단계? — "한 직선 위"라는 조건만으로는 점의 순서가 정해지지 않아요. 경우를 나눠야 해요.
📖 용어 노트 선분 $\overline{AB}$: A와 B를 잇는 유한한 선. 길이는 두 점 사이의 거리
조건 정리

주어진 것: $\overline{AB} = 16$, $\overline{BC} = 8$

B의 위치에 따라 $\overline{AC}$의 길이가 달라져요:

경우 1: A — B — C (B가 A, C 사이)
$\overline{AC} = \overline{AB} + \overline{BC} = 16 + 8 = 24$ cm

경우 2: A — C — B (C가 A, B 사이)
$\overline{AB} = \overline{AC} + \overline{CB}$ → $\overline{AC} = \overline{AB} - \overline{BC} = 16 - 8 = 8$ cm
⚠️ 경우 3은 없나요? B — A — C 순 (A가 B, C 사이)도 생각할 수 있어요. 그런데 이 경우 $\overline{BC} = \overline{BA} + \overline{AC}$이므로 $8 = 16 + \overline{AC}$ → $\overline{AC} = -8$이 되어 불가능해요. 따라서 두 경우만 유효해요.

2단계 · Page 2

경우 1: A — B — C 순

왜 이 단계? — 첫 번째 경우에서 M을 찾고 BM을 계산해요.
📖 용어 노트 중점: 선분을 정확히 반으로 나누는 점
경우 1 풀이
$\overline{AC} = 24$ cm
M은 $\overline{AC}$의 중점 → A에서 12 cm 떨어진 점

B는 A에서 16 cm 떨어진 점 (A — B — C 순)
M은 A에서 12 cm 지점, B는 A에서 16 cm 지점
→ M이 B보다 A에 더 가깝다

$\overline{BM} = \overline{AB} - \overline{AM} = 16 - 12 = \mathbf{4}$ cm

3단계 · Page 3

경우 2: A — C — B 순

왜 이 단계? — 두 번째 경우에서도 M의 위치를 찾아 BM을 구해요.
📖 용어 노트 중점: 선분을 정확히 반으로 나누는 점
경우 2 풀이
$\overline{AC} = 8$ cm
M은 $\overline{AC}$의 중점 → A에서 4 cm 떨어진 점

B는 A에서 16 cm 떨어진 점 (A — C — B 순)
M은 A에서 4 cm 지점, B는 A에서 16 cm 지점

$\overline{BM} = \overline{AB} - \overline{AM} = 16 - 4 = \mathbf{12}$ cm

4단계 · Page 4

두 경우를 모아 정리해요

왜 이 단계? — 두 경우 모두 가능하므로 두 값이 모두 답이에요.
📖 용어 노트 선분: 두 끝점이 있는 유한한 선
두 경우 비교
경우 1 (A — B — C): $\overline{BM} = 4$ cm
경우 2 (A — C — B): $\overline{BM} = 12$ cm

두 경우 모두 조건 $\overline{AB} = 16$, $\overline{BC} = 8$을 만족하므로 두 값 모두 정답이에요.

정답 + 풀이 흐름 요약

📖 용어 노트 직선 AB: 양방향으로 무한히 뻗는 선 · 반직선 $\overrightarrow{AB}$: A에서 시작해 B 방향으로 뻗는 선 · 선분 $\overline{AB}$: A와 B를 잇는 유한한 선 · $\angle AOB$: 꼭짓점이 O인 각 · $\perp$: 두 직선이 수직으로 만남 · $\parallel$: 두 직선이 서로 평행함
$\overline{BM} = 4$ cm 또는 $12$ cm
(B의 위치에 따라 두 경우 모두 가능)
전체 풀이 흐름 — 4단계
1
B의 위치에 따라 두 경우 나누기: A—B—C / A—C—B
2
경우 1: $\overline{AC} = 24$ → M은 A에서 12 cm → $\overline{BM} = 16 - 12 = 4$ cm
3
경우 2: $\overline{AC} = 8$ → M은 A에서 4 cm → $\overline{BM} = 16 - 4 = 12$ cm
4
두 경우 모두 유효 → 두 값 모두 정답
⚠️ 이 유형의 핵심 "한 직선 위에"라는 조건만으로는 점의 순서가 하나로 정해지지 않아요. B가 A, C 사이에 있는지 밖에 있는지 두 경우를 모두 생각해야 해요!
