문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 항: 수나 문자의 곱으로 이루어진 단위  ·  계수: 문자 앞의 수 ($3x$ → 계수 3)  ·  상수항: 문자 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수 1인 식  ·  분배법칙: $a(b+c)=ab+ac$
유형 7 · H3

$4(2x - 1) - k(x - 3) = 5x + 5$를 만족하는 상수 $k$를 구하세요.
🤔 잠깐, 손으로 풀어보세요 좌변을 전개하면 $x$와 상수항이 $k$를 포함한 형태로 나와요. 그 계수를 우변과 맞추면 $k$가 나와요.
1단계 · Page 1

좌변을 전개해요 ($k$ 포함)

왜 이 단계? — 분배법칙으로 좌변 두 괄호를 전개하면 $k$가 포함된 $x$항과 상수항이 나와요.
📖 용어 노트 분배법칙: $k(x-3) = kx - 3k$  ·  앞에 $-$가 붙으면: $-k(x-3) = -kx + 3k$
각 괄호 전개
$4(2x - 1) = 8x - 4$
$-k(x - 3) = -kx + 3k$
합치기
$8x - 4 - kx + 3k$
$x$항: $(8 - k)x$
상수항: $-4 + 3k$
즉: $(8 - k)x + (-4 + 3k)$

2단계 · Page 2

우변과 계수를 맞춰요

왜 이 단계? — 좌변 정리 결과와 우변 $5x + 5$의 계수가 같아야 해요. 이 조건으로 $k$를 구해요.
📖 용어 노트 계수 비교: 두 식이 항등식이면 같은 항의 계수가 같아요
$x$ 계수 비교

$(8 - k)x + (-4 + 3k) = 5x + 5$ 이므로:

$x$ 계수: $8 - k = 5$
$\Rightarrow k = 8 - 5 = 3$
🤔 검증 힌트 $k = 3$이면 상수항도 맞는지 확인해봐요: $-4 + 3 \times 3 = -4 + 9 = 5$ ✓

3단계 · Page 3

$k = 3$을 대입해서 확인해요

왜 이 단계? — 구한 $k$가 정말 맞는지 원래 식에 넣어서 좌변 = 우변을 확인해요.
📖 용어 노트 검증: 구한 답을 원래 조건에 대입해서 확인
$k = 3$ 대입 확인
$4(2x - 1) - 3(x - 3)$
$= 8x - 4 - 3x + 9$
$= 5x + 5$ ✓

우변 $5x + 5$와 일치해요!

정답 + 풀이 흐름 요약

📖 용어 노트 분배법칙  ·  동류항 정리  ·  계수 비교  ·  검증
$k = 3$
전체 풀이 흐름 — 3단계
1
전개: $4(2x-1) = 8x-4$,   $-k(x-3) = -kx+3k$ → 합: $(8-k)x + (-4+3k)$
2
계수 비교: $8-k = 5$ → $k = 3$  /  상수항 확인: $-4+9 = 5$ ✓
3
검증: $k=3$ 대입 시 $5x+5$ 확인 ✓
⚠️ 이 유형의 두 가지 핵심 ① $-k(x-3)$을 전개할 때 $-k \times (-3) = +3k$ (부호 조심)
② $x$ 계수로 $k$를 구한 후, 상수항으로 검증하면 실수를 잡을 수 있어요
