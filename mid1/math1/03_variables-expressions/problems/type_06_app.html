유형 6 — 일차식의 덧셈·뺄셈

괄호 전개 → 동류항 정리 흐름 익히기
가장 중요한 규칙 — 뺄셈 괄호의 부호 반전!
$(A) - (B)$에서 $(B)$ 앞에 $-$가 붙으면
괄호 안 모든 항의 부호가 바뀌어요.
$(5x + 2) - (3x - 1)$
$= 5x + 2 - 3x + 1$ ← $-1$이 곱해져서 $-1$이 $+1$로!
$= 2x + 3$

$(3x + 4) + (x - 2)$를 계산하세요.
덧셈 괄호는 그냥 없애도 돼요. 부호 변화 없음!
정답: $4x + 2$
$3x + 4 + x - 2 = (3+1)x + (4-2) = 4x + 2$
2
L
$(5x - 3) - (2x + 1)$을 계산하세요.
뒤 괄호 앞에 $-$가 있어요. 괄호 안 $2x$와 $+1$의 부호가 모두 바뀌어요.
정답: $3x - 4$
$5x - 3 - 2x - 1 = (5-2)x + (-3-1) = 3x - 4$
$(2x + 1)$ 앞에 $-$: $-2x - 1$ (부호 모두 반전!)
3
L
$(4a - 5) - (a - 3)$을 계산하세요.
$-(a - 3) = -a + 3$이에요. $-3$이 $+3$이 되는 것 주의!
정답: $3a - 2$
$4a - 5 - a + 3 = (4-1)a + (-5+3) = 3a - 2$
1
M
$(3x - 1) - (x - 4) + 2(x + 2)$를 계산하세요.
세 부분을 순서대로 처리해요. ① 뺄셈 괄호 풀기 ② 곱셈 분배 ③ 동류항 정리
정답: $4x + 7$
$= 3x - 1 - x + 4 + 2x + 4$
$= (3 - 1 + 2)x + (-1 + 4 + 4)$
$= 4x + 7$
2
M
다음 계산에서 틀린 부분을 찾고 바르게 고치세요.
$(4x - 3) - (2x - 5) = 4x - 3 - 2x - 5 = 2x - 8$
$-(2x - 5)$를 전개할 때 $-5$가 어떻게 되어야 하는지 확인해봐요.
틀린 부분: $-(2x - 5)$를 $-2x - 5$로 전개함
올바른 전개: $-(2x - 5) = -2x + 5$
바른 풀이: $4x - 3 - 2x + 5 = 2x + 2$
뺄셈 괄호: 안의 모든 부호가 바뀌어요. $-5$는 $+5$가 돼요!
3
M
$A = 2x - 3$, $B = x + 5$일 때, $2A - B$를 계산하세요.
$A$와 $B$에 식을 대입한 뒤, 분배와 동류항 정리를 해요.
정답: $3x - 11$
$2A - B = 2(2x - 3) - (x + 5)$
$= 4x - 6 - x - 5 = 3x - 11$
1
H
$3A - 2(A - B) = A + 2B$를 계산하여 확인하고,
$A = x - 1$, $B = 2x + 3$일 때의 값을 구하세요.
$3A - 2(A - B)$를 전개해서 $A + 2B$임을 확인한 뒤, $A$와 $B$에 식을 대입해요.
확인: $3A - 2A + 2B = A + 2B$ ✓
$A + 2B = (x-1) + 2(2x+3) = x - 1 + 4x + 6 = 5x + 5$
2
H
어떤 식 $X$에 $3x - 2$를 더해야 하는데, 실수로 빼서 $x + 4$가 되었어요.
바르게 계산한 결과를 구하세요.
$X - (3x - 2) = x + 4$이므로 $X$를 먼저 구하고, $X + (3x-2)$를 계산해요.
정답: $7x$
$X = (x + 4) + (3x - 2) = 4x + 2$
바른 계산: $(4x + 2) + (3x - 2) = 7x$
3
H
$A - \{2B - (A + B)\}$를 간단히 하세요. ($A$, $B$는 일차식)
안쪽 소괄호부터 먼저 풀어요. 그 다음 중괄호, 그 다음 전체 순서로요.
정답: $2A - B$
$A - \{2B - (A + B)\}$
$= A - \{2B - A - B\}$
$= A - \{B - A\}$
$= A - B + A$
$= 2A - B$
