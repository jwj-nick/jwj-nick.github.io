문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 항: 수나 문자의 곱으로 이루어진 단위  ·  계수: 문자 앞의 수 ($3x$ → 계수 3)  ·  상수항: 문자 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수 1인 식  ·  분배법칙: $a(b+c)=ab+ac$
유형 5 · H1

$ax + 3y - 2x + by - 1$을 간단히 한 결과가 $3x + y - 1$이 되려면
$a$, $b$의 값을 각각 구하세요.
🤔 잠깐, 손으로 풀어보세요 동류항을 정리하면 $x$ 계수와 $y$ 계수가 결정돼요. 그 계수를 결과와 비교하면?
1단계 · Page 1

동류항끼리 모아요

왜 이 단계? — 먼저 주어진 식의 동류항을 정리해 $x$ 계수와 $y$ 계수를 $a$, $b$로 표현해야 해요.
📖 용어 노트 동류항: 문자와 차수가 같은 항  ·  계수: 문자 앞의 수
주어진 식 정리

$ax + 3y - 2x + by - 1$에서 동류항을 묶어요:

$x$항: $ax$와 $-2x$ → 계수끼리: $(a - 2)x$
$y$항: $3y$와 $by$ → 계수끼리: $(3 + b)y$
상수항: $-1$

정리하면:

$(a - 2)x + (3 + b)y - 1$
🤔 다음 단계 힌트 이 식이 $3x + y - 1$과 같아야 해요. 같은 식이면 각 항의 계수가 일치해야 한다는 게 핵심이에요.

2단계 · Page 2

계수를 비교해요

왜 이 단계? — 두 식이 항등식이면 대응하는 계수가 모두 같아야 해요. 이 원리로 $a$, $b$를 구해요.
📖 용어 노트 항등식: 어떤 수를 넣어도 항상 성립하는 등식 → 양변의 각 계수가 같아야 함
계수 비교 원리

$(a - 2)x + (3 + b)y - 1 = 3x + y - 1$이면:

$x$ 계수:   $a - 2 = 3$
$y$ 계수:   $3 + b = 1$
상수항:   $-1 = -1$ ✓ (이미 일치)
$a$, $b$ 계산
$a - 2 = 3$  ⟹  $a = 5$
$3 + b = 1$  ⟹  $b = -2$
⚠️ 흔한 실수 $3 + b = 1$에서 $b = 1$로 쓰는 실수! 반드시 $b = 1 - 3 = -2$예요. 부호 주의.

3단계 · Page 3

대입해서 확인해요

왜 이 단계? — 구한 값을 원래 식에 넣어서 결과가 맞는지 확인하는 습관을 들여요.
📖 용어 노트 검증: 구한 답을 원래 조건에 대입해서 맞는지 확인
$a = 5$, $b = -2$ 대입
$5x + 3y - 2x + (-2)y - 1$
$= (5 - 2)x + (3 - 2)y - 1$
$= 3x + y - 1$ ✓

결과가 문제에서 요구한 $3x + y - 1$과 일치해요!

정답 + 풀이 흐름 요약

📖 용어 노트 동류항: 문자와 차수가 같은 항  ·  계수: 문자 앞의 수  ·  계수 비교법: 두 식이 같으면 대응 계수가 같다는 원리
$a = 5$,   $b = -2$
전체 풀이 흐름 — 3단계
1
주어진 식의 동류항을 정리해 $(a-2)x + (3+b)y - 1$ 형태로 만들기
2
결과 식 $3x + y - 1$과 계수 비교: $a-2=3$ → $a=5$,   $3+b=1$ → $b=-2$
3
검증: $a=5$, $b=-2$ 대입 시 $3x + y - 1$ 확인 ✓
⚠️ 이 유형의 핵심 포인트 미지수 $a$, $b$가 계수로 들어있을 때는 동류항을 먼저 정리해서 "계수를 $a$, $b$로 표현"하고, 그 다음 결과와 계수 비교를 해야 해요.
