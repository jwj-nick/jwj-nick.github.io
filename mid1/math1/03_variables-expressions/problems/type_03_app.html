유형 3 — 식의 값 계산

대입(substitution) — 문자 자리에 수를 넣고 계산하기
대입 핵심 규칙
음수를 대입할 때는 반드시 괄호를 써요!
$x = -2$일 때 $3x$를 계산할 때: $3 \times (-2)$로 써야 해요.
$3 \times -2$ (×) → $3 \times (-2) = -6$ (O)
1
L
$x = 3$일 때, $4x - 7$의 값을 구하세요.
$x$ 자리에 $3$을 그대로 넣고 계산해요.
정답: $5$
$4x - 7 = 4 \times 3 - 7 = 12 - 7 = 5$
2
L
$a = 2, b = -1$일 때, $3a + 2b$의 값을 구하세요.
$a$와 $b$ 자리에 각각 $2$와 $-1$을 대입해요. 음수는 괄호!
정답: $4$
$3a + 2b = 3 \times 2 + 2 \times (-1) = 6 + (-2) = 4$
3
L
$x = 5$일 때, $\dfrac{x}{5} + 1$의 값을 구하세요.
$\dfrac{x}{5}$에 $x = 5$를 대입하면 $\dfrac{5}{5}$예요.
정답: $2$
$\dfrac{x}{5} + 1 = \dfrac{5}{5} + 1 = 1 + 1 = 2$
1
M
$a = -3$일 때, $a^2 - 4a + 1$의 값을 구하세요.
$a^2 = (-3)^2 = 9$이에요. 음수를 제곱하면 양수가 되는 것 기억해요!
정답: $22$
$a^2 - 4a + 1$
$= (-3)^2 - 4 \times (-3) + 1$
$= 9 + 12 + 1 = 22$
$(-3)^2 = 9$ (양수!), $-3^2 = -9$ (음수!). 괄호 위치가 중요해요.
2
M
$x = \dfrac{1}{2}$일 때, $6x - 4$의 값을 구하세요.
$6 \times \dfrac{1}{2} = 3$이에요. 자연수와 분수의 곱!
정답: $-1$
$6x - 4 = 6 \times \dfrac{1}{2} - 4 = 3 - 4 = -1$
3
M
$x = 2, y = -1$일 때, $\dfrac{x - y}{x + y}$의 값을 구하세요.
분자, 분모에 각각 대입하고 따로 계산한 뒤 나눠요.
정답: $3$
분자: $x - y = 2 - (-1) = 2 + 1 = 3$
분모: $x + y = 2 + (-1) = 1$
$\dfrac{3}{1} = 3$

H
$x + y = 4$이고 $xy = 3$일 때, $x^2 + y^2$의 값을 구하세요.
$(x + y)^2 = x^2 + 2xy + y^2$이에요. 이 관계를 이용하면 $x^2 + y^2 = (x+y)^2 - 2xy$가 돼요.
정답: $10$
$x^2 + y^2 = (x+y)^2 - 2xy = 4^2 - 2 \times 3 = 16 - 6 = 10$
(이 풀이법은 나중에 곱셈공식에서 배우지만, 여기선 대입으로도 풀 수 있어요.)

H
$a = -2$일 때, $A = 3a^2 - 2a + 1$의 값을 구하고,
그 값을 다시 $b$로 놓고 $2b - 5$의 값을 구하세요.
두 단계로 나눠요. 먼저 $A$를 계산하고, 그 값을 $b$로 놓고 $2b-5$를 계산해요.
정답: $31$
$A = 3(-2)^2 - 2(-2) + 1 = 3 \times 4 + 4 + 1 = 12 + 4 + 1 = 17$
$b = 17$일 때, $2b - 5 = 2 \times 17 - 5 = 34 - 5 = 31$
3
H
$x = 1$일 때 $ax + b = 5$이고, $x = -1$일 때 $ax + b = -1$이에요.
$a$와 $b$의 값을 각각 구하세요.
$x = 1$을 대입한 식과 $x = -1$을 대입한 식, 두 식을 세워서 연립방정식처럼 풀어요.
정답: $a = 3, b = 2$
$x=1$: $a + b = 5$ ⋯ ①
$x=-1$: $-a + b = -1$ ⋯ ②
①+②: $2b = 4$ → $b = 2$
①에 대입: $a = 5 - 2 = 3$
