유형 5 — 동류항 정리

같은 문자, 같은 차수끼리 합치기
동류항이란? 문자와 차수가 모두 같은 항이에요.
$3x$와 $5x$ → 동류항 (둘 다 $x$의 일차항) → $8x$
$3x$와 $5x^2$ → 동류항 아님 (차수가 달라요)
$2x$와 $3y$ → 동류항 아님 (문자가 달라요)
동류항은 계수끼리만 더하거나 빼요.
1
L
$4x + 3 - 2x + 1$을 간단히 하세요.
$x$항끼리($4x$와 $-2x$), 상수항끼리($3$과 $1$)를 따로 모아요.
정답: $2x + 4$
$4x + 3 - 2x + 1 = (4-2)x + (3+1) = 2x + 4$
2
L
$5y - 3 + y - 2y + 7$을 간단히 하세요.
$y$가 붙은 항: $5y$, $y$, $-2y$ 세 개예요. 계수들을 더해요.
정답: $4y + 4$
$(5 + 1 - 2)y + (-3 + 7) = 4y + 4$
3
L
$3a + 2b - a + 4b$를 간단히 하세요.
$a$끼리, $b$끼리 따로 모아요. $a$와 $b$는 다른 문자라서 합칠 수 없어요.
정답: $2a + 6b$
$a$항: $3a - a = 2a$ / $b$항: $2b + 4b = 6b$ / 결과: $2a + 6b$
1
M
$2x^2 + 3x - x^2 + 4x - 5$를 간단히 하세요.
$x^2$항끼리, $x$항끼리 따로 모아요. $x^2$과 $x$는 차수가 달라서 합칠 수 없어요!
정답: $x^2 + 7x - 5$
$x^2$항: $2x^2 - x^2 = x^2$
$x$항: $3x + 4x = 7x$
상수항: $-5$
결과: $x^2 + 7x - 5$
$x^2$과 $x$는 동류항이 아니에요! 차수가 다르면 합칠 수 없어요.
2
M
다음 중 동류항끼리 바르게 묶은 것은?
① $3x$와 $3y$    ② $2x^2$와 $-x^2$    ③ $4x$와 $4x^2$    ④ $5$와 $5x$
동류항 = 문자가 같고 차수도 같아야 해요. 두 조건 모두 만족해야!!
정답: ②
① 문자 다름($x$ vs $y$) ✗
② $x^2$의 차수 2 = $x^2$의 차수 2, 문자도 같음 ✓
③ 차수 다름($x$는 1차, $x^2$은 2차) ✗
④ $5$는 상수항(차수 0), $5x$는 1차 ✗
3
M
$5x - 3y + 2x + y - 4$를 간단히 하세요.
$x$항, $y$항, 상수항 세 그룹으로 나눠서 계산해요.
정답: $7x - 2y - 4$
$x$항: $5x + 2x = 7x$
$y$항: $-3y + y = -2y$
상수항: $-4$
결과: $7x - 2y - 4$

H
$ax + 3y - 2x + by - 1$을 간단히 한 결과가 $3x + y - 1$이 되려면 $a$, $b$의 값을 구하세요.
동류항을 정리한 식의 계수를 주어진 결과와 맞춰요. $x$ 계수끼리, $y$ 계수끼리 같아야 해요.
정답: $a = 5, b = -2$
정리 후 $x$항: $(a-2)x$ / $y$항: $(3+b)y$ / 상수항: $-1$
$x$ 계수: $a - 2 = 3$ → $a = 5$
$y$ 계수: $3 + b = 1$ → $b = -2$

H
$3x^2 - 2x + 1$과 $-x^2 + 5x - 3$을 더하면 어떻게 되나요?
그리고 이 결과의 $x$ 계수는 무엇인가요?
두 식을 괄호 없이 이어 쓰고, 동류항끼리 정리해요.
결과: $2x^2 + 3x - 2$
$x^2$항: $3x^2 + (-x^2) = 2x^2$
$x$항: $-2x + 5x = 3x$
상수항: $1 + (-3) = -2$
$x$ 계수: $3$
3
H
어떤 식에서 $2x - 5$를 빼야 하는데 실수로 더했더니 $5x + 1$이 되었어요.
바르게 계산한 결과를 구하세요.
실수로 더한 결과에서 어떤 식을 구해요. $어떤 식 + (2x - 5) = 5x + 1$이니까 어떤 식 = $5x+1 - (2x-5)$예요.
정답: $x + 11$
어떤 식: $5x + 1 - (2x - 5) = 5x + 1 - 2x + 5 = 3x + 6$
바른 계산: $(3x + 6) - (2x - 5) = 3x + 6 - 2x + 5 = x + 11$
