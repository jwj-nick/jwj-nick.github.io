깊이 탐구 — 식 세우기 & 기호 생략

상황을 식으로 만들고, 수학 표기 규칙대로 깔끔하게 쓰는 두 단계를 연결해서 익혀요.

식 세우기 전략 — 3단계

1
상황 분석: 무엇이 변하는 양인지 문자로 정해요. "문자 $x$는 ___를 나타낸다"라고 먼저 정의!
2
관계 파악: 수량끼리 어떤 관계인지 확인 — 곱(단가×개수), 합(거스름돈), 비율(%), 거리÷속력 등
3
기호 생략: 식을 만든 뒤 수학 표기 규칙으로 정리 — 숫자 앞에, $\times$ 생략, $\div$ → 분수

기호 생략 7가지 규칙 체크리스트

1
$a \times b = ab$ (문자 사이 × 생략)
2
$3 \times a = 3a$ (숫자는 문자 앞에)
3
$1 \times a = a$ (계수 $1$ 생략)
4
$(-1) \times a = -a$ (계수 $-1$에서 $1$ 생략)
5
$a \times a = a^2$ (같은 문자 → 거듭제곱)
6
$a \div b = \dfrac{a}{b}$ (나눗셈 → 분수)
7
숫자끼리는 계수로 계산 먼저 — $2 \times 3 \times a = 6a$
M — 중간 난이도 문제
M1
가로가 $(a + 3)$ cm, 세로가 $b$ cm인 직사각형의 넓이를 기호 생략 규칙에 맞게 나타내세요.
넓이 = 가로 × 세로. 괄호가 있어도 곱셈 기호를 생략할 수 있어요!
정답: $(a+3)b$ cm²
$(a+3) \times b = (a+3)b$
괄호와 문자 사이의 $\times$도 생략해요.
M2
시속 $v$ km로 달리는 자동차가 $t$시간 동안 달렸을 때 남은 연료가 원래의 $\dfrac{1}{k}$이라면, 처음 연료량이 $L$리터일 때 남은 연료를 식으로 나타내세요.
$\dfrac{1}{k}$만큼 남았다는 건, $L$의 $\dfrac{1}{k}$배예요.
정답: $\dfrac{L}{k}$리터
$L \times \dfrac{1}{k} = \dfrac{L}{k}$
(거리나 속력은 남은 연료 계산에 필요 없어요 — 이 문제에선 "남은 비율"만 주어졌어요.)
M3
$a \div b \times (-2) \times a$를 기호 생략 규칙에 따라 가장 간단한 형태로 나타내세요.
왼쪽부터: $a \div b = \dfrac{a}{b}$, 그 다음 $\times (-2)$, 마지막으로 $\times a$
정답: $-\dfrac{2a^2}{b}$
$\dfrac{a}{b} \times (-2) \times a = \dfrac{(-2) \times a \times a}{b} = \dfrac{-2a^2}{b} = -\dfrac{2a^2}{b}$
H — 심화 난이도 문제
H1
정가 $p$원의 물건에 세금 10%가 붙고, 거기서 다시 멤버십 할인 $d$원을 받는다면 최종 가격을 식으로 나타내세요.
세금 10%가 붙으면 $p \times 1.1 = 1.1p$원이 돼요. 거기서 $d$원을 빼요.
정답: $(1.1p - d)$원
세금 포함 가격: $p + p \times 0.1 = 1.1p$
할인 후: $1.1p - d$원
H2
$a$, $b$, $c$가 각각 백의 자리, 십의 자리, 일의 자리인 세 자리 수를 식으로 나타내고, 이 세 자리 수에서 각 자리 숫자의 합을 빼는 식을 간단히 하세요.
세 자리 수: $100a + 10b + c$. 각 자리 합: $a + b + c$. 빼기!
정답: $99a + 9b$
$(100a + 10b + c) - (a + b + c) = 99a + 9b$
이 결과는 항상 9의 배수예요! (수의 신기한 성질)
H3
$(-a) \times (-a) \times b \div (-2a)$를 기호 생략 규칙에 따라 나타내세요. (단, $a \neq 0$)
$(-a)^2 = a^2$이에요. 음수 × 음수 = 양수. 마지막 $\div (-2a)$는 분모로!
정답: $-\dfrac{ab}{2}$
$(-a) \times (-a) = a^2$
$a^2 \times b \div (-2a) = \dfrac{a^2 b}{-2a} = \dfrac{ab}{-2} = -\dfrac{ab}{2}$

식 세우기에서 자주 하는 실수

✗
숫자를 문자 뒤에 씀: $a3$ (틀림)
✓
숫자는 반드시 문자 앞에: $3a$
✗
계수 $1$ 안 생략: $1x$ (어색한 표현)
✓
계수 $1$은 생략: $x$
✗
$(-1) \times x = -1x$ (틀림)
✓
$(-1) \times x = -x$ ($1$ 생략, 부호만)
✗
나눗셈 방향 혼동: $x \div 5 = \dfrac{5}{x}$ (틀림)
✓
나누어지는 수가 분자: $x \div 5 = \dfrac{x}{5}$
✗
$a \times a = 2a$ (틀림! $a$를 2배 한 게 아님)
✓
$a \times a = a^2$ (제곱!)

기호 복원(역방향)에서 자주 하는 실수

✗
$3ab$를 복원할 때 $3 \times a + b$로 오해
✓
$3ab = 3 \times a \times b$ (모두 곱하기)
✗
$-x^2$를 복원할 때 $(-x) \times (-x)$로 오해
✓
$-x^2 = (-1) \times x \times x$ (계수가 $-1$, 지수는 $x$에만)

이 두 유형이 연결되는 이유

수학에서 "식을 세운다"는 것과 "기호를 생략한다"는 것은 사실 하나의 흐름이에요.


상황 → (자연어로 관계 파악) → 기호 있는 식 → (생략 규칙 적용) → 수학 표기


예: "1개에 $a$원인 물건 3개의 값"
→ $a$원 × 3개
→ $a \times 3$
→ $3a$ 원

이 단원에서 배운 것이 쓰이는 곳

4단원 일차방정식: "식 세우기"가 방정식 세우기의 첫 단계예요.


5~6단원 함수: $y = 2x$처럼 기호 생략된 형태로 함수가 표현돼요.


현실 속: 스프레드시트 수식, 프로그래밍 변수가 모두 이 표기법을 씁니다. 알콰리즈미가 시작한 대수 언어예요!
