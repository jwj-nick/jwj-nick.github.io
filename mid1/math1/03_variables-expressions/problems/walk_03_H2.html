문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 항: 수나 문자의 곱으로 이루어진 식의 단위 (예: 3x, −2y, 5)  ·  계수: 항에서 문자 앞에 곱해진 수 (3x에서 3)  ·  상수항: 문자가 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수가 1인 식
유형 3 · H2

$a = -2$일 때, $A = 3a^2 - 2a + 1$의 값을 구하고,
그 값을 다시 $b$로 놓고 $2b - 5$의 값을 구하세요.
🤔 잠깐, 손으로 풀어보세요 두 단계로 나눠요. 1단계: $a = -2$를 $A$에 대입, 2단계: $A$의 값을 $b$로 놓고 $2b - 5$ 계산.
1단계 · Page 1

$a = -2$를 $A$에 대입해요

왜 이 단계? — 음수를 대입할 때 괄호를 반드시 써야 해요. 제곱 계산에 특히 주의!
📖 용어 노트 대입: 문자 자리에 수를 넣기  ·  음수 대입 주의: $(-2)^2 = 4$ (양수!), $-2^2 = -4$ (음수!)
$A = 3a^2 - 2a + 1$에 $a = -2$ 대입
$A = 3 \times (-2)^2 - 2 \times (-2) + 1$
$= 3 \times 4 - 2 \times (-2) + 1$
$= 12 + 4 + 1$
$= 17$
⚠️ 음수 대입 시 괄호 필수! $a = -2$를 대입할 때 반드시 괄호를 써요.
$3a^2$에서 $a = -2$: $3 \times (-2)^2 = 3 \times 4 = 12$ (O)
$3 \times -2^2 = 3 \times (-4) = -12$ (×) — 괄호가 없으면 부호가 달라져요!

2단계 · Page 2

$b = 17$로 놓고 $2b - 5$ 계산

왜 이 단계? — 1단계에서 구한 $A = 17$을 $b$라는 새 변수로 놓고 두 번째 식을 계산해요.
📖 용어 노트 대입: 문자 자리에 수를 넣기  ·  사칙연산 순서: 곱셈 먼저, 덧셈·뺄셈 나중
$b = 17$ 대입 → $2b - 5$
$2b - 5 = 2 \times 17 - 5$
$= 34 - 5$
$= 29$
💡 두 단계 대입의 핵심 1단계 결과 → 2단계 입력으로 연결되는 구조예요.
각 단계를 독립적으로 처리해야 실수가 없어요.

3단계 · Page 3

전체 흐름 한눈에 정리

왜 이 단계? — 두 단계가 어떻게 연결되는지 전체 흐름을 한눈에 확인해요.
📖 용어 노트 대입: 문자 자리에 수를 넣기  ·  사칙연산 순서: 거듭제곱 → 곱셈·나눗셈 → 덧셈·뺄셈
전체 흐름
$a = -2$
↓ 대입
$A = 3(-2)^2 - 2(-2) + 1 = 12 + 4 + 1 = 17$
↓ $b = A = 17$ 로 놓기
$2b - 5 = 2 \times 17 - 5 = 34 - 5 = 29$
⚠️ 흔한 실수 $A = 17$을 구하고 $b$로 놓지 않고, $a$ 값을 그대로 $b$에 대입하는 실수를 해요.
$2b - 5$에서 $b = A$ (1단계 결과) 이지, $b = a = -2$가 아니에요!

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 항: 수나 문자의 곱으로 이루어진 단위  ·  계수: 항에서 문자 앞에 곱해진 수  ·  상수항: 문자가 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수가 1인 식
$2b - 5 = 29$
($A = 17$, $b = 17$)
전체 풀이 흐름 — 2단계
1
$a = -2$ 대입: $A = 3(-2)^2 - 2(-2) + 1 = 12 + 4 + 1 = 17$
2
$b = 17$ 대입: $2b - 5 = 2 \times 17 - 5 = 29$
⚠️ 이 유형의 핵심 함정 음수 $-2$를 제곱할 때: $(-2)^2 = 4$ (양수), $-2^2 = -4$ (음수).
반드시 괄호를 써서 $3 \times (-2)^2$으로 계산해야 해요!
