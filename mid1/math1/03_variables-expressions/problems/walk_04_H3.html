문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 항: 수나 문자의 곱으로 이루어진 식의 단위 (예: 3x, −2y, 5)  ·  계수: 항에서 문자 앞에 곱해진 수 (3x에서 3)  ·  상수항: 문자가 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수가 1인 식
유형 4 · H3

어떤 다항식의 $x^2$의 계수는 $2$, $x$의 계수는 $-3$, 상수항은 $1$이에요.
이 다항식을 쓰고, 이 다항식의 차수와 항의 개수를 구하세요.
🤔 잠깐, 손으로 풀어보세요 계수 정보로 식을 만들어봐요.
"$x^2$의 계수가 2" → $2x^2$ 항이 있다는 뜻이에요.
1단계 · Page 1

계수 정보로 각 항을 만들어요

왜 이 단계? — "계수 × 해당 문자항" 형태로 각 항을 구성해요.
📖 용어 노트 계수: 항에서 문자 앞의 수  ·  상수항: 문자가 없는 항 (문자 없이 숫자만 있는 항)
각 항 구성
$x^2$의 계수 = $2$  →  $2x^2$ 항
$x$의 계수 = $-3$  →  $-3x$ 항
상수항 = $1$  →  $1$ 항
다항식 조립
$2x^2 + (-3x) + 1 = 2x^2 - 3x + 1$

2단계 · Page 2

차수와 항의 개수를 구해요

왜 이 단계? — 식을 완성했으니 차수와 항의 개수를 확인해요.
📖 용어 노트 차수: 다항식에서 각 항의 차수 중 가장 높은 것  ·  항의 개수: +나 - 로 구분된 항의 수  ·  상수항: 문자가 없는 항
$2x^2 - 3x + 1$의 분석
항: $2x^2$ (차수 2), $-3x$ (차수 1), $1$ (차수 0)
최고차수 = 2 → 차수: 2 (이차식)
항의 개수: 3개

3단계 · Page 3

역방향으로 확인해요

왜 이 단계? — 구성한 식에서 계수를 다시 읽어서 문제 조건과 맞는지 확인해요.
📖 용어 노트 계수: 항에서 문자 앞의 수  ·  상수항: 문자가 없는 항
$2x^2 - 3x + 1$ 에서 계수 읽기
$x^2$의 계수 = $2$ ✓ (문제와 일치)
$x$의 계수 = $-3$ ✓ (문제와 일치)
상수항 = $1$ ✓ (문제와 일치)
⚠️ 부호 주의 $-3x$에서 $x$의 계수는 $-3$이에요. $3$이 아니에요!
계수는 부호를 포함해서 써야 해요.

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 항: 수나 문자의 곱으로 이루어진 단위  ·  계수: 항에서 문자 앞에 곱해진 수  ·  상수항: 문자가 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수가 1인 식
$2x^2 - 3x + 1$
차수: 2  |  항의 개수: 3
전체 풀이 흐름 — 3단계
1
계수 정보 → 각 항: $2x^2$, $-3x$, $1$
2
다항식 조립: $2x^2 - 3x + 1$
3
차수 = 2 (최고차항 $2x^2$), 항의 개수 = 3
⚠️ 이 유형의 핵심 포인트 "계수 × 해당 문자항" 구조로 각 항을 만들어요.
계수가 음수이면 빼기(-) 부호가 붙어요. $-3x$처럼요.
