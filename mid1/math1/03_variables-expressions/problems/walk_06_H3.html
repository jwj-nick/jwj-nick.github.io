단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 항: 수나 문자의 곱으로 이루어진 단위  ·  계수: 문자 앞의 수 ($3x$ → 계수 3)  ·  상수항: 문자 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수 1인 식  ·  분배법칙: $a(b+c)=ab+ac$
유형 6 · H3

$A - \{2B - (A + B)\}$를 간단히 하세요. ($A$, $B$는 일차식)
🤔 잠깐, 손으로 풀어보세요 중괄호 $\{\}$ 안에 소괄호 $()$가 있어요. 어떤 괄호부터 먼저 풀어야 할까요?
1단계 · Page 1

안쪽 괄호 $()$부터 풀어요

왜 이 단계? — 괄호가 중첩될 때는 항상 가장 안쪽 괄호부터 전개해요. 바깥에서 안으로 가면 안 돼요!
📖 용어 노트 괄호 우선순위: 소괄호 () → 중괄호 {} → 대괄호 [] 순서로 안쪽부터 전개
소괄호 전개

$\{2B - (A + B)\}$에서 안쪽 $(A + B)$ 앞에 $-$가 붙어있어요:

$2B - (A + B)$
$= 2B - A - B$    ← $-(A+B) = -A - B$
$= -A + B$    ← $2B - B = B$
⚠️ 부호 반전 주의 $-(A + B) = -A - B$예요. 괄호 안의 모든 항의 부호가 반전돼요.

2단계 · Page 2

중괄호 $\{\}$ 전개해요

왜 이 단계? — 1단계에서 $\{2B - (A+B)\} = \{-A + B\}$로 정리했어요. 이제 중괄호를 전개해요.
📖 용어 노트 $-\{-A + B\} = A - B$ (중괄호 앞 $-$ → 안 모든 항 부호 반전)
전체 식에 적용

1단계 결과를 원래 식에 적용:

$A - \{2B - (A + B)\}$
$= A - \{-A + B\}$    ← 1단계 결과 대입
$= A - (-A + B)$    ← 중괄호 = 소괄호와 동일
$= A + A - B$    ← $-(-A) = +A$, $-(+B) = -B$
$= 2A - B$

3단계 · Page 3

전체 흐름을 한눈에 정리해요

왜 이 단계? — 중첩 괄호를 한 번에 다 쓰면서 확인하면 실수를 잡을 수 있어요.
📖 용어 노트 중첩 괄호 전개 순서: 안쪽 → 바깥쪽
전체 전개 한눈에
$A - \{2B - (A + B)\}$
$= A - \{2B - A - B\}$    ← 소괄호 전개
$= A - \{B - A\}$    ← $2B - B = B$ 정리
$= A - B + A$    ← 중괄호 전개 ($-\{B-A\} = -B+A$)
$= 2A - B$
⚠️ 실수 포인트 중괄호 앞 $-$ 부호를 처리할 때 $B - A$의 $-A$가 $+A$로 바뀌어요. 부호 반전을 빠뜨리면 틀려요.

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 중첩 괄호: 안쪽부터 전개  ·  뺄셈 괄호 부호 반전
$A - \{2B - (A+B)\} = 2A - B$
전체 풀이 흐름 — 3단계
1
안쪽 소괄호 전개: $2B - (A+B) = 2B - A - B = -A + B$
2
중괄호 전개: $A - \{-A + B\} = A + A - B$
3
동류항 정리: $A + A = 2A$ → 결과 $2A - B$
⚠️ 이 유형의 핵심 법칙 중첩 괄호는 반드시 안쪽부터 전개해요. 바깥 괄호를 먼저 전개하면 안 쪽 부호 처리가 뒤죽박죽이 돼요. 그리고 뺄셈 괄호는 항상 안의 모든 부호를 반전!
