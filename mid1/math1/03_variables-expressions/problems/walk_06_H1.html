문제 · Page 0

단계별로 풀어볼까요?

정답은 마지막에 확인해요. 먼저 스스로 생각해보고, 막히면 다음 단계로 넘어가세요.
📖 용어 노트 항: 수나 문자의 곱으로 이루어진 단위  ·  계수: 문자 앞의 수 ($3x$ → 계수 3)  ·  상수항: 문자 없는 항  ·  동류항: 문자와 차수가 같은 항  ·  일차식: 차수 1인 식  ·  분배법칙: $a(b+c)=ab+ac$
유형 6 · H1

$3A - 2(A - B) = A + 2B$임을 확인하고,
$A = x - 1$, $B = 2x + 3$일 때의 값을 구하세요.
🤔 잠깐, 손으로 풀어보세요 두 파트로 나눠요: ① 좌변 전개해서 우변과 같음을 확인, ② $A$, $B$에 식을 대입.
1단계 · Page 1

좌변 $3A - 2(A-B)$를 전개해요

왜 이 단계? — 분배법칙으로 $-2(A-B)$를 전개하고 동류항을 정리하면 우변 $A+2B$가 나와야 해요.
📖 용어 노트 분배법칙: $k(A-B) = kA - kB$  ·  음수 분배: $-2(A-B) = -2A + 2B$
좌변 전개
$3A - 2(A - B)$
$= 3A - 2A + 2B$    ← $-2 \times A = -2A$, $-2 \times (-B) = +2B$
$= A + 2B$

좌변 $= A + 2B$ = 우변 ✓ 확인 완료!

⚠️ 흔한 실수 $-2(A - B)$에서 $-B$에 $-2$를 곱하면 $+2B$가 돼요. $-2 \times (-B) = +2B$. 마이너스 × 마이너스 = 플러스!

2단계 · Page 2

$A$, $B$에 식을 대입해요

왜 이 단계? — 1단계에서 $3A - 2(A-B) = A + 2B$를 확인했으니, 간단한 $A + 2B$에 바로 대입하면 돼요.
📖 용어 노트 대입: 문자에 값(또는 식)을 넣는 것  ·  분배법칙: $2B = 2(2x+3)$
$A + 2B$에 대입

$A = x - 1$, $B = 2x + 3$이므로:

$A + 2B$
$= (x - 1) + 2(2x + 3)$
$= x - 1 + 4x + 6$    ← $2 \times 2x = 4x$, $2 \times 3 = 6$
$= 5x + 5$
💡 효율적인 전략 $3A - 2(A-B)$에 직접 대입하면 계산이 복잡해져요. 먼저 간단한 형태 $A + 2B$로 정리한 뒤 대입하는 게 더 빠르고 실수가 적어요.

3단계 · Page 3

계산 결과를 확인해요

왜 이 단계? — $5x + 5$가 정말 맞는지 동류항 정리를 다시 점검해봐요.
📖 용어 노트 동류항: 문자와 차수가 같은 항
검산

$x - 1 + 4x + 6$에서:

$x$항: $x + 4x = 5x$
상수항: $-1 + 6 = 5$
결과: $5x + 5$ ✓

정답 · Page ★

정답 + 풀이 흐름 요약

📖 용어 노트 분배법칙: $a(b+c)=ab+ac$  ·  동류항 정리  ·  대입: 문자에 값(또는 식)을 넣는 것
$A + 2B = 5x + 5$
($A = x-1$, $B = 2x+3$ 대입 시)
전체 풀이 흐름 — 3단계
1
좌변 전개: $3A - 2(A-B) = 3A - 2A + 2B = A + 2B$ → 우변과 같음 확인 ✓
2
$A + 2B$에 $A = x-1$, $B = 2x+3$ 대입: $(x-1) + 2(2x+3)$
3
전개·정리: $x - 1 + 4x + 6 = 5x + 5$
⚠️ 이 유형의 핵심 포인트 복잡한 식을 그대로 대입하지 말고, 먼저 간단히 정리한 뒤 대입해요. 계산 오류를 크게 줄일 수 있어요.
